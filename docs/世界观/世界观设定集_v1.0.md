## 1. 世界概览

### 1.1 世界分层

本游戏世界观采用**多层世界结构**，从凡界到仙界，共分九层：

| 层级 | 世界 | 说明 |
|------|------|------|
| 第一层 | 凡界（九州） | 凡人与低阶修士的活动舞台 |
| 第二层 | 灵界 | 真仙以上修士的主要活动区域 |
| 第三层 | 仙界 | 太乙金仙以上修士的活动区域 |
| 第四层 | 道界 | 大道本源所在，传说中至高存在居住地 |

### 1.2 时间尺度

- **凡界一年** = 仙界一日
- 纪元交替：每个纪元约持续 36,000 凡界年
- 大劫周期：每九纪元经历一次"轮回大劫"

### 1.3 核心概念

- **气运**：世界运转的根本能量，各势力争夺的核心资源
- **因果**：修士行为积累的业力，影响渡劫成功率
- **天道**：世界的最高法则，修仙者追求的终极目标

## 3. 地理与世界结构

### 3.1 凡界九州

- **中州**：修仙界中心，九大宗门所在地
- **东州**：东海之滨，灵兽出没
- **西州**：西域高原，炼器圣地
- **南州**：十万大山，秘境无数
- **北州**：北寒之地，冰雪修炼
- **幽州**：幽冥交界，亡灵横行
- **魔域**：魔族领地，正邪交战
- **妖域**：妖族栖息
- **海外**：散修乐园

### 3.2 特殊区域

- **九幽冥域**：亡魂归宿，黄泉之地
- **昆仑虚**：上古遗迹，机缘无数
- **东海归墟**：上古战场，沉没秘境
- **南疆蛊渊**：毒虫瘴气，危险禁地
- **北寒冰川**：万年玄冰，藏有古仙遗迹

## 7. 大劫设定

> 详见《剧情线与NPC设定.md》中的大劫倒计时章节

### 7.1 大劫本质

轮回大劫并非简单的天灾，而是**天道自我修复机制**：
- 每九纪元，天道积攒的因果浊气达到临界点
- 大劫通过毁灭与重生，清除浊气，重置世界
- 唯有超脱者，方能逃脱轮回

### 7.2 当前大劫进度

- **第一阶段（已完成）**：天象异变，灵气紊乱（游戏前期）
- **第二阶段（进行中）**：妖魔复出，因果清算（游戏中后期）
- **第三阶段（即将到来）**：天道崩塌，轮回重启（游戏后期）

### 7.3 大劫中的各方势力

- **正道宗门**：试图延缓大劫，保存文明
- **天魔教**：试图利用大劫夺取天道权柄
- **隐世古族**：持有对抗大劫的关键传承
- **玩家**：注定成为变数，书写新的结局
