### 第一章：凡人之始

**等级范围**：1-10级（淬体境-炼气境）
**对应境界**：淬体 → 炼气

#### 开篇剧情

> 你本是中州青云镇一名普通少年，每日上山砍柴为生。一日，你在后山悬崖下的石缝中，意外发现一具盘坐的枯骨，枯骨身旁散落着一枚古朴玉简与一柄锈迹斑斑的铁剑。当你触碰玉简的瞬间，一股温热的气流涌入体内——你感受到了"气感"。老道士曾说，这是踏入修仙之途的第一步……

#### 核心事件

**事件一：觉醒气感**
- 玩家在后山发现上古散修"云游子"的遗骸，获得《云游心法》与"游子剑"
- 系统提示：发现传承！你的修炼之路正式开始
- 引导任务：完成基础修炼教程

**事件二：第一次战斗**
- 村子附近出现妖兽伤人事件，玩家初战妖兽（野猪妖）
- 系统提示：战斗中自动积累灵气和经验
- 引导任务：击败5只妖兽

**事件三：仙缘初显**
- 青云镇来了一位仙师（天道宗外门弟子），检测灵根
- 玩家被发现拥有"五行灵根"，虽非天品，但可修炼
- 仙师留下一句话："三个月后，九宗收徒大会，去试试吧。"

#### 关键选择点

**选择一：拜师还是自学**
- A. 留在青云镇，跟随游子剑的指引自学修炼（独立修炼路线，获得额外机缘概率↑）
- B. 前往最近的修仙门派参加选拔（宗门引导路线，资源获取稳定↑）

**选择二：救助同伴**
- 镇上少年李狗蛋被妖兽抓走，玩家决定：
- A. 立刻追击救援（获得李狗蛋为追随者，但面临危险）
- B. 先提升实力再行动（错过李狗蛋，但安全获得初期资源）

#### 阶段结局

三个月后，你在仙师留下的测试石上留下了自己的名字。仙师看着你，微微点头：
> "你虽非天纵奇才，但心性坚韧，或许能在修仙路上走得更远。记住，境界只是起点，心性才是终点。"
> 你踏上了前往天道宗的路途，凡人之始，自此终结。

**阶段奖励**：
- 解锁宗门系统
- 获得称号：「初入道途」
- 获得称号：「青云之子」

#### 对应玩法解锁

| 解锁内容 | 说明 |
|----------|------|
| 挂机系统 | 离线自动修炼，积累灵气 |
| 洞府系统 | 拥有自己的修炼场所 |
| 基础副本「青云秘境」 | 适合淬体-炼气期玩家 |
| 基础装备锻造 | 学习炼制凡品装备 |
| 每日签到 | 领取日常资源 |

### 第二章：踏入仙途

**等级范围**：10-30级（筑基境-金丹境）
**对应境界**：炼气（10-20）→ 筑基（20-30）

#### 开篇剧情

> 你通过了天道宗外门选拔，成为一名外门弟子。宗门坐落于中州云雾山，亭台楼阁隐于云海之中。接引师兄告诉你：外门弟子多达万人，能进入内门的百中无一。你望着那云雾深处的金色大殿，心中暗下决心——三年后的宗门大比，便是你的机会。

#### 核心事件

**事件一：宗门大比预选**
- 每年一度的外门大比即将开始，前百名可获得进入内门候选资格
- 你在预选中遭遇各路高手，开始崭露头角
- 关键对手：来自东州的剑道天才「林惊羽」

**事件二：游子剑的秘密**
- 你在一次宗门任务中，偶然激活了游子剑的剑灵
- 游子剑原主人「云游子」的记忆碎片涌现——
- 原来云游子曾是天道宗内门弟子，因卷入宗门秘宝「天道图」的争夺而陨落
- 云游子临终前将毕生所学封入剑中，等待有缘人

**事件三：内门试炼**
- 三年一度的内门试炼开启，试炼之地：昆仑秘境
- 你在秘境中遭遇强敌，也获得了重要机缘：发现「天道图」的一角碎片
- 关键盟友：同门师姐「苏晴雨」

#### 关键选择点

**选择一：林惊羽的邀约**
- 林惊羽在宗门大比后向你伸出手：
- A. 接受，成为战友（获得林惊羽的好感，组建羁绊）
- B. 婉拒，独自修行（保持独立，获得神秘机缘）

**选择二：天道图碎片**
- 你发现的天道图碎片引起了长老注意：
- A. 交给宗门换取资源（获得大量贡献点，但卷入宗门阴谋）
- B. 藏匿自己研究（获得隐藏功法，但可能被追杀）

**选择三：内门试炼的危机**
- 秘境深处，你发现同门师兄被害的真相，凶手是内门弟子
- A. 揭露真相（获得正义声望，但树敌）
- B. 隐忍不发（获得暗中成长机会，但道心受损）

#### 阶段结局

你成功进入内门，但代价惨重——你卷入了宗门高层关于「天道图」的权力斗争。在拜入内门长老「天璇真人」门下后，你逐渐了解到一个惊天秘密：
> "天道图，乃是上古天道宗祖师从天道本源中抽取的一缕道韵所化。集齐天道图，便可窥见成仙的秘密……但每一块碎片的争夺，都伴随着血雨腥风。"
> 你握着游子剑，凝视着远方的云海。修仙之路，才刚刚开始。

**阶段奖励**：
- 解锁内门系统
- 获得称号：「内门弟子」
- 获得称号：「剑道初成」
- 收苏晴雨为羁绊角色

#### 对应玩法解锁

| 解锁内容 | 说明 |
|----------|------|
| 宗门任务系统 | 完成宗门委托获取贡献点 |
| 装备进阶锻造 | 蓝色/紫色装备制作 |
| 功法系统 | 学习更高级功法 |
| 灵根系统 | 检测并培养灵根 |
| 组队副本 | 多人协作挑战 |
| 灵兽蛋孵化 | 获得初始灵兽 |

### 第三章：宗门崛起

**等级范围**：30-50级（元婴境-化神境）
**对应境界**：金丹（30-40）→ 元婴（40-50）

#### 开篇剧情

> 成为内门弟子五年后，你已突破至金丹境，在宗门年轻一代中名列前茅。然而，平静的修仙界开始出现异变——北州出现神秘势力「天魔教」活动的迹象，南疆蛊渊封印松动。与此同时，九宗内部关于「天道图」碎片的争夺也愈演愈烈。作为天璇真人的亲传弟子，你被卷入了一场关乎宗门存亡的大事件……

#### 核心事件

**事件一：天璇真人失踪**
- 你的师父天璇真人在一次外出探索遗迹后失踪
- 宗门对外宣称是"闭关"，但你发现诸多疑点
- 线索指向：天魔教与宗门内部的勾结
- 你踏上寻找师父的旅途

**事件二：天魔教阴谋**
- 你在追查中发现，天魔教正在收集"上古封印"碎片
- 一旦收集完成，将释放被上古大能封印的"混沌魔兽·饕餮"
- 天魔教教主「厉青冥」的真实身份：原天道宗叛徒，天璇真人的师兄

**事件三：昆仑虚探索**
- 师父留下的线索指向昆仑虚深处
- 你在昆仑虚中获得关键信息：天璇真人被囚禁在"九幽冥域"
- 同行的还有御兽谷弟子「萧火火」与鬼道宗卧底「阿幽」

#### 关键选择点

**选择一：师父的真相**
- 你发现天璇真人当年也参与了"天道图"碎片的争夺，导致云游子陨落
- A. 继续营救，不计前嫌（获得天璇真人完整传承）
- B. 质问师父，选择自己认为正确的道路（获得独立成长机会）

**选择二：九幽冥域**
- 进入九幽冥域需要献祭一件珍宝：
- A. 献祭游子剑（获得冥器，但失去剑灵羁绊）
- B. 献祭天道图碎片（获得冥力，但引发追踪）
- C. 拒绝献祭，强闯（难度极高，但有隐藏奖励）

**选择三：阿幽的真实身份**
- 阿幽在关键时刻暴露是鬼道宗安插的卧底：
- A. 联手对敌（化解恩怨，获得鬼道宗友谊）
- B. 驱逐阿幽（避免风险，但失去潜在盟友）

#### 阶段结局

你成功救出天璇真人，但真相令你震惊——师父当年确实犯下过大错，而天魔教教主厉青冥正是为此复仇。在师父将毕生所学传授给你后，他选择了自我封印以赎罪：
> "因果轮回，从不饶人。我欠下的债，终究要还。你……替为师，将天道图碎片归还天道。"
> 你握着师父递来的最后一枚碎片，凝视着天际。九宗大比在即，天下风云将起。

**阶段奖励**：
- 解锁九宗大比系统
- 获得称号：「天璇传人」
- 获得称号：「冥域行者」
- 师父天璇真人化为「英灵」陪伴

#### 对应玩法解锁

| 解锁内容 | 说明 |
|----------|------|
| 九宗大比 | 年度PK赛事 |
| 灵兽进阶系统 | 灵兽突破与进化 |
| 翅膀系统 | 获得飞行翅膀 |
| 宗门战争 | 宗门对抗玩法 |
| 世界BOSS | 全服参与击杀 |
| 仙侣殿（婚姻系统） | 结识仙侣 |

### 第四章：风云际会

**等级范围**：50-70级（炼虚境-大乘境）
**对应境界**：化神（50-60）→ 炼虚（60-70）

#### 开篇剧情

> 九宗大比上，你代表天道宗出战，一路过关斩将，最终与天魔教少主「厉无心」决战于云顶天宫。这一战，你险胜半招，名震天下。然而，胜利的喜悦还未散去，一个更可怕的消息传来——饕餮封印已开始松动，第一次"劫前征兆"出现了……

#### 核心事件

**事件一：九宗大比巅峰对决**
- 你在决赛中战胜厉无心，但厉无心临败前说出惊天秘密：
- "你知道为什么天道宗能成为九宗之首吗？因为他们，才是当年封印饕餮的背叛者！"
- 你开始追查天道宗的历史黑幕

**事件二：饕餮封印松动**
- 昆仑虚深处，上古封印开始龟裂
- 饕餮的一缕意识向外传递：修仙界将迎来"清算之日"
- 各方势力开始明争暗斗，有人想阻止大劫，有人想利用大劫
- 世界频道：全服首次联动事件开启

**事件三：灵界通道开启**
- 灵气潮汐加剧，连接凡界与灵界的"飞升通道"短暂开启
- 你获得进入灵界的机会，但时间有限
- 灵界探索中，你发现了上古大能留下的"证道遗迹"

#### 关键选择点

**选择一：天道宗的黑历史**
- 你发现天道宗祖师确实参与了"饕餮背叛"事件
- A. 揭露真相，颠覆天道宗（获得反叛者支持，但被正道敌视）
- B. 隐忍不发，从内部改革（获得宗门信任，但背负骂名）
- C. 超然物外，专注修炼（保持中立，但失去改变格局的机会）

**选择二：厉无心的联盟**
- 饕餮觉醒在即，厉无心向你提议联合：
- A. 接受联盟，共同对抗饕餮（获得魔道资源，但被正道怀疑）
- B. 拒绝联盟，保持正道身份（获得正道支持，但腹背受敌）

**选择三：灵界探索**
- 证道遗迹中，你面临最终考验：
- A. 选择剑道（获得天外飞仙剑诀传承）
- B. 选择天道（获得天道宗至高心法）
- C. 选择自我（触发隐藏结局，获得前所未有的「自在道心」）

#### 阶段结局

饕餮的第一道封印彻底崩溃，混沌魔气开始侵蚀九州。你与正道七宗、魔道三教暂时结盟，共同守护最后的防线。在这场大劫前哨战中，你突破至大乘境，被誉为"末法以来的第一天才"：
> 大劫已至，天道不仁，以万物为刍狗。
> 而你，站在这个世界的十字路口，将书写属于你自己的传说。

**阶段奖励**：
- 解锁渡劫系统
- 获得称号：「天下第一天骄」
- 获得称号：「渡劫真人」
- 厉无心结为羁绊角色

#### 对应玩法解锁

| 解锁内容 | 说明 |
|----------|------|
| 渡劫系统 | 突破大乘，迎接天劫 |
| 灵界探索 | 进入灵界位面 |
| 证道遗迹 | 争夺成仙机缘 |
| 全服宗门战 | 服务器级对抗 |
| 仙界飞升 | 突破至真仙 |

### 第五章：渡劫飞升

**等级范围**：70-85级（渡劫境-真仙境）
**对应境界**：大乘（70-80）→ 渡劫（80-90）→ 真仙

#### 开篇剧情

> 饕餮魔气席卷九州，正邪两道的精英在"天魔战场"与混沌魔族展开决战。你的修为已达大乘巅峰，渡劫在即。然而，天劫并非最可怕的敌人——有人在暗中操控你的渡劫，试图将你扼杀在成仙的最后一步……

#### 核心事件

**事件一：天劫降临**
- 你选择在中州"九霄雷海"渡劫
- 天劫分为九重，一重比一重可怕
- 第九重天劫中，你看到饕餮的虚影——有人在利用天劫对你进行精神攻击
- 关键辅助：羁绊角色联手为你护法

**事件二：真仙劫真相**
- 渡劫成功后，你踏入真仙境，但记忆出现混乱
- 你发现自己的记忆被人篡改过，真正的身份并非普通散修
- 你是上古大能"道君"转世，前八纪元的记忆正在觉醒
- 道君当年参与了饕餮封印，却遭背叛，陨落转世

**事件三：灵界大洗牌**
- 饕餮本体开始挣脱封印，入侵灵界
- 真仙以上的修士必须做出选择：留守凡界对抗饕餮，还是飞升仙界逃避
- 你选择留下——因为你的因果，都在凡界

#### 关键选择点

**选择一：道君传承**
- 前世记忆觉醒，你获得道君的完整传承，但也承受着前世的因果
- A. 继承道君遗志，弥补当年遗憾（获得完整传承，但背负历史责任）
- B. 走出自己的道路，超脱前世因果（获得独立修行，但传承不完整）

**选择二：留守还是飞升**
- 仙界通道开启，真仙可以选择离开
- A. 留守凡界，对抗饕餮（获得凡界气运加成，但面临生死危机）
- B. 飞升仙界，从长计议（获得仙界资源，但失去凡界信任）

**选择三：饕餮的真相**
- 你发现饕餮并非纯粹的邪恶——它曾是天道的守护者，被背叛后才堕入混沌
- A. 尝试与饕餮沟通（极高风险，但可能化解恩怨）
- B. 坚定灭杀饕餮（获得正道支持，但难度极大）

#### 阶段结局

你成功飞升真仙境，并在凡界的决战中重伤饕餮，为世界争取了喘息之机。仙界的大门为你敞开，但你知道，这只是更大风暴的序幕：
> 真仙之名，不过是仙途起点。
> 灵界的征途，才是你真正的舞台。

**阶段奖励**：
- 解锁灵界系统
- 获得称号：「真仙」
- 获得称号：「道君转世」
- 获得仙界货币「仙玉」

#### 对应玩法解锁

| 解锁内容 | 说明 |
|----------|------|
| 灵界探索 | 在灵界建立势力 |
| 仙界飞升通道 | 突破至玄仙 |
| 真仙战场 | 真仙级PVP |
| 天道碎片收集 | 收集完整天道图 |
| 仙盟系统 | 仙界组织 |

### 第六章：灵界征途

**等级范围**：85-95级（真仙境-金仙境）
**对应境界**：真仙 → 玄仙 → 金仙

#### 开篇剧情

> 你飞升灵界后，发现这里与凡界截然不同——真仙满地走，金仙多如狗。你的真仙境修为在灵界不过是最底层。灵界各方势力盘踞，而凡界饕餮的威胁仍未解除。你必须在这里快速成长，才能在即将到来的终极决战中有一战之力。

#### 核心事件

**事件一：灵界宗门入门**

## 6. 大劫倒计时设定

### 6.1 大劫的真正面目

**本质：天道自我修复机制**

轮回大劫并非简单的天灾，而是**天道积攒了九个纪元的因果浊气达到临界点后的自我清洗**：

1. **浊气来源**：九个纪元以来，所有修士的因果孽债、怨念、执念都会渗透入天道本源，如同人体内的毒素
2. **大劫作用**：通过毁灭与重生，清除浊气，重置世界，让一切重新开始
3. **天玄子的阴谋**：天玄子试图利用大劫，清除天道中的其他存在，让自己成为新的天道意识

**饕餮的真相**：
- 饕餮原是天道的"清道夫"，负责清除天道的浊气
- 但被天玄子陷害，背负"混沌魔兽"的污名，被封印至今
- 饕餮并非想毁灭世界，它只是想恢复名誉，讨回公道

### 6.2 大劫时间线（玩家可感知）

### 6.3 各方势力对大劫的态度

**正道宗门（天道宗、万法仙宗、玄天剑宗等）**：
- 态度：阻止大劫，保存修仙文明
- 行动：加固封印、组织联军
- 隐患：内部有天玄子卧底，暗中推动有利于天玄子的大劫进程

**魔道（天魔教）**：
- 态度：利用大劫夺取权力
- 行动：收集封印碎片，故意松动封印
- 真相：厉青冥被天玄子利用，实际是在帮助天玄子的计划

**隐世古族（万佛寺、鬼道宗等）**：
- 态度：超然物外，但会自保
- 行动：关闭山门，保存实力
- 隐患：各有算盘，无法指望

**妖族（御兽谷等）**：
- 态度：寻求自保
- 行动：部分妖族倒向饕餮（饕餮曾庇护妖族）
- 隐患：可能成为饕餮的盟友

**灵界势力**：
- 态度：凡界蝼蚁，与我何干
- 行动：关闭通道，拒绝援助
- 真相：灵界同样受到威胁，只是还未波及

**玩家**：
- 位置：唯一变数
- 使命：打破天玄子的阴谋，改变命运
- 关键：道君转世的身份 + "遁去的一"

### 6.4 大劫各阶段剧情演变

**第一阶段（1-50级）：征兆初现**
- 触发：游戏开服活动"天象异变"
- 表现：世界频道播报天文奇观
- 玩法：新增"天象任务"，收集星象碎片
- 剧情：云游子剑灵首次预警

**第二阶段（50-70级）：封印松动**
- 触发：玩家首次击杀世界BOSS
- 表现：饕餮魔气影响地图，出现魔化怪物
- 玩法：新增"魔化副本"，击败魔化BOSS
- 剧情：厉无心首次提出合作

**第三阶段（70-85级）：通道撕裂**
- 触发：全服玩家平均等级达到要求
- 表现：灵界通道开启又关闭，世界陷入混乱
- 玩法：三界联动活动，跨服战斗
- 剧情：道君记忆大量觉醒

**第四阶段（85-95级）：魔气蔓延**
- 触发：特定服务器事件
- 表现：部分地图永久魔化，原住民NPC变化
- 玩法：魔化地图争夺战，守护最后的净土
- 剧情：天玄子现身

**第五阶段（95-100级）：大劫降临**
- 触发：主线剧情推进
- 表现：饕餮本体觉醒，终极世界BOSS战
- 玩法：全服联动，世界级战斗
- 剧情：饕餮与玩家对话，揭示真相

**终局（100级）**：
- 触发：完成所有主线
- 表现：道界之门开启
- 玩法：终极抉择，三种结局
- 结局：「天道独尊」「大道归一」「超脱轮回」
