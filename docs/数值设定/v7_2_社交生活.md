**洞府属性公式**:
```
total_charm = sum(decorations.charm)
total_comfort = sum(decorations.comfort)
charm_bonus = 1 + total_charm / 1000  // 每1000魅力=+10%灵石收益
comfort_bonus = 1 + total_comfort / 500  // 每500舒适度=+10%灵气速率
```

### 4.3 夫妻双修
```
base_spirit = player_spirit_rate × (1 + technique_bonus)
dual_cultivate_spirit = base_spirit × (1 + intimacy_bonus × 0.01) × (1 + realm_gap_penalty)
realm_gap_penalty = max(0, 1 - abs(realm_diff) × 0.05) // 境界差>5时惩罚
```

| 境界差 | 双修效率 |
|--------|---------|
| 0-2 | 100% |
| 3 | 90% |
| 4 | 80% |
| 5 | 70% |
| 6+ | 50%（最低） |
