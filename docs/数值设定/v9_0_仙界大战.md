### 3.2 战场地图

| 资源点 | 占领积分/s | 特殊效果 |
|--------|----------|---------|

```

// kill_bonus = 击杀敌对玩家累计积分（每击杀+20分）
```

## 6. PVP伤害计算（势力战/诸天战场）

### 6.1 伤害公式
```
PVP_damage = (player_atk × 0.8 - enemy_def × 0.3) × skill_multiplier × realm_penalty

realm_penalty = min(1.0, attacker_realm/ defender_realm)
// 境界差惩罚：低境界打高境界时伤害衰减
// 衰减上限：境界差3级以上时，伤害降至60%
```

### 6.2 防御衰减机制
```
// 连续被同一玩家攻击时，防御逐渐衰减
def_degradation = 0.1 × consecutive_hits
// 每次命中降低目标10%防御（叠加），离开战斗10秒后重置
```

### 6.3 境界压制表
| 境界差 | 伤害比例 | 说明 |
|--------|---------|------|
| 0-1 | 100% | 无惩罚 |
| 2 | 90% | 轻微压制 |
| 3 | 75% | 明显压制 |
| 4 | 60% | 严重压制 |
| 5+ | 50% | 极限压制（最低） |

## 7. 战争全局数值

### 7.1 战争参与度目标
```
目标：每周势力战参与率 > 60%
- 活跃势力成员数 > 300
- 诸天战场匹配时间 < 2分钟
```

### 7.2 资源点总产出
```
全服资源点每日总产出：
- 小型灵矿：50个 × 50,000 = 2,500,000灵石/天
- 中型灵石矿：20个 × 200,000 = 4,000,000灵石/天
- 大型灵石矿：5个 × 500,000 = 2,500,000灵石/天
- 天劫圣地：10个 × (突破丹价值) = 每日10个突破丹
```

### 7.3 奖励经济平衡
```
每周势力战总奖励：约 10,000,000 灵石
每周诸天战场总奖励：约 50,000,000 灵石
对比日活玩家灵石获取（普通）：约 1,000,000 灵石/天
→ 战争奖励为核心玩家提供3-5倍额外收益
```
