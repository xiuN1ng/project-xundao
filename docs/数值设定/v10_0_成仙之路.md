## 1. 飞升前置条件

### 1.1 飞升条件检查
| 条件 | 要求 | 达成难度 |
|------|------|---------|
| 境界 | 渡劫期大圆满（99阶） | ★★★★★ |
| 灵魂契合度 | ≥80% | ★★★ |
| 渡劫丹 | ×1 | ★★★ |
| 天劫洗礼次数 | 3次 | ★★★★ |
| 渡劫材料包 | ×1 | ★★★ |

### 1.2 天劫洗礼
```
天劫洗礼需求 = base_tribulation_cost × (1 + past_attempts × 0.5)
past_attempts = 飞升失败次数
首次飞升成功率 = 50% + 灵魂契合度×0.5%
```
| 飞升尝试 | 成功率 | 渡劫丹消耗 | 失败惩罚 |
|---------|-------|----------|---------|
| 第1次 | 50-90% | 1个 | 10%当前灵气+24小时冷却 |
| 第2次 | 60-95% | 1个 | 5%当前灵气+12小时冷却 |
| 第3次 | 80-98% | 1个 | 3%当前灵气+6小时冷却 |
| 第4次+ | 95%+ | 1个 | 1%当前灵气+3小时冷却 |

## 8. 关键公式

### 8.1 飞升成功率
```
base_success_rate = 0.5
soul_match_bonus = soul_match_pct × 0.005
past_failure_penalty = past_failures × 0.05
realm_quality_bonus = avg_realm_stat × 0.00001

final_success_rate = min(0.99, base_success_rate + soul_match_bonus - past_failure_penalty + realm_quality_bonus)
```

### 8.2 仙气累积速率
```
immortal_spirit_rate = (
    base_spirit_rate × 10  // 飞升后10倍基础
    + realm_bonus(8.0)    // 仙人境界加成
    + building_bonus      // 仙盟建筑
    + technique_bonus      // 仙术加成
) × (1 + alliance_member_count × 0.05)
```

### 8.3 仙术效果
```
skill_effect = base_effect × skill_level × (1 + immortal_level/50)
// 仙术效果随境界和技能等级双维度成长
```

```
仙气用途:
```

```
天道境玩家极限属性（理论值）:
```
