### 1.2 难度递增公式
```
tower_monster_hp = base_hp × (塔层难度倍率)
tower_monster_atk = base_atk × (塔层难度倍率)
tower_monster_def = base_def × (塔层难度倍率)

// 每层实际难度（非线性递增）
layer_difficulty = layer_multiplier × (1 + (layer - 1) × 0.02)
// 即每层+2%额外难度
```

## 5. 关键公式

### 5.1 战力推荐
```
recommended_player_power = tower_monster_hp × 0.6 + tower_monster_atk × 0.4
stable_clear_power = recommended_power × 1.5  // 稳定通关所需战力
// 例：50层怪物HP=100M → 稳定通关需玩家HP≈150M+
```

### 5.2 扫荡逻辑
```
// 玩家可扫荡已通关的最高层以下任意层
sweep_rewards = normal_clear_rewards × 0.8  // 扫荡获得80%正常奖励
sweep_cost = 0（免费扫荡）
sweep_limit = 无限制（可重复扫荡）
```

### 5.3 排行榜评分
```
tower_score = max_floor_reached × 10000 + (100000000 - clear_time_seconds)
// 分数 = 层数×10000 + (换算时间分)
```
