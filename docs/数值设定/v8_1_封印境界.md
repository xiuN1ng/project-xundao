**版本**: v8.1  
**系统**: 封魔渊（Sealed Demon Abyss）  
**定位**: 后期（渡劫期后）循环挑战内容，魔晶为独有货币  

---

## 1. 核心资源：魔晶（Demon Crystal）

### 1.1 魔晶定位
- 独立货币，仅产出于封魔渊
- 用于兑换高阶稀有道具、装备强化材料
- **不与灵石互通**，形成独立经济循环

### 1.2 魔晶产出公式
```
base_crystal = floor(monster_max_hp × 0.002 × realm_tier)
bonus_crystal = base_crystal × min(kill_streak / 10, 3.0)
final_crystal = base_crystal + bonus_crystal
```

## 2. 封魔渊副本结构

### 2.1 副本参数
| 参数 | 数值 |
|------|------|
| 解锁条件 | 渡劫期大圆满（第99阶）|
| 进入消耗 | 0（免费，但有每日进入次数限制）|

| 第二层封印 | 11-20 | 2.5x | 通关第一层封印 |

| 第四层封印 | 31-40 | 15.0x | 通关第三层封印 |

## 6. 关键公式

### 6.1 伤害计算
```
final_damage = (player_atk - monster_def × 0.5) × skill_multiplier × realm_bonus
if (crit): final_damage ×= 1.5
if (breakthrough): final_damage ×= 1.2（破防伤害）
```

### 6.2 每日重置逻辑
```
// 每日05:00重置
if (current_time >= next_reset_05:00):
    player.daily_sealed_abyss_entries = 3
    // 保留未完成的层数进度，不重置通关记录
```

### 6.3 挑战难度推荐
```
recommended_power = monster_hp × 0.8 + monster_atk × 0.2
// 玩家战力 >= recommended_power × 1.5 时可稳定通关
```

## 7. 数值平衡性校验

### 7.1 魔晶经济闭环
```
玩家每日收益（平均）:
- 3次全通关深渊1-50: 约 800,000 魔晶/天
- 封魔令牌回本周期: 200,000 / 3,000 ≈ 67天（首次购买）
- 魔皇血脉回本周期: 200,000 / 1,000 ≈ 200天（性价比低，付费导向）
```

### 7.2 战力平衡检查
```
渡劫期大圆满玩家战力（参考）:
- HP: ~300,000（境界×realm_bonus5.0+装备加成）
- ATK: ~35,000
- DEF: ~15,000

深渊50层BOSS需要战力: ~20,000,000 HP → 需装备+灵兽+法宝协同
```

## 8. 与其他系统的协同

| 协同系统 | 效果 |
|---------|------|
| 境界系统 | 渡劫期才可进入，确保玩家成熟度 |
| 法宝系统 | 魔器为独立装备栏，不与仙器冲突 |
| 成就系统 | 各封印区通关触发成就 |
| 称号系统 | 封魔称号提供属性加成 |
| 灵兽系统 | 灵兽可协助挑战，提升输出 |
