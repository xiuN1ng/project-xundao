## 二、核心问题清单

### P0 - 阻断级问题 (必须修复)

#### 1. 【前端】深渊副本 API 路由缺失 🚨
- **问题**: `AbyssDungeonPanel.vue` 调用 `/api/abyssDungeon/*` 系列API，但 `server/backend/routes/abyssDungeon.js` **文件不存在**
- **影响**: 玩家点击深渊副本任意按钮都会得到404错误，游戏直接报错
- **证据**: `server/backend/server.js:76` 注册了路由但文件不存在
- **解决方案**: 
  1. 创建 `routes/abyssDungeon.js`，实现以下API:
     - `GET /api/abyssDungeon/config` - 获取副本配置
     - `GET /api/abyssDungeon/list` - 获取副本列表
     - `POST /api/abyssDungeon/enter` - 入场
     - `POST /api/abyssDungeon/battle` - 战斗
     - `POST /api/abyssDungeon/nextLayer` - 下一层
  2. 参考 `realm_dungeon_system.js` 和 `resource_dungeon_system.js` 的后端数据
- **工作量**: 中
- **负责人**: [B]

#### 2. 【前端】VIP系统完全是 Stub 🚨
- **问题**: `VipPanel.vue` 仅34行，hardcoded数据 `vipLevel=1`，无任何API调用
- **影响**: VIP系统形同虚设，无法激励付费；月卡/VIP特权无实际效果
- **证据**: 组件只有静态数据，无 `fetch/axios` 调用
- **解决方案**:
  1. 对接 `/api/vip` 路由（已存在于backend）
  2. 实现VIP等级展示、特权列表、升级按钮
  3. VIP每日礼包领取功能
  4. VIP专属副本入口
- **工作量**: 中
- **负责人**: [F]

#### 3. 【前端】世界BOSS面板是 Stub 🚨
- **问题**: `WorldBossPanel.vue` 仅28行，`boss={name:'年兽',hp:75}` 是hardcoded
- **影响**: 世界BOSS是重要的社交/全服活动，完全无法使用
- **证据**: 组件无任何API调用；后端 `worldboss_system.js` 和 `routes/worldBoss.js` 都存在
- **解决方案**:
  1. 对接 `/api/worldBoss` 路由
  2. 实现BOSS列表、刷新时间、攻击按钮、伤害排名
  3. 实现BOSS血条、战斗结算、击杀奖励
- **工作量**: 中
- **负责人**: [F]

#### 4. 【前端】周卡/月卡面板是 Stub 🚨
- **问题**: `WeekPanel.vue` 仅23行，只有两个静态卡片显示"周卡¥30""月卡¥98"
- **影响**: 重要付费入口无法使用，直接损失收入
- **证据**: 组件无任何逻辑，点击无任何反应
- **解决方案**:
  1. 对接支付API（参考 `PaymentPanel.vue` 的首充逻辑）
  2. 实现周卡: 每日领取100灵石，持续7天
  3. 实现月卡: 每日领取500灵石+专属礼包，持续30天
  4. 添加"已续费"状态和剩余天数显示
- **工作量**: 小
- **负责人**: [F]

## 五、本轮新增方案摘要

### 新功能方案：装备套装收集系统 [P1-2]
**背景**: 当前装备只有单件属性，缺乏长期收集目标。  
**方案**: 实现套装UI，玩家收集指定件数激活套装效果。  
**核心数值** (参考梦幻西游套装设计):
- 2件套: 基础属性+10%
- 4件套: 特殊技能解锁  
- 6件套: 套装特效（光环/粒子）
- 套装类型: 凡品/良品/精品/极品/神话 (5档)

### 新功能方案：缺灵石引导系统 [P2-3]
**背景**: 新玩家经常卡在"不知道去哪获取灵石"。  
**方案**: 当灵石<100时，顶部资源栏出现问号图标，点击展开引导菜单。  
**引导菜单内容**:
1. 挂机修炼 → 稳定产出灵石
2. 通关副本 → 高效获取灵石+装备
3. 每日任务 → 任务奖励面板
4. 商城特惠 → 每日折扣礼包

## 六、结论

本轮体验发现了 **4个P0阻断问题**，其中最严重的是 **深渊副本API完全缺失**（玩家点击即报错）。另外 **3个面板是Stub**（VIP/世界BOSS/周卡月卡），严重影响付费转化。

建议 **优先级顺序**: P0-1 → P0-2 → P0-3 → P0-4 → P1系列 → P2系列
