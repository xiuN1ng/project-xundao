## 使用方式

### 1. GameImage 组件

```vue
<template>
  <!-- 使用资源路径 -->
  <GameImage path="ui.iconHp" />
  <GameImage path="player.male.mortal" />
  <GameImage path="skill.fireball" />
</template>

<script>
import GameImage from '@/components/GameImage'
export default { components: { GameImage } }
</script>
```

### 2. 资源辅助函数

```javascript
import { 
  getPlayerImage, 
  getEquipmentIcon, 
  getGlowByQuality,
  getRealmEffect 
} from '@/utils/resourceHelper'

// 获取主角立绘
const playerImg = getPlayerImage('male', 3)  // 筑基期

// 获取装备图标
const swordImg = getEquipmentIcon('sword', 'epic')

// 获取灵兽品质光效
const glow = getGlowByQuality('legendary')

// 获取境界特效
const effect = getRealmEffect(4)  // 金丹
```

### 3. 直接引用

```javascript
import Resources from '@/assets/resources'

// 在模板中直接使用
<img :src="Resources.ui.iconHp" />
```
