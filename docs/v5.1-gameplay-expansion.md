# 挂机修仙 v5.1 玩法扩展

**版本**: v5.1  
**主题**: 洞天福地、炼器、炼丹系统升级  
**预计开发周期**: 3周

---

## 1. 洞天福地系统

### 1.1 福地类型

| 福地名称 | 等级 | 特殊效果 | 开启条件 |
|----------|------|----------|----------|
| 灵泉眼 | 1 | 灵气产出+20% | 筑基期 |
| 悟道崖 | 2 | 功法经验+15% | 金丹期 |
| 炼心洞 | 3 | 心魔抵抗+25% | 元婴期 |
| 藏经阁 | 4 | 功法领悟概率+10% | 化神期 |
| 聚宝盆 | 5 | 掉落率+15% | 炼虚期 |
| 轮回池 | 6 | 死亡不掉落 | 合体期 |
| 封神台 | 7 | 渡劫成功率+10% | 大乘期 |

### 1.2 福地建设

#### 建设消耗
- 灵石：根据福地类型和等级
- 材料：专属建筑材料
- 时间：建设时长

#### 升级机制
- 每个福地可升级至10级
- 升级提升效果数值
- 每次升级解锁新功能

### 1.3 福地探索

#### 探索消耗
- 体力：每次探索消耗
- 时间：冷却机制

#### 探索奖励
- 灵石袋
- 稀有材料
- 功法残页
- 随机奇遇触发

### 1.4 福地归属

- 洞天福地可占领
- 占领者获得产出加成
- 可被其他玩家抢夺
- 护宗大阵保护

## 2. 炼器系统升级

### 2.1 器灵系统

#### 器灵类型
| 器灵品质 | 概率 | 效果 |
|----------|------|------|
| 普通 | 60% | 基础属性 |
| 优秀 | 25% | 属性+10% |
| 稀有 | 10% | 属性+20% |
| 史诗 | 4% | 特效+1 |
| 传说 | 1% | 特效+2 |

#### 器灵培养
- 器灵升级：消耗同类器灵或材料
- 器灵进化：特定条件下进化为更高品质
- 器灵技能：每个器灵拥有独特技能

### 2.2 法宝进阶

#### 进阶路线
```
普通法宝 → 灵器 → 宝器 → 仙器 →神器
```

#### 进阶条件
- 等级要求
- 材料需求
- 成功率机制

| 进阶类型 | 成功率 | 失败惩罚 |
|----------|--------|----------|
| 普通→灵器 | 80% | 返还50%材料 |
| 灵器→宝器 | 60% | 返还30%材料 |
| 宝器→仙器 | 40% | 返还20%材料 |
| 仙器→神器 | 20% | 返还10%材料 |

### 2.3 炼器配方

#### 新增配方
- 武器：青虹剑、太乙金仙剑、翻天印
- 防具：玄冰甲、金缕玉衣、天地法袍
- 饰品：乾坤圈、混天绫、九龙神火罩

#### 配方获取
- 任务奖励
- 副本掉落
- 交易市场购买
- 玩家交易

### 2.4 炼器特殊效果

| 效果名称 | 触发条件 | 效果描述 |
|----------|----------|----------|
| 吸血 | 攻击时 | 恢复造成伤害的5% |
| 反射 | 受击时 | 反弹10%伤害 |
| 冰冻 | 技能攻击 | 15%概率冰冻敌人2秒 |
| 燃烧 | 技能攻击 | 15%概率造成持续伤害 |
| 闪避 | 被攻击 | 10%概率完全闪避 |
| 暴击 | 攻击时 | 伤害+50% |

## 3. 炼丹系统升级

### 3.1 丹方系统扩展

#### 新增丹方
| 丹药名称 | 功效 | 材料 |
|----------|------|------|
| 悟道丹 | 功法经验+50% | 悟道茶叶×3, 灵草×10 |
| 渡劫丹 | 渡劫伤害-20% | 雷劫木×5, 灵乳×3 |
| 破境丹 | 突破成功率+15% | 天地灵果×2, 仙露×5 |
| 合体丹 | 灵气上限+30% | 龙血草×5, 麒麟角×2 |
| 延寿丹 | 寿命+100年 | 蟠桃×1, 人参果×1 |
| 轮回丹 | 死亡原地复活 | 阎罗草×3, 鬼面花×3 |

#### 丹方品质
- 普通：基础效果
- 优秀：效果+20%
- 稀有：效果+40%
- 传说：效果+60%，额外特效

### 3.2 炼丹技术

#### 火力控制
- 文火：成丹率高，药效温和
- 武火：成丹率低，药效猛烈
- 三味真火：特殊丹药必需

#### 手法技巧
- 搅拌：影响丹药均匀度
- 时机：影响丹药品质
- 封印：影响丹药保存时间

### 3.3 丹炉升级

| 丹炉等级 | 品质加成 | 特殊效果 |
|----------|----------|----------|
| 青铜炉 | +5% | 无 |
| 白银炉 | +10% | 失败返还20%材料 |
| 黄金炉 | +15% | 失败返还30%材料 |
| 玉清炉 | +20% | 必定出双份 |
| 紫金炉 | +25% | 有概率出三份 |
| 八卦炉 | +30% | 稀有丹方概率翻倍 |

### 3.4 药材系统

#### 新增药材
- 灵草类：龙血草、凤尾草、冰晶草
- 矿石类：千年寒铁、万年温玉
- 特殊类：九天玄露、混沌之气

#### 药材培育
- 灵田升级提升产量
- 特殊气候影响品质
- 杂交培育新品种
