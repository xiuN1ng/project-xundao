# 第一部·第四章·第十节

## 夜归青云·暗流涌动

云无咎离开天璇峰时，已是后半夜。

夜色深沉如墨，漫天星斗点缀在苍穹之上。月亮不知何时已躲入云层之后，天地间一片昏暗，只有山间的萤火虫在草丛间明灭不定，为这片寂静的夜增添了几分幽微的光亮。

他沿着蜿蜒的山路独自前行，脑海中却依然回荡着方才在天璇峰上经历的一切。

天璇真人……

那个看上去儒雅温和的中年男子，修为深不可测，心思更是难以捉摸。他明明有能力将自己碾成齑粉，却偏偏没有动手。他明明可以强行夺取游子剑，却只是提出用丹药交换。他甚至还以一种极为隐秘的方式，传递了关于"证道遗迹"的信息。

这一切，都让云无咎感到困惑不已。

"他究竟是什么人？"

"他是云游子前辈的仇人，还是……另有隐情？"

"那句'孩子'，又是什么意思？"

云无咎想不通。

太多的疑问，太少的答案。

他深吸一口气，将这些杂念暂时压下，加快脚步向青云峰走去。

回到住处时，东方已经泛起了鱼肚白。

晨曦的第一缕光芒穿透云层，将天边染成淡淡的金红色。新的一天，即将开始。

云无咎推开房门，却发现自己毫无睡意。

他在榻边坐下，将游子剑横放于膝前，低头凝视着那柄锈迹斑斑的古剑。

三个月的时光，让这柄剑发生了微妙的变化。剑身上的锈迹淡了几分，隐约能看出曾经锋利的剑刃轮廓。剑柄处那个模糊的"叶"字铭文，在晨光中若隐若现，仿佛在诉说着它不为人知的过往。

"前辈……"

云无咎轻声呢喃。

"您究竟遭遇了什么？天璇真人又是什么人？那段记忆中的天玄子，与天璇峰有什么关系？"

游子剑沉默不语。

他想要再次进入剑中，与云游子的残念交流，却发现自己的神识已经疲惫到了极点。今夜经历了太多——青峰岭的激战、与林惊羽的对话、枯骨的记忆传递、天璇真人的召见……每一件事都消耗了他大量的心神，此刻他已经无力再进行任何高强度的神识交流。

"罢了，先休息吧。"

云无咎叹了口气，将游子剑放在枕边，和衣躺下。

疲惫如潮水般涌来，很快便将他淹没。

---

这一觉，云无咎睡到了日上三竿。

当他再次睁开眼睛时，阳光已经从窗棂的缝隙中斜斜照入，在地面上投下几道明亮的光斑。

云无咎坐起身，只觉浑身上下酸痛无比，像是被马车碾过一般。那是昨夜与青鳞狼激战留下的后遗症，当时全凭一口气撑着，此刻松懈下来，才觉出不对劲。

他从包袱中取出一枚回灵丹，吞服下去，又运转了几个周天的《云游心法》，那股疲惫感才渐渐消散。

"今日开始，要更加努力修炼了。"

他低声自语。

距离外门大比还有三个月。在那之前，他必须尽可能地提升自己的修为。

赵乾坤的威胁依然存在。这个纨绔子弟睚眦必报，绝不可能轻易放过他。而天璇峰上的那位神秘真人，似乎也对他手中的游子剑虎视眈眈。

更别提那个隐藏在历史深处的"天玄子"，以及那座传说中的"证道遗迹"……

这一切，都让他感到一种前所未有的紧迫感。

"我必须变强。"

云无咎握紧拳头，目光坚定。

他翻身下床，简单洗漱之后，便推门而出。

清晨的青云峰，空气清新，鸟语花香。

云无咎沿着熟悉的山道向峰顶走去。那里是他每日晨练的地方，也是他最不受打扰的修炼圣地。

一路上，他遇到了不少外门弟子。

那些人见到他，神色各异——有的投来好奇的目光，有的则窃窃私语，更有几个明显是赵乾坤手下的人，远远地瞪了他一眼，便转身离去。

云无咎恍若未闻，脚步不停。

他知道，自己的处境并不乐观。赵乾坤的威胁、林惊羽的关注、天璇真人的召见……这一切都让他成为了众人瞩目的焦点。有的人对他好奇，有的人对他嫉妒，还有的人对他虎视眈眈。

但他并不在意。

别人怎么看他，与他何干？他只需要做好自己该做的事——修炼，变强，然后揭开所有的真相。

来到峰顶，云无咎照例在青石上盘膝而坐，开始每日的晨练。

灵气从四面八方汇聚而来，沿着经脉缓缓流入丹田。他闭上双眼，将心神沉入修炼之中，渐渐地，外界的一切都离他远去。

然而，就在他即将进入入定状态的时候，一道不和谐的声音忽然传入耳中。

"哟，这不是云师弟吗？这么早就来修炼了？真是勤奋啊。"

云无咎眉头微皱，睁开眼睛。

只见峰顶小径上，七八名外门弟子正鱼贯而来。为首的那个尖嘴猴腮、满脸谄媚，正是赵乾坤的第一号狗腿子——孙麻。

"云师弟昨日在青峰岭可真是威风啊。"孙麻皮笑肉不笑，"听说遇到了林师兄？啧啧，真是好运气。"

云无咎面色平静，没有接话。

他知道，孙麻是来试探的。赵乾坤虽然当面不敢动手，却派手下的人来打探消息，想要看看他的虚实。

"可惜啊，"孙麻叹了口气，脸上却挂着一丝幸灾乐祸的笑意，"有些人自以为攀上了高枝，殊不知树大招风。林师兄是何等人物，岂是他一个小小的散修能高攀的？"

"就是就是。"旁边一个虎背熊腰的弟子附和道，正是钱虎，"也不照照镜子，看看自己是什么东西。"

几人哄笑起来，笑声中满是嘲讽与恶意。

云无咎静静地坐在那里，目光平静如水。

他看着眼前这些人，忽然觉得有些可笑。

他们以为言语上的挑衅能伤害到他？他们以为赵乾坤的势力能压垮他？他们根本不知道，他经历了什么，他背负着什么，他又将走向何方。

这些人，在他眼中，不过是跳梁小丑罢了。

"说完了吗？"

云无咎淡淡开口。

孙麻一愣，似乎没想到他会这样回应。

"云无咎，你别不识好歹！"钱虎瞪大眼睛，"赵师兄只是让你知道，在这天道宗，有些人是你得罪不起的！"

"我知道了。"云无咎点点头，"还有别的事吗？"

他的语气平淡，像是在询问一件无关紧要的小事。

孙麻等人面面相觑，一时不知如何接话。

他们本以为云无咎会愤怒、会害怕、会惶恐，却没想到对方竟然如此平静。那种平静不是装出来的，而是一种发自内心的、从骨子里透出来的从容。

仿佛他们这些人，根本不值得他多看一眼。

"你……"

孙麻正要说什么，却被身后一人拉住了。

那是一个沉默寡言的青年，面容冷峻，站在人群最后面，却给人一种莫名的压迫感。

"走吧。"

那青年只说了两个字，便转身离去。

孙麻咬了咬牙，似乎有些不甘，但还是带着人离开了。

云无咎目送他们离去，目光微微闪动。

那个沉默的青年……他从未见过。但对方身上的气息，却让他隐隐感到一丝不安。那不是普通外门弟子能有的气息，更像是……

"算了，先不管这些。"

云无咎收回目光，重新闭上眼睛。

无论如何，当务之急是尽快提升修为。

只有足够强大，才有资格去追寻真相。

只有足够强大，才有资格去应对即将到来的风暴。

而现在，他唯一能做的，就是修炼。

日复一日。

夜复一夜。

直到那一天的到来。
