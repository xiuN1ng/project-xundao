# 第一部·第一章·第七节：山中苦修

## 一、日复一日

翌日清晨，云无咎开始了他的修炼生涯。

他没有告诉任何人他已经开始修炼了——包括李狗蛋，包括云爷爷，甚至包括那些对他议论纷纷的镇上百姓。他只是像往常一样，天不亮便起身，出门砍柴，日落方归，看起来与往日毫无二致。

然而只有他自己知道，每天夜里，他都在房中盘膝而坐，按照《云游心法》的法诀，一点一点地吸纳着周身的灵气。

修炼是枯燥的。

第一层"感应灵气"，说起来简单，做起来却需要日复一日的坚持。他必须学会在纷乱的思绪中保持心神的宁静，必须学会用意识去触碰那看不见摸不着的灵气，必须学会将自己融入天地之间，与万物合而为一。

云游子前辈留下的玉简中写道：*"气感者，修炼之始也。感而不觉，是为愚夫；觉而不悟，是为凡品；悟而不行，是为空谈。唯感之、觉之、悟之、行之，方为正道。"*

他将这句话写在了一张纸上，贴在床头，每日诵读。

第一个七天，他将《云游心法》第一层修炼至小成。

那一日深夜，他如往常一般在房中打坐，忽然感觉到丹田中的气流比往常浓郁了几分。那气流不再是若有若无的一缕，而是凝聚成了一个隐约可见的气旋，在丹田处缓缓转动。

成了。

他睁开眼，眼中闪过一丝喜色。

第二层"吸纳灵气"，比第一层难了不止十倍。他必须将感应到的灵气引入体内，在丹田中凝聚成气旋。这说起来容易，做起来却需要极高的专注力和极稳的心性。一旦分神，灵气便会四散而去，前功尽弃。

他失败了无数次。

每一次失败，他都重新来过。

他告诉自己：云游子前辈用三百年等来了他，他不能辜负这份机缘。

第二个七天，他将第二层修炼至入门。

第三个七天，第二层小成。

第四个七天，第二层大成。

第五个七天，第二层圆满，正式踏入第三层——凝气化液。

这一个月的修炼，让他深刻体会到了什么叫"修仙之难"。五行灵根固然给了他比常人更好的资质，却也意味着他要比常人付出更多的努力。五种灵气同时吸纳，同时炼化，其中的难度远超单灵根或双灵根的修士。

好在《云游心法》足够稳健平实，一步一个脚印，从不冒进，这才让他能够稳步提升。

这一日，他像往常一样在屋后的老槐树下打坐，忽然听见镇中传来一阵喧哗声。

"不好了！不好了！山里的妖兽下山了！"

"快跑啊！妖兽来了！"

喊叫声、哭嚎声、脚步声乱成一片。

云无咎霍然睁开双眼，只见远处的山坡上烟尘滚滚，一头硕大的黑影正朝着镇子的方向冲来。那黑影的速度极快，眨眼间便冲入了镇边的稻田，将大片的稻禾踏为齑粉。

正是那头野猪妖！

它的后腿上还缠着一圈干涸的血痂——那正是云无咎一个多月前砍出的伤口。

可此刻，那伤口竟已痊愈了。

云无咎的瞳孔骤缩。

它又来了。

而且这一次，它似乎比上次更加狂暴、更加凶悍。

## 二、妖兽复仇

来不及多想，云无咎一跃而起，将游子剑从腰间抽出，朝镇子的方向飞奔而去。

"爷爷！"

他心中只有一个念头——爷爷还在家里！

他一路狂奔，穿过混乱的人群，冲进那条熟悉的小巷。远远地，他便看见茅屋前围了一群人。

"云老头！云老头！"

"快，快抬进去！"

"伤得重不重？"

云无咎挤开人群，冲进茅屋。

只见云爷爷躺在地上，额头上有一道血痕，显然是被什么东西砸中了。他的意识还算清醒，见云无咎进来，便挣扎着想要坐起来。

"无咎……你没事就好……妖兽……快跑……"

"爷爷！"云无咎扑上前，将老人扶起，眼眶通红，"您怎么伤的？"

"没事……被落石擦了一下……"云爷爷摆了摆手，"倒是你，那头畜生怕是被你砍了一刀后怀恨在心，今儿才下山来寻仇。都是爷爷没用，没本事让你早点离开这穷地方……"

"爷爷，别说了。"云无咎强忍泪水，将老人安置好，转身便朝门外走去。

"无咎！你做什么！"云爷爷惊呼。

"我去杀那畜生。"

他的声音很轻，却带着一股不容置疑的坚定。

他知道，以他现在的实力，正面对抗这头妖兽无异于以卵击石。可他也知道，若他不去，那畜生便会继续在镇中肆虐，不知还要伤了多少人。

更重要的是——那畜生是冲着他来的。

这笔账，该由他来还。

## 三、打谷场之战

镇口，打谷场。

人群已经四散奔逃，唯有那头野猪妖还在横冲直撞，将周围的房屋撞得七零八落。它浑身缠绕着比上次更浓的黑气，双目赤红如血，口中滴着腥臭的涎水，每一次冲撞都带着惊天动地的声势。

"无咎哥！"

人群中传来李狗蛋撕心裂肺的喊声。他不知何时又跑了回来，此刻正站在人群最外围，急得满脸通红。

"无咎哥！快跑！你打不过它的！"

云无咎没有回应。

他站在打谷场中央，握紧游子剑，直面那头朝他冲来的妖兽。

"嗷——！"

野猪妖似乎认出了他，赤红的眼睛里闪过刻骨的仇恨。它四蹄猛然蹬地，庞大的身躯如同一辆失控的战车，带着滚滚烟尘，朝他狂奔而来！

"孽畜！"

云无咎一声厉喝，身形骤然暴起！

他知道自己打不过它。

可他也知道，这一战，他不能退。

剑光凛冽，游子剑划出一道凌厉的弧光，直取妖兽的脖颈！

"铛——！"

金铁交鸣之声震耳欲聋。那野猪妖的反应极快，竟在电光石火间抬起前蹄，硬生生挡下了这一剑！

剑锋与獠牙相撞，迸发出刺目的火花。云无咎只觉一股磅礴的巨力从剑身传来，震得他虎口发麻，险些握不住剑柄。他的身形被那股巨力撞得倒飞出去，在空中翻了几个跟头，方才勉强稳住身形。

好强的力量！

这头畜生比一个月前更强了！

野猪妖赤红的双眼死死盯着他，口中发出震怒的嘶吼。四蹄刨地，它再次朝云无咎冲来——

这一次，它的速度更快了。

云无咎咬紧牙关，身形急退。他知道自己绝不能被它正面撞上——那冲力，只怕连人带剑都要被撞成粉碎。

他一边退，一边思索着对策。

硬拼？不行。他的力量远不如这头畜生，正面对抗只有死路一条。

引诱？也不行。这里是打谷场，四周都是人群，若他将妖兽引向人群，后果不堪设想。

那……

就在他一筹莫展之际，他的目光忽然落在了手中的游子剑上。

剑身上泛着一层淡淡的青芒。

那是灵气的光芒。

自从他在石台上触碰玉简、觉醒气感之后，这柄剑便似乎发生了某种变化。它不再是一柄普通的铁剑，而是一柄蕴含着灵性的法剑。

他想起云游子前辈说过的话：*"剑中自有灵性，汝若能与它心意相通，日后必成大器。"*

心意相通……

他闭上眼，将全部心神集中在剑身上。

那股温热的感觉再次出现了。

像是有什么东西在剑中苏醒，正在用一种极轻极弱的声音对他说话。

*"孩子……借你一臂之力……"*

那声音苍老而温和，带着几分熟悉的味道。

是云游子前辈！

那柄剑中，竟真的藏着云游子前辈的剑灵！

"前辈！"云无咎在心中喊道，"请助我一臂之力！"

剑身微微颤动，像是在回应他的呼唤。

下一刻，一股磅礴的力量从剑身涌入他的体内！

那力量并不属于他，却与他体内的灵气完美融合，带着他挥出了这一剑——

"游子九式，第一式——游子吟！"

一道璀璨的剑芒划破长空！

那剑芒长达三丈，青光凛冽，如同一条咆哮的青龙，直直斩向那头野猪妖！

"噗——！"

剑芒斩落，野猪妖的头颅应声飞出！

黑色的血液喷涌而出，染红了半片天空。那庞大的身躯摇晃了两下，轰然倒地，砸起漫天的尘土。

一剑毙命。

四周鸦雀无声。

所有人都呆呆地望着那具身首异处的妖兽尸体，望着那单膝跪地的少年，一时竟不知该作何反应。

"这……"

"他一剑杀了妖兽？"

"那是什么剑法？！"

"那少年……竟是个修仙者？！"

云无咎握着游子剑，单膝跪地，大口大口地喘息着。

方才那一剑，几乎耗尽了他全部的灵气。他的脸色苍白如纸，冷汗湿透了衣衫，眼前一阵阵发黑。

但他笑了。

他赢了。
