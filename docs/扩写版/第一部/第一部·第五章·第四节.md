# 第一部·第五章·第四节：惊羽来访

## 一、白衣剑客

云泽刚走出几步，一道清朗的声音忽然从旁边响起。

"云师弟。"

那声音如同山涧清泉，流过青石，在喧闹的人群中显得格外悦耳。

云泽停下脚步，转头看去。

只见一个身穿白衣的少年正朝他走来。那少年约莫十七八岁，面容清秀，气质出尘，周身萦绕着一股淡淡的剑意。他腰间悬着一柄细长的剑，剑柄上刻着"惊羽"二字，在阳光下泛着冷冽的光芒。

林惊羽。

东州剑道天才，入门三年便已名震天道宗。传说他的剑法天赋百年难遇，曾在一次宗门任务中一剑斩杀三头筑基境巅峰的妖兽，名声大噪。

更重要的是，在去年的宗门大比预选赛中，林惊羽与云泽曾有过一次交手——虽然那次是平局收场，但林惊羽对云泽手中的游子剑产生了浓厚的兴趣。

"林师兄。"云泽拱手行礼。

林惊羽走到他面前，目光从他腰间的游子剑上一扫而过，眼中闪过一丝不易察觉的光芒。

"方才的事，我都看到了。"他压低声音说道，"赵乾坤不是善茬，他在内门有人。这次试炼中，你要小心他。"

云泽点了点头："多谢林师兄提醒。"

"不必客气。"林惊羽看着他，目光中透着几分探究，"我找你，其实是想说另一件事。"

"什么事？"

林惊羽环顾四周，确认没有人注意之后，才压低声音说道："昆仑秘境中，危险重重。到时候若是遇到妖兽，记得不要硬拼。"

他说这话时，语气平淡，仿佛只是在说一件寻常小事。但云泽却从中听出了几分弦外之音。

"林师兄这话，似乎不只是说妖兽吧？"他轻声问道。

林惊羽微微一愣，随即露出一丝赞许的神色。

"你果然是个聪明人。"他说道，"我就直说了——这次试炼，赵乾坤的真正目标不是妖兽，是你。"

"我知道。"云泽回答得很平静。

"你知道？"林惊羽有些意外，"那你还……"

"有些事，躲是躲不掉的。"云泽抬起头，望向远处的天璇峰，"既然躲不掉，不如正面迎上。"

林惊羽沉默了片刻。

他看着眼前这个比自己大不了几岁的少年，忽然发现对方身上有一种他很少见到的东西——那是一种沉稳到近乎冷酷的气质，仿佛这世上没有什么能让他动容。

但他知道，这种沉稳不是天生的，而是经历过大风大浪之后磨砺出来的。

"云师弟。"林惊羽忽然开口，"你我二人虽然只见过几面，但上次交手，让我印象很深。"

"哦？"云泽微微挑眉，"愿闻其详。"

"你的剑法……很特别。"林惊羽斟酌着用词，"我修炼天外飞仙剑诀多年，自问对剑道一途还算有些心得。但你的剑法，却让我有种……说不清道不明的感觉。"

他顿了顿，目光落在云泽腰间的游子剑上："你那柄剑，不是普通的剑吧？"

云泽没有回答，只是静静地看着他。

两人的目光在空中交汇，一个是好奇与探究，一个是警惕与审视。

片刻之后，林惊羽忽然笑了。

"罢了，你不想说，我也不勉强。"他拱了拱手，"只是昆仑秘境中，我希望能与你合作。"

"合作？"云泽有些意外，"林师兄修为高深，为何要与区区在下合作？"

林惊羽没有直接回答，只是若有所思地看了一眼那柄游子剑。

"因为我想知道，那柄剑背后的真相。"他的声音透着几分复杂，"我在宗门典籍中看到过一些记载。游子剑……是太古年间一位剑道大能的佩剑，据说后来成为了天道宗的镇宗之宝的剑胚。"

"但后来不知为何，这柄剑从天道宗消失了。"他看向云泽，"再出现时，就是在你手中。"

云泽心中一动。

这些信息，与云游子剑灵告诉他的完全吻合。林惊羽能知道这些，说明他在宗门典籍上下过一番苦功。这个人，远比他表面上看起来的要深沉。

"林师兄想知道什么真相？"云泽问道。

林惊羽摇了摇头："现在还说不清楚。只是有一种直觉告诉我，这柄剑的背后，藏着一个很大的秘密。"

他看向云泽，眼中闪过一抹复杂的光芒："而这个秘密，或许与天道宗的过去有关。"

云泽沉默了。

他不知道该不该相信林惊羽。这个人来得太突然，说的话又太过巧合——他怎么会知道自己手中的剑不是凡品？又怎么会恰好在这个时机出现，提出合作？

是巧合？还是有人指使？

"云师弟。"林惊羽似乎看出了他的疑虑，"我知道你对我有所戒备。这是正常的，换作是我，也会如此。"

他拱了拱手："但我可以告诉你一件事——我对天璇真人，没有半分好感。"

"此话怎讲？"

林惊羽沉吟片刻，似乎在斟酌该不该说出口。

"三年前……"他终于开口，"我有一个师兄，死在了昆仑秘境里。"

"那次秘境试炼，天璇真人是主持者。而我的师兄，死因成谜，宗门最后给出的结论是'意外身亡'。"

"但我不信。"林惊羽的眼中闪过一抹寒光，"我师兄的实力我清楚，他不可能犯下那种低级的错误。"

"你是说……"

"我什么都没说。"林惊羽打断他，"但我希望你能明白——这次试炼，我和天璇真人，不是同路人。"

云泽看着他，目光中多了几分审视。

片刻之后，他忽然笑了。

"好。"他点了点头，"那便合作。"

林惊羽微微一愣，似乎没想到他会答应得这么干脆。

"你……就这么信我？"

"不是信你。"云泽摇了摇头，"是因为你的仇人，和我的仇人，是同一个人。"

他的目光落在远处天璇峰的方向，眼中闪过一抹深沉的光芒。

"敌人的敌人，未必是朋友。但至少……可以暂时同行。"

林惊羽看着他，忽然笑了。

"有意思。"他说道，"云师弟，你比我想象的有趣多了。"

两人相视一笑，默契尽在不言中。

## 二、旧怨添新仇

就在这时，一道阴沉的声音从旁边传来。

"林惊羽，你在这里做什么？"

云泽转头看去，只见赵乾坤正站在不远处，脸色铁青地望着他们二人。

方才他被云泽当众下了面子，正憋着一肚子气无处发泄。现在又看到云泽与林惊羽站在一起，心中更是嫉恨交加。

林惊羽可是内定的核心弟子，地位远在他之上。整个天道宗外门，没有几个人敢招惹他。而云泽这个从穷山沟里出来的土包子，竟然能与林惊羽平起平坐！

"林师兄。"赵乾坤走上前去，脸上挤出一个讨好的笑容，"您怎么和这种人混在一起？"

"这种人？"林惊羽挑了挑眉，"哪种人？"

"就是……"赵乾坤瞥了云泽一眼，压低声音说道，"就是云泽这种人啊。一个从乡下来的穷小子，连灵根都是中等品阶，能有什么出息？您和他站在一起，只会掉身份……"

"够了。"林惊羽打断他，脸色冷了下来，"赵乾坤，我与谁交往，还轮不到你来指手画脚。"

赵乾坤脸色一变："林师兄，我不是这个意思……"

"你是什么意思，我心里清楚。"林惊羽看了他一眼，目光中透着几分不屑，"你与云师弟之间的过节，我略有耳闻。三年前的事，究竟是谁对谁错，在场的人都心知肚明。"

"你……"赵乾坤脸色涨红。

"怎么？不服气？"林惊羽淡淡地看了他一眼，"昆仑秘境中，各凭本事。你若是觉得自己实力足够，大可以在秘境里与云师弟公平较量。但若是想要玩什么阴招损招……"

他的声音忽然冷了下来，透着几分凛冽的寒意："我林惊羽的剑，可不是吃素的。"

赵乾坤的脸色瞬间变得惨白。

林惊羽的实力他很清楚——入门三年便已名震天道宗，一剑斩杀三头筑基境巅峰妖兽的事迹，至今仍是外门弟子口中的传奇。以他的实力，要杀自己不过是易如反掌。

"我……我知道了……"赵乾坤讪讪地说了一句，然后灰溜溜地转身离去。

待赵乾坤走远，林惊羽才收回目光，转头看向云泽。

"方才那番话，是真心话。"他说道，"昆仑秘境中，你若遇到危险，可以来找我。"

"多谢。"云泽拱了拱手。

"不必客气。"林惊羽微微一笑，"咱们现在是合作关系，互相照应是应该的。"

他看了一眼远处天璇峰的方向，眼中闪过一抹复杂的光芒。

"而且……我也想看看，天璇真人到底在打什么主意。"

说完，他便转身离去，白衣飘飘，很快消失在人群中。

云泽望着他的背影，心中思绪万千。

林惊羽……这个人，比他想象的要复杂得多。

他说的那些话，到底有几分真、几分假？他与天璇真人之间的仇怨，是真的存在，还是刻意编造出来博取信任？

云泽不知道。

但他知道一件事——在这场试炼中，每一个人都可能是朋友，也可能是敌人。他不能轻易相信任何人，但也不能把所有人都推到对立面。

在这盘棋局中，他需要更多的棋子。

而林惊羽，或许就是其中之一。

## 三、登舟在即

午时将近，飞舟即将起飞。

参加试炼的三十六名弟子陆续登上飞舟。云泽随着人流走上甲板，找了一个不起眼的角落站定。

他环顾四周，将甲板上的人影一一记在心里。

三十六名弟子中，大致可以分为几类。

第一类是那些出身世家、背景深厚的弟子。他们穿着华贵，腰间的兵器和佩戴的饰品都透着不凡的气息。这类人约占总数的三成，通常实力不俗，且身后都有长辈撑腰。

第二类是那些天赋异禀、被宗门重点培养的天才弟子。他们大多穿着朴素，但周身气息内敛，眼神锐利，一看就不是等闲之辈。林惊羽就是这类人的代表。

第三类是那些实力一般、全凭运气被选中的普通弟子。他们站在人群中，神色紧张，左顾右盼，明显是第一次参加这种级别的试炼。

而云泽属于哪一类？

从表面上看，他属于第三类——一个出身贫寒、资质平平的普通外门弟子。但实际上……

他低下头，看了一眼自己腰间的游子剑。

剑鞘上的花纹已经被他用刀痕刻意划花，看起来就像一柄普通的凡剑。但在剑鞘深处，那柄游子剑正静静地散发着微弱的灵光，与他体内的真气遥相呼应。

"无咎小子。"剑灵的声音在脑海中响起，"小心点。这艘船上，有几道气息不太对劲。"

"几道？"云泽在心中问道。

"至少三道。"剑灵说道，"那两个站在船头的内门弟子，体内的真气运转方式很奇怪，不像是天道宗的正统功法。还有一个站在角落里的小个子，身上的气息时隐时现，显然是用了什么特殊的敛息术。"

"你觉得他们是什么人？"

"不好说。"剑灵沉吟片刻，"但有一点可以肯定——他们不是来参加试炼的。"

云泽心中一凛。

不是来参加试炼的？那他们的目的是……

"无咎小子，你要有心理准备。"剑灵的声音透着几分凝重，"这场试炼，恐怕比预想的还要复杂。"

云泽默默地点了点头。

就在这时，天空忽然暗了下来。

一艘巨大的飞舟从云层之上缓缓降落，船头站着一个身穿紫色长袍的中年男子。那男子面容威严，目光如电，浑身上下散发着一股令人心悸的威压——那是化神境强者独有的气息！

"是天璇真人！"人群中有人惊呼。

云泽的眼神骤然一凝。

天璇真人。

他终于见到了这个人。

三年前，云游子的记忆碎片让他看到了当年那场背叛的真相。天璇真人——当时还只是一个意气风发的内门弟子，为了天道图碎片，不惜勾结外人，设计陷害云游子，最终将其围杀于荒野之中。

而如今，当年的凶手已经成为了天道宗的化神境长老，位高权重，受万人敬仰。

天璇真人的目光从人群中扫过，最终落在了云泽身上。

那一瞬间，云泽感觉到一股无形的压力笼罩了自己。那压力并不强烈，却如同一座大山压在心头，让他几乎喘不过气来。

这是试探。

云泽咬紧牙关，体内的真气疯狂运转，硬生生将那股压力抗了过去。

片刻之后，天璇真人收回目光，淡淡开口：

"诸位弟子，昆仑秘境十年开启一次，是宗门给予你们的机缘，也是考验。望诸位珍重此机会，在秘境中努力修炼，为宗门争光。"

"此次试炼，由本座亲自主持。"天璇真人的声音回荡在广场上空，"试炼时间为期一个月。一个月后，活着出来的弟子，皆可获得内门候选资格。"

"另外，"天璇真人话锋一转，"此次试炼设置了一道额外关卡——星辰阁遗迹。那是昆仑秘境的核心禁地，里面藏有上古年间留下的珍贵传承。敢于挑战者，可自行进入。但本座要提醒诸位——"

他的目光再次扫过人群，最终定格在云泽身上。

"那里面的危险，远超你们的想象。若是没有足够的实力，切勿轻易涉险。"

这话说得意味深长。

云泽与天璇真人的目光在空中交汇，心中警铃大作。

天璇真人知道。

他知道云泽身上有云游子的传承，知道他必然会去星辰阁遗迹。甚至，他说出这番话，就是为了引诱云泽前往。

这是一个阳谋。

但云泽没有退路。

星辰阁遗迹中有云游子留下的更多记忆碎片，有关于天道图碎片的秘密——更重要的是，有他必须知道的真相。

"走吧。"林惊羽的声音在耳边响起，"上飞舟。"

云泽深吸一口气，压下心中翻涌的情绪，跟随人流登上了天道云舟。
