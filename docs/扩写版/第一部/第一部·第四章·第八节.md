# 第一部·第四章·第八节

## 深夜召见·天璇真人

回到天道宗时，已是深夜。

云泽与张小胖分别后，独自回到青云峰住处。

他本想早些休息，却发现自己无论如何都静不下心来。今日的经历，实在太过震撼。那具枯骨传递给他的记忆、林惊羽透露的剑道传承、以及天玄子这个名字……太多太多的疑问，充斥着他的脑海。

他盘膝坐于榻上，闭上眼睛，试图运转《云游心法》来平复心神。

灵气在经脉中缓缓流转，滋养着他疲惫的身躯。约莫过了小半个时辰，他的呼吸渐渐平稳，心神也安定了下来。

"罢了，先睡吧。"

他叹了口气，正要躺下休息，忽然听见门外传来一阵急促的脚步声。

"砰砰砰——！"

敲门声很重，带着一股不容拒绝的急切。

云泽眉头一皱，起身走到门边，打开房门。

门外站着一个身穿青色道袍的外门弟子。那弟子约莫十七八岁，面容普通，但神色间透着一丝紧张。他见云泽开门，连忙躬身行礼。

"云师弟，深夜打扰，还请见谅。"

"师兄客气了。"云泽回了一礼，"不知师兄深夜来访，有何要事？"

那弟子左右看了看，压低声音道："出大事了。天璇峰……天璇真人出关了！他点名要见你！"

云泽心中一凛。

天璇真人？那个云游子前辈口中的仇人？

他为何会在这个时候出关？又为何要点名见自己？

无数念头在脑海中翻涌，但云泽面上却不动声色。

"多谢师兄告知，我这就去。"

他换了一身干净的道袍，又将游子剑负于背后，随那弟子一同出门。

走出住处的那一刻，夜风扑面而来，带着几分寒意。

云泽抬头望向远处的天璇峰。

那座山峰比青云峰更加巍峨险峻，山势如剑，直插云霄。峰顶云雾缭绕，在夜色中若隐若现，隐约可见几点灯火在云雾中闪烁。

那是天璇峰所在的方向。

云泽深吸一口气，跟着那弟子向天璇峰走去。

一路上，他的脑海中不断回想着云游子前辈的话——

"你要小心天璇峰。不只是天璇真人，还有天璇峰上的一切。"

"我的仇人……远比你想象的要可怕得多。"

云游子前辈的仇人，便是天璇真人。而今日那段记忆中，天玄子背叛叶孤城的画面，同样与天璇峰脱不了干系。

这两者之间，是否有什么联系？

天璇真人召见自己，又是为了什么？

无数疑问涌上心头，却没有人能给他答案。

穿过几座山峰，又沿着一条蜿蜒的石阶向上攀登，约莫走了小半个时辰，两人终于来到了天璇峰脚下。

"云师弟，我只能送你到这里了。"那弟子停下脚步，神色恭敬中带着几分畏惧，"再往上，便是天璇真人的道场，宗门弟子不得擅自进入。你自己上去吧。"

云泽点点头，独自沿着石阶向上攀登。

每走一步，他都能感受到一股无形的压力从峰顶倾泻而下。

那是上位者的威压，是修为远超自己之人不经意间散发的气息。那压力沉重而压抑，仿佛一座无形的大山，压在他的肩头，让他每迈出一步都要耗费极大的力气。

他知道，这是天璇真人在试探自己。

但他没有退缩。

他咬紧牙关，一步一步向上攀登。汗水从额头滚落，打湿了他的衣领。双腿像是灌了铅一样沉重，但他依然没有停下。

终于，他来到了峰顶宫殿前。

那是一座古朴而庄严的宫殿，青砖黛瓦，飞檐翘角，在夜色中散发着淡淡的灵光。宫殿大门敞开着，内部灯火通明。

云泽定了定神，迈步走了进去。

殿内空旷而幽深，两侧立着数根雕刻着繁复纹路的石柱，柱顶燃着长明灯，散发着柔和的光芒。空气中弥漫着一股淡淡的檀香气息，闻之让人心神宁静。

在大殿正中央，一个身穿灰色道袍的中年男子负手而立，背对着他。

那男子身形清瘦，气质儒雅，看上去就像一个普通的教书先生。他的背影微微有些佝偻，却透着一股说不出的沧桑，仿佛承载着无数不为人知的往事。

云泽站在殿门口，躬身行礼。

"天璇真人，弟子云泽，求见真人。"

那中年男子没有转身。

他只是静静地看着殿外的夜空，似乎在欣赏着什么。

良久，他才开口。

"云泽……好名字。"

他的声音平淡，听不出喜怒。

"无咎者，不犯错误，不惹祸端也。"

他缓缓转过身来。

云泽终于看清了天璇真人的面容。

那是一张方正的脸庞，眉眼间透着一股说不出的沧桑。五官端正，甚至称得上清俊，但那双眼睛却深邃得像是两口望不见底的古井，其中藏着太多太多云泽看不懂的东西。

"抬起头来。"天璇真人说道。

云泽依言抬头，与天璇真人的目光对视。

刹那间，一股浩瀚如海的神识从天璇真人身上涌出，如同一道无形的巨浪，将云泽整个人笼罩其中！

云泽只觉浑身一僵，仿佛被一股无形的力量禁锢住了。

那神识太强了，强得如同山岳，如同深渊。他感觉自己就像一只蝼蚁，站在巨人面前，连呼吸都变得困难起来。

他知道，这是天璇真人在试探自己。

他咬紧牙关，拼命抵抗着那股威压，额头上青筋暴起。

但他的修为与天璇真人相差太远。那股威压如同实质，压得他浑身骨骼都在作响，仿佛随时都会被碾成碎片。

就在他快要支撑不住时——

体内丹田中忽然涌出一股温热的气流。

那气流来自游子剑！

就在危机时刻，那柄锈迹斑斑的古老长剑中，竟自行涌出一道若有若无的剑意，与云泽体内残余的灵力交融在一起，在他周身形成一道薄薄的防护罩。

与此同时，他眉心处那道若隐若现的金色印记，也微微闪烁了一下。

那是道君转世的印记——他虽不自知，却在关键时刻本能地护住了他。

天璇真人瞳孔微缩。

他猛地收回神识，看着云泽的目光中，多了几分意味深长的审视。

"有趣……"

他喃喃自语，声音低得几乎听不见。

片刻后，他开口问道："你手中的剑，可否借我一观？"

云泽心中一凛。

他知道，自己手中的游子剑，绝不能让天璇真人看到。

那剑中封印着云游子的残念，而云游子正是被眼前此人陷害之人！

若让天璇真人看到游子剑……

云泽不敢想象会发生什么。

"回真人，"他躬身道，"弟子那柄剑只是寻常凡剑，恐怕入不了真人法眼。"

天璇真人眉头微皱，似乎对他的回答并不满意。

"你不必瞒我。"他淡淡说道，"你腰间那柄剑，绝非寻常之物。老夫方才用神识探查过，那剑中蕴含着一股极为精纯的剑意……那是上古剑道的痕迹。"

他向前踏出一步，目光如炬。

"你若肯将剑交给老夫一观，老夫可赐你一枚'凝元丹'，助你突破炼气境四层。丹药虽小，但以你如今的修为，足以让你省去半年苦修之功。"

凝元丹！

云泽心中一震。

凝元丹是二品丹药中极为珍贵的存在，对于炼气境的修士而言，简直是可遇而不可求的宝物。一枚凝元丹，足以让一个炼气境初期的修士在短短数日内突破一个小的境界，而且没有任何副作用。

天璇真人竟然舍得用这种东西来换取游子剑的一观？

此人心思，深不可测！

云泽正不知如何作答，忽然，大殿外传来一阵喧哗。

"天璇师兄！你出关了怎么也不通知一声？我等正要为你接风洗尘呢！"

数道身影从殿外走入，为首的是一个鹤发童颜的老者。那老者精神矍铄，目光炯炯有神，一看便是修为深厚之辈。

正是天道宗另一位内门长老——天机真人。

天璇真人面色不变，向天机真人微微颔首："不过是闭关数年而已，何须劳师动众。"

天机真人笑道："师兄此言差矣！你闭关这几年，宗门事务繁杂，我等可没少操心。来来来，今日既然出关，定要与师兄好好叙叙旧！"

他说着，目光有意无意地扫过云泽，眼中闪过一丝异色。

天璇真人淡淡道："此子是我叫来的，有些事要问他。你先带人去偏殿等候，我稍后就来。"

天机真人眉头微皱，似乎想说什么，但看了看天璇真人的神色，终究没有开口。

"那……好吧。"

他向云泽招招手："小子，跟我来。"

云泽如蒙大赦，连忙躬身行礼，随天机真人退出大殿。

走出大殿的那一刻，他暗暗松了口气，后背已被冷汗浸透。

天璇真人的修为……太可怕了！

方才那道神识威压，简直如山如海，让他连反抗的念头都生不出来。若非游子剑中的剑意及时涌出护住了他，只怕他当场就要出丑。

更重要的是……

天璇真人为何会对他的剑如此感兴趣？

难道……他已经察觉到了什么？

无数疑问涌上心头，让云泽心乱如麻。

他不知道等待自己的，将是怎样的命运。

但他知道，自己的修仙之路，远比他想象的要危险得多。

而他唯一能做的，就是不断地变强，不断地追寻真相。

直到有一天，他能够不再仰人鼻息，不再任人摆布。
