# 第一部·第四章·第六节

## 幽谷深处·古庙残骸

山谷深处，云雾缭绕，幽暗逼仄。

云泽紧跟在林惊羽身后，沿着一条几乎被灌木淹没的小径向谷底深处走去。脚下的落叶厚厚的，踩上去发出沙沙的声响。四周静得出奇，连虫鸣鸟叫都消失了，只有两人的脚步声在寂静的山谷中回响。

越往深处走，空气中的血腥味就越浓，夹杂着一股若有若无的腐臭。那气味并不新鲜，像是已经存在了很久，挥之不去。

地面上的落叶间，散落着不少妖兽的残骸——有体型如牛的野猪，有成群的野鹿，还有几具已经白骨化的巨大骨架。从残骸的分布来看，这些妖兽并非死于一处，而是死在不同的位置，像是被什么东西从各个方向吸引过来，最终死在了这里。

"这些妖兽……都是被那头青鳞狼杀死的？"云泽皱眉道。

"不止。"林惊羽摇头，俯身查看一具野鹿的残骸，"看这伤口，不像是青鳞狼所为。它的爪子虽然锋利，但留下的伤口应该是撕裂状。而这些伤口……"

他用剑尖挑开野鹿颈部的皮毛，露出下面断裂的颈骨。那伤口平滑如镜，像是被什么极为锋利的东西一击切断。

"更像是被某种更强大的东西，一击毙命。"

林惊羽的声音低沉，眸中闪过一丝凝重。

云泽心中一凛。

能让林惊羽都感到凝重的东西，那得是多可怕的存在？

两人又前行了约莫半个时辰。

山路越来越崎岖，周围的树木也越来越高大茂密，将头顶的阳光遮得严严实实。空气中弥漫着一股阴冷潮湿的气息，像是从未有过阳光照进来的幽冥之地。

忽然，眼前豁然开朗。

那是一处隐秘的山谷，三面环山，只有一条狭窄的缝隙通向外界。谷中央有一片不大的湖泊，湖水清澈，却隐隐泛着一丝幽蓝的光芒。

最诡异的是那湖泊上空——明明没有风，湖面却泛着细密的涟漪，像是有什么东西在水底缓缓游动。

但真正让两人心惊的，是湖边的一处石台。

那石台由青灰色的岩石垒成，看上去年代久远，表面的石纹已经被岁月侵蚀得模糊不清。石台之上，赫然盘坐着一具枯骨。

那枯骨保持着打坐的姿势，周身萦绕着一层淡淡的光芒，像是一层看不见的屏障，将它与外界隔绝开来。它的骨架宽大，即便已经化为枯骨，依然能看出当年的伟岸身姿。

枯骨面前，摆着一柄断成两截的长剑，以及一枚散发着微弱灵光的玉简。

更诡异的是——

那枯骨的眼眶中，竟有两团幽幽的蓝色火焰在跳动！

那火焰没有温度，没有热度，却散发着一股说不出的阴森与诡异。它像是有生命一般，缓缓转动着，似乎在注视着眼前的两个不速之客。

"骨……骨煞？！"

云泽浑身一震，下意识握紧了手中的游子剑。

骨煞是修仙界的一种邪物，通常是修为高深的修士在陨落后，怨念不散，附着在尸骨之上形成的怪物。它们嗜血成性，会主动攻击活物，极难对付。

然而，云泽很快发现了不对。

"不对……这气息……不是骨煞！"

骨煞的气息是阴冷、邪恶、嗜杀的。但眼前这具枯骨，虽然眼眶中燃烧着鬼火，却并没有散发出那种令人作呕的邪气。反而有一股若有若无的……悲凉？

林惊羽的脸色也变了。

他的身体骤然绷紧，周身的剑意不自觉地涌出，在身前形成一道半透明的剑气屏障。他死死盯着那具枯骨，眸中闪过一丝难以置信的震惊。

"这气息……这股剑意……"

他的声音忽然变得有些颤抖。

"怎么可能！"

就在此时，那枯骨眼眶中的蓝色火焰跳动得更加剧烈了。

片刻后，一道苍老而嘶哑的声音，在这方天地间响起：

"叶……孤城……的后人……？"

那声音断断续续，像是从很远很远的地方传来，又像是从无尽的岁月中穿越而来。每一个字都带着沉重的沧桑，仿佛承载着千万年的悲欢离合。

云泽浑身一震。

叶孤城的后人？什么意思？

他下意识转头看向林惊羽。

林惊羽面色凝重，沉声道："你是何人？为何知道我的来历？"

那枯骨的眼眶中，蓝焰跳动得更加剧烈了，仿佛在回忆着什么遥远的往事。

"叶孤城……万古剑道第一人……太古三剑之师……"

它的声音断断续续，却带着一股难以言喻的悲凉与沧桑。

"我……等这一天……等了太久太久了……"

忽然，那两团蓝焰猛地一涨，枯骨周身的光芒骤然璀璨起来！

与此同时，一股无形的力量从枯骨身上涌出，化作一道若有若无的光丝，穿过林惊羽，直直没入云泽的眉心！

"无咎——小心！"

林惊羽大喊一声，连忙伸手想要阻止。然而那光丝的速度快得惊人，他根本来不及反应。

云泽只觉眉心一凉，一股陌生的力量便已涌入他的识海。

下一刻——

他的意识被拉入了一片混沌之中。

眼前是一片模糊的光影，像是被水波扭曲的倒影。那光影不断变幻，渐渐勾勒出一幅幅画面——

画面中，一个意气风发的青年，手持长剑，立于九天之上。

他身穿白衣，长发如瀑，面容俊朗，眸中却有一股睥睨天下的气势。他的身形不算高大，却给人一种顶天立地的感觉，仿佛天地之间，唯他一人。

他的对面，是一头庞大无比的混沌巨兽。

那巨兽遮天蔽日，身形如山，獠牙如戟，一双血红的眼眸中燃烧着无尽的贪婪与嗜血。它仅仅是站在那里，便让整片天地都在颤抖。

那是——饕餮！

混沌七魔兽之一，传说中能够吞噬一切的恐怖存在！

"饕餮！"

画面中，那青年开口，声音清朗，却带着一股不容置疑的威严。

"你作恶多端，涂炭生灵。今日我叶孤城便代天行道，将你封印！"

他挥剑而出。

那一剑，仿佛凝聚了天地之力。日月为之失色，星辰为之黯淡。剑光如银河倒悬，璀璨夺目，照亮了整片天际。

饕餮咆哮连连，巨大的爪子挥舞，撕裂虚空。然而在那剑光面前，它的抵抗显得如此苍白无力。

"吼——！！！"

饕餮发出凄厉的惨叫，庞大的身躯被剑光笼罩，缓缓坠入虚空之中。

封印将成！

然而就在这一刻——

一道阴森的黑影从暗处闪出！

那黑影速度极快，化作一道黑色的流光，狠狠刺入了青年的后背！

"你——！"

青年回头，难以置信地看着那个偷袭之人。

那人身穿一袭儒雅的道袍，面容俊朗，嘴角挂着一丝阴冷的笑意。他的气质出尘，看上去就像一个得道高人，谁也无法将他与偷袭者联系在一起。

然而此刻，他的脸上却挂着一抹得意的冷笑。

"叶孤城，你太天真了。"

他低声说道，声音中带着一丝嘲讽。

"你以为，凭你一己之力，就能真正封印饕餮？你以为，你是救世主，是英雄？"

他俯身，在青年耳边低语。

"你只是一颗棋子罢了。"

"天玄子……是你……！！！"

青年怒吼一声，眼中满是愤怒与不甘。

然而他的身躯已经开始坠落，意识在最后一刻化为千丝万缕，融入了那柄陪伴他征战一生的长剑之中。

"天玄子……我……记住了……"

画面消散。

云泽猛地睁开双眼，浑身冷汗淋漓。

他的胸口剧烈起伏，心脏像是要跳出胸腔一般。那段残破的影像虽然只是短短一瞬，却给他带来了极大的冲击。

那是他第一次如此清晰地"看到"叶孤城。

那个万古剑道第一人，那个以一己之力封印混沌魔兽的绝世强者，竟然被人偷袭陨落！而且，偷袭他的人，竟然是一个表面儒雅、内心阴险的伪君子！

天玄子……

这个名字，像是一根刺，深深扎入了云泽的心中。

而更让他震惊的是，枯骨传递给他的信息中，还有一段关于游子剑来历的记载——

原来，当年叶孤城封印饕餮时，共铸三柄神剑：天道剑、游子剑、飞仙剑。游子剑中封印的，正是叶孤城的一缕剑魂残识。

而天道剑，则被叶孤城留给了自己的大弟子，也就是后来天道宗的创派祖师。

然而，天玄子背叛后，窃取了天道剑，以天道剑之力横扫八荒，建立了自己的势力，成为天道宗的幕后黑手。

也就是说——

天道宗的根基，从一开始，就是建立在阴谋与背叛之上的！

"原来如此……原来如此……"

云泽喃喃自语，心中掀起惊涛骇浪。

他终于明白了云游子为何会死在青云镇后山，明白了游子剑为何会流落到那偏僻之地。

云游子发现了天玄子的秘密，他是被灭口的！

而自己手中的游子剑，承载着叶孤城的剑魂残识，也是天玄子必欲得之而后快的东西！

"无咎！"

林惊羽的声音打断了他的思绪。

云泽抬头，只见林惊羽面色凝重地看着他，眼中满是担忧。

"你没事吧？刚才那是怎么回事？"

云泽犹豫了一下。

林惊羽是叶孤城的后人——这一点，从枯骨的话语中已经可以确定。而方才那段残破的记忆，似乎也与林惊羽有关。

他是否应该将这一切告诉林惊羽？

沉默片刻后，云泽开口了。

"林师兄，我方才……看到了一段残破的记忆。"

他将那段记忆简要描述了一遍。

林惊羽听完，脸色变得极为难看。

"天玄子……原来竟有此人……"

他忽然闭上双眼，似乎在消化这个惊天的真相。

良久，他才睁开眼，目光如剑。

"无咎师弟，你今日所见所闻，绝不可对任何人提起。天玄子此名……恐怕牵涉极大，你我如今的实力，还远远不足以与其对抗。"

云泽重重点头。

他知道，从这一刻起，自己已经被卷入了一场远超想象的风暴之中。

天璇峰、饕餮、游子剑、自己的身世之谜……所有的线索，都在指向一个隐藏在历史深处的巨大阴谋。

而他自己，作为道君转世，作为"遁去的一"，恐怕正是这场阴谋中，最关键的一枚棋子。

但棋子……也可以成为执棋人。

云泽握紧游子剑，眼中闪过一丝坚定。

"天玄子……不管你是谁，我都会找到你，揭开这一切的真相！"

这一刻，少年的心中，一颗追求大道的种子，悄然生根发芽。
