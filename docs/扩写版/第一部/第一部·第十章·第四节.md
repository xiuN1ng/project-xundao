# 第一部·第十章·第四节：初入灵界

## 一、通道彼岸

穿越灵界通道的感觉，很奇妙。

那不是普通的移动，而是一种近乎灵魂层面的剥离与重组。起初，云泽感到自己的身体变得无比轻盈，仿佛被什么东西托着，缓缓向上飘去。那感觉就像是在梦中飞翔，周围是一片流动的紫色光海，无数细小的光点在那光海中浮沉，如同漫天飞舞的萤火虫。

然后，那光海开始收缩。

无数光点向他涌来，穿透他的皮肤、血肉、骨骼，最终在他的丹田中汇聚成一团灼热的光球。那光球在他体内旋转、压缩、升华，带给他一种前所未有的充实感。

他感到自己的经脉在被重塑。

那些原本淤塞的穴位被逐一打通，那些原本狭窄的经脉被拓宽数倍。他能清晰地感受到，天地灵气正以前所未有的速度涌入他的身体，比凡界浓郁了何止十倍。

这种提升是惊人的。

但也伴随着巨大的痛苦。

经脉扩张的过程就如同被滚烫的烙铁反复灼烧，每一次扩张都会引发一阵剧痛。那疼痛从丹田蔓延至四肢百骸，仿佛有千万只蚂蚁在啃噬他的血肉。

云泽咬紧牙关，一声不吭地承受着。

他不知道自己在这光海中飘荡了多久。时间在这里似乎失去了意义，短则一瞬，长则百年，他无法分辨。

终于，那紫色的光海开始消散。

周围的景象逐渐变得清晰起来——首先映入眼帘的是一片澄澈的蓝天，然后是远处连绵起伏的山脉，接着是脚下坚实的土地。

云泽的双脚落地的那一刻，他感到一阵眩晕袭来。

他单膝跪地，大口大口地喘着粗气。冷汗从他的额头滑落，打湿了他的衣襟。那穿越通道的剧痛虽然已经消退，但余韵犹在，他的身体还在微微颤抖。

过了好一会儿，他才缓过劲来。

他站起身，打量着四周。

这是一片荒凉的山谷。

山谷不大，方圆不过百丈，四周被陡峭的岩壁环绕。岩壁上光秃秃的，几乎没有植被，只有一些耐寒的苔藓贴在岩石的背阴处，泛着暗绿色的微光。

天色已经微微发暗，显然他穿越通道用了不少时间。

山谷中弥漫着一股淡淡的灵气——不，不应该叫"淡淡的"，应该说是一股"浓郁的"灵气。与凡界相比，这里的灵气浓度至少高出了二十倍。如果凡界的灵气是一杯水，那么灵界的灵气就是一桶水，两者根本不在同一个量级。

而更让他惊讶的是，这里的灵气中隐隐带着一丝金色的光芒。

那是道韵的气息。

道韵，是比灵气更高层次的存在。它无形无质，却能被修士感知和吸收。拥有道韵的灵气，能够加速修士的修炼，提升修士的悟性，甚至能够帮助修士领悟天地法则。

在凡间，道韵几乎是传说。

而在灵界，道韵却随处可见。

"这就是灵界……"

云泽喃喃自语，心中涌起一股难以言喻的震撼。

他闭上眼睛，深深吸了一口气。

那灵气顺着他的口鼻流入，在经脉中运转了一个大周天，然后缓缓沉入丹田。

只是一口，便抵得上他在凡间打坐一个月。

如果能在这里修炼……

他的修为将会以难以想象的速度提升。

但他知道，现在不是修炼的时候。

他初来乍到，对这个世界一无所知。这里是灵界的什么地方？附近有没有修士聚居的城池？太虚宫又在哪个方向？这些问题的答案，他一个都不知道。

而且，他身上的气息与灵界的"原住民"不同。

他是凡界飞升上来的，身上带着凡界特有的气息。对于灵界的修士来说，他就像是一个来自异乡的陌生人，很容易被辨认出来。

如果不尽快融入这个世界，他很可能会成为众矢之的。

一个从凡界飞升上来的"土包子"，身上却带着如此浓郁的灵气波动——这种反差，足以引起任何人的觊觎。

云泽睁开眼睛，目光扫视着四周。

山谷的出口在北面，那里有一条蜿蜒的小路，沿着岩壁向上延伸，消失在远处的山脊之后。

他决定沿着那条路走。

希望能找到有人烟的地方。

## 二、太虚宫的方向

沿着山路走了大约一个时辰，天色已经完全暗了下来。

灵界的夜空与凡间不同。

凡间的夜空是漆黑的，繁星点点，月光如水。而灵界的夜空却是深紫色的，那紫色与灵气潮汐时的天空一模一样，但更加深邃、更加纯粹。在那紫色的天幕上，悬挂着三轮月亮——一轮金色、一轮银色、一轮透明。三轮月亮呈三角排列，共同照亮着大地，使得灵界的夜晚并不像凡间那样黑暗。

借着月光和星光，云泽继续前行。

他的脚步很轻，几乎没有发出任何声响。那是他多年来在山林间行走养成的习惯——走路不发出声响，可以在遇到危险时争取到宝贵的反应时间。

山路越来越陡峭，两旁的景致也在不断变化。

从最初的荒凉山谷，到后来的稀疏树林，再到现在的密集竹林。那竹林长得极为茂盛，竹竿通体碧绿，竹叶在夜风中沙沙作响，仿佛有什么东西在林间窃窃私语。

云泽放慢脚步，将神识向前探去。

他的神识在这灵界中似乎变得更加敏锐了——或许是灵气浓度高的缘故，他能感知的范围比在凡间扩大了数倍。短短片刻，他便将前方数里之内的情况尽收眼底。

在竹林深处，有一座破旧的山神庙。

庙很小，只有一间正殿，供奉着一尊不知名的神像。庙门半掩，里面透出微弱的火光——那是有人在里面生火取暖。

云泽犹豫了一下，最终还是向那庙宇走去。

他需要找人问路。

而且，他也有一个问题需要解答：这里是灵界的什么地方？距离太虚宫有多远？

推门而入，一股温暖的气息扑面而来。

庙内生着一堆篝火，火焰在夜风中跳跃，将四周的墙壁映照得忽明忽暗。篝火旁坐着三个人——两个老者，一个少年。

那两个老者看起来都有七八十岁的模样，须发皆白，脸上布满皱纹。他们穿着粗布麻衣，乍一看像是普通的凡人，但云泽从他们身上感知到了微弱的灵气波动——那是修士特有的气息。

那两个老者的修为不高，大约只有炼气境初期的水平。

而那个少年则不同。

那少年看起来只有十五六岁的模样，面容清秀，眉宇间透着一股灵气。他穿着一身灰白色的道袍，道袍上绣着一个模糊的图案——那图案像是一座山峰，又像是一朵云彩，云泽从未见过。

少年的修为比那两个老者高得多，大约在筑基境巅峰左右。

三人显然没想到会有人在这深夜来访。

当云泽推门而入的那一刻，他们齐齐转过头，目光警惕地看向门口。

"你是谁？"其中一个老者开口问道，声音沙哑。

"过路的。"云泽拱了拱手，"打扰诸位了。"

他环顾四周，找了一个角落坐下，与那三人保持着一定的距离。

那两个老者对视一眼，见云泽没有表现出明显的敌意，便稍稍放松了些许。但他们的目光依然时不时地飘向云泽，带着几分戒备。

唯有那个少年，一直盯着云泽看。

"你……是从凡界来的？"少年忽然开口。

云泽心中一凛，面上却不动声色。

"你怎么知道？"

"你的气息不对。"少年说，"灵界的修士，身上都有一种特殊的'灵韵'。那种灵韵是长期吸收灵界灵气形成的，已经渗透进了骨子里。而你身上没有那种灵韵，反而有一种……"

他顿了顿，似乎在斟酌措辞。

"有一种……从下界带上来的'土气'。"

云泽无言以对。

这少年的观察力倒是敏锐得很。

"是，我是刚从凡界飞升上来的。"他索性大方承认，"初来乍到，对灵界一无所知。想向几位打听一些事情，不知道方便不方便？"

三人再次对视一眼。

"你想打听什么？"另一个老者问道。

"这里是什么地方？"云泽说，"距离太虚宫有多远？"

话音落下，庙内的气氛骤然一变。

那两个老者的脸色刷地变白，像是听到了什么可怕的消息。而那个少年的眼神也变得微妙起来，他上下打量着云泽，眼中闪过一丝复杂的光芒。

"太虚宫？"那少年重复了一遍，声音里多了几分警惕，"你问太虚宫做什么？"

"我要去那里。"云泽说。

"去太虚宫？"少年的眉头皱了起来，"你是想拜入太虚宫门下？"

"算是吧。"

少年沉默了。

他低下头，似乎在思考着什么。

过了好一会儿，他才重新抬起头。

"这里是灵界的边陲之地，名为'落星谷'。"他说，"从这里到太虚宫，大约有三万里的路程。以你的脚程，不眠不休也要走上三个月。"

三万里。

三个月。

云泽在心中默默计算着这个距离。

"不过……"少年话锋一转，"如果你只是想拜入太虚宫，我劝你还是趁早打消这个念头。"

"为什么？"

"因为太虚宫不收外人。"少年说，"他们是灵界最顶尖的大势力之一，宗门内的弟子要么是灵界本土的天才，要么是各方势力的精英子弟。像你这种从凡界飞升上来的……"

他摇了摇头，没有说下去。

但那意思已经很明显了——像他这种没有背景、没有靠山的凡界飞升者，在灵界就是最底层的存在。太虚宫那样的地方，根本不会正眼瞧他。

"那依你之见，我应该怎么做？"云泽问。

少年沉默了一会儿。

"太虚宫每年都会在各地开设'招募大会'，招收杂役弟子。"他说，"所谓杂役弟子，就是做一些打杂跑腿的活计，身份低微，地位底下。但至少能留在太虚宫的势力范围内，接触到更高深的功法。"

"如果你真的想进太虚宫，这或许是一条路。"

云泽点点头。

"多谢。"

他站起身，向那三人拱了拱手。

"今夜叨扰了，告辞。"

说完，他便向门外走去。

"等等。"

少年忽然出声叫住了他。

云泽停下脚步，转过身。

少年从怀里掏出一枚玉牌，向他抛来。

云泽伸手接住，低头一看。

那是一枚普通的通行令牌，令牌正面刻着一个模糊的图案——与他道袍上的图案一样，像是一座山峰，又像是一朵云彩。

"这是什么？"

"我的通行令牌。"少年说，"凭这枚令牌，你可以参加三个月后的招募大会。届时你拿着它去太虚宫山门外的报名处，会有人接待你。"

云泽怔住了。

他与这少年素不相识，对方为何要平白无故地帮他？

仿佛是看出了他的疑惑，少年咧嘴一笑。

"我叫叶轻尘。"他说，"你是这几年来，第一个敢在灵界问路问到我头上的凡界飞升者。光是这份胆量，就值得我帮你一把。"

他的笑容里带着几分玩味。

"而且……你的眼神让我想起了一个人。"

"谁？"

叶轻尘没有回答。

他站起身，拍了拍袍角的灰尘，向门外走去。

"后会有期。"

他的身影消失在夜色之中，留下一串轻快的脚步声。

那两个老者对视一眼，也纷纷起身，告辞离去。

庙内只剩下云泽一人。

他低头看着手中的那枚玉牌，陷入了沉思。

叶轻尘。

太虚宫。

招募大会。

这灵界的水，比他想象的还要深得多。

但无论如何，他已经迈出了第一步。

接下来，他要做的，就是在这灵界中生存下去，然后一步步接近他的目标——天道图的碎片。

而三个月后的太虚宫招募大会，将是他的第一个跳板。

云泽将玉牌收入怀中，走出了山神庙。

外面的夜空依然深邃，三轮月亮高高悬挂，将大地照得一片银白。

他抬起头，深深地吸了一口灵界清新的空气。

"灵界……"

"我来了。"
