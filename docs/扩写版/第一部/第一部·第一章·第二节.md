# 第一部·第一章·第二节：山中之变

## 一、死地求生

那头野猪妖如同一辆失控的战车，带着滚滚烟尘，朝云无咎狂奔而来！

它的四蹄踏在落叶上，发出潮湿而黏腻的声响，每一步都像是在地面上敲响沉闷的战鼓。那庞大的身躯足有半人高，冲击力可想而知——若是撞上了，只怕连人带树都要被撞飞出去。

云无咎没有时间多想。

在那电光石火的一瞬，他的身体比脑子更快做出了反应——他猛然朝左侧一滚，整个人几乎是贴地飞扑出去的。野猪妖从他原本站立的位置呼啸而过，带起的劲风刮得他面颊生疼，像是被人狠狠扇了一巴掌。

他的后背撞在一棵老树上，树干被他撞得剧烈晃动，枯叶簌簌落下。他顾不得疼痛，连滚带爬地站起身来，握紧柴刀，死死盯着那头已经调转身形的妖兽。

野猪妖似乎没料到他能躲开这一冲，四蹄在地面上刨出几道深深的沟壑，扬起一片尘土。它的赤红双眼死死盯着他，像是两团燃烧的火焰，透着刻骨的仇恨与贪婪。

它再次冲了过来。

这一次，它的速度更快，攻势更猛。

云无咎咬紧牙关，身形急退。他知道自己绝不能被这东西正面撞上——那獠牙、那冲力，若是撞实了，他这把骨头只怕要碎成渣。可他也不能一直退——背后便是悬崖绝壁，退无可退。

他的目光在四周快速扫了一圈。

身后三丈外有一块大青石，青石旁边是一道陡坡，坡下是密密麻麻的灌木丛。再往左几步远，有一棵被雷电劈断的老松，树干粗壮，足够挡住野猪的正面冲击。

那就是他的机会。

他不再犹豫，身形一闪，朝那棵老松的方向冲去。

野猪妖显然没料到他不跑反进，赤红的眼睛闪过一丝茫然。但它很快便反应过来，四蹄猛然蹬地，再次朝他发起冲锋——

云无咎侧身一闪，堪堪从野猪的身侧掠过。他的动作快得像是一阵风，柴刀在手中划出一道弧线，狠狠砍在了野猪的后腿上！

"铛——！"

金铁交鸣之声震耳欲聋！

云无咎只觉虎口一震，整条手臂都被那股反震力震得发麻，柴刀险些脱手飞出。他低头一看，柴刀的刀刃上崩出了一道米粒大的缺口，而那野猪的后腿上——仅仅留下了一道浅浅的白痕，连皮都没破！

好硬的皮！

他的心一下子沉到了谷底。

这头畜生的皮，比他记忆中还要厚得多。上一次他能砍伤它，是因为那畜生当时正背对着他，他偷袭得手；可这一次，它有了防备，他的柴刀根本伤不了它分毫。

野猪妖吃痛（虽然没破皮，但那一刀砍得它骨头都在打颤），发出一声震耳的嘶吼，赤红的双眼几乎要喷出火来。它猛然转身，四蹄在地上刨出深深的沟壑，再次朝云无咎冲来。

这一次，它的速度更快了。

云无咎来不及躲闪，只能本能地举起柴刀格挡——

"砰——！"

一声闷响，他的身体像是一只断线的风筝，被那股巨力撞飞了出去。他的后背重重撞在那块大青石上，石头都被撞得晃了几晃。一股腥甜涌上喉头，他一张嘴，吐出一口血沫。

剧痛从后背蔓延到全身，像是有无数根针在扎他的五脏六腑。他挣扎着想要爬起来，却发现自己的肋骨断了两根，每一次呼吸都像是在用刀子割他的肺。

"嗷——！"

野猪妖仰天长嚎，像是在宣告胜利。它的赤红双眼居高临下地俯视着云无咎，口中滴下腥臭的涎水，一滴一滴地落在云无咎的脸上，恶心而灼热。

它不再急着冲上来，而是一步一步地朝他逼近，像是一只玩弄猎物的猫，要将恐惧一点一点地注入猎物的血液之中。

云无咎躺在地上，胸膛剧烈起伏，每一次呼吸都伴随着撕裂般的疼痛。他握紧柴刀的手在发抖——不是因为恐惧，而是因为绝望。

打不过。

跑不了。

他真的要死在这里了吗？

就在他几乎要放弃的那一刻，他的脚踝忽然感觉到了一丝异样。

那股温热的气息又出现了。

它从脚下的泥土深处渗出，若有若无地拂过他的脚踝，像是某种召唤，又像是某种牵引。那感觉说不清道不明，却令他的心跳不由自主地加快了几分。仿佛是有什么东西在地底深处苏醒，正在用一种极为隐秘的方式呼唤着他。

那感觉他在一个月前便曾感受过。

那一次，他在这座山上砍柴时，无意间触碰到了那股温热的气息。它从脚下的泥土中涌出，驱策着他朝北面的那道断崖走去。若非那一次的经历，他也不会发现那座隐藏在崖壁之中的石台，更不会遇到那位留下传承的前辈。

可是那一次，他并没有来得及探索那道断崖的秘密。

他只是顺着那股气息的牵引，找到了那条隐蔽的小径，却因为心中挂念着家中等待的云爷爷，没有贸然深入，便匆匆下了山。

而今日，在生死存亡的关头，那股气息再次出现了。

它比上一次更加浓烈，更加急切，像是一双无形的手，正在拼命地朝他挥舞着，呼唤着他快跑、快逃、快去那里——

那里有救他命的东西！

云无咎的心猛然一震。

他没有时间去深想这异象的来由。他只知道，这是他唯一的机会。

他用尽全身力气，猛地朝侧面滚去——

"嗷——！"

野猪妖扑了个空，愤怒地嘶吼起来。它显然没料到这已经半死不活的猎物还能动弹，四蹄刨地，再次朝他冲来。

云无咎咬紧牙关，撑着柴刀，拼命朝那道断崖的方向爬去。

一下、两下、三下……

他的手掌被尖锐的石头划破，鲜血染红了泥土；他的肋骨刺入肺叶，每一次移动都像是在用烙铁烫他的五脏六腑。可他不敢停下，不能停下——那畜生的蹄声就在身后，一下一下，像是催命的鼓点，越来越近。

三丈。

两丈。

一丈——

他终于爬到了那道断崖的边缘。

那断崖不算极高，约莫二三十丈，然而崖壁陡峭如削，下临深涧，寻常人根本上不去。平日里村民砍柴只敢在崖下活动，从不敢攀援而上。崖壁上爬满了青藤和苔藓，湿滑无比，若是失足跌落，必死无疑。

然而此刻，那密密麻麻的青藤，却成了云无咎唯一的救命稻草。

野猪妖冲到崖边，猛然刹住了脚步。

它虽然凶悍，却也知道悬崖的厉害。那双赤红的眼睛死死盯着云无咎，口中发出愤怒的喘息，却不敢再往前一步。

云无咎靠在崖壁上，大口大口地喘息着。鲜血从他的嘴角溢出，顺着下巴滴落在地上，染红了一小片枯叶。他望着那头不敢上前的妖兽，嘴角忽然扯出一丝苦笑。

这畜生，倒也怕死。

可他也没有高兴得太早——这断崖虽挡住了野猪，却也将他的退路彻底封死了。他爬不上崖壁，又不敢跳下去，如今已是进退两难的绝境。

那畜生不愿离去，只是蹲在崖边，赤红的眼睛死死盯着他，像是在等待什么。

等他耗尽力气，等他失足坠落。

他深吸一口气，强撑着站起身来。

然后，他做了一件让那野猪妖都始料未及的事——

他一把抓住崖壁上的青藤，纵身跃下了悬崖！

## 二、崖壁秘径

风声呼啸，灌入耳中。

云无咎只觉得自己像是被一只巨手狠狠抛下了深渊，耳畔是呼啸的风声和剧烈的心跳。崖壁上那些青藤缠绕在他手臂上，死死勒住他的皮肉，痛得他几乎要叫出声来——可他没有松手。他不敢松手。

一下、两下、三下……

他像是一只被风刮落的枯叶，在崖壁上剧烈地摆荡。好几次他的脚都踩空了，踩在了虚空之中，全靠那根青藤才没有坠落下去。他的手掌被藤条勒得血肉模糊，鲜血顺着胳膊流下来，染红了半边袖子。

终于，在第五次剧烈摆荡之后，他的脚终于踩到了实地。

那是一块狭窄的岩台，勉强能容两人并排站立。脚下是万丈深渊，抬头是直插云霄的崖壁，身侧是密密麻麻的青藤和苔藓。云无咎靠在崖壁上，大口大口地喘息着，每一次呼吸都像是在用刀子割他的肺。

他不知道自己掉下去了多深，也不知道这里是崖壁的什么位置。他只知道，那个方向——

那股温热的气息，正是从头顶上方某处渗透下来的。

他抬起头，眯着眼睛朝上方望去。

晨光从崖壁的缝隙中倾泻而下，将一切都染上了一层朦胧的金色。在那光芒的尽头，他隐约看见了一道狭窄的岩缝，岩缝之后似乎藏着什么东西。

那气息，便是从那里传来的。

他顾不上身上的伤势，咬着牙朝那道岩缝攀爬过去。

岩壁湿滑，脚踩不稳，他便用手攀；手也被青藤勒得皮开肉绽，他便用牙齿咬住藤条，硬生生地将自己拽了上去。好几次他险些失足坠落，好几次他觉得自己再也爬不动了——可那股气息就像是一盏明灯，在黑暗中指引着他，让他一次次地从绝望中燃起希望。

终于，他的手指触碰到了那道岩缝的边缘。

那岩缝极窄，仅容一人侧身通过。他的身体贴着冰冷而粗粝的岩壁，一点一点地挤了进去。岩壁上的碎石刮过他的脸颊和手臂，留下一道道血痕，他却浑然不觉。

挤了约莫十几步，前方的光线忽然一亮。

他钻出岩缝，眼前豁然开朗——

他来到了一处极为隐蔽的石台之上。

那石台三丈见方，平整如削，像是被人以大法力一剑削平。四周云雾缭绕，将这方寸之地与外界隔绝开来，恍若遗世独立的洞天福地。头顶的阳光从云缝中倾泻而下，在石台上投下斑驳的光影，氤氲着朦胧的仙气。

而石台的中央，赫然盘坐着一个人影。

云无咎的脚步一下子僵住了。

那是一个人。

不，那曾经是一个人。

如今只剩下一具枯骨，裹在一件早已腐烂成絮的青袍之中，盘膝而坐，双手交叠置于膝上，姿态安详如入定老僧。枯骨的眉骨依然高耸，眼窝深陷，似乎还在凝视着这片他守护了三百年的天地。那腐朽的衣袍上，依稀可见暗金色的纹路，那是某种极为高深的法袍，曾伴主人走过无数风雨，如今却只化作一缕尘埃。

最令人惊异的是，这具枯骨周身萦绕着淡淡的白雾，那股温热的气息，正是从这枯骨之中散发而出。

而在那枯骨的膝前，整整齐齐地摆放着两样东西。

一柄铁剑。

一枚玉简。

云无咎呆呆地站在原地，大口大口地喘息着。鲜血从他的嘴角和鼻孔中汩汩流出，在石台上留下一串殷红的印子。他的肋骨断了两根，每一次呼吸都像是在用烙铁烫他的五脏六腑，疼得他眼前一阵阵发黑。

可他还是强撑着没有倒下。

他望着那具枯骨，一时竟不知该作何反应。

这是谁？

他为何会死在这里？

他在这里坐了多少年？

为何他的尸骨之上，没有半点腐朽的痕迹？

为何他的周身，会有这般温热的气息？

无数疑问涌上心头，然而没有人能给他答案。

石台寂静，唯有山风呜咽，吹过枯骨，吹过锈剑，吹过那枚泛着微光的玉简。仿佛还能听见三百年前的那个夜晚，狂风骤雨，一名重伤的青年踉踉跄跄地攀上这座断崖，在这方寸之地坐化而死。临死前，他将毕生所学封入玉简，将平生佩剑留待有缘人，然后，便再也没有然后了。

三百年的风霜，三百年的寂寞，三百年的等待……

都化为眼前这具枯骨，这柄锈剑，这枚玉简。

云无咎的心中忽然涌起一股难以言喻的悲凉。

他缓步上前，在那枯骨前三尺处站定，恭恭敬敬地躬身一礼。

"前辈……"

他的声音沙哑而虚弱，在寂静的石台上回荡。

"晚辈云无咎，误入宝地，惊扰了前辈安眠，还望前辈恕罪。"

他轻声说道，声音在寂静的石台上回荡。

枯骨无言，唯有那淡淡的白雾在周身缓缓流转。

云无咎等了片刻，见并无异状，这才将目光落在了那枚玉简之上。

那玉简古朴温润，虽历经沧桑，却仍泛着淡淡的莹光。简身刻着几道模糊的纹路，像是某种上古符文，隐隐透着一股令人心悸的气息。

他伸出手，犹豫了一下，还是轻轻触碰了一下那玉简的边缘——

刹那间。

一道璀璨的青芒从玉简之中骤然绽放！

那光芒刺目至极，刺得云无咎不由自主地闭上了双眼。紧接着，一股滚烫的热流从指尖涌入，如同决堤的洪水，沿着他的经脉疯狂奔涌！

那热流所过之处，云无咎只觉浑身的血肉都在颤栗。每一寸肌肤、每一根骨骼、每一条经脉，都被那股热流冲刷、淬炼、蜕变。他从未经历过这种感受，疼痛中夹杂着酥麻，灼热中蕴含着清凉，像是有什么东西正在他体内脱胎换骨，将他从一个凡人改写为一个……可以修炼的存在。

他张开嘴想要呼喊，却发不出任何声音。

他想要挣脱，想要逃跑，却发现自己的身体早已失去了控制。那股力量太过强大，太过温柔，却又无法抗拒，如同命运的洪流，将他裹挟其中。

他的身体剧烈地颤抖起来，像是一叶扁舟被卷入惊涛骇浪之中。

那股热流在他体内奔涌了不知多久——或许是片刻，又或许是千百年——终于缓缓平息下来。

热流散去的那一刻，云无咎猛地睁开了双眼。

他的眼睛亮得惊人，像是有两颗星星藏在了瞳孔深处。

他低头看了看自己的双手——皮肤比从前白皙了几分，隐隐透着光泽。他能感觉到血液在血管中奔涌，心跳比从前有力了许多，浑身上下似有使不完的力气。

最奇异的是，他的肋骨处传来一阵阵麻痒——那两根断骨，竟在以肉眼可见的速度愈合着！

而他的丹田中，不知何时多了一丝极细极弱的气流。

那气流如同春蚕吐丝，纤细得几乎无法察觉，却实实在在地存在着。它在他丹田处缓缓凝聚，旋转，壮大，如同混沌初开时的那一缕元气。

气感。

他有了气感！

云无咎的呼吸骤然急促起来。

云爷爷曾对他说过：*"这世间，有一种人，生来便能感应天地灵气的存在。这种人，便是所谓的有灵根者。而那些没有灵根的人穷其一生，也无法感知到灵气的存在。气感，是踏入修仙的第一步。有了气感，方能吸纳灵气、淬炼肉身、筑就道基。没有气感，一切皆是妄谈。"*

而如今，他有了气感。

他，云无咎，青云镇砍柴少年，竟是那万中无一的有灵根者！

在这青岚山的断崖之上，在这具枯骨之前，在这柄锈剑与这枚玉简的见证下，他终于迈出了这改变命运的第一步！

他深吸一口气，将喜悦压下，抬头望向天空。

云雾已散，金色的阳光从云缝中倾泻而下，将整座石台笼罩在一片璀璨的光芒之中。他站在那光芒里，身形单薄却挺拔如松，腰间别着那柄锈剑，衣袂飘飘，恍惚间竟有几分仙家弟子的气度。

可就在此时，他忽然察觉到一丝异样。

他抬起头，瞳孔骤然紧缩——

枯骨周围的白雾正在缓缓旋转，勾勒出一个虚幻的人形轮廓。

那轮廓渐渐清晰。

是一名青年男子的模样。

他身着青袍，眉目清俊，气度超然，负手而立，周身萦绕着淡淡的剑气。那剑气凌厉无匹，仿佛能将山河斩断、将日月倾覆，却又在他眉宇间化作了洒脱与不羁。他的目光深邃而清澈，像是看透了世间一切，又像是将一切都不放在心上。

云无咎呆呆地望着那道虚影，喉咙发紧，一个字也说不出来。

那虚影似乎察觉到了他的存在，缓缓转过头来，望向了他。

四目相对。

那是一双极为清澈的眼睛。清澈得像是山巅的湖水，又深邃得像是无底的深渊。眼底有沧桑、有疲惫、有不甘，更有一丝久违的欣慰——仿佛等待了太久太久，终于等到了那个有缘人。

虚影望着他，嘴唇微微翕动。

一道苍老而温和的声音在他脑海中响起：

*"有缘人……你终于来了。"*
