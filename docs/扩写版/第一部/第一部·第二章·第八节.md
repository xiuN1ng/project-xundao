# 第一部·第二章·第八节：心镜迷阵

## 一、幻象

云无咎的脚步在镜前一尺处停住了。

不是他想要停下，而是脚仿佛被什么东西钉在了原地一般，一步也迈不动。

那面镜子就矗立在他面前，高约三丈，宽约两丈，表面光滑如水银，却泛着淡淡的银光。镜框由某种不知名的黑色金属铸成，框上雕刻着繁复的花纹，那些花纹像是盘绕的蛇，又像是流动的水，看得久了便觉得头晕目眩。

而镜中映照出的，不是他的倒影。

而是一个他无比熟悉的人。

云爷爷。

老人就坐在一张旧木椅上，穿着那件打满补丁的灰色棉袄，花白的头发在脑后挽成一个髻，脸上沟壑纵横，却带着他记忆中最温暖的笑容。老人手里端着一碗冒着热气的米粥，那粥熬得稠稠的，上面还卧着一个煎得金黄的荷包蛋——是云爷爷的独门做法，整个青云镇只有他会做这样的粥。

"咎儿，饿了吧？"

老人的声音从镜中传来，和记忆中一模一样，沙哑而温和，带着一股让人无法拒绝的亲切。

"来，趁热吃。"

他招了招手，示意云无咎过去。

那声音太真实了。

真实得让云无咎的心脏猛地揪紧。

他想起了那天早上——他出门砍柴，云爷爷还坐在门槛上抽旱烟，叮嘱他"早点回来，路上小心"。他想起了那碗温在灶台上的粥，想起老人布满皱纹的笑脸，想起那些相依为命的日日夜夜。

他想起了最后一次见到云爷爷的场景。

那是测灵根那天，老人执意要送他到村口。他走了很远，回头一看，白发苍苍的老人还站在老槐树下，朝他挥手。那身影越来越小，越来越模糊，最终消失在了晨雾之中。

那是他见云爷爷的最后一面。

他甚至不知道老人现在怎么样了。是还在家里等他？还是已经……

"咎儿，你在想什么？"

镜中的云爷爷站起身来，端着那碗粥，朝他缓缓走近。

"来，吃了再说。吃了饭再走。"

老人的声音像是从极遥远的地方传来，穿越了无尽的岁月，带着三百六十多个日日夜夜的思念。

云无咎的脚步动了。

他不由自主地朝那面镜子迈了一步。

只是一步。

可就是这一步，让他浑身的汗毛都竖了起来。

不对。

他的脑海中忽然闪过一道电光。

云爷爷从不做荷包蛋。

老人说过，煎荷包蛋费油，平日里一个都舍不得吃，只有逢年过节才舍得做上一个。而镜中这碗粥上，赫然卧着两个荷包蛋。

两个。

云无咎的脚步停住了。

他死死盯着镜中那张和蔼的面孔，盯着那碗冒着热气的粥，盯着那双浑浊却温暖的眼睛。

镜中的云爷爷还在朝他招手，还在笑着说"来，吃了再说"。

可那张笑脸，此刻看来，竟有几分说不出的诡异。

那笑容太完美了。完美得像是画上去的。真正的云爷爷笑起来的时候，眼角会皱成一团，露出一口缺了门牙的牙床。而镜中这个，笑得无懈可击，却总觉得少了点什么。

少了那种历经沧桑的厚重感。

少了那种柴米油盐的烟火气。

少了那种"我在这里等你"的笃定。

"你不是爷爷。"

云无咎的声音很轻，却很坚定。

镜中的画面顿了一下。

那张笑脸像是被冻住了一般，笑容僵在了脸上。

"咎儿，你说什么呢？"镜中的云爷爷歪了歪头，笑容依旧和蔼，"我就是爷爷啊。你不认识我了？"

"我不认识你。"

云无咎退后一步。

"你不是我爷爷。"

"我爷爷已经走了。"

他的声音有些发涩，却依然坚定。

"测灵根那天，我离开青云镇的时候，爷爷来送我。他站在村口的老槐树下，朝我挥手。他什么都没说，就只是笑，那种舍不得却又不敢表露的笑。"

"后来我杀了那头大野猪，回村的时候，他没有迎出来。我当时就觉得不对劲。"

"再后来，我听说爷爷在我离开后的第三天就……就走了。"

他的声音哽了一下。

"他走的时候很安详，嘴角还带着笑。村里的人说，他是在睡梦中走的，没有受罪。"

"他说——让我不要回头，往前走。"

"我不会回头的，爷爷。"

他抬起手，朝那面镜子轻轻一推。

"但我也不会被幻象困住。"

他的手穿过了镜面。

那看似坚实的镜面在他触碰的一瞬间便如水面般荡开涟漪，镜中的云爷爷、那碗粥、那张旧木椅，都像是被打碎的玻璃一般，化作点点星光，消散在空气中。

幻象破了。

心镜破了。

前方的道路重新出现在他眼前。

云无咎收回手，在原地站了片刻。

他的眼眶有些发红，却没有流泪。

他知道那只是幻象。可幻象中的那碗粥、那张笑脸、那声"咎儿"，却依然在他脑海中回荡，挥之不去。

他会永远记得云爷爷。

可他不会被那份记忆困住。

他深吸一口气，压下心中的酸涩，迈步继续向前。

他知道，这只是第一面镜子。后面的三十五面镜子，才是真正的考验。

---

## 二、执念

石林中的光线很暗，只有从石柱顶端透进来的微弱阳光，能见度不过三四丈。

云无咎沿着石柱之间的缝隙缓缓前行，目光警惕地扫视着四周。

空气中弥漫着一股淡淡的雾气，那雾气带着一股奇异的甜味，吸得久了便让人觉得昏昏沉沉。他不得不打起精神，小心翼翼地避开那些雾气较为浓密的地方。

第二面镜子出现在五丈之外。

这面镜子泛着淡淡的金光，镜框上的花纹也与第一面不同，像是一条盘旋的金龙，气势恢宏。

而镜中映照出的，是一个人影。

那是一个身着青袍的中年男子，面容清癯而刚毅，眉宇间透着一股不怒自威的气势。他的身旁斜倚着一柄铁剑，那铁剑锈迹斑斑，却隐隐透着一股不屈的锋芒，仿佛随时都会破鞘而出，斩尽世间不平事。

是云游子。

镜中的云游子和他记忆中一模一样——同样的眉眼，同样的气度，同样的青袍和铁剑。他就那样盘坐在镜中，面前摆着一壶酒、一只杯，慢条斯理地自斟自饮。

"小家伙。"

那声音从镜中传来，苍老而温和，透着几分醉意。

"你来了。"

云无咎的喉咙一紧。

这是他最敬重的恩人。那位在断崖石台上枯坐了三百年、只为等待一个有缘人的前辈。那位将毕生所学倾囊相授、将游子剑托付于他的恩人。

"前辈。"

他的声音有些发颤。

"老夫等了三百年，终于等到了你。"镜中的云游子站起身来，朝他招了招手，"过来，让老夫好好看看你。"

他招了招手，示意云无咎过去。

云无咎的脚步动了。

他不由自主地朝那面镜子走去，眼睛死死盯着镜中那张熟悉的面孔。

云游子前辈……他最敬重的恩人。他把毕生所学传授给了自己，把游子剑托付给了自己，甚至在最危急的关头耗尽了最后的残魂，只为给他打开一条生路。

他想要上前。

他想要再听一次前辈的教诲，哪怕只是幻象也好。

可是——

他忽然顿住了。

脚步在镜前一尺处停住，像是被什么东西拉住了似的。

不对。

有什么地方不对。

他的目光死死盯着镜中那张和蔼的面孔。

那面容确实是云游子前辈的样子，那气度也确实是云游子前辈的气度。可镜中的云游子，却少了一样东西。

眼神。

真正的云游子前辈，眼神中始终带着一丝深沉的忧郁。

那是因为他死在了阴谋之中，死在了自己人的背叛之下。他的眼神里永远有一道化不开的阴影，那是三百年冤屈的积累，是永远无法弥合的伤痛。

镜中的这个"云游子"，眼神清澈如水，波澜不惊。

那不是真正的云游子。

那只是他心中"最敬重的恩人"的幻象，而非恩人本人。

"你也不是前辈。"

云无咎退后一步，声音平静。

"前辈走的时候跟我说——'活下去，活得越久，修为越高，你能接触到的层次就越高。到那时，真相自然会浮出水面'。"

"前辈要我活下去，不是要我沉溺在过去的幻象里。"

他再次抬手。

这一次，他没有犹豫。

他的手掌带着破空的劲风，狠狠拍在镜面之上。

"碎。"

心镜应声而裂，化作漫天星光消散在空气中。

## 三、执念深重

第三面镜子。

镜中是一群孩童，他们指着他嘲笑——"没爹没娘的野种"、"扫把星"、"丧门星"……

他看了一眼，一掌拍碎。

第四面镜子。

镜中是他站在天道宗的山门外，被人推搡着赶出来——"你不配"、"滚回你的穷山沟去"、"一个孤儿也想修仙？做梦！"

他看了一眼，一掌拍碎。

第五面镜子。

镜中是云爷爷躺在病床上，弥留之际，嘴里念叨着他的名字——"咎儿……咎儿……你在哪儿……"

他的手指在镜面上停了一瞬。

只是一瞬。

然后，他的手掌还是拍了下去。

"爷爷走了。"

他低声说。

"我没能见他最后一面。这是我这辈子最大的遗憾。"

"可是——"

他的声音微微一顿，随即变得坚定起来。

"他不会想看到我被这个遗憾困住的。他要我往前走，不是要我往后看。"

心镜破碎。

第六面。第七面。第八面……

每一面镜子都映照出他内心深处的某种执念。

那些执念有的清晰、有的模糊，有的让他愤怒、有的让他悲伤、有的让他恐惧。可无论是什么，他都一一看穿，一一击碎。

因为他知道，这些都是假的。

真正的人不会回来。真正的事无法改变。他唯一能做的，就是不被这些幻象所迷，不被这些执念所困，坚定地往前走。

第十面镜子。

第十五面镜子。

第二十面镜子。

每一面镜子都被他一一击破，每一种执念都被他一一看穿。

他的心中只有一个念头——往前。

可是渐渐地，他发现了一些规律。

第一面镜子映照的是对亲情的执念。第二面是对恩情的执念。第三面到第五面，是对自卑和恐惧的执念。而从第六面开始，镜中的内容开始发生变化——

第六面，是他站在一座巍峨的山巅，俯瞰着脚下的芸芸众生。他的身旁站满了人，所有人都对他毕恭毕敬。可在那人群之中，他却找不到一个熟悉的面孔。

他看了一眼，拍碎。

第七面，是他在天道宗中步步高升，成为人人羡慕的内门弟子、核心弟子、甚至长老。可当他想要联系云爷爷的时候，却发现老人早已不在人世，而他甚至没能回去送他最后一程。

他的手掌在镜面上停了一瞬。

只是一瞬。

然后，还是拍了下去。

"我会变强的。"

他低声说。

"可我不会为了变强而忘记我是谁。"

第八面。第九面。第十面……

第二十五面镜子出现在他面前。

这面镜子泛着淡淡的黑光，与之前的那些都不同。

镜中是一片黑暗。

无尽的、深邃的、令人窒息的黑暗。

那黑暗像是一头无形的巨兽，正张着血盆大口，等待将他吞噬。

然后，黑暗中亮起了一双眼睛。

那是一双极为阴冷的眼睛。

眼睛的主人站在黑暗深处，看不清面容，只能看见那双眼睛泛着幽幽的绿光，像是深渊中潜伏的恶鬼。那目光落在云无咎身上，让他浑身的汗毛都竖了起来。

"云游子的传承者……"

一个阴恻恻的声音从黑暗中传来。

"你以为，你逃得掉吗？"

云无咎的心脏猛地一缩。

那声音太熟悉了。

和他在无数个深夜噩梦中听到的那个声音一模一样。

那是云游子前辈的仇人。

是那个让前辈含冤而死的人。

是那个前辈用尽最后的力气警告他"处处提防"的人。

他不知道那人在天道宗是什么身份，不知道对方有多强，甚至不知道对方的名字。可他知道，那个人此刻就在那座云雾缭绕的山峰之上，俯瞰着这一切。

"你以为你逃得掉吗？"

那声音再次响起，带着几分嘲弄。

"你不过是云游子留下的一个棋子罢了。等你进了天道宗，等你接触到越来越多的秘密，你就会发现——一切都在我的掌控之中。"

"你的每一步，都是我安排好的。"

"你的命运，早在三百年前就已经注定。"

"云游子当年没能逃出我的手掌心。你以为，你比他强吗？"

那声音带着令人毛骨悚然的自信，像是一只猫在戏弄着一只走投无路的老鼠。

云无咎站在镜前，一动不动。

他的拳头攥得死紧，指甲深深嵌入掌心，刺痛感一阵阵地传来。

愤怒。

不甘。

恐惧。

各种情绪在他胸腔中翻涌，让他几乎喘不过气来。

可是——

他深吸一口气。

然后，他笑了。

那是一个极轻极淡的笑，带着几分不屑，几分嘲弄。

"掌控我的命运？"

他抬起头，直视着镜中那双阴冷的眼睛。

"你在说笑话吗？"

"我从青云镇走到这里，一路上遇到多少困难，闯过多少难关，你知道吗？"

"我没有背景，没有资源，没有任何人帮我。我唯一有的，就是我自己。"

"可我照样走到了这里。"

"你说我的每一步都是你安排好的？"

他抬起手，掌心亮起一团淡淡的青光。

"那你安排得可真够差的。"

他一掌拍下。

心镜轰然碎裂！

那黑暗中的眼睛发出一声怒吼，随即被漫天的星光吞没。

## 四、出口

第三十六面镜子出现在他面前。

这面镜子与之前的三十五面都不同。

之前的镜子都是映照内心的执念和恐惧，而这一面——

这面镜子映照的是他自己。

不是过去的自己，不是未来的自己，而是现在的自己。

一个穿着破旧布衣、背负游子剑、满身伤痕却目光坚定的少年。

他站在镜前，与镜中的自己对视。

镜中的自己也在看着他。

那眼神里有疲惫、有伤痛、有孤独，却也有一丝倔强的光芒——那是无论如何都不肯服输的倔强，是哪怕跌倒了也要爬起来继续向前的不屈。

"我认识你。"

云无咎轻轻说。

"你是云无咎。青云镇的砍柴少年。云爷爷的孙子。云游子前辈的传人。"

"你出身贫寒，没有背景，没有资源，什么都要靠自己去争取。"

"可你也有别人没有的东西。"

"你有云爷爷从小教给你的坚韧和善良。"

"你有云游子前辈传授给你的功法和剑术。"

"你有游子剑的庇护和加持。"

"最重要的是——"

他抬起手，将手掌按在镜面上。

"你有你自己。"

镜面荡开涟漪，像是水面被投入了一颗石子。

镜中的自己朝他微微一笑，那笑容里没有了之前的疲惫和伤痛，取而代之的是一种前所未有的平静。

然后，镜中的自己化作点点星光，没入了他的掌心。

那些星光温热而柔和，像是来自另一个自己的拥抱。

与此同时，一道耀眼的金光从他掌心涌出，将整面镜子吞没。

当金光散去的时候，云无咎发现自己已经站在了石林的出口处。

他的面前是一片开阔的平台，阳光明媚，白云悠悠。

他走出石林了。

## 五、余波

平台上已经站了不少人。

有的神色如常，有的脸色苍白，有的浑身是汗，显然在石林中都经历了一番苦战。还有一些人坐在地上，或靠着石柱喘息，或闭目养神，显然是被那心镜折腾得不轻。

李狗蛋和张小胖也在其中。

李狗蛋一见到他便跳了起来，连滚带爬地朝他跑过来，一脸劫后余生的庆幸："无咎哥！你可算出来了！你在里面走了多久？俺在里面转了大半天，差点没出来！"

"我也是。"张小胖跟在他后面，同样是一脸后怕，"那些镜子真是太邪门了，俺看到了俺爹临终前的样子……他、他躺在床上，拉着我的手说……说……"

他的声音哽住了，说不下去了。

李狗蛋连忙拍着他的背安慰他："没事了没事了，都过去了。咱们不是都出来了吗？"

云无咎朝两人点了点头，没有多说什么。

他只是静静地站在阳光下，闭着眼睛，让微风拂过他的面颊。

心镜试炼，让他对自己的认识更深了一层。

他知道自己有很多执念，有很多放不下的东西。

可他也知道，正是这些执念和放不下，让他成为了现在的自己。

自卑、恐惧、愤怒、不甘……

这些负面情绪他一样不少。

可他也有别的东西。

坚韧、善良、不屈、重情。

这些才是他真正的底色。

心性测试，他通过了。
