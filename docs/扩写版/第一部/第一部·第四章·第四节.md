# 第一部·第四章·第四节

## 御风而行·青峰岭

翌日清晨，天色微明。

云无咎早早起床，将游子剑负于背后，又检查了一遍行囊中的丹药和符篆，确认无误后，便出了门。

张小胖已经在青云峰脚下等着他了。

这胖子的心情显然不错，脸上的笑容就没断过。他穿着一身利落的短打，腰间挂着一个小巧的药篓，背上还背着一个鼓鼓囊囊的大包袱，也不知道装了什么。

"云兄弟！这边！"他远远地招手。

云无咎走上前去，两人打过招呼后，便一同踏上了前往青峰岭的路途。

天道宗山门以东三百里处，便是青峰岭。

这个距离若是步行，少说也要走上两三天。但对于修士而言，只需施展御风术，便能大大缩短行程。当然，以云无咎如今炼气境三层的修为，御风术只能勉强施展，速度和持久性都有限。好在张小胖也是炼气境二层，两人的速度倒是差不多。

两人踏风而行，脚尖点地，轻轻一跃便是数丈之遥。

晨风拂面，带着山间特有的草木清香。云无咎深吸一口气，只觉心旷神怡。

这是他第二次离开天道宗山门。上一次，还是三个月前初来报到之时。那时他人生地不熟，只顾着紧张，根本无暇欣赏沿途的风景。如今故地重游，心境已然不同。

脚下的山川河流飞速后退，大约两个时辰后，一片连绵起伏的丘陵地带出现在两人眼前。

那便是青峰岭。

站在高空俯瞰，只见满目苍翠，峰峦叠嶂，无数山峰如同一颗颗翠绿的宝石，散落在这片广袤的大地之上。一条清澈的溪流从山间蜿蜒而过，在阳光下闪烁着粼粼波光。

空气中弥漫着浓郁的灵气，比天道宗山门内还要充沛数倍。云无咎忍不住深吸了一口，只觉浑身经脉都舒展开来，仿佛置身于灵气的海洋之中。

"不错不错！"张小胖也是一脸享受，"这里的灵气比宗门浓郁多了！难怪那些妖兽愿意待在这儿！"

云无咎点点头，目光扫视着下方的地形。

按照任务要求，他们需要在三日之内，采集至少十株二品灵药"青灵草"，以及任意数量的其他灵药。青灵草多生长于山间阴湿之地，尤其是溪流两岸的岩石缝隙中，最为常见。

"先去东边的那片山谷。"云无咎指向地图上标记的一处位置，"那里的青灵草最多。"

两人找了一处平缓的山坡降落，步行向山谷深处走去。

山间小径崎岖难行，两侧是茂密的灌木丛，不时有野兔、山鸡等小兽从草丛中窜出，惊起一片飞鸟。

一路上，他们遇到了不少同样来采集灵药的外门弟子。大家各忙各的，倒也相安无事。偶尔有人认出云无咎，投来几道意味深长的目光，却也无人上前搭话。

"那个就是云无咎吧？听说赵乾坤要对付他。"

"嘘，小声点，别让他听见。"

"怕什么？他一个散修，还能翻了天不成？"

窃窃私语声传入耳中，云无咎却恍若未闻，脚步不停。

张小胖凑过来，压低声音道："云兄弟，这几天你可得小心点。赵乾坤的人到处在打听你的消息，说不定已经知道你接了青峰岭的任务。"

"我知道。"云无咎点点头，神色平静。

他早就料到了这一步。赵乾坤那种睚眦必报的性格，既然放出话来要对付他，就绝不可能善罢甘休。此次青峰岭之行，注定不会太平。

但他并不害怕。

修仙之路，本就是与天争命，与人争锋。若连这点挑战都应付不了，还谈什么追寻大道？

两人沿着山间小径前行，约莫走了小半个时辰，终于抵达了那片山谷。

山谷两侧是高耸的青峰，中间是一片开阔的谷地。谷中有一条清澈的小溪缓缓流过，溪水清凉见底，能看见水草在水底轻轻摇曳。

"就是这里了。"张小胖指着溪流两岸，"青灵草多长在溪边的岩石缝隙里，咱们分开找，采集完了在谷口汇合。"

云无咎点点头，与张小胖约定好汇合的时间和地点后，便各自向不同的方向走去。

他沿着溪流慢慢前行，目光在两岸的岩石缝隙间扫视，寻找着青灵草的踪迹。

青灵草是一种二品灵药，叶片呈青绿色，边缘泛着淡淡的金纹，采摘时需要连根拔起，保留完整的根系，否则药效会大打折扣。

走了约莫一刻钟，云无咎终于在一块青石旁边发现了几株青灵草。

他蹲下身，小心翼翼地将青灵草连根拔起，放入背后的药篓之中。动作轻柔而熟练，显然这三个月来没少做这种事。

采完这几株，他又继续向前走去。

一路上，他又陆续发现了几处青灵草的生长地，采集的数量渐渐多了起来。按照这个进度，完成十株的任务目标并不困难，甚至还有余裕可以多采一些，拿回宗门兑换灵石。

然而，就在云无咎专心采药的时候，四周的气氛却渐渐变得诡异起来。

起先，他只是觉得周围的虫鸣声变少了。原本时不时能听到的鸟叫、蛙鸣、溪水潺潺的声音，都渐渐消失，整片山谷陷入了一种诡异的寂静之中。

然后，他闻到了空气中那股若有若无的血腥味。

那味道很淡，淡得几乎难以察觉。但对于曾经在山林中生活过多年的云无咎而言，这种味道再熟悉不过——那是妖兽的血的气息。

他的后背倏地绷紧了。

云无咎停下手上的动作，静静地站在原地，目光警惕地扫视着四周。

谷中静得出奇，连风都停了。溪水依旧在流，却发出一种说不出的沉闷声响，像是被什么东西压抑着。

那血腥味越来越浓，夹杂着一股若有若无的腐臭。地面上散落着一些细碎的白色骨头——那是小兽的残骸，已经被什么东西啃食得只剩碎片。

云无咎的手悄悄摸向背后的游子剑。

就在这时，一声震天的咆哮忽然从山谷深处传来！

"吼——！！！"

那咆哮声低沉而狂暴，震得山谷两侧的岩石簌簌发抖。无数碎石从山坡上滚落，砸入溪水之中，溅起一片水花。

地面剧烈震动，林木摇晃，仿佛有什么庞然大物正在向这边冲来。

云无咎瞳孔骤缩。

他猛地抽出游子剑，只见前方的灌木丛剧烈摇晃，一道巨大的黑影正以惊人的速度向这边逼近！

下一刻，一头足有两丈高的巨大妖兽从林木间冲出！

那是一头浑身覆盖着青色鳞甲的巨狼，体型比普通野狼大了数倍不止。它的皮毛呈现出一种诡异的青黑色，每一片鳞甲都泛着幽幽的冷光，在阳光下闪烁着金属般的质感。

最可怕的是那双眼睛——血红一片，没有丝毫理智，只有纯粹的嗜血与狂暴。那不是普通野兽的目光，而是一种被什么东西激发、扭曲之后的疯狂。

"一星妖兽……青鳞狼！"

云无咎失声惊呼。

青鳞狼是青峰岭的特有妖兽，以皮糙肉厚、凶猛善战著称。通常栖息在深山之中，轻易不会出现在外围区域。但此刻这头青鳞狼不知为何离开了巢穴，而且明显带着一股狂暴的气息，像是被什么东西激怒了。

它张开血盆大口，露出两排锋利的獠牙，獠牙上沾着未干的血迹，显然刚刚饱餐过一顿。

它死死盯着云无咎，眸中闪过一丝狡黠与贪婪。

下一刻，它猛地向云无咎扑来！
