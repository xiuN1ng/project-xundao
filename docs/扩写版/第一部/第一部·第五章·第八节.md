# 第一部·第五章·第八节：星辰阁前

## 一、意外相遇

云泽继续在森林中穿行。

他的身体还很虚弱，每走一步都能感觉到四肢百骸传来的酸痛。但他咬紧牙关，一步一步地向前挪动着，不敢有丝毫停顿。

噬魂三兽的追踪让他消耗了大量的体力，而方才与赤蟒的一战更是让他伤上加伤。如今的他，就像是一台运转过度的机器，随时都可能崩溃。

"无咎小子，前面有一条小溪。"剑灵的声音响起，"你应该去清洗一下伤口，顺便补充一些水分。"

云泽点了点头，顺着剑灵指引的方向走去。

穿过一片茂密的灌木丛，一条清澈的小溪出现在眼前。溪水潺潺流淌，在岩石间跳跃，发出悦耳的声响。阳光从树冠的缝隙中洒落，在水面上投下斑驳的光影。

云泽在溪边蹲下，捧起一掬清水，浇在脸上。

清凉的溪水让他精神为之一振，昏沉的脑袋也清醒了几分。他脱下外衣，检查了一下身上的伤口——右手手腕处的裂痕比之前更明显了，左手手臂上也有几道被树枝划破的伤痕。

"还好，都是皮外伤。"剑灵说道，"休息一下，吃点东西，应该能恢复。"

云泽从怀中取出几枚干粮，就着溪水吃了下去。

那是他在进入秘境前准备的干粮，虽然口感粗糙，但热量很高，足以补充体力。

吃完之后，他靠在溪边的一块大石上，闭上眼睛，开始调息运功。

体内的真气缓缓运转，沿着经脉流淌，修复着受损的身体。虽然恢复的速度很慢，但总比什么都不做强。

不知过了多久，忽然，一道声音从身后传来。

"云师弟。"

云泽的眼睛猛然睁开。

他几乎是本能地弹起身来，手掌按上游子剑的剑柄，身形戒备地转向声音传来的方向。

"别紧张，是我。"

一道白色的身影从树林中走出，正是林惊羽。

"林师兄？"云泽微微皱眉，"你怎么在这里？"

"我来找你。"林惊羽走到他面前，目光从他身上的伤痕上一扫而过，"你受伤了？"

"小伤，不碍事。"云泽摇了摇头，"林师兄是怎么找到我的？"

"我跟着你的气息。"林惊羽在他身旁坐下，"方才在这片区域感知到了一场战斗的波动，便循着痕迹找了过来。没想到竟然是你。"

"战斗？"

"嗯，是噬魂三兽的气息。"林惊羽的眼中闪过一丝惊讶，"你竟然从它们手中逃脱了？"

"不只是逃脱。"云泽淡淡地说，"赤蟒死了。"

"什么？！"林惊羽的脸色骤变，"你杀了赤蟒？"

"嗯。"

林惊羽盯着他看了半晌，眼中闪过一抹复杂的神色。

"云师弟……你的实力，恐怕比我想象的要强得多。"他说道，"噬魂三兽联手足以对抗金丹境修士，你能杀死其中一头，说明你的真实战力已经无限接近金丹境了。"

"运气好而已。"云泽摇了摇头，"若不是赤蟒落单，我绝无胜算。"

"运气也是实力的一部分。"林惊羽说道，"更何况，你方才还受了这么重的伤。换作是我，恐怕早就撑不住了。"

云泽没有说话，只是静静地看着他。

片刻之后，他开口问道："林师兄，你来找我，是有什么事吗？"

林惊羽沉吟片刻，从怀中取出一张羊皮纸。

"我找到了这个。"他将羊皮纸递到云泽面前，"是一张昆仑秘境的地图。"

云泽接过地图，展开一看。

只见地图上画着密密麻麻的线条和标记，标注着昆仑秘境各处要地的位置。在地图的东北方向，有一个用红色墨水圈出的标记，旁边写着"星辰阁"三个字。

"这是……"

"这是我师父留给我的。"林惊羽说道，"他当年曾经进入过昆仑秘境，在里面发现了一些秘密。这张地图，就是他当年绘制的。"

"你师父？"云泽微微皱眉，"令师是……"

"我师父是内门的一位长老。"林惊羽说道，"他老人家在十年前去世了，临终前将这张地图交给了我，让我有机会的话，去星辰阁看看。"

"看看什么？"

"看看那里到底藏着什么秘密。"林惊羽的眼中闪过一抹深沉的光芒，"我师父说，二十年前，天道宗发生了一件大事。那件大事改变了很多人的命运，也埋下了很多秘密。而那些秘密的核心，就在星辰阁。"

云泽心中一震。

二十年前——那不正是云游子被害的时间吗？

"林师兄。"他抬起头，直视林惊羽的眼睛，"你告诉我这些，是想要什么？"

林惊羽沉默了片刻。

"我想要真相。"他说道，"我师父死前告诉我，他这辈子做过一件错事，一件让他后悔终生的错事。他说他没有勇气去弥补，只能把希望寄托在我身上。"

"什么错事？"

"我不知道。"林惊羽摇了摇头，"他没告诉我，只说让我去星辰阁看看，那里会有答案。"

"所以你才想要与我合作？"

"一部分原因吧。"林惊羽点了点头，"另一部分原因……是因为你。"

"我？"

"你的剑法很奇怪。"林惊羽盯着他，"你用的剑法，我从未见过，但我总觉得它与我师父留给我的剑谱有几分相似。"

云泽心中一凛。

相似？

游子九式是云游子独创的剑法，与天道宗其他剑派都不同。林惊羽的师父又怎么会留下与之相似的剑谱？

"林师兄。"他忽然开口，"令师的道号是什么？"

"家师道号……凌霄。"林惊羽说道，"是天道宗凌霄峰的峰主。"

"凌霄……"云泽在心中默念这个名字，忽然想起剑灵曾经提起过。

"无咎小子。"剑灵的声音在脑海中响起，语气透着几分凝重，"凌霄子这个名字，老夫有些印象。"

"什么印象？"

"二十年前，他是云游子的师弟。"剑灵说道，"当年云游子被害的时候，他也在场。"

云泽的瞳孔骤然收缩。

林惊羽的师父，竟然是云游子的师弟？

"林师兄。"他深吸一口气，"有件事，我觉得有必要告诉你。"

"什么事？"

"你师父口中的'那件错事'……"云泽直视他的眼睛，"可能与二十年前的一场背叛有关。"

"背叛？"林惊羽的眉头紧锁，"什么背叛？"

"二十年前，天道宗有一位天才弟子，名叫云游子。"云泽的声音很低，"他被人陷害，死在了昆仑秘境。而害死他的人……"

他没有说下去，但林惊羽已经明白了他的意思。

"你是说……我师父与那件事有关？"

"我不知道。"云泽摇了摇头，"但你师父让你去星辰阁，必然有他的道理。也许到了那里，一切就会有答案。"

林惊羽沉默了。

他低下头，似乎在消化这些信息。

过了很久，他才重新抬起头。

"云师弟。"他的声音透着几分沙哑，"谢谢你告诉我这些。"

"不必客气。"云泽说道，"我们是合作伙伴，不是吗？"

林惊羽看着他，忽然笑了。

"是。"他站起身来，朝云泽伸出手，"走吧，我们一起去星辰阁。"

云泽握住他的手，借力站起身来。

两人相视一笑，便朝着地图上标注的方向走去。

## 二、星路漫漫

三日之后。

云泽和林惊羽终于抵达了星辰阁遗迹的入口。

这一路上，他们遇到了无数艰难险阻——有毒瘴弥漫的沼泽，有凶兽出没的峡谷，有迷雾笼罩的山谷。但两人都凭借着过人的实力和默契的配合，一一闯了过来。

云泽的伤势在林惊羽的帮助下恢复了大半，虽然还无法发挥全部实力，但应付一般的危险已经不成问题。

此刻，他们正站在一道巨大的石门之前。

那石门高约五丈，宽约三丈，通体由一种不知名的黑色岩石雕琢而成。门楣上刻着三个古朴的大字——星辰阁。

三个字龙飞凤舞，笔走龙蛇，透着一股苍凉而古老的气息。

"这就是星辰阁……"林惊羽喃喃道，"师父果然没有骗我。"

云泽站在石门前，目光从那三个字上一扫而过。

他的心中涌起一股奇异的感觉——那石门虽然沉默无声，却仿佛在向他诉说着什么。一种若有若无的熟悉感从石门深处传来，让他的心跳不由自主地加快了几分。

"无咎小子。"剑灵的声音在脑海中响起，带着几分激动，"这里的气息……和当年一模一样！"

"当年？"

"当年云游子就是在这里发现了天道图碎片。"剑灵说道，"他用生命为代价，将碎片封印在了星辰阁最深处。而这道石门，就是封印的关键所在。"

"封印的关键？"云泽微微皱眉，"什么意思？"

"意思是……"剑灵顿了顿，"只有修炼了《云游心法》的人，才能打开这道石门。"

云泽看着眼前的石门，忽然明白了什么。

天璇真人让他活着、让他成长、让他修炼《云游心法》……一切的一切，都是为了这一天。

为了让他来打开这道石门。

"云师弟？"林惊羽见他神色有异，出声问道，"怎么了？"

"没什么。"云泽摇了摇头，"只是在想，这道石门该怎么打开。"

"你有办法？"林惊羽眼睛一亮。

云泽没有回答，而是走上前去，将手掌按在石门上。

他的掌心传来一阵微微的刺痛——那是石门中的封印在试探他，试图确认他是否修炼过《云游心法》。

片刻之后，石门发出一阵低沉的轰鸣声，缓缓向两侧打开。

"竟然真的打开了！"林惊羽惊讶出声，"云师弟，你是怎么做到的？"

"运气。"云泽淡淡地说，"走吧，进去看看。"

两人并肩走入石门。

门内是一条长长的走廊，走廊两侧的墙壁上刻满了壁画和符文。那些壁画描绘的是上古年间的场景——有神仙飞天，有妖兽肆虐，有英雄拔剑，有天地崩塌……

"这些壁画……"林惊羽放慢脚步，仔细观看，"记载的是什么？"

云泽也在看那些壁画。

第一幅壁画：一只巨大的妖兽盘踞在天地之间，那妖兽有龙头、龟身、鳞甲，正是传说中的饕餮。壁画上，饕餮并非邪恶之物，而是在清除天地之间的因果浊气，维护天道运转。

第二幅壁画：一名身穿白衣的剑客拔剑而起，那剑客面容模糊，但气势惊天，一剑划破苍穹，将饕餮镇压在了大地之下。

云泽看着那白衣剑客的身影，心中忽然涌起一股奇异的感觉。

那身影……为何如此熟悉？

"无咎小子。"剑灵的声音忽然响起，带着几分颤抖，"那个白衣剑客……是云游子的师父！"

"什么？"云泽心中一震。

"是星辰子！"剑灵的声音透着难以抑制的激动，"云游子的师尊！当年太古之战的主导者！"

云泽看着那幅壁画，心中翻涌起惊涛骇浪。

云游子的师父……竟然是太古年间镇压饕餮的英雄？

那岂不是说，云游子的师门，与天道宗的创立有着莫大的关联？

"继续走。"他压下心中的震惊，朝走廊深处走去。

## 三、壁画真相

两人沿着走廊一路前行，将两侧的壁画一一看过。

壁画的内容越往后越是触目惊心。

第三幅壁画：饕餮被背叛，魔气冲天，无数修士陨落。

第四幅壁画：白衣剑客与一名黑袍身影激战正酣。那黑袍身影的面容被黑气笼罩，看不清长相，但云泽注意到——那黑袍身影的胸口，佩戴着一枚令牌。

那令牌的形状，正是——天道宗的令牌！

"这……"林惊羽的脸色骤变，"这是什么意思？太古年间，背叛饕餮的人中，有天道宗的祖师？！"

云泽没有说话，只是静静地看着那枚令牌。

他的脑海中快速闪过无数念头。

天道宗建立于上古年间，历史悠久，传承万载。但从未有人知道，这个庞然大物的根基，竟然建立在背叛之上。

"云师弟。"林惊羽的声音有些颤抖，"这些壁画记载的是真的吗？"

"应该是真的。"云泽说道，"这些壁画与……一些我了解到的信息完全吻合。"

"你了解到的信息？"

云泽沉默了片刻。

"林师兄。"他开口，"有些事，我觉得现在应该告诉你了。"

"什么事？"

"二十年前被陷害的那个天才弟子云游子……"云泽直视他的眼睛，"他是星辰子的传人。"

林惊羽的瞳孔骤然收缩。

"星辰子的传人？"他的声音透着难以置信，"可是星辰阁不是早就……"

"早就覆灭了？"云泽摇了摇头，"星辰阁只是隐入了暗处，并没有真正消亡。云游子就是星辰阁最后的传人。"

"而害死他的人……"他的声音忽然冷了下来，"正是如今天道宗位高权重的天璇真人。"

"天璇真人？！"林惊羽惊呼出声，"这怎么可能！他可是化神境的长老！"

"正因为他是化神境的长老，才能只手遮天地隐藏真相二十年。"云泽的眼中闪过一抹寒光，"当年他为了抢夺天道图碎片，与外人勾结，设计陷害了云游子。云游子临死前，用生命为代价将碎片封印在了星辰阁深处。而那道封印……"

他看向走廊尽头的方向。

"只有修炼了《云游心法》的人才能打开。"

林惊羽的脸色变得苍白。

他想起了师父临终前的那些话——"我这辈子做过一件错事"，"我没有勇气去弥补"……

难道师父所说的"错事"，就是与那场背叛有关？

"林师兄。"云泽的声音打断了他的思绪，"我不知道令师当年是否参与了那场背叛。但有一件事我可以确定——星辰阁遗迹里，藏着所有的真相。"

"你要去找那个真相吗？"林惊羽问道。

"必须去。"云泽的语气很坚定，"那里有我需要的东西。"

林惊羽沉默了片刻，忽然笑了。

"好。"他说道，"那我也去。"

"你确定？"云泽看着他，"里面的危险，可能远超你的想象。"

"我师父让我来这里，就是想要我找到真相。"林惊羽的眼中闪过一抹坚定，"无论那个真相是什么，我都要亲眼看到。"

云泽看着他，忽然伸出手。

"那就一起走。"

林惊羽握住他的手。

"一起走。"

两人并肩向走廊尽头走去。
