# 第一部·第八章·第四节：昆仑结集

## 一、魔域之门

九宗联军抵达昆仑虚外围时，已是十日之后。

这一路上，联军昼行夜宿，翻山越岭，穿过了数不清的城镇与村庄。每到一处，都会引起当地的轰动——毕竟，这支队伍中有太多名震天下的强者，随便拎出一个放到地方上都是呼风唤雨的存在。

然而，当联军终于抵达昆仑虚外围时，那些欢呼声便戛然而止了。

因为眼前的景象，实在太过骇人。

昆仑虚。

传说中远古战场的遗址，天地间最神秘的禁地之一。

这里的天空是暗红色的，像是被无尽的鲜血染过。那红色浓稠得几乎凝成实质，笼罩在整片区域上空，让人看了便觉得压抑得喘不过气来。大地上到处是焦黑的痕迹，像是被天火焚烧过，又像是被雷电劈裂过。残垣断壁星罗棋布，有的如同一柄柄指向天空的利剑，有的则像是一具具横陈的尸骨，在暗红的天光下显得格外狰狞。

空气中弥漫着一股腐臭与硫磺混合的气味，浓郁得让人几欲作呕。那气味像是来自地狱深处，带着死亡与毁灭的气息。

而最令人胆寒的，是那几乎要凝成实质的魔气。

魔气如同黑色的浓雾，在地面上翻涌奔腾，时而汇聚成巨大的漩涡，时而如同无数头饥饿的野兽在嘶吼咆哮。身处其中，灵气运转都会受到极大的阻碍，仿佛每走一步都要耗费平常数倍的力气。

"这里的魔气浓度，比外界高了何止百倍……"林惊羽皱着眉头说道，脸色有些发白。

他是剑修，最依赖灵气战斗。在这种魔气浓郁的环境中，他的实力至少要打个七八折。

"不只是你。"天虚真人的声音从前方传来，"所有人都一样。在这里，化神境的强者只能发挥出元婴境的实力，而元婴境……怕是连金丹境都不如。"

"那我们岂不是……"一名年轻的弟子脸色煞白。

"所以才需要速战速决。"天虚真人说，"拖得越久，对我们越不利。"

先锋队三千余人在这魔气边缘停了下来。

前方那道巨大的裂缝就矗立在那里——那是昆仑虚的入口，一道横亘于天地之间的狰狞伤口。裂缝中不断涌出浓郁的魔气，如同沸腾的浓烟，向四周蔓延。

云泽站在队伍最前方，目光穿透层层魔气，望向那道裂缝深处。

什么都看不见。

只有黑暗。

无尽的、幽深的、仿佛要将一切吞噬的黑暗。

但他却能感觉到，在那黑暗的最深处，有一个庞大无比的存在正在缓缓苏醒。那存在的每一次呼吸，都会引发魔气的一次剧烈波动，仿佛整个昆仑虚都在为它的觉醒而颤抖。

饕餮。

它就在那里。

等待着他们的到来。

"先锋队，随我进入！"天虚真人大手一挥。

先锋队三千余人列成战斗阵型，缓缓向那道裂缝逼近。

## 二、阵前点兵

裂缝入口处，魔气比外围更加浓郁。

当先锋队踏入裂缝的那一刻，所有人都感觉到了明显的压力——那是来自魔气的侵蚀，如同一只无形的巨手，正从四面八方挤压着他们的身体和灵魂。

"结阵！"天虚真人沉声下令。

先锋队的修士们迅速变换阵型，围成一个圈，将天虚真人护在中央。每个人都将灵气外放，在体外形成一层淡淡的光罩，勉强抵御着魔气的侵蚀。

"这里的环境比老夫预想的还要恶劣。"天虚真人皱着眉头说道，"若继续深入，怕是有不少弟子撑不住。"

"师叔，让我和云师弟在前方开路吧。"林惊羽站出来说道，"我们两个修为高一些，抵抗魔气侵蚀的能力也强一些。"

"好。"天虚真人点头，"你们两个在前方开路，厉无心负责策应。遇到危险，立刻后退，不要恋战。"

"是！"

林惊羽拔出长剑，剑身上泛着淡淡的青光。那是他将灵气注入剑身的征兆——在外界，这种程度的灵气外放对他而言轻而易举，但在这里，他却需要倾尽全力才能勉强维持。

"云师弟，走吧。"林惊羽回头看了云泽一眼。

云泽点了点头，将游子剑握在手中。

剑身微微发热，仿佛在回应着他的召唤。

奇怪的是，在这魔气浓郁的环境中，游子剑不仅没有受到影响，反而隐隐透出一丝暖意。那暖意如同涓涓细流，从剑柄流入他的掌心，再沿着经脉蔓延至全身，让他的灵气运转变得顺畅了许多。

这是怎么回事？

云泽心中一动，想起了厉无心对他说过的话——"游子剑中封存着饕餮的一缕神识"。

难道是因为这个缘故？

他来不及多想，因为前方的魔气中已经出现了动静。

"前方有东西！"厉无心忽然出声示警。

众人循声望去，只见前方的魔气忽然剧烈翻涌起来，紧接着，一个巨大的黑影从魔气深处缓缓浮现。

那是一头难以用言语形容的怪物。

它的身形如同一座小山，浑身覆盖着漆黑的鳞甲。那些鳞甲边缘锋利如刀，在暗红的天光下闪烁着金属般的光泽。它的四肢粗壮如柱，爪子深深嵌入地面，仿佛要将整个大地都撕裂。

而最令人胆寒的，是它的头颅。

那头颅像是狮子，又像是巨龙，一张血盆大口占据了整个面部的三分之二，獠牙如山岳般层层叠叠，光是一根獠牙就有数丈之高。

"是……是饕餮？！"一名年轻的弟子惊恐地喊道。

"不。"厉无心摇了摇头，神情凝重，"这不是饕餮本体。这只是它身上分裂出来的一缕残念。"

"饕餮的残念？！"

"饕餮被封印了三百年，早已不是当年那个无敌的存在。"厉无心说，"它的大部分力量都被封印压制，只有一小部分力量能够渗透出来。这些渗透出来的力量，凝聚成了这些分身。"

"换句话说——"

厉无心的话还没说完，那头怪物已经朝他们冲了过来。

"散开！"天虚真人大喝一声。

先锋队的修士们立刻四散而开，堪堪避开了怪物的第一次冲击。

但那怪物的速度太快了。

它庞大的身躯如同移动的山岳，却有着与之完全不相称的敏捷。四蹄猛然蹬地，它瞬间调转方向，朝最近的一名弟子扑去。

"啊——！"

那名弟子惨叫一声，被怪物的巨爪拍飞出去，重重地撞在岩壁上，口中喷出一大口鲜血。

"张师弟！"附近的弟子惊呼出声。

"不要慌乱！结阵迎敌！"天虚真人的声音如同惊雷，在众人耳边炸响，"所有化神境长老，随老夫一同出击！"

十余道身影同时飞出，向那头怪物围杀而去。

剑气、掌风、法术，如雨点般向怪物轰去。

但那怪物的防御力惊人得可怕——普通攻击打在它身上，只能留下一道浅浅的白印，根本无法造成实质性的伤害。

"这东西的皮也太厚了吧！"林惊羽一边闪避一边骂道。

"让我来！"云泽忽然大喝一声。

他身形一闪，瞬间掠至怪物身侧，游子剑高高举起，剑身上爆发出璀璨的青芒。

"游子九式——第三式——游子吟！"

剑光凛冽，如同一轮弯月，直直斩向怪物的脖颈！

"噗——！"

剑光切入怪物的鳞甲，如同切豆腐一般，将其整个脖颈斩断！

黑色的血液喷涌而出，那庞大的身躯摇晃了两下，轰然倒地。

"好剑法！"天虚真人大喜，"云泽，不愧是云游子的传人！"

云泽没有回答，只是静静地望着那具尸体。

那怪物的头颅滚落在地上，一双赤红的眼睛依然睁着，仿佛还带着死前的不甘。

"这只是开始。"厉无心走到他身边，声音低沉，"前方还有更多。"

云泽点了点头，将游子剑收回剑鞘。

他感觉到了。

在这头怪物倒下的地方，一缕淡淡的黑气正缓缓飘向昆仑虚深处。

那是饕餮残念消散后留下的痕迹。

它正在回归本体。

## 三、血战魔域

先锋队继续深入。

但仅仅前进了不到百丈，异变便再次发生。

地面突然剧烈震动起来。

"轰隆隆——"

震耳欲聋的轰鸣声从四面八方传来，仿佛有无数头巨兽在地底咆哮。

紧接着，无数根黑色的触须从地底破土而出，如同雨后春笋一般，瞬间布满了整片天地。

"魔物袭击！"

"结阵！结阵！"

先锋队的修士们反应极快，迅速结成防御阵型。剑气、掌风、法术如雨点般向那些触须轰去。

但那些触须仿佛是用钢铁铸就的一般坚硬，普通攻击打在上面，只能留下一道浅浅的白印。

"都让开！"林惊羽大喝一声，手中长剑出鞘。

"天外飞仙剑诀——一剑隔世！"

一道璀璨的剑光划破长空，如同银河坠落，将数十根触须一剑斩断。

"好剑法！"

众人纷纷叫好。

但下一刻，所有人都笑不出来了。

因为那些被斩断的触须，竟然以肉眼可见的速度重新生长出来，而且数量比之前更多了！

"没用的。"厉无心脸色阴沉，"这些触须不是普通的魔物，它们是饕餮的一部分。只要饕餮不死，这些触须就永远不会停止生长。"

"那该怎么办？！"天虚真人急道。

"只有一个办法。"厉无心看向那道光柱的方向，"直捣黄龙。找到饕餮本体，从根源上解决它。"

"可是饕餮的力量……"

"我去。"云泽忽然开口。

众人一愣。

"饕餮的力量，不是普通的攻击能够对付的。"云泽握紧了游子剑，"但我有这个。"

他扬了扬手中的游子剑。

"游子剑是上古神兵，或许能克制饕餮的魔气。"云泽说，"而且——饕餮认识我。"

"什么？！"众人震惊。

"在昆仑秘境的时候，我就曾与饕餮的残念有过接触。"云泽说，"它认出了我身上的某种东西——道君印记。"

"道君？！"天虚真人大惊，"你是道君转世？！"

云泽没有回答。

他只是转身看向那道光柱的方向，眼中闪过一丝决然。

"天虚师叔，联军主力就拜托你了。"他说，"拖住这些魔物，为我争取时间。"

"你一个人去？"林惊羽急道，"太危险了！我和你一起！"

"不行。"云泽摇头，"饕餮的觉醒需要道君之力来压制。你们去了，反而会成为拖累。"

"可是——"

"林师兄。"云泽拍了拍林惊羽的肩膀，"照顾好苏师姐。如果我回不来，替我告诉她……"

"别说了！"林惊羽打断了他，"你会回来的。"

云泽笑了笑，没有再说什么。

他转身，向那道光柱的方向疾驰而去。

他的身影很快消失在浓浓的魔气之中。

身后，是三千名修士浴血奋战的身影。

身前，是未知的深渊与真相。

饕餮在等待着他。

而他，已经准备好了。
