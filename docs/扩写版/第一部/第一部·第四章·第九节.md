# 第一部·第四章·第九节

## 偏殿疑云·隐秘传音

天机真人将云泽带到偏殿后，便离去了。

"你在这里等着。"老人丢下这句话，便转身走了出去，似乎有意的给两人独处的空间。

偏殿不大，陈设简单，只有一张檀木桌、几把椅子，以及墙上挂着的一幅山水画。殿内燃着几盏长明灯，光线柔和，将整个偏殿笼罩在一片温暖的光晕之中。

云泽独自坐在殿中，心中思绪万千。

他知道，以自己如今的修为，根本无力与天璇真人对抗。方才那道神识威压，已经让他清楚地认识到了这一点。那是一种压倒性的力量差距，大到他甚至生不出反抗的念头。

但他也隐隐察觉到，天璇真人的态度似乎有些奇怪。

若真是云游子前辈口中的"仇人"，天璇真人应该会毫不犹豫地对他出手，或者至少表现出明显的敌意。但从方才的接触来看，天璇真人看他的眼神中，除了审视之外，还有一种说不清道不明的情绪。

那不是单纯的觊觎，更像是……在寻找什么。

"云游子前辈说，他的仇人是天璇真人。但天璇真人陷害他的原因，却从未说明。"

云泽陷入沉思。

"如今看来，这其中恐怕还有更深的隐情……"

他想起今日在青峰岭山谷中，那具枯骨传递给他的记忆。

那段记忆中，天玄子是背叛叶孤城的元凶。但天璇峰上坐镇的人是"天璇真人"，而非"天玄子"。

这两个名字……是同一个人，还是不同的存在？

若是同一个人，那天璇真人的真实身份，就远比表面看上去要复杂得多。

若是不同的人……那天璇真人与天玄子之间，又是什么关系？

云泽越想越是困惑。

就在他百思不得其解的时候，偏殿的门忽然被推开了。

一道身影走了进来。

云泽抬头一看，竟然是天璇真人本人。

老人已经换了一身衣服，穿着一件素净的灰色长袍，看上去与方才威严的气势截然不同，更像是一个慈祥的长辈。

他没有说话，只是静静地看着云泽，目光深邃，仿佛能看穿一切。

云泽连忙站起身，躬身行礼。

"真人。"

天璇真人摆了摆手，示意他坐下。

"坐吧，不必拘礼。"

云泽迟疑了一下，还是坐了回去。

天璇真人在他对面坐下，抬手一挥，一道淡淡的光芒从殿内升起，在两人之间形成一道若有若无的屏障。

那是隔音禁制，可以防止两人的对话被人偷听。

云泽心中一凛。

天璇真人为何要设置隔音禁制？他要对我说什么？

老人静静地看着他，似乎在打量着什么。

良久，他忽然开口。

"云泽，你可知道，你与我一位故人……长得很像？"

云泽心中一震。

"故人？"

他从未想过，天璇真人会用这样的语气与他说话。那语气中，没有威严，没有审视，只有一种淡淡的感慨，和一丝极淡极淡的……怀念？

"什么故人？"云泽问道。

天璇真人没有回答。

他只是静静地看着云泽，目光中的情绪复杂得让人捉摸不透。

片刻后，他幽幽叹了口气。

"罢了，你不必知道这些。"

他忽然站起身，向云泽走去。

云泽下意识想要后退，却发现自己的身体像是被什么东西定住了一般，动弹不得。

天璇真人走到他面前，抬起手，轻轻按在了他的眉心之上。

那手掌温热而干燥，带着一股说不出的力量。

下一刻，一股温热的气流从天璇真人的掌心涌入他的识海。

那气流极为柔和，并不像方才的神识威压那般霸道，反而像是一道春风，轻轻拂过他的识海。

与此同时，一道清晰的声音在他脑海中响起——

"孩子……"

那是一个苍老而温和的声音。

"你若想知道真相，便去昆仑虚的'证道遗迹'。那里有你需要的一切答案。"

"但要小心……在你足够强大之前，不要让任何人知道你的真正身份。否则……你将死无葬身之地。"

话音落下，那股温热的气流也随之消散。

天璇真人收回手掌，转身离去。

他走到殿门口时，忽然停下脚步，却没有回头。

"今日之事，不许对任何人提起。"

他的声音恢复了方才的平淡与威严。

"去吧。"

云泽如梦初醒，呆呆地站在原地。

他方才听到了什么？

天璇真人称他为"孩子"？

他让自己去"证道遗迹"寻找真相？

更重要的是……天璇真人的语气，分明是在……保护他？

这究竟是怎么回事？！

云泽百思不得其解。

他想要追上去问个明白，却发现天璇真人的身影已经消失在殿外的夜色之中。

他独自站在偏殿中，心中翻涌着惊涛骇浪。

天璇真人的态度，太过诡异。

他分明知道云游子的仇人就是自己——若云游子前辈所言属实的话。但，他却不但没有对云泽出手，反而以一种极为隐秘的方式，传递了这样的信息。

这其中，究竟藏着什么秘密？

还有那句"不要让任何人知道你的真正身份"——

什么身份？道君转世？还是其他什么他不知道的东西？

云泽不知道。

但有一件事他可以确定——

自己身上的秘密，恐怕远比他想象的要复杂得多。

云游子的仇人，天璇真人……似乎并不是表面上那么简单。

而他自己，作为道君转世，作为"遁去的一"，也被卷入了这场跨越九纪元的巨大阴谋之中。

窗外，夜色深沉，星月无光。

云泽独自走出天璇峰，沿着石阶一步步向下走去。

夜风吹在他脸上，带着几分寒意，却让他渐渐冷静了下来。

他深吸一口气，将方才的一切压在心底。

无论天璇真人是敌是友，无论这其中有多少隐秘，他现在唯一能做的，就是不断地变强。

只有足够强大，才能揭开所有的真相。

只有足够强大，才能保护自己在意的人。

只有足够强大，才能在这场跨越纪元的博弈中，不被碾成齑粉。

"昆仑虚……证道遗迹……"

云泽轻声呢喃，将这几个字牢牢记在心里。

昆仑虚，那是传说中的万山之祖，位于中州极西之地。据说那里是上古仙人遗留的遗迹，蕴含着无数远古的秘密与机缘。无数修士梦寐以求，想要踏入那片神秘之地，却终生无缘得见。

而天璇真人，却让他去那里寻找真相。

这说明什么？说明天璇真人知道一些他不知道的事情。说明那个"证道遗迹"中，藏着他身世之谜的关键线索。

可问题是，天璇真人为什么要告诉他这些？

若天璇真人真是云游子前辈口中的"仇人"，他应该恨不得将自己碎尸万段才对。可他非但没有动手，反而在暗中指点他方向，这其中必有蹊跷。

云泽越想越是困惑。

他想起天璇真人看着他的眼神——那眼神中没有敌意，没有杀气，反而有一种说不出的……怀念？

"你与我一位故人长得很像。"

天璇真人的这句话，在他脑海中回响。

故人？什么故人？与自己长得很像的人，会是谁？

他想起自己的身世。他从小在青云镇长大，由云爷爷抚养成人。对于自己的父母，他一无所知。云爷爷只说他是被人遗弃在镇子后山的婴儿，被云爷爷捡回来抚养长大。

难道说，他的身世，竟与天璇真人有某种关联？

无数的念头在云泽脑海中翻涌，让他一时间心乱如麻。

"罢了，想再多也没用。"

他摇摇头，将这些杂念暂时压下。

无论真相如何，他现在唯一能做的，就是不断地变强。只有足够强大，才有资格去追寻答案，才有资格去揭开所有的秘密。

总有一天，他会踏上昆仑虚那片神秘的土地，找到"证道遗迹"，揭开所有的真相。

但不是现在。

现在，他还太弱了。

前路漫漫，荆棘遍布。

但云泽的眼中，没有恐惧，只有坚定。

无论前方有什么在等待着他，他都要一步步走下去。

直到揭开所有的真相。

直到站在这个世界的巅峰。
