# 第一部·第四章·第九节

## 偏殿疑云·隐秘传音

天机真人将云泽带到偏殿后，便离去了。

"你在这里等着。"老人丢下这句话，便转身走了出去，脚步声渐渐远去，最终消失在走廊尽头。

偏殿不大，陈设简单，只有一张檀木桌、几把椅子，以及墙上挂着的一幅山水画。殿内燃着几盏长明灯，光线柔和，将整个偏殿笼罩在一片温暖的光晕之中。空气中弥漫着一股淡淡的檀香气息，那香气清幽而绵长，嗅之令人心神宁静。

然而云泽的内心，却怎么也宁静不下来。

他独自坐在殿中，双手交叠放在膝上，指节不自觉地攥紧又松开。方才那道神识威压给他带来的震撼尚未消散，那是一种压倒性的力量差距，大到他甚至生不出反抗的念头——就像是一只蚂蚁面对滔天巨浪，连颤抖的资格都没有。

但更让他困惑的，是天璇真人方才看他时的那种眼神。

那不是单纯的审视，也不是他想象中的杀意。若真是云游子前辈口中的"仇人"，天璇真人应该会毫不犹豫地对他出手，或者至少表现出明显的敌意。但从方才的接触来看，老人看他的眼神中，除了那一闪而过的威压之外，还有一种说不清道不明的情绪。

那不是敌意。

更像是……在寻找什么。

或者说，在辨认什么。

"云游子前辈说，他的仇人是天璇真人。但天璇真人陷害他的原因，却从未说明。"

云泽陷入沉思。

他想起今日在青峰岭山谷中，那具枯骨——叶孤城——传递给他的记忆碎片。记忆中，天玄子背叛了叶孤城，是导致他被困三百年的元凶。但天璇峰上坐镇的人是"天璇真人"，而非"天玄子"。

这两个名字……是同一个人，还是不同的存在？

若是同一个人，那天璇真人的真实身份，就远比表面看上去要复杂得多。三百年前他还叫"天玄子"，三百年后却成了"天璇真人"，这其中发生了什么？他为何要改名换姓？若是不同的人……那天璇真人与天玄子之间，又是什么关系？师徒？同门？还是其他什么更深的牵扯？

云泽越想越是困惑，只觉眼前蒙着一层重重迷雾，怎么也看不清楚。

殿外的夜色透过雕花的窗棂渗进来，在地面上投下一片斑驳的光影。灯火摇曳，那些光影也随之轻轻晃动，像是水波在无声地荡漾。

就在他百思不得其解的时候，偏殿的门忽然被轻轻推开了。

一道身影悄无声息地走了进来。

云泽下意识地抬起头，只见来人正是天璇真人本人。

老人已经换了一身衣服，不再是方才那身威严的玄青道袍，而是一件素净的灰色长袍，布料看上去轻薄而柔软，透着一股子洗尽铅华的沧桑。灯光映在他脸上，将那些皱纹照得分外清晰，却也让他的神情显得比方才和蔼了许多，不再是那个高高在上的天璇峰掌峰真人，更像是一个饱经风霜的寻常老人。

他没有说话，只是静静地看着云泽。

那目光深邃而幽远，仿佛能穿透他的皮肉、血脉，一直望进他的骨髓里去。那眼神太过复杂，复杂到云泽根本读不出其中究竟是敌是友、是善是恶。

云泽连忙站起身，躬身行礼。

"真人。"

天璇真人摆了摆手，示意他坐下。

"坐吧，不必拘礼。"

他的声音也比方才平和了许多，少了那股拒人于千里之外的冷淡，多了几分说不出的温和。

云泽迟疑了一下，还是坐了回去。但他的脊背始终绷得笔直，双手在袖中悄悄攥紧，随时准备应对任何可能的突发状况。

天璇真人缓缓走到他对面坐下，抬手一挥。

一道淡淡的光芒从殿内升起，在两人之间形成一道若有若无的屏障。那屏障近乎透明，若非云泽目力过人，只怕根本察觉不到它的存在。他伸手轻轻触了触，指尖传来一阵微凉的感觉，像是触碰到了一层薄薄的水幕。

隔音禁制。

这种禁制他曾在宗门典籍中读到过，是一种极为高明的防窃听手段。一旦禁制升起，这殿中的一丝一毫动静都不会传到外面去。

天璇真人为何要设置隔音禁制？他要对我说什么？是灭口，还是……

云泽心中一凛，后背不自觉地渗出一层冷汗。

老人却仿佛没有察觉他的紧张，只是静静地看着他，目光中的情绪复杂得让人捉摸不透。

沉默在两人之间蔓延。

殿内只有灯火偶尔爆出的噼啪声，除此之外，一片死寂。

良久，天璇真人忽然开口了。

"云泽。"

"弟子在。"

"你可知道……"老人的声音低沉而缓慢，像是从极深极远的地方传来，"你与我一位故人，长得很像？"

云泽心中猛地一震。

"故人？"

他从未想过，天璇真人口中竟会说出这样的话。那语气中，没有威严，没有审视，只有一种淡淡的感慨，和一丝极淡极淡的……怀念？

"什么故人？"云泽追问道，声音不自觉地提高了几分。

天璇真人却没有回答。

他只是静静地看着云泽，目光从他的眉眼移到鼻梁，又从鼻梁移到嘴唇，像是在描摹一幅看不见的画像。那眼神中的情绪太过复杂，有欣慰，有感慨，有怅然，还有一丝极深极深的……哀伤？

那哀伤让云泽莫名地感到一阵心悸。

片刻后，老人幽幽地叹了口气。

"罢了。这些事，你还不需要知道。"

他忽然站起身，向云泽走去。

云泽下意识地想要后退，却发现自己的身体像是被什么东西定住了一般，动弹不得。体内灵力的运转仿佛被什么东西锁住，任他如何挣扎都无济于事。那是一种近乎绝望的无力感，就像一只雏鸟被苍鹰的利爪摁住，除了等待命运的裁决之外，别无选择。

天璇真人走到他面前，抬起那只枯瘦而布满皱纹的手，轻轻按在了他的眉心之上。

那手掌温热而干燥，手感粗糙而苍老，却带着一股说不出的力量。那力量不是杀意，也不是威压，而是一种难以言喻的……温暖？

下一刻，一股温热的气流从天璇真人的掌心涌入云泽的识海。

那气流极为柔和，并不像方才的神识威压那般霸道，反而像是一道春风，轻轻拂过他的识海，所过之处，如沐春光。那感觉太过舒适，舒适到云泽几乎要沉沉睡去。

与此同时，一道苍老而温和的声音在他脑海中清晰地响起——

"孩子……"

仅仅两个字，却让云泽的鼻根猛地一酸。

那语气太过温和，温和得不像是从一个仇人嘴里说出来的。那是一种长辈对晚辈说话的口吻，带着几分怜惜，几分慈爱，还有几分说不清道不明的……心疼？

"你若想知道真相，便去昆仑虚的'证道遗迹'。那里有你需要的一切答案。"

"但要小心……在你足够强大之前，不要让任何人知道你的真正身份。否则……你将死无葬身之地。"

话音落下，那股温热的气流也随之消散。

天璇真人收回手掌，退后一步。

他看着云泽的神情微微一变——那眼神中的温和与怜惜已经消失不见，取而代之的，是一片深不见底的淡漠。

"今日之事，不许对任何人提起。"

他的声音恢复了方才的平淡与威严，仿佛方才的一切都只是云泽的幻觉。

"去吧。"

云泽如梦初醒，呆呆地站在原地。

方才那是……真的吗？

天璇真人称他为"孩子"？他让自己去"证道遗迹"寻找真相？更重要的是，天璇真人的语气，分明是在……保护他？

可若他真是云游子前辈口中的"仇人"，为何要这样做？

难道……云游子前辈的仇人，并非眼前这位天璇真人？

还是说，这其中另有隐情？

无数个问号在云泽脑海中翻涌，让他一时间心乱如麻。

他张了张嘴，想要问些什么，却见天璇真人已经转过身去，望向窗外的夜色，不再看他。

那背影佝偻而苍老，在摇曳的灯火中显得格外孤寂。

"真人……"

"去吧。"老人的声音再次响起，不带丝毫感情，"有些人，有些事，知道得越少，活得越久。"

云泽沉默了片刻。

他深深地向老人行了一礼，然后转身向殿门走去。

走到门口时，他忍不住回头望了一眼。

天璇真人依旧背对着他，佝偻的身影映在窗纸上，像是一截燃尽了灯芯的残烛。

那一瞬间，云泽忽然觉得这位老人很孤独。

孤独得像是独自在黑暗中行走了千年的旅人，没有同伴，没有归处，只有无尽的黑暗与他为邻。

他轻轻推开门，走了出去。

夜风迎面扑来，带着几分深秋的凉意，吹得他打了个寒噤。

他站在偏殿门外，深吸了一口气，又缓缓吐出。只觉胸腔中的那股郁结稍稍松动了些，但心中的困惑却有增无减。

天璇真人的态度太过诡异。

他分明知道云游子的仇人就是自己——若云游子前辈所言属实的话。但，他却不但没有对云泽出手，反而以一种极为隐秘的方式，传递了那样的信息。

这其中，究竟藏着什么秘密？

还有那句"不要让任何人知道你的真正身份"——

什么身份？道君转世？还是其他什么他不知道的东西？

云泽不知道。

但有一件事他可以确定——自己身上的秘密，恐怕远比他想象的要复杂得多。

云游子的仇人，天璇真人……似乎并不是表面上那么简单。

而他自己，作为道君转世，作为"遁去的一"，已经被卷入了这场跨越九纪元的巨大阴谋之中。

窗外，夜色深沉，星月无光。

云泽独自走出偏殿，沿着天璇峰的石阶一步步向下走去。

脚下的石阶冰凉而坚硬，每一步都发出轻微的声响，在寂静的夜色中显得格外分明。夜风吹在他脸上，带着几分寒意，却也让他的头脑渐渐冷静了下来。

他深吸一口气，将方才的一切压在心底。

无论天璇真人是敌是友，无论这其中有多少隐秘，他现在唯一能做的，就是不断地变强。

只有足够强大，才能揭开所有的真相。

只有足够强大，才能保护自己在意的人。

只有足够强大，才能在这场跨越纪元的博弈中，不被碾成齑粉。

"昆仑虚……证道遗迹……"

云泽轻声呢喃，将这几个字牢牢刻在心里。

昆仑虚，那是传说中的万山之祖，位于中州极西之地。据说那里是上古仙人遗留的遗迹，蕴含着无数远古的秘密与机缘。无数修士梦寐以求，想要踏入那片神秘之地，却终生无缘得见。

而天璇真人，却让他去那里寻找真相。

这说明什么？说明天璇真人知道一些他不知道的事情。说明那个"证道遗迹"中，藏着他身世之谜的关键线索。

可问题是，天璇真人为什么要告诉他这些？

若他真是云游子前辈口中的"仇人"，他应该恨不得将自己碎尸万段才对。可他非但没有动手，反而在暗中指点他方向，这其中必有蹊跷。

云泽越想越是困惑。

他想起天璇真人看着他的眼神——那眼神中没有敌意，没有杀气，反而有一种说不出的……怀念？

"你与我一位故人长得很像。"

这句话在他脑海中反复回响。

故人？什么故人？与自己长得很像的人，会是谁？

他想起自己的身世。他从小在青云镇长大，由云爷爷抚养成人。对于自己的父母，他一无所知。云爷爷只说他是被人遗弃在镇子后山的婴儿，被云爷爷捡回来抚养长大。

难道说，他的身世，竟与天璇真人有某种关联？

无数的念头在云泽脑海中翻涌，让他一时间心乱如麻。

"罢了，想再多也没用。"

他摇摇头，将这些杂念暂时压下。

无论真相如何，他现在唯一能做的，就是不断地变强。只有足够强大，才有资格去追寻答案，才有资格去揭开所有的秘密。

总有一天，他会踏上昆仑虚那片神秘的土地，找到"证道遗迹"，揭开所有的真相。

但不是现在。

现在，他还太弱了。

前路漫漫，荆棘遍布。

但云泽的眼中，没有恐惧，只有坚定。

无论前方有什么在等待着他，他都要一步步走下去。

直到揭开所有的真相。

直到站在这个世界的巅峰。

夜风呼啸而过，吹起他的衣袂和发丝。

天璇峰上，一盏孤灯在夜色中摇曳。

灯下，天璇真人静静伫立，望着那道渐渐远去的单薄身影，浑浊的眼中，有泪光一闪而过。

"……像，太像了。"

他喃喃低语，声音沙哑而苍老。

"你若泉下有知，可会怪我？"

无人应答。

只有夜风呜咽，像是某个古老灵魂的叹息。
