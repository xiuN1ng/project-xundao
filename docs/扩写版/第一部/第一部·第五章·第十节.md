# 第一部·第五章·第十节：真相大白

## 一、背刺

"找死！"

那声音冰冷而充满杀意，云无咎只来得及侧身一闪，那剑光便擦着他的肩膀掠过，带起一蓬血雾！

他猛然回头，只见殿堂的入口处，站着一个身穿紫袍的身影。

那身影面容阴沉，目光如电，正是——

天璇真人！

"没想到，你还真能通过三关。"天璇真人缓步走入殿堂，眼中闪烁着贪婪的光芒，"星辰子的传承，还有这块天道图碎片……今日，都归本座了！"

云无咎抹去嘴角的血迹，目光冷冽："天璇真人……你果然来了。"

"本座等这一天，已经等了二十年了。"天璇真人一步步逼近，浑身的化神境威压尽数释放，如同一座大山压向云无咎，"二十年前，云游子那小子坏了本座的大事。今日，本座总算能弥补这个遗憾了。"

"二十年前……"云无咎直视着他，"你陷害云游子前辈，就是为了这块天道图碎片？"

"陷害？"天璇真人冷笑一声，"云游子那个冥顽不灵的老顽固，明明有着绝世天赋，却偏偏要守着那些该死的规矩，不肯与本座合作。他死有余辜！"

"你……！"云无咎的拳头紧紧攥起。

"怎么？想为云游子报仇？"天璇真人的嘴角勾起一丝嘲弄，"就凭你一个筑基境的小辈，也配？"

他抬起手，一道紫色的剑光在掌心凝聚，散发着令人心悸的威能。

"小子，你以为本座为什么要收你为徒？为什么要让你成长到今天？"

他一步步逼近，声音透着彻骨的寒意："就是为了今天！为了让你替本座打开封印，为了让本座坐收渔翁之利！"

"你以为你很聪明？"天璇真人嗤笑，"你不过是本座养的一颗棋子罢了！"

云无咎直视着他，目光中没有恐惧，只有冷静。

"棋子？"他的嘴角微微上扬，"天璇真人，你太小看我了。"

"更何况——"他缓缓拔出游子剑，"你以为我没有防备你吗？"

话音未落，殿堂之外忽然传来两声长啸！

紧接着，两道身影从殿堂入口处掠入，落在云无咎身旁。

林惊羽！

苏晴雨！

"林惊羽？苏晴雨？"天璇真人的脸色微变，"你们两个……竟然背叛本座？！"

林惊羽冷冷地看着他："天璇真人，二十年前你陷害云游子师兄的真相，我们已经知道了。"

苏晴雨也开口道："云游子师兄是你的同门师兄，你为了天道图碎片，竟然勾结外人，设计陷害他，导致他含冤而死。这样的人，也配做天道宗的长老？！"

天璇真人的脸色彻底阴沉下来。

"好……好……好……"他连说三个好字，声音透着彻骨的寒意，"没想到本座养了二十年的棋子，如今也要反噬了。"

他的目光在三人身上扫过，最终定格在云无咎身上："你以为凭你们三个，就能拦得住本座？"

话音落下，一股滔天的气势从天璇真人身上爆发出来。

那是化神境巅峰的威压！

在这股威压之下，林惊羽和苏晴雨的脸色齐齐一变。即便是金丹境的苏晴雨，在这股威压面前也感受到了巨大的压力。

"无咎小子，你疯了吗！"剑灵在脑海中急切地喊道，"他是化神境！你们三个加起来也不是他的对手！"

云无咎没有回答，只是缓缓举起了手中的游子剑。

"剑灵前辈。"他轻声说道，"当年云游子是一个人对抗他们一群，对吗？"

剑灵沉默了。

"既然他能做到——"云无咎的眼中闪过一抹决然之色，"那今天，就让我来替他，完成这场未竟之战！"

## 二、殊死一搏

"游子第七式——归去来兮！"

云无咎体内的真气疯狂涌动，道君印记同时绽放出耀眼的光芒。

游子剑在手中剧烈颤抖，仿佛感应到了主人心中那股决绝的战意。

剑光冲天而起，化作一道银色的长虹，直取天璇真人！

天璇真人冷哼一声，抬手便是一道紫色的剑光迎了上去。

两股力量在空中碰撞，发出震耳欲聋的轰鸣。整个殿堂都在剧烈颤抖，仿佛要崩塌一般。

然而，就在剑光碰撞的瞬间，云无咎忽然变换了招式。

"游子第八式——此心安处是吾乡！"

这一招，是云无咎从未施展过的招式。

三年前，云游子的残念曾告诉他，游子九式共有九招，但最后两招——第八式和第九式，是云游子在临死前才领悟出来的。这两招威力极大，但代价也极大。每施展一次，都会消耗大量的生命力。

此刻，云无咎毫不犹豫地施展出了第八式。

剑光化作无数道银色的丝线，从四面八方向天璇真人包裹而去。那丝线看似轻柔，却蕴含着毁天灭地的力量。

天璇真人的脸色终于变了。

"这是……游子九式的第八式？！"他惊呼出声，"你怎么会——"

话未说完，那银色丝线已经缠绕上了他的身体，顿时，他的动作变得迟缓起来。

"林师兄！苏师姐！"云无咎怒吼，"动手！"

林惊羽与苏晴雨对视一眼，毫不犹豫地拔剑出击。

"天外飞仙剑诀——一剑惊鸿！"

林惊羽的剑光如同一道流星，划破长空，直取天璇真人的要害。

与此同时，苏晴雨也动了。

"碧波千重浪！"

她的法术化作一道道碧绿色的波浪，向天璇真人席卷而去。

三人的攻击同时到达！

天璇真人发出一声怒吼，体内的真气疯狂涌动，强行挣脱了银色丝线的束缚。

但他还是慢了一步。

林惊羽的剑光在他的肩膀上留下了一道血痕。苏晴雨的法术击中了他的后背，让他踉跄了一下。

"可恶！"天璇真人怒喝一声，抬手便是三道剑光分别向三人攻去。

云无咎急忙举剑格挡，只觉得一股巨大的力量传来，他的身体被震得连退数步，虎口发麻。

"林师兄！苏师姐！"云无咎大喊，"我来拖住他，你们快去拿碎片！"

"什么！"林惊羽脸色大变，"你一个人怎么可能——"

"快去！"云无咎怒吼，"再不走就来不及了！"

林惊羽与苏晴雨对视一眼，眼中闪过挣扎之色。

但他们知道，云无咎说的是对的。以他们的实力，根本无法与化神境的天璇真人抗衡。若是留在这里，只会拖累云无咎。

"云师弟，你一定要活着！"苏晴雨咬牙说道。

说罢，她与林惊羽转身向光幕之后冲去。

"想走？"天璇真人大怒，想要追上去。

然而云无咎却挡在了他面前。

"你的对手是我。"

天璇真人的脸色阴沉到了极点："小子，你以为凭你一个筑基境，能拦得住本座多久？"

"不试试怎么知道？"云无咎的嘴角微微上扬。

天璇真人不再废话，直接全力出手。

紫色的剑光如同狂风暴雨，向云无咎倾泻而去。

云无咎咬紧牙关，将游子九式施展到了极致，拼命抵挡着天璇真人的攻击。

一招、两招、三招……

每一次碰撞，云无咎都会受到不同程度的伤害。他的身上已经布满了伤口，鲜血染红了他的衣衫。

但他依然没有倒下。

"第十招！第十五招！第二十招！"

云无咎不知道自己挡了多少招。他的意识已经开始模糊，身体也快要达到极限。

但他依然在坚持。

因为他知道，每多坚持一秒，林惊羽和苏晴雨就多一秒钟的时间。

"第二十五招！"

终于，云无咎再也支撑不住，被天璇真人一剑击飞，重重地撞在了墙壁上。

"无咎小子！"剑灵急切地喊道。

云无咎挣扎着想要站起来，但他的身体已经不听使唤了。

天璇真人缓步向他走来，眼中闪烁着冰冷的杀意。

"小子，你很有骨气。"他冷冷地说道，"但骨气不能当饭吃。今日本座就送你上路！"

他举起手中的剑，正要刺下——

忽然，一道金光从天空中落下！

那金光蕴含着极其强大的力量，直接将天璇真人逼退了数步！

"什么！"天璇真人脸色大变。

## 三、尘埃落定

只见光幕之后，林惊羽和苏晴雨的身影重新出现。

在他们手中，各自握着一块散发着金色光芒的玉片。

那正是——天道图碎片！

但只有一块。

"天璇真人！"林惊羽大声说道，"我们已经将碎片带出来了！你若是杀了云师弟，这块碎片……我们就当场毁掉！"

天璇真人的脸色变得铁青。

他万万没想到，这两个金丹境的小辈，竟然敢威胁他。

但天道图碎片对他实在太重要了。他不能冒这个险。

"好……好……"天璇真人咬牙切齿，"今日之事，本座记下了！"

说罢，他的身影化作一道紫光，消失在了殿堂之中。

云无咎瘫坐在地上，看着天璇真人离去的方向，终于松了一口气。

"云师弟！"林惊羽和苏晴雨急忙跑过来，将他扶起。

"无妨……"云无咎摇摇头，"只是受了些伤，养几日就好。"

他的目光落在林惊羽手中的天道图碎片上，眼中闪过一丝欣慰。

"拿到了……"他喃喃道，"终于拿到了……"

"无咎小子。"剑灵的声音在脑海中响起，透着几分激动，"你做到了。"

云无咎微微一笑，随即眼前一黑，昏了过去。

## 四、余波未平

三日后。

昆仑秘境试炼结束。

这一次的试炼，三十六名弟子进入秘境，最终活着出来的只有二十三人。剩下的十三人，永远地留在了那片神秘的土地上。

而云无咎，因为成功通过星辰阁遗迹的三关考验，获得了星辰阁的道统传承，成为了这次试炼中最大的赢家。

更重要的是，他获得了天道图碎片的第一块！

虽然这块碎片在天璇真人手中还有数块，但云无咎相信，随着时间的推移，他一定能将所有的碎片都收集齐全。

在离开昆仑秘境的那一天，林惊羽和苏晴雨一起来到了云无咎面前。

"云师弟。"林惊羽说道，"这次试炼，若不是你，我们恐怕都活不下来。"

云无咎摇摇头："若不是你们在关键时刻出手相助，我也不可能拿到碎片。这份恩情，我记在心里。"

苏晴雨微微一笑："云师弟不必客气。说起来，我们也算是并肩作战的战友了。日后若有什么需要帮忙的，尽管开口。"

云无咎点点头："好。一言为定。"

三人相视一笑，默契尽在不言中。

天璇真人……我一定会找上门的。

云无咎望向天璇峰的方向，眼中闪过一抹坚定的光芒。

等着吧。天璇真人。总有一天，我会让天下人知道，二十年前那场背叛的真相！
