# 第一部·第五章·第一节：内门试炼

## 一、晨钟消息

青云峰后山，晨雾未散。

一道修长的身影盘膝坐在崖边的一块青石之上，周身萦绕着淡淡的灵光。晨风拂过，吹动他垂落肩头的黑发，却吹不散他眉宇间那股沉稳如渊的气息。

三年。

云无咎缓缓睁开眼睛，眸中精光一闪即逝，随即恢复成深邃平静的墨色。他低头看了看自己的手掌——那双手比三年前宽厚了许多，虎口处布满厚厚的茧子，指节粗大而有力，是常年握剑磨出来的痕迹。掌心隐约可见一道淡淡的纹路，那是他体内真气运转时留下的印记。

从当初那个连灵气的流动都感知不到的懵懂少年，到如今筑基境巅峰、半步金丹的修为——这三年来，他经历了太多。

天道宗青云峰后山这片崖坪，是他每日打坐修炼的地方。这里灵气浓郁，地势又偏僻，少有人来打扰。他喜欢这里的清净，喜欢晨风拂面时那股清冽的感觉，更喜欢独自面对天边云海翻涌时，心中涌起的那种辽阔。

三年时间，说长不长，说短不短。

放在寻常修士身上，从淬体境突破到筑基境巅峰，至少也要二三十年的苦功。而他只用了三年——这个速度，足以让整个天道宗外门为之震动。

但云无咎自己知道，这三年他经历了什么。

每一个深夜，当其他弟子早已酣睡之际，他仍在灯下苦读《云游心法》的口诀，一字一句地参悟其中奥义。每一次宗门任务，他都主动请缨前往最危险的地方，在生死边缘磨砺自己的意志和战力。每一次与人交手之后，无论胜败，他都会反复复盘，寻找自己的不足之处。

除了《云游心法》的修炼之外，他还在云游子剑灵的指点下，将游子九式剑诀修炼到了第七式。七式剑招，已是当年云游子巅峰时期的水准。这个进度让剑灵都啧啧称奇，说他"悟性虽非绝顶，但毅力之坚，实乃老夫生平仅见"。

而更让他意外的是，随着修为的提升，他体内的道君印记也开始微微发热。那枚印记藏在他胸口的位置，平日里毫无动静，但每当他运转真气时，便会隐隐散发出温热的气息，仿佛在回应着他的成长。

"无咎小子。"剑灵的声音在脑海中响起，带着几分慵懒，"今日怎么起这么早？太阳还没出来呢。"

"习惯了。"云无咎轻声回答，"睡不着。"

"睡不着？"剑灵似乎感知到了什么，"是因为那件事？"

云无咎没有说话，但他的沉默便是最好的回答。

那件事——三年前昆仑秘境中，云游子的记忆碎片让他看到了当年背叛的真相。天璇真人，那个在宗门内德高望重的内门长老，那个在所有人眼中都是正道楷模的师父级人物，竟然是当年陷害云游子的罪魁祸首。

而他自己，因为修炼了云游子留下的《云游心法》，因为拥有云游子的游子剑，因为与云游子有着说不清道不明的因果联系，无意中成为了天璇真人眼中的刺。

这三年来，他无时无刻不在小心翼翼地隐藏着自己的实力。他知道，以他现在的修为，根本无法与化神境的天璇真人抗衡。若是过早暴露，等待他的只有一个下场——死。

所以他选择隐忍。

在外人面前，他是一个天赋不错、但还算守规矩的外门弟子。他不争不抢，不引人注目，该表现的时候表现，该藏拙的时候藏拙。三年来，他成功地让所有人认为他只是一个"还不错"的外门弟子，而非一个足以威胁到某些人的存在。

但他知道，这一天迟早会来。

因为他修炼的《云游心法》太过特殊。只要他继续修炼下去，他的实力迟早会达到一个让天璇真人无法忽视的地步。而到那时，隐忍便不再是一种选择。

"无咎小子。"剑灵的声音打断了他的思绪，"有件事我觉得有必要提醒你。"

"什么？"

"你体内的道君印记……"剑灵的语气变得凝重起来，"它成长的速度比我预想的要快得多。按理说，以你现在的修为，那枚印记不应该有任何反应。但它不仅有了反应，而且反应越来越强烈。"

云无咎心中一凛："这意味着什么？"

"意味着你的前世正在苏醒。"剑灵沉默了片刻，"或者说，意味着那枚印记正在与你的灵魂逐渐融合。这个过程可能会加速你实力的提升，但同时也可能带来一些……不可预知的后果。"

"什么后果？"

"我不确定。"剑灵的声音透着几分无奈，"道君转世这种事，老夫也是头一次遇见。你是九纪元轮回以来唯一一个成功转世的道君，没有任何经验可以借鉴。唯一能告诉你的是——小心行事，不要过分依赖那枚印记的力量。"

云无咎默默地点了点头。

就在这时，后山的方向忽然传来一阵清脆的钟声。

那钟声悠扬而绵长，在群山之间回荡，惊起了无数栖息的飞鸟。

"集合钟？"云无咎微微皱眉，"这个时辰敲集合钟……是有什么大事发生了？"

他站起身来，拍了拍衣袍上并不存在的灰尘，然后沿着崖边的小径向后山外门的方向走去。

## 二、钟声如雨

集合钟一响，外门弟子们纷纷从各自的住处走出，向青云峰下的广场汇集。

云无咎随着人流来到广场时，那里已经聚集了不少人。放眼望去，黑压压一片，少说也有两三百号弟子。他们或站或坐，三三两两地交头接耳，脸上都带着几分好奇和兴奋。

"听说了吗？这次是内门试炼的消息！"

"内门试炼？不是十年才举行一次吗？上次举行还是十年前呢！"

"是啊，我入门才三年，还没赶上一回。上次试炼我还太小，连参加资格都没有。"

"这次可是十年一遇的盛会！据说内门极为重视，派了好几位长老来主持！"

云无咎站在人群边缘，静静地听着周围的议论声。

内门试炼。

这四个字对于天道宗的外门弟子而言，意味着一次改变命运的机会。

十年一次，是外门弟子叩开内门大门的唯一通道。一旦通过，便能脱离外门弟子身份，成为天道宗真正的核心一员，享用内门的上等修炼资源，甚至有机会拜入化神境强者门下。

而若是错过这次机会，便只能再等十年。十年对于凡人而言或许不算什么，但对于修士而言，却是至关重要的时间。修炼一途，如逆水行舟，不进则退。十年的光阴，足以让许多人的青春和天赋付诸东流。

"云师弟！"

一道熟悉的声音从人群中传来。

云无咎循声望去，只见一个身穿青色长袍的青年正朝他走来。那青年约莫二十出头，面容清秀，气质温和，腰间悬着一柄普通的铁剑，正是与他关系不错的云帆师兄。

"云帆师兄。"云无咎拱手行礼。

云帆快步走到他身旁，脸上带着笑意："我就知道你会来。怎么样，听说了吗？内门试炼的消息！"

"略有耳闻。"云无咎点点头，"具体情况如何？"

"我也是刚得到消息。"云帆压低了声音，凑近他耳边说道，"据说是今天早上内门那边传出来的——这次试炼的名单已经定了，总共有三十六个名额，咱们外门占了十二个。"

"十二个？"云无咎微微挑眉，"外门弟子上千人，只取十二个？"

"是啊，门槛极高。"云帆苦笑着摇了摇头，"据说要求是筑基境以上才有资格参加，而且还要经过层层筛选，最终由内门长老亲自点名确认。整个外门符合条件的不超过五十人，能被选中参加试炼的，无一不是其中的佼佼者。"

他说着，忽然拍了拍云无咎的肩膀，眼中闪过一丝羡慕："不过云师弟你可不用担心。以你的实力，这次试炼必定榜上有名！"

云无咎没有接话，只是淡淡地笑了笑。

就在这时，广场前方的高台之上，忽然有几道身影凭空出现。

那几人身穿各色长袍，气息深不可测，往那里一站，便有一股无形的威压笼罩全场。人群中的议论声顿时小了下去，所有人都下意识地抬起头，望向那几道身影。

"是内门的长老！"云帆的声音透着几分紧张，"竟然一次性来了三位！"

云无咎的目光从那几道身影上一一扫过，最终停在了居中那位身穿紫色长袍的中年男子身上。

那男子面容威严，目光如电，浑身上下散发着一股令人心悸的气息——那是化神境强者独有的威压，让人在他面前不由自主地生出敬畏之心。

天璇真人。

这个名字在云无咎脑海中闪过，如同一块冰落入沸水之中，激起一阵难以遏制的怒意。但他的面色依然平静如水，只是垂在身侧的手掌微微攥紧了一瞬，随即又松开。

"诸位弟子。"天璇真人的声音响起，不高不低，却清晰地传入每个人的耳中，"今日召集诸位，是有一件大事宣布。"

他的目光从人群中扫过，最终落在了某个方向——云无咎所在的方向。

那一瞬间，云无咎感觉到一股无形的压力笼罩了自己。那压力并不强烈，却如同一座大山压在心头，让他几乎喘不过气来。

这是试探。

云无咎咬紧牙关，体内的真气下意识地想要运转抵抗，但被他强行压制了下去。他只是微微低下头，做出一副恭敬聆听的模样，将那股压力承受了下来。

片刻之后，天璇真人收回目光，嘴角似乎微微上扬了一下，随即恢复如常。

"三年一度的内门试炼，终于要如期举行了。"他的声音回荡在广场上空，"这一次，试炼的地点是——昆仑秘境。"

此言一出，全场哗然。

"昆仑秘境！"

"竟然是昆仑秘境！"

"那可是九死一生的禁地啊！"

人群中爆发出一阵骚动，有人兴奋，有人担忧，有人面露惧色。

昆仑秘境，是上古年间留下的一处神秘空间，据传与天界有着千丝万缕的联系。秘境中灵气浓郁，生长着无数珍稀灵药，也栖息着强大至极的上古妖兽。是天道宗用来磨砺弟子的绝佳场所，但同时也是一处九死一生的禁地。

"肃静！"天璇真人抬起手，压下了众人的喧哗，"昆仑秘境十年开启一次，是宗门给予你们的机缘，也是考验。望诸位珍重此机会，在秘境中努力修炼，为宗门争光。"

"此次试炼，由本座亲自主持。"他的声音透着不容置疑的威严，"试炼时间为期一个月。一个月后，活着出来的弟子，皆可获得内门候选资格。"

"另外——"他话锋一转，"此次试炼设置了一道额外关卡——星辰阁遗迹。那是昆仑秘境的核心禁地，里面藏有上古年间留下的珍贵传承。"

他的目光再次扫过人群，最终定格在云无咎身上。

"敢于挑战者，可自行进入。但本座要提醒诸位——那里面的危险，远超你们的想象。若是没有足够的实力，切勿轻易涉险。"

云无咎与天璇真人的目光在空中交汇。

那一瞬间，他分明看到了对方眼中闪过的一丝意味深长的光芒。

是警告？是威胁？还是……另有图谋？

云无咎心中警铃大作。

他太了解天璇真人了。三年前那场背叛的真相还历历在目，这个老贼绝不可能无缘无故地说这番话。他提起星辰阁遗迹，必然有他的目的。

"师弟？"云帆见他神色有异，低声唤道，"你怎么了？"

"没什么。"云无咎收回目光，神色恢复如常，"只是在想，这趟昆仑秘境，怕是不太平。"

云帆没有听清他的话，只是拍了拍他的肩膀："放心吧云师弟，以你的实力，只要小心行事，一定能平安回来的。"

云无咎没有回答。

他抬起头，望向天璇峰的方向，眼中闪过一抹深沉的光芒。

天璇真人……你到底在打什么算盘？

## 三、暗流涌动

晨会结束后，弟子们三三两两地散去，各自回去准备试炼的事宜。

云无咎没有急着离开，而是独自一人回到了后山的崖坪上。

他在那块青石上坐下，闭上眼睛，神识沉入体内，试图与剑灵交流。

"剑灵前辈。"他在心中呼唤。

"老夫在。"剑灵的声音立刻响起，"你也在想天璇老贼刚才那番话？"

"嗯。"云无咎点点头，"他说得太刻意了。星辰阁遗迹是昆仑秘境的核心禁地，按照惯例，这类禁地通常不会主动向弟子提及。他不但提了，还特意强调了'敢于挑战者可自行进入'——这分明是在引诱某些人上钩。"

"你怀疑他在针对你？"剑灵问道。

"不只是怀疑。"云无咎的眼中闪过一抹冷意，"这三年来，我一直在隐藏实力，从来没有在他面前表现出任何异常。但他今天看我的那一眼……"

他没有说下去，但剑灵已经明白了他的意思。

"你是说，他已经盯上你了？"

"或许从一开始，他就盯上我了。"云无咎的声音透着几分苦涩，"毕竟，我修炼的是《云游心法》，用的是游子剑。这两样东西都与他当年陷害云游子有关。只要我还在天道宗一天，他就一天不会放下戒心。"

剑灵沉默了片刻，才缓缓开口："无咎小子，你说的没错。当年云游子的事，天璇老贼一直是主谋。他不会允许任何威胁存在。而你……恰好就是那个威胁。"

"但他为什么不直接对我动手？"云无咎问道，"以他的实力，要杀我不过是易如反掌。"

"因为他还有顾虑。"剑灵说道，"你现在是天道宗的正式外门弟子，若是无缘无故被杀，必然会引起宗门的调查。更重要的是……"

"更重要的是什么？"

"更重要的是，他需要你。"剑灵的声音透着几分讥讽，"你修炼了《云游心法》，而《云游心法》是打开星辰阁遗迹封印的唯一钥匙。没有你，他进不了星辰阁。"

云无咎心中一震。

他忽然想起了三年前，在昆仑秘境中看到的那段记忆碎片——当年云游子发现天道图碎片之后，为了不让碎片落入天璇真人之手，用自己的生命为代价，将那块碎片封印在了星辰阁遗迹最深处。

而那道封印，只有修炼了《云游心法》的人才能打开！

"你是说……"云无咎的声音变得低沉，"天璇真人让我活着，是为了利用我打开封印？"

"没错。"剑灵说道，"以他的实力，那块碎片早就该取走了。但他取不走——因为那块碎片被云游子的残念封印着。只有修炼了《云游心法》的人，才能打开那道封印。"

"所以他才在星辰阁遗迹这件事上做文章。"云无咎喃喃道，"他是在引诱我主动进入遗迹。"

"对。"剑灵说道，"这老贼打得一手好算盘——让你主动走进去，然后在你打开封印的那一刻，再将你杀死。这样一来，他既能拿到碎片，又能神不知鬼不觉地除掉你这个隐患。"

云无咎的拳头紧紧攥起。

原来如此。

天璇真人让他活着，让他成长，让他修炼《云游心法》……不是为了保护他，而是为了有一天——利用他打开那道封印，然后将他弃如敝履！

"混蛋！"他在心中低骂一声。

"无咎小子，你打算怎么办？"剑灵问道。

云无咎沉默了片刻，忽然轻轻笑了。

那笑容里没有半分温度，冷冽得如同寒冬腊月的冰凌。

"既然他想利用我，那就让他利用。"他的眼中闪过一抹锐利的光芒，"但能不能利用得动，还要看他有没有这个本事。"

"你打算将计就计？"

"为什么不呢？"云无咎站起身来，望向远方天璇峰的方向，"星辰阁遗迹中，有云游子前辈留下的更多记忆碎片，有关于天道图碎片的秘密——更重要的是，有他当年留下的真相。这些东西，我必须拿到手。"

"但你可有想过，天璇老贼不会让你轻易得逞？"

"想过。"云无咎的声音平静而坚定，"但有些事，明知不可为而为之。这不是鲁莽，是担当。"

剑灵沉默了很长时间。

"好。"最终，剑灵只吐出了一个字，却重如千钧，"有骨气。当年云游子若能有你这番心智，或许也不至于落到那般下场。"

"剑灵前辈，"云无咎忽然问道，"你恨天璇真人吗？"

剑灵沉默了。

那沉默里藏着太多东西——有愤怒，有悲伤，有不甘，有遗憾。二十年的囚禁，二十年的等待，二十年的孤独守候，都化作了一声沉重的叹息。

"恨。"最终，剑灵只吐出了一个字。

"那就够了。"云无咎望向远方，"这笔账，总要有人去算。"

他站在崖边，晨风拂过他的衣袍，将他的身影衬托得格外孤傲。

天璇真人……你等着。

这笔账，我云无咎记下了。
