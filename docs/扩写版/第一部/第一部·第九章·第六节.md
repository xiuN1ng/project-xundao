# 第一部·第九章·第六节：故人相见

## 一、山门惊变

从昆仑虚到天道宗，云泽御剑飞行了整整一日。

当夜幕降临时，那座熟悉的山门终于出现在他的视野之中。远远望去，天道宗的山门在夜色中灯火通明，无数盏长明灯沿着石阶一路向上延伸，如同一条璀璨的长河，将整座山峰点缀得宛如仙境。

但云泽的目光却没有落在那些灯火上。

他的目光，落在了山门之外那个孤零零的身影上。

那人身着青色道袍，腰悬长剑，正负手立于山门外的那棵老槐树下。夜风吹动他的衣袂，将他的长发吹得猎猎作响，却吹不动他脸上那凝重的神色。

是林惊羽。

仿佛是感应到了什么，林惊羽忽然抬起头，望向远方的夜空。

下一刻，他的身体猛地一震。

"云师弟！"他的声音从山风中传来，带着难以置信的惊喜，"你——你怎么——"

他的话说到一半，忽然戛然而止。

因为他看清了。

看清了云泽眉心处那道金色的印记。

看清了云泽周身那股若有若无的金色光晕。

看清了云泽整个人的气质，在这一夜间发生了翻天覆地的变化。

那种变化很难用言语形容——不是外貌的改变，而是某种更深层的东西。云泽站在那里，与昨日离开时并无二致，但他给人的感觉却完全不同了。

若是说从前的云泽是一柄藏在鞘中的剑，锋芒内敛，不显山露水；那么此刻的他，便是剑已出鞘，寒光毕露，让人不敢直视。

那是炼虚境强者才有的气势。

"你突破了？！"林惊羽的声音在山风中回荡，带着压抑不住的震惊，"你突破了化神境？"

云泽微微一笑，落在他身前。

"侥幸。"他说，"有些际遇。"

"际遇？"林惊羽的目光依然停留在他的眉心处，那里道君印记正散发着若有若无的金光，"你这际遇，也太……"

他忽然说不下去了。

因为他想起了昆仑虚之战后，饕餮对他说过的那番话。

"道君转世，真正的力量只有在进入证道遗迹之后才能觉醒。"

"若他能通过遗迹中的三道考验，便可彻底继承道君的传承。"

"若他不能——"

饕餮没有说完那句话，但林惊羽明白其中的意思。

如今看来，云泽不仅通过了考验，还获得了他意想不到的收获。

"你……真的做到了。"林惊羽望着云泽，眼中闪过复杂的神色——有惊喜，有欣慰，也有几分说不清道不明的感慨。

"做到了。"云泽点头，"但这只是开始。"

他抬起头，望向山门深处那片灯火辉煌的方向："天玄子的事……宗门里知道多少？"

林惊羽的神色一凝。

"说来话长。"他压低声音，"进去再说——苏师姐一直在等你。"

## 二、情深不语

青云峰洞府门前，苏婉晴已经等在那里。

夜色中，她的身影单薄而纤细，一袭月白色的长裙在夜风中轻轻飘动。她的长发披散在肩头，面容在月光下显得格外清丽，却又带着一丝藏不住的疲惫。

当她看到云泽的那一刻，整个人都僵住了。

她张了张嘴，似乎想说什么，却又什么都说不出来。她向前走了两步，却又骤然停下，仿佛不敢相信眼前的一切是真的。

"无咎……"

她的声音很轻，轻得像是怕惊碎了一个梦。

云泽望着她，心中忽然涌起一阵难以言喻的酸涩。

不过短短数日，这个曾经在昆仑秘境中与他并肩作战的女子，却像是苍老了好几岁。她的眼眶有些发红，眼下有淡淡的青影，显然已经很久没有睡过一个安稳觉了。

她一直在担心他。

一直在等他。

"我回来了。"云泽走上前，轻声道。

苏婉晴望着他，眼眶渐渐泛红。

"你知不知道……"她的声音开始颤抖，"你知不知道我有多担心你……"

"你留下一张字条就走了，连句话都不肯多说……你知道我等了多久吗……"

"我以为……我以为你出了什么事……我每天晚上都睡不着觉，一闭上眼睛就做噩梦……"

她的声音越来越哽咽，泪水终于夺眶而出。

云泽沉默了。

他知道自己这样做很自私。让那些关心他的人担心，从来都不是他的本意。但彼时彼刻，他别无选择——证道遗迹的事情不能泄露，他必须独自前往。

可此刻看到苏婉晴泪流满面的样子，他才真正意识到，自己的离开给她带来了多大的伤害。

"对不起。"他轻声道。

苏婉晴摇了摇头，抬起头，泪眼朦胧地望着他："我不要你说对不起……我只要你平安……"

"只要你平安就好……"

云泽望着她，心中忽然涌起一股从未有过的情绪。

那情绪来得如此强烈，如此汹涌，以至于他自己都有些措手不及。

他忽然伸出手，将苏婉晴轻轻揽入怀中。

苏婉晴的身体僵了一下，随即软了下来。

她将脸埋在他的胸口，泪水浸湿了他的衣襟。她的肩膀在微微颤抖，却再也没有发出声音——只是静静地伏在他怀中，像是找到了这世上最安全的港湾。

云泽没有说话。

他只是轻轻抬起手，放在她的后背上，一下一下地轻抚着。

夜风吹过两人身旁，带来远处山林间的草木清香。月光如水，静静地洒落在两人身上，将他们的身影拉得很长很长。

良久，苏婉晴终于从他的怀中退了出来。

她抬起头，用袖子擦了擦眼角的泪痕，脸上浮现一丝羞赧的红晕。

"我……我失态了……"她低着头，不敢看他。

"没有。"云泽摇了摇头，嘴角浮现一丝淡淡的笑意，"你很好。"

"一直都很好。"

苏婉晴的脸更红了。

一旁的林惊羽看着这一幕，嘴角抽搐了一下，默默地转过身去，假装在看远处的夜景。

## 三、风起云涌

片刻之后，三人进入洞府，开始商议正事。

"宗门里的情况，比你想象的更复杂。"林惊羽开门见山，"天璇真人叛宗的消息传出后，宗内高层分成两派——一派认为天璇真人罪有应得，另一派则认为另有隐情。"

"还有一派……"他的目光落在云泽身上，"在等你回来。"

"等我？"云泽微微挑眉。

"是的。"林惊羽道，"天璇真人被封印之前，将你收为亲传弟子。这一点，整个天道宗都知道。"

"如今他被定性为叛徒，而你作为他的弟子，自然就成了众人议论的焦点。"

"有人认为你与他同流合污，应当一并处置；有人认为你只是被蒙蔽，可以网开一面；还有人认为……"

他顿了顿，似乎在斟酌措辞。

"认为什么？"云泽问。

"认为你是道君转世，身负重大使命，不应被宗门内部的争斗所牵连。"林惊羽说，"这一派以青云峰首座青虚子为首，正竭力为你争取清白。"

"但他们能做的，也只是拖延时间而已。"

"若你不能给宗门一个交代，用不了多久，宗门高层便会对你采取行动。"

云泽沉默了。

他早就料到会有这一天。天璇真人叛宗的消息在修真界掀起了轩然大波，而作为天璇真人唯一的亲传弟子，他自然无法置身事外。

更何况——他还背负着"道君转世"这个敏感的身份。

"掌门怎么说？"他问道。

"掌门至今未表态。"林惊羽摇了摇头，"但据我所知，掌门对天璇真人一事似乎也有疑虑。他并未下令对你采取强制措施，只是在观望。"

"观望什么？"

"观望你接下来的行动。"林惊羽望着他，"以及……观望你是否能证明自己的清白。"

云泽沉默了良久。

他抬起头，看向窗外那轮皎洁的明月，心中思绪万千。

他知道，自己必须尽快做出决断。

天璇真人的事情，终究需要一个了结。而那个了结的方式，将决定他在天道宗的未来——甚至，将影响整个修真界的格局。

"我明白了。"他说，"明日一早，我会去见掌门。"

"你要主动说明证道遗迹的事？"林惊羽问。

"是的。"云泽点头，"事已至此，隐瞒已经没有意义。"

"我会在宗门高层面前，当众揭露天玄子的真面目。"

"将三百年前的真相——全部说出来。"

林惊羽和苏婉晴对视一眼，都从对方眼中看到了震惊。

"你确定？"林惊羽问，"天玄子是天道宗的开派祖师，在宗门中的地位根深蒂固。若你贸然揭露真相，很可能会引发更大的动荡。"

"我确定。"云泽的目光平静而坚定，"因为那是天璇真人临终前托付给我的使命。"

"他说，'替我把真相告诉世人'。"

"我答应了他。"

"无论前方有多少艰难险阻，我都会说到做到。"

他的声音不大，却带着一股不可动摇的力量，仿佛一座巍峨的山峰，矗立在夜风之中。

林惊羽望着他，忽然想起了九宗大比时的那一战。

那时云泽也是这样——面对强敌，明知不敌，却依然亮剑。

那种倔强，那种不屈，那种明知不可为而为之的执念，正是他最欣赏云泽的地方。

"好。"他点了点头，"明日一早，我陪你去。"

"我也去。"苏婉晴道。

云泽看了她一眼，微微摇头："苏师姐，你不必——"

"你不必替我做决定。"苏婉晴打断他，目光坚定，"我是你的师姐，你的事，就是我的事。"

"无论前路如何，我都会站在你身边。"

云泽望着她，心中涌起一阵暖意。

他没有再拒绝。

"好。"他说，"那便一起。"

夜风从窗外吹入，带起苏婉晴鬓边的几缕发丝。月光洒落，在她的脸上投下柔和的光影。

云泽望着她，忽然发现，她的眼角有了一丝笑意。

那是许久以来，他第一次在她脸上看到的笑容。
