# 第一部·第五章·第三节：飞舟将发

## 一、七日之期

七日后。

昆仑秘境开启之日。

天道宗山门之外的广场上，人山人海，热闹非凡。

这一日的天气格外晴朗，万里无云，碧空如洗。阳光从头顶洒落，将广场上的青石地面照得泛着淡淡的金色光芒。广场四周插满了各色旗帜，在晨风中猎猎作响，那上面绣着天道宗的标志——一轮冉冉升起的明日，下方是一柄横空出世的长剑。

天道宗的历史可以追溯到上古年间，传承至今已逾万年。作为中州最古老的宗门之一，天道宗在整个修仙界都享有崇高的地位。宗门的创始祖师据传是一位渡劫成功的大能，飞升仙界之后，留下道统，庇佑苍生。

当然，这些都只是传说罢了。对于大多数普通弟子而言，天道宗只是一个修炼的地方——一个可以让他们有机会踏上仙途、改变命运的地方。

此刻，广场上已经聚集了近千名弟子。

他们大多穿着青色的外门弟子服，腰间悬着各式各样的兵器，三三两两地聚在一起，交头接耳地议论着什么。人群中偶尔传来几声大笑，引得周围的人纷纷侧目。

而在广场的正中央，一艘巨大的飞舟静静地悬浮在半空。

那飞舟通体银白，长约三十丈，宽约十丈，船身如同一片倒扣的云朵，边缘处镶嵌着一圈淡蓝色的灵石。船身上刻满了繁复的符文，那些符文散发着微弱的光芒，在阳光下若隐若现，透着一股神秘而威严的气息。

天道云舟。

这是天道宗的镇宗之宝之一，据传是上古年间某位炼器大师的毕生心血。整艘飞舟由千年寒铁打造，船身上的符文是远古时期流传下来的护山大阵，能够抵御化神境以下的任何攻击。

飞舟的甲板上，已经站着不少身影。那些都是参加此次试炼的内门弟子和长老，气息深不可测，让站在下方的外门弟子们心生敬畏。

云无咎也在人群中。

他穿着一身洗得发白的青色长袍，腰间悬着那柄毫不起眼的游子剑，混在人群之中毫不出众。三年来，他已经习惯了这种低调——不引人注目，不招摇过市，不给人留下任何把柄。

他的目光从飞舟上一一扫过，最终停在了船头的那道紫色身影上。

天璇真人。

那个男人负手而立，身穿一袭紫色长袍，头戴玉冠，面容威严。他的目光淡淡地从下方的弟子们身上扫过，所过之处，众人纷纷低下头去，不敢与之对视。

只有云无咎，直直地站在那里，与那道目光在空中交汇。

只是一瞬间，云无咎便移开了视线。他低下头，做出一副恭敬的模样，仿佛刚才那一瞬间的对视只是巧合。

但他的心中，已经翻涌起惊涛骇浪。

那一瞬间的对视，他分明感受到了天璇真人目光中的深意——那不是简单的"宗主看向弟子"的目光，而是一种审视，一种估量，更是一种……期待。

期待？

云无咎心中冷笑。

天璇老贼，你在期待什么？期待我乖乖地走进你的圈套？还是期待我帮你打开那道封印？

无论你在期待什么，我都不会让你如愿。

"云师弟！"

一道声音从身后传来，打断了他的思绪。

云无咎回过头，只见一个身穿青色长袍的青年正朝他走来。那青年约莫二十出头，面容清秀，气质温和，正是与他关系不错的云帆师兄。

"云帆师兄。"云无咎拱手行礼。

"师弟，你可真沉得住气。"云帆走到他身旁，压低声音说道，"这都什么时候了，你还一副老神在在的模样。你看看周围的人，哪个不是紧张得手心冒汗？"

云无咎淡淡一笑："紧张有什么用？该来的总会来。"

"你这性子……"云帆无奈地摇了摇头，"也不知道该说你稳重还是该说你心大。"

他顿了顿，又压低了声音："对了，名单我打听过了。这次参加试炼的共有三十六人，全部是外门中的佼佼者，修为最弱的也有筑基境初期。咱们外门一共占了十二个名额，其他的都是内门弟子。"

"内门弟子也参加？"云无咎微微皱眉，"这不对吧？内门试炼不是只针对外门弟子吗？"

"是针对外门弟子没错。"云帆点点头，"但这次情况特殊。据说是因为昆仑秘境太过危险，内门特意安排了一些弟子参与，好从旁协助。"

"协助？"云无咎的嘴角微微勾起一丝弧度，"我倒觉得更像是……监视。"

云帆愣了一下，随即露出一个苦笑："师弟，有些话心里知道就行了，别说出来。"

云无咎没有回答，只是静静地望向飞舟的方向。

监视也好，协助也罢，他都不在乎。他在乎的只有一件事——这次试炼，他能不能活着回来。

## 二、人群百态

巳时将近，广场上的人越来越多。

除了参加试炼的弟子之外，还有许多前来送行的弟子和长老。他们站在广场四周，形成了一圈厚厚的人墙，将飞舟所在的区域围得水泄不通。

人群中不时传来各种议论声。

"听说这次试炼的地点是昆仑秘境？那地方可是出了名的凶险，十个人进去能活着出来三个就不错了！"

"可不是嘛！昆仑秘境里有不少上古妖兽，据说还有化形级别的大妖！那些东西可不是咱们这些筑基境的小辈能对付的！"

"你们说，这次试炼有几个人能活着回来？"

"难说……反正我觉得，能有一半人活着出来就不错了。"

"那些被选中参加试炼的人也是倒霉。昆仑秘境十年才开启一次，本以为是什么天大的机缘，谁知道竟是送死的机会。"

"也不能这么说。昆仑秘境虽然危险，但里面确实有不少好东西。我听说那里面有上古年间留下的灵药和法宝，若是能活着出来，说不定能得到意想不到的收获。"

"得了吧，有命拿也得有命享才行。那些东西再好，那也得有命消受不是？"

云无咎静静地听着周围的议论声，神色不变。

"师弟，你紧张吗？"云帆忽然问道。

"不紧张。"云无咎摇了摇头。

"真的？"

"真的。"云无咎转过头，看向他，"师兄，你觉得这次试炼，最危险的是什么？"

云帆愣了一下，沉吟片刻才回答："昆仑秘境里的妖兽？"

"不。"云无咎摇了摇头，"最危险的从来不是妖兽。"

"那是什么？"

云无咎没有直接回答，只是意味深长地看了他一眼，然后转身向飞舟的方向走去。

云帆愣在原地，望着他的背影，心中忽然涌起一股莫名的寒意。

最危险的从来不是妖兽……

他忽然想起了一些事——一些关于宗门内部的明争暗斗，关于师徒之间的尔虞我诈，关于那些隐藏在平静水面之下的暗流涌动。

云师弟……你到底知道些什么？

## 三、故人相逢

云无咎正往飞舟方向走去，忽然一道熟悉的声音从旁边响起。

"哟，这不是咱们外门的'天才'吗？"

那声音带着几分阴阳怪气的腔调，听得人浑身不舒服。

云无咎脚步一顿，转头看去。

只见一个身穿青色长袍的青年正朝他走来。那青年约莫二十出头，面容冷峻，眉宇间透着几分孤傲，嘴角挂着一丝不屑的笑意。他的腰间悬着一柄长剑，剑鞘上刻着一个古朴的"赵"字，在阳光下泛着淡淡的寒光。

赵乾坤。

三年前那个在外门中横行霸道的小霸王。

三年过去了，赵乾坤的变化不大，依然是那副目中无人的模样。但云无咎注意到，他的气息比三年前浑厚了许多——筑基境后期，距离筑基境巅峰只有一步之遥。

三年时间，从筑基境初期修炼到筑基境后期，这个速度不算慢。但比起他而言……

云无咎在心中冷笑一声。

"赵师兄。"他拱了拱手，语气平淡，"好久不见。"

"别叫我师兄！"赵乾坤的脸色瞬间阴沉下来，"我比你入门早三年，你该叫我师兄——不对，你根本没资格叫我师兄！"

他向前逼近一步，目光阴冷地打量着云无咎："三年不见，你还是这副穷酸样。穿得破破烂烂的，腰间挂个破铁剑，就这德行也敢来参加内门试炼？"

周围渐渐有人注意到了这边的动静，纷纷围拢过来看热闹。

"那是赵乾坤师兄吧？他和那个云无咎有过节？"

"你不知道？三年前外门大比的时候，赵师兄和云无咎起了冲突，差点打起来。后来虽然被长老拦下了，但梁子算是结下了。"

"原来如此……赵师兄这是要当众羞辱他啊！"

"可不是嘛！云无咎虽然是外门弟子中的佼佼者，但比起赵师兄还是差了一截。赵师兄可是筑基境后期，距离巅峰都不远了！"

议论声传入耳中，赵乾坤脸上的笑意更浓了几分。

他很享受这种众星捧月的感觉。尤其是在云无咎面前——这个三年前敢当众顶撞他的穷小子，今天他就要让所有人看看，得罪他赵乾坤是什么下场！

"云无咎。"赵乾坤的声音压低了几分，透着几分威胁，"我劝你识相点。这次昆仑秘境，危险重重，若是出了什么'意外'，你这条小命可就交代在里面了。"

"你若是乖乖地把那柄破剑交出来，再当众给我磕三个响头，叫我一声爷爷……"他的嘴角勾起一丝残忍的笑意，"我或许可以考虑，在秘境里'照应'你一下。"

云无咎静静地站在那里，听他说完。

他的表情很平静，没有愤怒，没有恐惧，也没有不屑。只是平静——像是在看一场无聊的表演。

"说完了？"他开口，声音很轻。

赵乾坤愣了一下。

他本以为云无咎会愤怒，会恐惧，会当场发作——无论哪一种，他都有办法应对。但这种平静的态度，却让他有些措手不及。

"你……"他刚要开口，却见云无咎已经转身离去。

"站住！"赵乾坤怒吼一声，"本少爷还没说完，你敢走？！"

云无咎头也不回。

"赵师兄。"他的声音从人群中传来，不高不低，却清晰地传入每个人的耳中，"秘境之中，各凭本事。你若是想与我为难，我奉陪到底。但若是想要我低头……"

他回过头，看了一眼赵乾坤，嘴角勾起一丝淡淡的弧度。

"抱歉，我这人没那个习惯。"

说完，他便头也不回地向飞舟走去，留下一脸铁青的赵乾坤站在原地。

"好……好……好……"赵乾坤咬牙切齿，"云无咎，你给我等着！昆仑秘境里，我一定会让你死得很难看！"
