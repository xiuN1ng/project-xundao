# 第一部·第七章·第二节：规则

## 一、天虚真人

天虚真人立于英灵碑前，灰袍在山风中微微拂动。

老人的面容清癯而威严，颧骨高高耸起，皮肤薄得几乎能看见底下青筋的走向。但他的眼睛却一点都不老——那双眼睛深邃得像两口古井，井底仿佛藏着无尽的星辰，让人不敢直视。凡是与他目光相遇的弟子，都不由自主地低下头去，仿佛那一眼便能将自己看穿、看透、看进骨头缝里去。

他站在那里，周身没有一丝气息泄露。明明是肉眼凡胎的身形，却让人觉得那不是一个人，而是整座山岳、整片苍穹、整个天地压在了那里。有几个修为尚浅的弟子甚至觉得胸口发闷、呼吸急促，不得不暗暗运起功法抵抗那股无形的威压。

云无咎也在暗中运功。

他体内的灵气自发地流转起来，在经脉中形成一道薄薄的屏障，将那股威压隔绝在外。他能感觉到身边的林惊羽也在做着同样的事——两人对视一眼，都从对方眼中看到了一丝凝重。

这便是大乘境巅峰的实力。

仅仅只是无意识散发出的气场，便足以让金丹境的弟子们如临大敌。云无咎不由得想起天璇真人——那个同样站在修真界顶端、却最终选择自我封印的老人。同样的境界，不同的选择，结局也截然不同。

"本届九宗大比，共设三轮。"

天虚真人开口了。

他的声音不高，却清清楚楚地传入每一个人的耳中。那声音不大，却像是从四面八方同时响起，分不清是从哪个方向传来——这是修为到了极致才能做到的境界，声由心生，无处不在。

"第一轮，海选。"

老人抬手，向身后的石碑一指。

"英灵碑下设有一道问心阵。阵中幻境万千，考验的是心境与意志。凡通关者，方有资格进入第二轮。"

人群中响起一阵低低的议论声，像是平静的湖面被投入了一颗石子，涟漪层层扩散。

"问心阵？那不是天道宗用来筛选弟子的阵法吗？"

"听说那阵法厉害得很，专门针对心魔发起进攻，心性不坚的人根本过不去……"

"怕什么，咱们都是金丹境的高手，心境能差到哪里去？"

云无咎静静地听着，没有说话。

问心阵他并不陌生。三年前，他就是在这道阵法中艰难通过试炼，才得以成为天道宗的外门弟子。彼时的他，面对的是内心深处最脆弱的那些角落——孤独、恐惧、对命运的怨恨。而三年后的今天，他已不再是当年那个在幻境中手足无措的少年。

但问心阵的可怕之处，不在于它有多难，而在于它有多真。它不会给你任何虚假的幻象，你所看到的每一幕，都是自己内心深处最不愿面对的东西。三年前他能通过，靠的不是实力，而是那股子倔强的、不肯认输的劲儿。

那股劲儿，是云爷爷给他的。

老人总说，咎者，过也，亦可解作改。往后你便叫无咎——无有过错，无有灾殃。无论从前如何，往后的路，你自个儿走。路是人走出来的，不是命注定的。

他将这句话刻在心底，用了三年时间去真正理解它。

"第二轮，淘汰赛。"

天虚真人的声音再次响起，将众人的思绪拉回到当下。

"海选过关者，两两捉对，胜者晋级，败者淘汰。最终决出十六人，进入第三轮。"

老人顿了顿，目光扫过全场。那目光平静得像一潭死水，却又深不见底，让人无法揣测其中的喜怒。

"淘汰赛采用抽签制，赛前才会公布对阵名单。也就是说——"

他故意停顿了一下。

"你们无法选择对手，也无法预知自己的命运。"

人群中又是一阵骚动，比方才更加嘈杂。

云无咎微微眯起眼睛。

抽签制……这意味着即便是同门弟子，也可能在淘汰赛中相遇。天道宗若有三名弟子通过海选，他们之间便有可能在第二轮自相残杀。这规则看似残酷，却也恰恰体现了修真界的本质——在这个弱肉强食的世界里，没有人能永远依赖宗门的光环护体。

他侧过头，看了看身边的林惊羽。

林惊羽的表情很平静，仿佛对这规则早有预料。那双星目中闪烁着锐利的光芒，像是一柄出鞘的长剑，寒光内敛，却随时可能刺向敌人的咽喉。

"师兄。"云无咎低声道，"若是抽到同门……"

"那就全力以赴。"

林惊羽的回答干脆利落，没有一丝犹豫。

"同门切磋，点到为止是礼让，生死相搏才是尊重。"他淡淡道，"若是在大比中还要藏私，那便是对对手最大的侮辱。"

云无咎点了点头，没有再说什么。

他理解林惊羽的想法。对于林惊羽这样的剑修来说，剑就是他的第二条命，剑道就是他毕生追求的道路。在剑道面前，一切人情世故都要让路。这是林惊羽的信念，也是他的骄傲。

"第三轮，巅峰对决。"

天虚真人的声音忽然低沉了几分，像是从九天之上压下来的一道雷霆。

"十六人，捉对厮杀，最终决出本届大比的冠军。"

"规则只有一条——"

老人的目光忽然变得锐利起来，像是两柄出鞘的利剑，直直地刺向天空。

"生死无论。"

这三个字一出，全场骤然安静。

方才还在窃窃私语的弟子们，此刻全都闭上了嘴巴。空气仿佛凝固了一般，压在每个人的肩头，让人喘不过气来。山风依然在呼啸，卷起漫天的尘土与落叶，但那些声响此刻都变得遥远起来，像是从另一个世界传来的回音。

有几个弟子的脸色变了。

他们显然没有料到，这届大比的规则会如此严苛。"生死无论"四个字，意味着在擂台上杀死对手是合法的行为——只要不是使用违禁手段、不违背大比的根本规则，那么无论你用什么方法击败对手，都是被允许的。

有人在人群中悄悄后退了一步。

那一步很小，小到几乎无人注意。但云无咎看见了。他看见那个弟子的手在微微发抖，额头上渗出了细密的汗珠，在阳光下闪闪发亮。

他没有嘲笑那个人。

他只是想起了三年前的自己。

彼时的他，还只是青云镇上一个懵懂无知的少年，连修仙是什么都不清楚。他以为自己这辈子都会在青云镇度过，砍柴、采药、照顾云爷爷，然后娶妻生子，度过平凡的一生。

是天道宗改变了他的命运。

而今天，命运再次站在了十字路口。"生死无论"——这四个字意味着，从他踏上擂台的那一刻起，他就必须做好将性命豁出去的准备。不是对手死，就是自己亡。没有第三条路可走。

"九宗大比，不是儿戏。"

天虚真人的声音不大，却字字清晰，像是一颗颗钉子，钉入每个人的心底。

"修真之路，本就是与天争命、与地争锋、与人争雄。每一场战斗，都是生死之间的较量。你们手中的剑，斩断的不只是对手的经脉——也可能是自己的前程，甚至是自己的性命。"

老人顿了顿。

他环顾四周，目光从每一位弟子的脸上掠过。那目光很慢，像是在审视什么，又像是在寻找什么。

"怕了？"

他忽然问了一句。

那语气很轻，轻得像是在问一个无关紧要的问题。但就是这轻飘飘的一个字，却让在场所有人都感受到了其中的分量。

怕了？怕了你就退出。

没人退出。

山风呼啸而过，卷起漫天的尘土与落叶。天机台上，九宗的弟子们笔直地站着，目光坚定，像是九座不可动摇的山峰。他们来自不同的宗门、不同的地域、不同的背景，但此刻，他们都做出了同样的选择——留下来，直面这场生死之争。

天虚真人看着这一幕，满意地点了点头。

"好。"

只一个字。

那声音里带着欣慰，带着期许，也带着一丝难以察觉的感慨。老人仿佛从这些年轻的面孔上，看到了自己曾经的影子——那个意气风发、不知天高地厚、以为凭着一腔热血就能闯出一片天地的少年。

岁月如水，逝者如斯。

当年那个少年，如今已是大乘境巅峰的绝世强者，站在了凡界的最顶端。可即便是他，也依然要面对岁月的侵蚀、面对天道的审判、面对渡劫的生死一关。修真之路，何处是尽头？

"既然无人退出，那便依此规则。"

天虚真人收回思绪，转身向英灵碑走去。

老人的脚步很慢，每一步都踏得很稳。灰袍的下摆在石板上拖曳，发出轻微的沙沙声。那背影看上去有些佝偻，却自有一股顶天立地的气势——仿佛他不是在走向一块石碑，而是在走向整个天地，走向那冥冥之中早已注定的命运。

他走到英灵碑前，伸出右手，按在了石碑的表面。

那石碑通体漆黑如墨，表面却光滑如镜，像是一块被无数岁月打磨过的黑玉。天虚真人的手掌按上去的瞬间，石碑上那些密密麻麻的名字仿佛都颤抖了一下——像是感应到了主人的到来，正在向他致意。

"开。"

一声低喝。

那漆黑如墨的石碑，忽然亮了起来。

金色的光芒从石碑的底部缓缓升起，像是有一条金龙在石碑内部游动。金光所过之处，那些刻在石碑上的名字便仿佛活了过来——每一个名字都在轻轻颤抖，散发出或强或弱的光芒，像是在向自己的主人致敬，又像是在诉说着一段段早已尘封的往事。

"那是……英灵碑在响应？"

人群中有人低声惊呼，声音里带着掩饰不住的震撼。

"传说英灵碑是用上古神兽的骸骨铸成，能够感应参赛者的实力与心性。"旁边有人压低了声音解释道，"天虚真人以大乘境的修为催动它，这是在向所有人展示天道宗的底蕴啊……"

云无咎的目光落在那些闪烁的名字上。

他看到了许多陌生的名字，也看到了一些熟悉的名字——那些名字属于曾经的强者，属于在历届大比中崭露头角的天才人物。他们有的已经陨落，有的已经飞升，有的则成为了震慑一方的绝世强者。但无论他们如今身在何处，这些名字都永远刻在了英灵碑上，与天道宗同寿，与日月同辉。

这是何等的荣耀，又是何等的寂寞。

荣耀的是名字被后人铭记，寂寞的是刻碑的人早已化作一抔黄土。修真者穷其一生追求长生不老，可到头来，能被记住的终究只是那几个名字而已。更多的人，连同他们的故事，都淹没在了岁月的长河中，再也无人问津。

他的目光继续向上移动，最终停在了石碑最顶端的一行字上。

那是一个极其古老的名字。

笔画苍劲有力，像是用剑尖一笔一画刻上去的。字迹的边缘已经有些模糊，但依然能感受到那字迹中蕴含的磅礴气势——那是一种超越了大乘境、超越了一般修士想象极限的威压，仿佛只要看上一眼，便会被那气势彻底压垮。

"云游子。"

云无咎的嘴唇无声地动了动。

三个字，却重逾千钧。

那是他师祖的名字。

那是刻在英灵碑最顶端、唯一一个不属于任何一届大比、却被历代天道宗弟子敬仰至今的名字。传说云游子在两千年前，以一人之力独战三大邪宗，最终与敌人同归于尽。他的骸骨散落在昆仑山脉的深处，他的名字却被天道宗永远铭记，刻在了英灵碑的最高处，俯瞰着后世每一位参赛者。

而他——云无咎——修炼的是云游子留下的心法与剑法，体内流淌的是云游子传承给他的道君印记。从某种意义上说，他与云游子之间，早已超越了两千年的时光，建立起了一种冥冥之中的联系。

他不知道自己能不能像云游子那样，成为震慑一方的绝世强者。

但他知道，只要他还活着，只要他还在这条路上走下去，他就永远不会放弃。

"游子九式……"

他低头看了看自己的手掌。

三年来，他用这九式剑法击败了无数对手，也在这九式剑法中找到了属于自己的道。"游子"二字，他终于有了更深的理解——不是流浪，不是漂泊，而是即便身在远方、心有归处的那种笃定。

天虚真人收回按在石碑上的手。

金色的光芒渐渐收敛，那些名字也重新归于沉寂。但每个人都知道，那块石碑中蕴含的力量，从未消失过。它只是沉睡着，等待着下一次被唤醒的时刻。

"海选，将在一个时辰后开始。"

天虚真人的声音再次响起。

"所有人，就地休整。"

说罢，老人转身离去，消失在山道尽头。

## 二、暗流

人群渐渐散开，各自寻找地方休息。

天机台四周设有九座观礼台，分别对应九大宗门。那观礼台皆以汉白玉砌成，雕梁画栋，金碧辉煌，彰显着各宗的底蕴与气派。但大多数弟子并没有立刻登上观礼台，而是三三两两地聚在一起，低声议论着方才公布的规则。

"生死无论……这也太狠了吧？"

一个身穿玄天宗服饰的弟子皱着眉头说道，脸色有些发白，"我还以为九宗大比只是切磋技艺，没想到竟然要拿命来拼……"

"怕了？"旁边一个剑阁的弟子嗤笑一声，"怕了就趁早滚回去，别在这里丢人现眼。"

"谁说我怕了？"玄天宗弟子顿时涨红了脸，"我只是觉得，这规则未免太不近人情了些……"

"修真界哪有什么人情可讲？"剑阁弟子撇了撇嘴，"你今日对敌人讲人情，明日敌人可不会对你讲人情。九宗大比的意义，就在于让咱们提前适应这种生死一线的环境。想活着？那就拼命变强。"

云无咎站在天道宗的队伍中，静静地听着这些议论。

他没有参与讨论，只是默默地观察着四周的人群。他的目光从玄天宗的队伍掠过，又扫过剑阁的阵列，最终停在了万法门的方向——

那里，一个身穿黑色长袍的青年正静静地站着，目光同样在人群中搜寻着什么。

两人的目光在半空中相遇。

只是一瞬间。

那青年的眼神很冷，冷得像是从深渊中凝视过来。那不是寻常的冷漠，而是一种历经磨难、看透世事之后才会有的冰冷。那眼神让云无咎想起了昆仑虚中的那场大战，想起了天璇真人自我封印时的决绝背影。

那是将一切都抛弃了的人才会有的眼神。

也是最危险的人才会有的眼神。

"厉无心……"

云无咎的瞳孔微微收缩。

那青年正是厉无心——天魔教的少主，本届九宗大比最神秘也最令人忌惮的对手。三年前的昆仑秘境试炼中，云无咎曾与厉无心有过一面之缘，彼时的厉无心还只是一个被心魔困扰的少年，在主角的帮助下才得以脱困。

但此刻的厉无心，与三年前判若两人。

他的身量比三年前高了一大截，肩膀也变宽了，整个人像是一柄被反复淬炼过的利剑，锋芒毕露。他站在万法门的队伍中，周身没有任何多余的动作，只是静静地站在那里，便已给人一种无形的压迫感。

仿佛他本身就是一座深渊，正在无声地吞噬着周围的一切光线与温度。

林惊羽也注意到了厉无心。

"那就是厉无心？"他低声问道，声音里带着一丝凝重。

"嗯。"云无咎点头。

"看起来……确实不简单。"林惊羽的眼中闪过一丝凝重，"他身上的气息……比我想象的更强。"

"师兄打算在决赛之前与他交手吗？"

"那要看他配不配了。"林惊羽淡淡道，"我的目标是冠军。如果他在决赛之前就被淘汰了，那只能说明他不够格。"

林惊羽没有说话，只是轻轻拍了拍云无咎的肩膀。

那动作很轻，却传递了千言万语。两人之间的默契不需要言语来维系——三年的并肩作战，早已让他们成为彼此最坚实的后盾。无论大比中遇到什么样的对手、什么样的困境，他们都不会退缩。

因为这是他们的道。

各自的道，也是共同的道。

山风依然在吹，卷动着众人的衣袍。天机台四周，九宗的旗帜在风中猎猎作响，每一面旗帜都代表着一个宗门的传承与荣耀。海选即将开始，属于他们的战斗，即将打响。
