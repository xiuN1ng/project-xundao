# 第一部·第四章·第一节

## 青云峰顶·晨曦

清晨的青云峰，薄雾如纱，缠绕在群峰之间。

云无咎盘膝坐于峰巅一块凸起的青石之上，身形纹丝不动，宛如一尊雕塑。晨曦的第一缕光芒穿透云海，将漫天云霞染成金红之色，壮阔如画。山风凛冽，吹得他单薄的青袍猎猎作响，他却浑然不觉。

呼吸吐纳，灵气如丝如缕，自四面八方汇聚而来，沿着经脉缓缓流入丹田。

三个月了。

他在心中默数。自踏入天道宗的那一天起，已整整过去九十三日。这九十三个日夜，他几乎将每一刻都用在了修炼之上——清晨习剑，上午打坐，午后研读《云游心法》，夜里与剑中残念对话。旁人看来枯燥乏味的日子，他过得却无比充实。

游子剑横放膝前，剑身斑驳，锈迹深沉。

这柄从青云镇后山石缝中偶然寻得的锈剑，在三个月的蕴养之下，似乎起了某种微妙的变化。那层看似厚重的锈迹比之初时淡了几分，剑身上隐隐透出一层若有若无的清光——那是灵力浸润之后才会出现的光泽。普通人或许看不出来，但云无咎日日与其为伴，感知得清清楚楚。

灵气在经脉中运转了一个大周天，云无咎缓缓睁开双眼。

他低头看向自己的掌心，感受着丹田中那比三月前充沛了何止三倍的灵力，唇角微微牵动了一下。那股灵力温热而绵长，在丹田深处凝聚成一团淡淡的云雾之形，正是《云游心法》修炼到一定境界后才会呈现的气象。

三个月，从淬体境七层突破至炼气境三层。

这个速度，放在整个天道宗外门，都算得上出类拔萃。寻常散修从淬体境七层突破到炼气境三层，少说也要一两年的苦功，而他只用了三个月。

但云无咎并没有因此骄傲。

他深知，自己之所以进步神速，归根结底是因为《云游心法》的精妙，以及游子剑中云游子前辈残留的修炼心得。若论资质悟性，他并不比那些世家子弟强多少。若论资源背景，他更是在外门弟子中垫底的存在。

修仙之路，如同逆水行舟。稍有懈怠，便会被他人远远甩在身后。

他想起离开青云镇的那个清晨。

李狗蛋站在镇口的老槐树下，使劲朝他挥手。那小子测灵根时因问心阵中心性浮躁，被淘汰至杂役堂，临别时拍着他的肩膀说："无咎，你可得好好修炼！等你在内门站稳脚跟，别忘了拉兄弟一把！"

狗蛋的嘱托，他一直记着。

云无咎站起身，将游子剑负于背后，正要继续晨练剑术，忽然听见身后传来一阵杂乱的脚步声和喧哗声。

"让开让开！赵师兄来了！"

"都聋了吗？赵师兄的路你们也敢挡？"

脚步声由远及近，夹杂着粗暴的呵斥声。云无咎眉头微皱，侧身望去，只见十余名外门弟子正沿着青石小径鱼贯而来，人群纷纷避让两侧，神色恭敬地垂首而立。

人群之中，一个锦衣少年负手而行，步履从容，神态倨傲。

那少年约莫十七八岁，面白无须，五官倒是生得不差，偏偏眉眼间透着一股阴鸷之气，让人看了便觉不舒服。他身穿一件绣着金纹的青色长袍，腰间悬着一枚刻有"赵"字的羊脂玉佩，脚蹬云纹靴，浑身上下透着一股养尊处优的傲气——那是世家子弟才有的派头，寻常散修学都学不来。

正是外门一霸，赵乾坤。

赵乾坤的身后，还跟着两个面容各异的随从。

走在左边那个尖嘴猴腮、笑起来满脸谄媚的，名叫孙麻，是赵乾坤的第一号狗腿子，专门负责为其出谋划策、跑腿办事。走在右边那个虎背熊腰、面目凶悍的，名叫钱虎，专职负责吓唬人，在外门弟子中名声极差。

三人一行，气势汹汹，直奔峰顶方向而来。

周围的弟子纷纷退避，大气都不敢出，生怕惹恼了这位赵师兄。

云无咎站在原地没有动。

他不是不知道赵乾坤的威名，也不是不知道得罪此人的后果。但这里是青云峰顶，是他每日晨练的地方，是他先来的，凭什么要给别人让路？

他只是微微侧身，给来人让出了半条通道，神色平静如常。

然而赵乾坤走到他身前时，却停下了脚步。

"咦？"

赵乾坤的目光在他身上停留了一瞬，随后落在那柄负在他背后的锈剑之上。他眉头微挑，嘴角勾起一丝玩味的笑意。

"这不是那个从穷乡僻壤来的野小子吗？"

他的语气轻慢，像是在打量一只误入闹市的野猴。

"叫什么来着……云无咎？"

云无咎面色平静，抱拳道："赵师兄。"

他没有多说什么，也没有辩解什么。这是天道宗，是修仙界，讲究的是实力为尊。赵乾坤的叔叔是内门长老赵玄机，在宗门中势力庞大，远不是他一个初入仙门的散修能够招惹的存在。

面对这种人，多说无益，忍是上策。

"炼气境三层了？"赵乾坤上下打量了他一眼，似乎在评估一件货物的价值，"三个月从淬体境七层突破到炼气境三层，倒是有几分运气。"

他向前踱了一步，居高临下地俯视着云无咎。

"不过这运气……在本少爷面前可不管用。"

他伸出手，拍了拍云无咎的肩膀。力道不轻不重，却带着一股说不出的轻蔑与羞辱意味。

"本少爷劝你一句，在这天道宗，最好学会低头。你那点出身，放在外门连根葱都算不上。要不是宗门不允许私斗，你以为你能在这青云峰上站稳三天？"

周围的弟子们噤若寒蝉，没人敢出声。

赵乾坤的威名，是用一次次打压外门弟子建立起来的。被他欺负过的散修不计其数，有些人甚至被打压到主动退出宗门。如今他盯上了云无咎这个没背景的散修，众人都已经在心里给云无咎判了死刑。

云无咎垂着眼睛，一言不发。

他的手在袖中微微攥紧，指甲几乎嵌进掌心。

赵乾坤见他不说话，脸上露出满意的神色，正要再训斥几句，却忽然听见远处传来一阵细微的风声。

那风声很轻，轻得几乎难以察觉。但赵乾坤身为炼气境后期的修士，对周遭灵气的波动极为敏感。他脚步骤然一顿，转头向峰顶边缘望去。

只见一道白衣身影，不知何时已立于峰巅边缘的一块巨石之上。

那是一名青年，约莫二十出头，身着天道宗核心弟子才有资格穿戴的白色剑修袍，背负一柄通体晶莹的玉剑。他身形颀长，笔直如剑，长发束于脑后，随风轻扬。

他剑眉星目，面容清隽，气质冷峻疏离，周身萦绕着一股凛冽至极的剑意——那剑意内敛而深沉，仿佛一柄藏于鞘中的绝世神兵，不动则已，动则惊天。

晨曦的光芒从他身后洒落，将他整个人镀上了一层淡金色的轮廓，看上去宛若画中仙人。

"林……林师兄？！"

赵乾坤的脸色瞬间大变。

他连忙收起方才的嚣张姿态，挺直了脊背，恭敬地向那白衣青年行礼："赵乾坤见过林师兄！"

周围的弟子们也纷纷躬身行礼，一个个神色激动。

林惊羽。

东州剑道第一天才，天道宗外门弟子心中的传奇人物。年仅二十便已是金丹境剑修，一手"天外飞仙剑诀"出神入化，在天道宗乃至整个九宗年轻一辈中都声名显赫。

传闻他曾在一次秘境探索中，以一剑之威独战三头化神境妖兽，将三头足以毁山灭河的化神大妖尽数斩杀，名震修仙界。

然而这位传奇剑修素来行事低调，常年闭关修炼，极少在外门露面。今日忽然出现在青云峰，着实让众人大吃一惊。

林惊羽从巨石上飘然落下，目光淡漠地扫过众人。

他的视线从赵乾坤身上掠过，没有任何停留，仿佛只是在看一个无关紧要的路人。

然而，当他的目光落在云无咎身上时，脚步骤然一顿。

他的目光直直盯着云无咎背后那柄锈迹斑斑的长剑，眸中闪过一丝难以掩饰的震动。

那是一种极为复杂的神情——有震惊，有疑惑，有审视，还有一丝极淡极淡的……敬畏？

"这位师弟。"

林惊羽向前踏出一步，声音清冷如泉，却带着一丝不易察觉的急切："你这柄剑……可否借我一观？"

此言一出，四下皆惊。

赵乾坤瞪大了眼睛，难以置信地看着这一幕——林惊羽，天道宗外门第一剑修天才，竟然主动向一个刚入门的散修借剑？

而且，他看那柄锈剑的眼神……分明是如临大敌、如见鬼神的凝重！

那柄锈迹斑斑的破剑，有什么值得看的？

众人面面相觑，心中充满了疑惑。

云无咎也愣了一下。

他没想到林惊羽会对自己的剑感兴趣。他更没想到，这位传说中的天才剑修，会用如此郑重的语气向他提出请求。

但他没有犹豫太久。

他伸手解下背后的游子剑，双手递上："林师兄请看。"

林惊羽接过游子剑。

他的动作很轻，像是在捧着一件易碎的珍宝。

他将剑横于面前，指尖轻轻拂过剑身。那剑身上的锈迹斑驳，触手冰凉，看上去就像一柄再普通不过的凡剑。但林惊羽却像是感知到了什么，眉头越皱越紧。

忽然，他指尖微微用力。

一股若有若无的剑意从那锈迹斑斑的剑身上透出——那剑意极为微弱，若不是刻意感知，几乎察觉不到。但它纯粹、古老、深邃，蕴含着一种说不清道不明的玄奥，仿佛来自另一个遥远的时代。

林惊羽的瞳孔骤缩。

他深吸一口气，翻转剑身，目光落在剑柄处一个模糊的铭文上。

那铭文因年代太过久远，已几乎看不真切。但在场之人皆是修士，目力远超常人，在场众人屏息凝神，依稀能辨认出那是一个古老的篆字——

"叶……"

林惊羽倒吸一口凉气。

他的脸色，在这一刻彻底变了。

那不再是审视一个普通师弟的目光，而是带着几分郑重、几分警惕，以及……几分说不出的激动。

他猛地抬头，看向云无咎："此剑……你是从何得来？"

云无咎犹豫了一下，如实道："师弟出身中州边陲一小镇，在家乡后山采药时，偶然从一块石缝中获得。"

"中州边陲……后山……石缝……"

林惊羽喃喃重复，若有所思。

他闭上双眼，似乎在脑海中搜索着什么。片刻后，他睁开眼，看向云无咎的目光中多了几分深意。

"果然如此。"

他将游子剑还给云无咎，神色郑重："这位师弟，我有一言相告。"

云无咎连忙躬身："请师兄指教。"

林惊羽看了他一眼，声音低沉："此剑绝非寻常之物。你日后修炼剑道，务必善加保管，切不可轻易示人。"

他顿了顿，又补充道："三月后的外门大比，你若能进入前十，可来东峰寻我。我有些修炼上的心得，或许对你有用。"

说罢，他不再理会旁人，御剑而起，化作一道白光，消失在云海之中。

全场死寂。

众人呆呆地看着林惊羽消失的方向，半晌回不过神来。

赵乾坤站在原地，脸色精彩极了——从最初的震惊，到难以置信，最后定格在一种深深的嫉恨之上。

林惊羽，天道宗第一剑修天才，竟然主动向这个乡下来的野小子抛出橄榄枝？还邀请他大比后去东峰？

这让他这个长老的侄子情何以堪？

"云无咎！"

赵乾坤咬牙切齿，声音几乎是从牙缝里挤出来的。

"好，很好！你给本少爷等着！"

他一甩袖子，带着孙麻和钱虎气冲冲离去。临走前，他狠狠瞪了云无咎一眼，那眼神中满是怨毒与阴狠，仿佛要将他的模样刻进骨头里。

云无咎站在原地，看着赵乾坤远去的背影，心中微微一沉。

他知道，自己今日算是彻底得罪了赵乾坤。这个纨绔子弟心胸狭窄，睚眦必报，日后少不了麻烦。

但他并不后悔。

林惊羽对游子剑的反应，让他心中涌起了一个强烈的念头——这柄剑的来历，恐怕远比他想象的要复杂得多。而他自己，也已经被卷入了一场不知何时开始的漩涡之中。

"此剑绝非寻常之物……"

他低头看向手中游子剑，轻声问道："前辈，您究竟是什么来历？"

游子剑沉默不语。

唯有剑身上那若有若无的清光，在晨曦中微微闪烁，似乎在诉说着它不为人知的过往。

云无咎握紧剑柄，心中暗暗发誓：无论这柄剑背后藏着怎样的秘密，他都要一步步追寻下去。

前路漫漫，但他已无退路。
