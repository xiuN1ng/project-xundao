# 第一部·第二章·第六节：云来镇

## 一、风云际会

云来镇是距离天道宗最近的凡人聚居点。

说是镇子，实际上更像是一个由客栈、酒楼、茶馆组成的补给站，专门为每三年一度的收徒大会而生。平日里这里冷冷清清，街上连个行人都少见；可一旦到了收徒大会的前夕，整座镇子便像是被注入了活力，从四面八方涌来的少年少女将每一条街道挤得水泄不通。

云无咎三人找了大半天，才在镇子角落里找到一间还有空房的客栈。

"三位客官，实在不好意思，"掌柜的是一个精瘦的中年人，满脸歉意，"这几日客人太多，小店的房间都住满了。就剩这一间上房，三位要是不嫌弃，就将就着住吧。"

那"上房"其实就是一间逼仄的小屋，摆了三张木板床便转不开身，窗户小得只能透进一丝光亮，空气里弥漫着一股潮湿的霉味。

可张小胖却满不在乎："行啊行啊，有地方睡就不错了！"

李狗蛋也点头附和："就是就是，比露宿野外强多了。"

安顿好住处，三人便上街打探消息。

云来镇的街道上人头攒动，到处都能听到关于天道宗收徒大会的议论。

"听说了吗？今年来参加收徒大会的足有三千多人！"

"三千多？最后能录取多少？"

"据说只有一百个名额，三千多人抢一百个位置，简直是神仙打架！"

"不止不止，俺还听说今年有世家的子弟来了好几个，都是测出了上品灵根的绝世天才！"

"上品灵根？那咱们这些普通人还有戏吗？"

"谁知道呢，天道宗收徒不是只看灵根的，还得看心性。看缘分。"

云无咎默默听着这些议论，心中暗暗记下。

三千多人，只录取一百人。

这个竞争比预想中还要激烈。

---

## 二、张灵轩

正当三人边走边聊的时候，前方忽然传来一阵喧哗。

"快看快看！那是谁家的少爷？好大的排场！"

"那马车好华贵啊！俺这辈子都没见过那么漂亮的马车！"

"还有那些护卫，个个都是练家子，身手不凡啊！"

云无咎循声望去，只见街对面停着一辆装饰华丽的马车。那马车通体漆黑，拉车的是两匹浑身雪白的高头大马，马蹄上钉着银钉，一看便是价值不菲的神骏。

马车的帘子掀开，从里面走下一个人来。

那是一个十六七岁的少年，身穿一袭月白色的锦袍，腰悬玉佩，面容俊朗，气质出尘。他的眉眼间带着一丝傲气，却并不让人觉得讨厌，反而有种理所当然的从容。

"是张灵轩！"李狗蛋惊呼，"无咎哥，你还记得吗？测灵根那天，他测出了上品木灵根！"

云无咎当然记得。

张灵轩，张家庄的少爷。据说他的父亲是当地有名的富商，家财万贯，富甲一方。而他自己更是天资卓绝，从小便展现出了过人的修炼天赋。这次来天道宗参加收徒大会，张家庄上下自然是倾尽家财支持，生怕这位少爷在外面受了委屈。

张灵轩走下马车，目光在人群中扫过。

他的视线掠过那些衣着华贵的世家子弟，掠过那些瑟缩在角落里的寒门少年，最后落在了——

云无咎身上。

两道目光在人群中相遇。

张灵轩的眼神里带着一丝好奇，又带着一丝探究。他显然认出了云无咎——那个在青云镇测灵根时和他一起被周仙师点名的人。

他朝云无咎微微颔首，嘴角带着一丝淡淡的笑意。

那笑意里没有恶意，只有一种世家子弟特有的从容与自信。

云无咎也朝他点了点头，算是打了招呼。

然后两人便各自移开了目光，继续各自的路途。

"无咎哥，你认识他？"李狗蛋凑过来，好奇地问。

"不认识。"云无咎摇了摇头，"只是见过一面。"

"哦。"李狗蛋若有所思地点了点头，"俺瞧他那人还不错，至少没像其他世家子弟那样鼻孔朝天。"

"人不可貌相。"云无咎淡淡道，"交情如何，得相处了才知道。"

## 三、传送阵

第二日一早，收徒大会正式开始。

天道宗在云来镇外的空地上设立了一座临时的传送阵，将通过灵根测试的少年直接送往云雾山下的集合点。

那传送阵极为壮观——一块方圆十丈的青石地面，四周矗立着八根雕刻精美的石柱，每根石柱上都刻满了密密麻麻的符文。平日里这阵法看不出什么异样，可一旦启动，那些符文便会亮起幽幽的蓝光，将整个广场照得如同白昼。

云无咎和李狗蛋、张小胖三人早早便来到了广场上等候。

广场上已经聚集了数百名少年男女，年龄大多在十二到十八岁之间，穿着各异，神态各异，却都带着同样的紧张与期待。

"各位参加收徒大会的弟子，"一个身着青色道袍的中年修士站在传送阵前，高声宣布，"天道宗三年一度的入门选拔即将开始。请通过灵根测试的弟子依次上前，准备传送。"

"这次选拔分为三个环节：心性测试、悟性测试、以及最终的试炼。通过这三个环节的弟子，将正式成为天道宗的外门弟子。"

"未能通过的弟子，也不用灰心。修仙之路千万条，天道宗只是其中之一。日后若有缘，咱们再见。"

"现在，请各位依次上前。"

人群中响起一阵窃窃私语，随即有人开始朝传送阵走去。

云无咎深吸一口气，迈步上前。

他的身后，李狗蛋紧紧跟着，虽然脸色有些发白，却还是咬着牙往前走。张小胖则一脸轻松，甚至还有心情跟旁边的人聊天打屁。

三人一起踏上了传送阵。

一道白光闪过，云无咎只觉眼前一花，身体像是被什么东西托着，轻飘飘地飞了起来。

下一瞬，他睁开眼，发现自己已经身处另一个地方。

那是一座巨大的山峰脚下。

山峰高耸入云，云雾缭绕，山腰处隐约可见飞檐翘角的宫殿建筑。空气中弥漫着浓郁的灵气，比青云镇浓了何止十倍，吸一口便觉得神清气爽。

"这就是……天道宗？"李狗蛋瞪大了眼睛，满脸不可置信。

"不是天道宗，"张小胖同样是一脸震撼，"是天道宗的山门所在……天道宗还在上面呢！"

云无咎望着那云雾深处的山峰，握紧了拳头。

他来了。

从青云镇到天道宗，从凡人到修士，从懵懂少年到寻道路上的行者——

他的修仙之路，从这一刻起，正式开始。
