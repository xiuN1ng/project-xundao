# 第一部·第十章·第二节：渡劫准备

## 一、劫云聚

三月之期，转瞬即至。

这一天，天道宗山门上空万里无云，烈日当空，碧蓝的天幕上连一丝云彩都没有。按理说，这样晴朗的天气，正是渡劫的绝佳时机——劫云需要凝聚，晴空万里可以给修士更多准备的时间。

然而，当云泽登上青云峰顶的那一刻，他却感到一股无形的压力从天而降。

那压力不是来自天空，而是来自脚下——来自地脉深处那股正在疯狂涌动的灵气潮汐。那潮汐如同一锅被煮沸的水，不断地翻涌、撞击、膨胀，仿佛随时都会冲破地表的束缚。

而随着那潮汐的涌动，天空也开始变色了。

起初只是地平线尽头出现了一抹淡淡的紫光，那紫光微弱得几乎不可察觉，像是有人在天边点燃了一盏遥远的灯。

然后，那紫光开始扩散。

它从西边的天际向东方蔓延，如同打翻的墨水浸染宣纸，速度越来越快，范围越来越广。短短半个时辰，那紫光便笼罩了大半个天空，将原本碧蓝的天幕染成了一片深邃的紫黑。

云泽站在峰顶，仰头望天。

那紫黑色的云层正在缓缓旋转，仿佛有什么东西在那云层深处苏醒。云层旋转的速度起初很慢，慢到几乎看不出变化，但随着时间推移，那旋转越来越快，渐渐形成了一个巨大的漩涡。

那漩涡的中心，是一片绝对的黑暗。

那黑暗深不见底，仿佛能够吞噬一切光线和物质。每当云泽的目光试图探入那片黑暗时，他都会感到一阵眩晕——那感觉就像是在凝视深渊，而深渊也在凝视他。

劫眼。

那是天劫凝聚的征兆。

"来了。"

林惊羽的声音从身后传来。

云泽没有回头。他依然仰着头，凝视着那缓缓旋转的劫云。

林惊羽走到他身旁，也抬头望去。

"比我想的还要恐怖。"他说，声音里难得有了一丝凝重，"这劫云的规模，比寻常天劫大了何止十倍。我曾见过一位化神境巅峰的前辈渡劫，他的劫云只有这十分之一大小。"

"因为他是化神境。"云泽说，"而我是道君转世。"

他说得很平静，像是在陈述一个事实。

但那事实的重量，只有他自己知道。

道君转世，这是他在证道遗迹中觉醒的身份。这个身份赋予了他超越常人的天赋和机缘，却也让他成为了天道的眼中钉。

天道不允许变数的存在。

而他，就是那个变数。

所以，这天劫不仅是修为的天劫，更是命运的天劫。天道要借这次机会，将他彻底抹杀。

"你怕吗？"林惊羽问。

云泽沉默了一会儿。

"怕。"他说，"但怕也没用。"

他从腰间解下游子剑，将剑横在身前。

那柄剑陪伴了他多年，从青云镇后山的悬崖下，到天道宗的山门，再到昆仑虚的深处。剑身上布满了大大小小的缺口和裂纹，那是无数次战斗留下的痕迹。但即便如此，那剑中的剑灵——云游子的残念——依然在微微跳动，仿佛随时都会苏醒。

"云游子前辈。"云泽低声说，"这一战，拜托了。"

剑身微微发热，像是在回应他的话。

## 二、故人皆至

日过正午，劫云越聚越厚。

天道宗山门内的修士们纷纷停下手中的事务，仰头望天，脸上写满了震惊与敬畏。他们中的大多数人从未见过天劫，更别提如此规模的天劫。那黑色的劫云如同一只巨大的怪兽，盘踞在天道宗上空，遮天蔽日，仿佛要将整个凡界都吞噬进去。

"这是谁在渡劫？怎会有如此规模的劫云？"

"莫非是哪位大能即将飞升？"

"不对，这劫云的颜色不对——寻常天劫的劫云是金色的，而这却是紫黑色的……"

"难道是有人要渡传说中的'天魔劫'？"

修士们议论纷纷，却没有人敢靠近青云峰。

天道宗的高层早已下令：青云峰方圆十里之内，禁止任何人靠近。违者，以叛宗论处。

这道命令，是天道宗宗主亲自下的。

云泽在天道宗的这几年，立下过不少功劳——九宗大比亚军、昆仑虚之战的功臣、道君转世的觉醒者。这些功劳足以让他在天道宗获得最高的礼遇，尽管他即将渡劫的是一场足以让天道都为之侧目的恐怖天劫。

而作为渡劫者的他，此刻正盘膝坐在青云峰顶的一块巨石上，闭目调息。

巨石周围，已经被林惊羽布下了一圈剑阵。

那剑阵不算复杂，但胜在精巧。三十六柄飞剑按照特定的位置插入地面，剑身微微发光，形成一道无形的屏障。那屏障能够挡住外来的攻击，却不会干扰天地灵气的流动——这是林惊羽特意设计的，专门用于护法。

除了剑阵之外，还有另外两道身影站在不远处。

苏婉晴和厉无心。

苏婉晴带来了她这三年炼制的所有丹药——九转护魂丹、千年续命露、万象归元丸……每一颗都是极品，每一颗都价值连城。她将这些丹药整整齐齐地码放在云泽身旁的一个玉盒里，然后退到一旁，默默地注视着他。

厉无心则带来了天魔教的秘法。

"天劫九重，第九重有精神攻击。"他说，"这一关最难熬。你身上的道君印记太招摇了，天道一定会借此攻击你的神魂。我这里有一门'锁魂术'，可以暂时封住你的神魂波动，让天道摸不清你的底细。"

"代价是什么？"云泽问。

"代价？"厉无心笑了笑，"代价就是，在锁魂术生效期间，你不能使用任何神识类功法。"

云泽点点头。

不能使用神识功法，总比在第九重天劫中神魂俱灭要好。

"还有一件事。"厉无心忽然说。

他从怀里掏出一枚黑色的令牌，令牌上刻着一个狰狞的"厉"字。

"这是我父亲的令牌。"他说，"当年他在昆仑虚之战中，以身化魔，与饕餮的魔气融为一体。从那以后，他的意识便一直存在于饕餮体内，偶尔还能通过这枚令牌与我交流。"

"如果你在灵界遇到什么解决不了的麻烦，就捏碎这枚令牌。我父亲的残念会指引你方向。"

他将令牌塞进云泽手里。

云泽握着那枚温热的令牌，一时无言。

他知道这枚令牌对厉无心的意义。那是厉青冥留在这世上的最后一丝联系，是厉无心对父亲的全部念想。如今他将这枚令牌交给自己，相当于将自己最后的底牌交了出来。

"你不留着？"云泽问。

"留着做什么？"厉无心耸耸肩，"我又不打算死。"

他说得轻描淡写，好像这枚令牌真的只是一块无关紧要的破铜烂铁。

但云泽能感受到，令牌上残留着一丝极其微弱的温度——那是厉青冥残念的气息，也是父子之间最后的羁绊。

"谢谢。"云泽说。

厉无心摆摆手，转身走开了。

## 三、劫云压顶

午时三刻，劫云终于凝聚完毕。

那劫云已不再是当初的紫黑色，而是变成了纯粹的黑——一种深不见底的、仿佛能够吞噬一切光线的黑。在那黑暗之中，有无数细小的电弧在跳跃，发出噼里啪啦的声响，像是有什么东西正在那云层深处酝酿。

然后，第一道雷劫落下了。

那雷劫是一道金色的光柱，从劫云中心的那片黑暗中直直劈下，所过之处，空气被瞬间电离，形成一道刺目的白光。那白光比日头还要亮上万倍，照得天地间一片惨白，仿佛连影子都被那光芒彻底淹没。

云泽睁开眼睛。

他站起身，游子剑已在手中。

他没有躲避，也没有防御，而是迎着那道雷劫，踏前一步。

剑光一闪。

一道凌厉的剑气从游子剑剑身上迸射而出，迎着那道金色光柱撞了上去。

轰——

一声巨响。

剑气与雷劫在半空中碰撞，激荡出一圈肉眼可见的冲击波。那冲击波向四周扩散开去，所过之处，草木尽皆化为齑粉，岩石尽皆崩裂为碎屑。林惊羽布下的剑阵在这冲击波面前剧烈摇晃，三十六柄飞剑发出尖锐的嗡鸣，像是在悲鸣。

第一重天劫，云泽安然度过。

但这只是开始。

第二道雷劫紧随其后，比第一道更粗、更亮、更加凶猛。

然后是第三道、第四道、第五道……

一道接一道的雷劫从天而降，如同密集的鼓点，敲击着云泽的身心。每一道雷劫都比前一道更加恐怖，到后来，那雷劫已经不再是单纯的光柱，而是化作了各种形态——有的是咆哮的雷兽，有的是咆哮的雷龙，有的是从天而降的雷山。

云泽的剑法也在不断进化。

从最初的游子九式，到后来的云游心法，再到证道遗迹中觉醒的道君传承。他的剑法越来越凌厉，越来越精妙，到最后，每一剑都仿佛蕴含着天地的至理。

但即便如此，他的身体也在以肉眼可见的速度衰弱。

雷劫的威力是递进的。一重比一重重。到第五重天劫时，他的经脉已经开始吃不消了。每一道雷劫落下，都会在他的经脉中留下一道焦黑的伤痕。那些伤痕不断累积，到后来，他的整个身体都在冒着青烟，像是一块被烧红的铁。

第六重天劫落下时，他的膝盖已经弯曲了。

第七重天劫落下时，他的手臂开始颤抖。

第八重天劫落下时，他几乎要被那恐怖的威压彻底压垮。

"云泽！"

苏婉晴在远处喊了一声，声音里满是焦急。

她想要冲上去，却被林惊羽一把拉住。

"别去！"林惊羽的声音很低，但很坚定，"现在去只会害了他。天劫认人，一旦有外人介入，劫云的威力会成倍增加。"

苏婉晴咬着嘴唇，眼眶通红。

她只能眼睁睁地看着那道单薄的身影，在狂暴的雷劫中苦苦支撑。

第九重天劫，终于降临了。

那是一道与众不同的雷劫。

它不是金色的，也不是白色的，而是——透明的。

那透明的光柱从劫云深处飘落，像是一缕轻烟，又像是一片羽毛。它没有任何声响，没有任何威压，看起来甚至有些……温柔。

但云泽的脸色却在瞬间变得煞白。

因为他知道，这道看似温柔的天劫，才是真正致命的东西。

精神攻击。

那是直接作用于神魂的攻击，绕过一切肉体防御，直击灵魂深处。

他要面对的，不再是肉身的考验，而是道心的审判。

透明的光柱越来越近。

云泽深吸一口气，将厉无心给他的"锁魂术"催动到极致。

他的神魂波动在一瞬间被压制到了最低点，几乎与天地融为一体。

而就在那光柱即将触及他眉心的一刹那——

他看到了饕餮的影子。

那是一头无法用言语形容的恐怖巨兽。它的身躯比山岳还要庞大，遮天蔽日，一眼望不到边。它的眼睛是两团燃烧的火焰，散发着无尽的贪婪与毁灭。那贪婪与毁灭仿佛要将整个世界都吞噬殆尽。

但下一瞬，那巨兽的影子却突然变化了。

它的身躯在缩小，那庞大的轮廓渐渐变得清晰，最终凝聚成了一个人形——一个云泽无比熟悉的人形。

那是……他自己。

一个与他长得一模一样的人，站在他面前，嘴角挂着一抹嘲讽的笑意。

"你以为你能逃掉吗？"那人问他，声音与他一模一样，"天道也好，道君也罢，你终究只是这世界的一颗棋子。从你出生的那一刻起，命运就已经被写定了。"

"你注定要死在天玄子的手里。"

"你注定要成为他的踏脚石。"

"你注定……"

"够了。"

云泽的声音很轻，却带着一种不容置疑的坚定。

他向前走了一步，直视着那与自己一模一样的影子。

"我命由我不由天。"

"就算天道要灭我，我也要与天道斗上一斗。"

"就算命运早已写定，我也要在那命运之外，杀出一条血路来。"

他的眉心处，道君印记骤然亮起，散发出耀眼的金光。

那金光将饕餮的影子彻底吞没，也将那透明的天劫光柱彻底驱散。

劫云开始消散。

那盘踞在天道宗上空数个时辰的黑色云层，正在以肉眼可见的速度瓦解。阳光重新洒落，照在那满目疮痍的青云峰上，也照在云泽那张苍白的脸上。

他单膝跪地，大口大口地喘着粗气。

但他的嘴角，却勾起了一抹笑容。

第一道天劫——

他渡过了。
