# 第一部·第一章·第六节：爷爷的话

## 一、门槛上的灯火

是夜，月明星稀。

云无咎回到茅屋时，云爷爷正坐在门槛上等他。

老人已年过七旬，头发花白，脸上沟壑纵横，身子骨却还算硬朗。他披着一件破旧的棉袄，手里拄着一根桃木拐杖，正眯着眼睛望着夜空中的星斗，不知在想些什么。门槛旁边的石墩上摆着一盏油灯，火苗摇曳，在夜风中投下明明暗暗的光影。

"爷爷，我回来了。"云无咎快步上前，伸手想要扶老人起身。

"回来了。"云爷爷咧嘴一笑，露出几颗残缺的牙齿，"今儿的事，狗蛋都跟我说了。五行灵根，好，好，好！"

他一连说了三个"好"字，眼角的皱纹都舒展开来，浑浊的眼睛里似乎有什么东西在闪烁——是欣慰，是骄傲，更有一丝说不清道不明的感慨。

"爷爷也替你高兴！"他拍了拍云无咎的手，"我就知道，我云老头捡回来的孙子，不是寻常人物！"

"爷爷……"云无咎心中一暖，却又有些愧疚。

他想说：爷爷您身体不好，不用担心我。

他想说：爷爷我不会丢下您的。

他想说：爷爷您放心，我一定会出人头地，将来让您过上好日子。

可话到嘴边，却一个字也说不出来。

老人似乎看穿了他的心思，拍了拍他的肩膀，笑道："傻孩子，哭什么？爷爷又不是要死了。高兴的事儿，哭丧着脸做什么？"

"我没哭。"云无咎别过头去，用袖子蹭了蹭眼角。

"没哭就好。"云爷爷站起身，拉着他往屋里走，"进屋说话，夜里凉，别在外头站着。"

## 二、那碗白米粥

屋内陈设简陋，一张木床、一张方桌、几把条凳、一口灶台，便是全部的家当。墙角堆着几袋粮食，袋口扎得紧紧的，那是镇上富户周老爷家送来的——今日云无咎测出五行灵根的消息传开后，周老爷便派人送了两袋米、一罐盐、几尺粗布，说是恭喜云家出了人才，略表心意。

云爷爷让他在桌边坐下，自己则走到灶台边，掀开锅盖，从里面端出一碗还冒着热气的白米粥。那粥做得很稠，米粒饱满，上面还卧着两个煎得金黄的荷包蛋。

"吃吧。"老人把碗推到他面前，"今儿测灵根，肯定累坏了吧？先垫垫肚子。"

云无咎的鼻子忽然有些发酸。

他知道这碗粥的分量。家中贫寒，米缸常年见底，细米粥通常是逢年过节才有的待遇，更别说还有两个荷包蛋。那是云爷爷攒了许久的鸡蛋，平日里一个都舍不得吃。

"爷爷，这……"

"别说了。"云爷爷打断他，重新坐回门槛上，点起旱烟，"你长大了，该往外走。爷爷不拦你。"

烟雾袅袅升起，模糊了老人的面容。

"只是记住一件事——"

云无咎抬起头。

"无论走多远，都别忘了自己是谁。修仙也好，成仙也罢，都不能丢了良心。"

老人浑浊的眼睛忽然变得清明起来，直直地望着他：

"记住了吗？"

云无咎重重地点了点头："记住了。"

"好。"云爷爷满意地笑了笑，"去吃吧。吃完早点睡，明儿开始，你便要加紧修炼了。三个月后，爷爷送你去云雾山。"

"爷爷，您不用送……"

"怎么，嫌弃爷爷老了？"老人佯怒，"你小子翅膀硬了，就不想让爷爷送你了？"

"我不是这个意思……"

"那就让爷爷送。"老人的声音忽然低了下去，带上了几分不易察觉的哽咽，"爷爷送你去云雾山，看你进天道宗的大门……这辈子，爷爷就知足了。"

云无咎望着老人那满是皱纹的脸庞，心中一阵酸涩。

他想说：爷爷您身体不好，经不起折腾。

他想说：爷爷您就在镇上等我，我一定会回来看您的。

可他知道，这些话说了也没用。老人执意要送他，他便由着他。

他端起那碗粥，一口一口地吃着。

粥很烫，热气扑在脸上，熏得他眼眶有些发热。他低着头，不敢让老人看见他的表情。

吃完最后一口，他放下碗，深深地吸了口气。

"爷爷，"他抬起头，望着老人，"您放心。我一定会好好修炼，一定会有出息的。等我在天道宗站稳脚跟，就来接您过去享福。"

"享福？"老人失笑，"爷爷这把老骨头，还享什么福？只要你好好的，爷爷就比什么都强。"

他站起身，走到墙角的柜子前，从里面取出一个小布包，递到云无咎手中。

"这是什么？"云无咎接过布包，入手沉甸甸的。

"打开看看。"

云无咎依言打开布包，只见里面整整齐齐地叠放着十几两碎银，还有几串铜钱。那是老人一辈子的积蓄——他知道，这些钱本就不多，如今全拿出来，便是他的全部家当了。

"爷爷，这钱您留着，我不能要——"

"拿着。"云爷爷打断他，"你在外面闯荡，哪有不要钱的道理？这些银子不多，可也够你路上花用的了。省着点花，别乱花钱。"

"可是爷爷，您一个人在镇上——"

"爷爷有手有脚，还饿不死。"老人摆摆手，"倒是你，一个人在外面，可别委屈了自己。吃饱穿暖，比什么都重要。"

云无咎握着那个布包，喉头哽咽，半晌说不出话来。

## 三、别了，青云镇

那一夜，云无咎很晚才睡。

躺在床上，他翻来覆去地睡不着，脑子里乱糟糟的，全是今天发生的事。

测灵根、石台奇遇、云游子前辈的传承、三个月后的入门试炼……这一切来得太快，快得让他有些措手不及。

他想起测灵根时照灵镜中那片五彩斑斓的光芒，想起中年仙师脸上那难以置信的神色，想起镇上百姓那羡慕嫉妒的目光，想起李狗蛋激动得语无伦次的模样……

五行灵根。

他真的是五行灵根。

他真的可以修仙了。

他翻了个身，将手贴在胸口——那枚玉简被他贴身收着，此刻微微发热，像是一颗小小的心脏在跳动。

《云游心法》、游子剑、云游子前辈的传承……

他不知道那位前辈口中的"仇人"究竟是谁，也不知道天道宗里究竟藏着怎样的阴谋。他只知道，从今往后，他必须变得更强。

强到足以保护云爷爷。

强到足以查清真相。

强到足以……不辜负那位前辈的期望。

他闭上眼，在黑暗中默默发誓。

三个月。

他要用这三个月的时间，将《云游心法》修炼到极致，将自己的根基打得牢牢实实。

三个月后，他要去天道宗。

去追寻前方的路。

去揭开命运的神秘面纱。

去走出一条……属于他自己的道路。

窗外，月光如水，洒在青云镇的茅屋上，洒在连绵起伏的青岚山上，洒在这片生他养他的土地上。

他在这里生活了十五年。

这里的每一棵树、每一块石头、每一条小路，他都再熟悉不过。

可从今往后，这一切都将渐行渐远。

他不知道自己什么时候才能再回来。

也许一年，也许十年，也许……永远。

他想起云爷爷方才说的那句话：*"无论走多远，都别忘了自己是谁。"*

不会忘的。

他永远不会忘记。

这里是他的根。

是他出发的地方。

也是他无论走多远，都要回来的地方。

他翻了个身，终于沉沉睡去。

那一夜，他做了一个梦。

梦中，他站在一片茫茫云海之上，衣袂飘飘，仗剑而立。

远处，有一座巍峨的山峰直插云霄，峰顶宫殿林立，仙鹤盘旋。

一名看不清面容的身影站在他对面，周身萦绕着黑色的雾气，像是一座深不可测的深渊。

那身影开口，声音阴恻恻的：

"云游子的传承者……你以为，你逃得掉吗？"

云无咎猛然惊醒。

窗外，东方已泛起鱼肚白。

他额头上满是冷汗，心跳如擂鼓。

那是梦吗？

还是……某种预兆？

他不知道。

但他知道，有些事情，比他想象中更加复杂。
