# 第一部·第二章·第七节：天道宗

## 一、山门之前

传送阵的白光散去，云泽睁开眼，发现自己已经站在了另一片天地之中。

第一眼望出去，他便怔住了。

眼前是一座巨大的山峰。

那山峰比他这辈子见过的任何山都要高、都要大、都要……不像山。它像是一柄直刺苍穹的利剑，又像是一根撑起天地的柱子，仿佛天便是被这根柱子顶着的。山体呈青黑色，岩石裸露处泛着幽幽的光泽，像是被无数年的风雨打磨过的铠甲。山腰以上全被乳白色的云雾遮住，只隐约能看见几处飞檐翘角的宫殿建筑从云层中探出来，红墙绿瓦，金碧辉煌，恍若传说中天宫的景象。

而最令人震撼的，是那股灵气。

空气仿佛都是甜的。

不是真的甜，而是一种难以形容的舒适感。每一口呼吸都像是在饮用甘泉，每一个毛孔都在贪婪地舒张。云泽能清晰地感觉到，周遭的空气中漂浮着无数细小的光点——那是灵气，是天地间最纯粹的能量。它们如游鱼戏水，如蜂蝶采蜜，在他的周身缓缓流转，顺着他的呼吸渗入他的皮肤，渗入他的血肉，渗入他的骨髓。

《云游心法》在体内不由自主地运转起来。

那感觉太过美妙了。

平时在青云镇修炼时，感应灵气需要静心凝神许久才能捕捉到几缕若有若无的气息；而在这里，灵气到处都是，多得像空气一样，伸手便能抓一大把。他的丹田处微微发热，那缕原本纤细如丝的气流，在浓郁灵气的滋养下正缓缓壮大，像是久旱逢甘霖的禾苗，拼命汲取着水分。

太舒服了。舒服得他几乎想要就地躺下，好好睡一觉。

"小家伙，这地方的灵气浓郁得吓人……"

那道苍老而温和的声音忽然在他脑海中响起。

云泽的身体僵住了。

那是云游子前辈的声音。

他的心脏猛地揪紧，鼻腔一酸，眼眶倏地发热。前辈的残魂不是已经消散了吗？那道守护了他数百年、最终化为星尘而去的身影，此刻却仿佛又出现在了他面前，用那熟悉的语气在他耳边低语。

可他知道那只是幻觉。

前辈已经不在了。那个在断崖石台上盘膝枯坐了三百年、只为等待一个有缘人出现的青袍剑客，已经彻底消散在了天地之间。

他深吸一口气，将那股酸涩压回心底。

前辈不在了，但他留下的剑还在、心法还在、遗志还在。这就够了。

"无咎哥，你怎么了？"

李狗蛋的声音从旁边传来，带着几分担忧。

云泽回过神来，发现自己不知何时已经红了眼眶。他连忙别过头去，用袖子飞快地蹭了一下眼角，装作是被风沙迷了眼的样子。

"没什么。"他的声音有些哑，"就是……这里的风有点大。"

李狗蛋狐疑地看了他一眼，却也没多问。他自己正忙着四处张望呢——这座山峰、这片广场、这些云雾，对于一个从没出过远门的乡下少年来说，每一样都是这辈子没见过的新鲜事物。

张小胖则已经兴奋得不能自已，两只眼睛瞪得溜圆，嘴里不停地"哇"、"啊"、"天哪"地叫唤，活像个没见过世面的刘姥姥。

"这就是天道宗？"李狗蛋喃喃道，仰着头望着那云雾深处的宫殿，嘴巴张得能塞进去一个鸡蛋。

"这还不是天道宗本体，"张小胖从旁插嘴，声音里带着一丝敬畏，"这只是山门外的接引广场。真正的天道宗在上面呢，被护山大阵遮着，咱们这些外人根本上不去。等成了正式弟子，才能进去。"

云泽没有说话。

他只是静静地打量着四周，将这里的一切刻入脑海。

广场极为开阔，方圆足有数十丈，地面铺着巨大的青石板，每一块都打磨得平平整整，接缝处严丝合缝，踩上去连脚都感觉不到凹凸。广场的边缘种着一圈古松，那些松树粗得要两三人才能合抱，枝干遒劲，针叶繁茂，树龄怕是已有数百年之久。松树下立着几块巨大的石碑，碑身乌黑如墨，上面刻着密密麻麻的金色文字，在阳光下泛着幽幽的光芒。

广场上已经聚集了上千名少年男女。

年龄大多在十二到十八岁之间，穿着各异，气质各异。

有的穿着华贵的锦袍，衣料轻薄如纱，袖口领口绣着精致的花纹，身边还跟着挑着行李的仆从，一看便是世家大族出来的子弟。那锦衣少年昂首阔步地走在广场上，下巴扬得高高的，目不斜视，仿佛脚下的青石板都是为他铺的一般。

有的穿着朴素的布衣，背上背着一个瘪瘪的包袱，身边空无一人，走路时还时不时回头张望，生怕跟丢了什么似的。这些是寒门子弟，大多孤身一人闯荡，脸上写满了紧张与不安。

有的三五成群地聚在一起低声交谈，时不时发出一阵笑声，显然是来自同一个地方的同乡；有的独自站在角落里，双手紧紧攥着衣角，嘴唇抿成一条线，显是在极力压抑着内心的紧张。

还有的……

云泽的目光忽然一凝。

广场的另一侧，站着七八个身穿统一青袍的少年。那青袍的款式与来接引他们的萧云所穿极为相似，只是布料和做工稍逊一筹，腰间也没有挂玉牌。他们站成一个扇形，将一个锦衣少年护在中央。

那锦衣少年……

云泽认出来了。

是张灵轩。

昨日在云来镇上见过的那位张家少爷。此刻他身边围了一圈人，明显是张家派来护送的随从。

似乎感应到了云泽的目光，张灵轩转过头来，朝他这边看了一眼。

两道目光隔空相遇。

张灵轩的嘴角微微勾起，朝他点了点头——和昨日在街上那个点头如出一辙，淡淡的、礼貌的，带着几分世家子弟特有的从容。

云泽也朝他点了点头，然后便移开了目光。

"那谁啊？"李狗蛋凑过来，压低声音问道，"无咎哥，你怎么老跟那人点头？"

"不认识。"云泽说，"只是见过一面。"

"见过一面就点头打招呼？无咎哥，你这可不像是你的作风啊……"

"多一个朋友总比多一个敌人强。"云泽淡淡道。

李狗蛋眨了眨眼，似懂非懂地点了点头。

张小胖在一旁嗤笑一声："得了吧，你们俩就是想太多。什么朋友不朋友的，等进了天道宗，谁管你是谁？到时候都是竞争关系，一个比一个恨不得把对方踩下去。"

这话虽然刻薄，却也不无道理。

云泽没有反驳，只是默默将目光投向了那座云雾缭绕的山峰。

三千多人，只录取一百人。

这意味着，广场上至少有两千九百人，将在三关试炼之后被淘汰出局。

而他，必须在这一百人之中。

## 二、接引弟子

"各位参加收徒大会的弟子，请到石碑前登记。"

一个清朗的声音在广场上空响起。

众人循声望去，纷纷踮起脚尖往声音传来的方向张望。

只见云雾深处，一个身影正缓缓飘来。

那是一个身着青色道袍的年轻修士。他踏云而来，足尖轻点地面，周身萦绕着淡淡的白雾，身形潇洒如风。那年轻修士约莫二十岁出头，面容清俊，剑眉入鬓，一双眼睛清澈如水，却又深不见底。他的头发用一根玉簪束在脑后，腰间挂着一枚羊脂玉牌，玉牌上刻着一个古朴的"萧"字，在阳光下熠熠生辉。

他稳稳地落在广场中央的石台上，衣袂飘飘，像是画中走出的仙人。

"在下天道宗内门弟子萧云，今日负责接引诸位。"那年轻修士朝众人拱了拱手，脸上带着温和的笑容，"诸位远道而来，辛苦了。在入门试炼开始之前，请先到那几块石碑前登记个人信息。"

他的声音不高，却清晰地传入每个人的耳中，像是在耳边说话一样，毫无费力之感。单是这一手，便让台下的少年们齐齐吸了一口凉气——这便是修仙者的本事吗？光凭声音便能压住上千人的嘈杂？

李狗蛋张大了嘴巴，半晌才合拢来，小声嘀咕道："俺滴个亲娘，这人……是人还是仙啊……"

张小胖白了他一眼："废话，人家都说了是天道宗内门弟子，当然是仙了。"

"俺就是没见过这么像仙的人嘛……"

云泽默默打量着萧云。

这位内门弟子的修为深不可测。他看不出对方是什么境界，只觉得那双眼睛像是两口古井，深邃而平静，仿佛无论丢进去什么东西都会无声无息地沉没。这种感觉他曾在云游子前辈的残魂身上感受到过——那是经历过无数风雨之后沉淀下来的平静，是"曾经沧海难为水"的超然。

强者。

这个萧云，是个真正的强者。

众人的目光随着萧云的视线望去，只见广场边缘整整齐齐地排列着八块石碑。那些石碑高约一丈，宽约三尺，通体乌黑如墨，表面光滑如镜。碑身上刻着密密麻麻的金色符文，符文之间有一个手掌大小的凹陷掌印，掌印下方还有一个更小的凹槽，像是一个放东西的地方。

"那石碑是做什么用的？"人群中有人小声问道。

"那是身份识别碑，"萧云解释道，脸上依然挂着那副温和的笑容，"诸位只需将手掌按在掌印上，再将灵根测试时拿到的令牌放入凹槽，石碑便会自动记录诸位的信息。"

"这也太神奇了吧……"李狗蛋又是一阵惊叹。

张小胖扯了扯他的袖子："别大惊小怪的，给人笑话。"

云泽低头看了看自己手中的令牌。

那是一枚巴掌大小的木牌，正面刻着"天道宗收徒大会"六个字，背面刻着他自己的名字、籍贯和灵根信息。这枚令牌是当初在青云镇测灵根时周仙师发给他的，说是参加收徒大会的唯一凭证，没有它便进不了天道宗的大门。

他将令牌小心翼翼地揣在怀里，跟在人群后面，缓缓朝石碑走去。

队伍很长，排在他前面的是数十名少年男女，年龄都和他差不多，有的好奇地打量着石碑，有的紧张地攥着手中的令牌，有的则在低声交谈，互相打探对方是哪里人、测出了什么灵根。

"我是临安府的，火属性中品灵根。"

"我是清河县的，水属性下品。"

"唉，我是土属性下品，怕是没什么希望了……"

"别灰心，听说天道宗收徒不只是看灵根，心性和悟性也很重要。"

"真的吗？"

"当然是真的，我有个远房亲戚的表哥，三年前参加过，他说……"

云泽默默听着这些对话，心中暗暗记下。

修仙界果然是一个看天赋的地方。这些少年千里迢迢赶来，有的甚至走了一两个月才到云来镇，一个个满怀希望，削尖了脑袋想往天道宗里钻。可最后能进去的，不过是凤毛麟角的那么几个人。

更多的人，只是一次又一次地碰壁，一次又一次地失望，最终带着落寞回到故乡，继续过那面朝黄土背朝天的日子。

这就是修仙界的残酷。

但他不打算做那大多数人。

队伍缓缓前移，很快就轮到了他。

云泽走上前去，将手中的令牌放入石碑下方的凹槽中。令牌大小刚刚好，严丝合缝地嵌了进去，与石碑融为一体。

然后，他将右手按在了掌印之上。

掌印冰凉，入手如同触着一块寒玉。他能感觉到掌印下方有什么东西在涌动，像是在读取他手掌上的纹路、读取他体内的灵气信息。

一息。

两息。

三息。

石碑忽然亮了起来。

不是普通的光——那光芒五彩斑斓，交织缠绕，仿佛有五种颜色在他掌心同时绽放。金色、青色、蓝色、红色、黄色，五行之色，轮番流转，最终缓缓融合在一起，化作一片柔和的混沌之光。

那混沌之光越来越亮，越来越盛，最终竟透出碑面，将云泽的整只手掌都笼罩其中！

"咦？"

旁边传来萧云的一声轻咦。

云泽抬起头，看见那位内门弟子的眼中闪过一丝讶异——那是很少见到的情绪。萧云从头到尾都一副云淡风轻的模样，无论看到什么都不动声色，可此刻却因为他的灵根而露出了片刻的动容。

"五行俱全？"萧云的声音微微一提，"中品之上？"

周围的少年们也纷纷投来目光。

"五行灵根？那岂不是金木水火土全有？"

"五种属性都能修炼？这……这也太离谱了吧？"

"俺听说过五行灵根，据说万中无一，整个中州几十年都不一定能出一个！"

"可五行灵根不是出了名的难修炼吗？博而不精，什么都沾一点，什么都不精，进境慢得要死……"

"就是就是，贪多嚼不烂，还不如单灵根实在。"

"那位师兄都测出上品单灵根了，不还是乖乖在这儿排队？五行灵根有什么了不起的……"

云泽没有理会那些议论。

他从石碑前退开，将令牌取出，转身离开。

他的脚步平稳，神色从容，仿佛刚才那番动静与他毫无关系。

但在心底，他却暗暗思忖：云游子前辈说过，五行灵根确实难修炼，需要同时兼顾五种属性的平衡，比单灵根要慢上许多。但前辈也说过，五行灵根并非没有优势——当五行平衡之后，修炼任何属性的功法都能事半功倍，而且体内灵气远比单灵根浑厚，持久战中占据绝对优势。

所谓"博而不精"，不过是那些不懂五行灵根奥妙之人的偏见罢了。

他不需要向任何人解释。

修仙这条路，从来不是看别人怎么说，而是看自己怎么走。

## 三、山路见闻

登记完成后，萧云带着众人沿着山路往上走。

从广场到住处的这段路，是云泽第一次近距离接触天道宗的外围。

山路是用青石板铺成的，宽约两丈，平整得像是用尺子量过。石板与石板之间的缝隙极小，踩上去连脚都感觉不到凹凸，显然是被人用法术打磨过的。山路两侧是茂密的树林，那些树和青云镇后山的树完全不同——树干粗壮得要三四人才能合抱，树皮呈青黑色，表面隐隐泛着光泽，像是涂了一层油脂；枝叶繁茂得遮天蔽日，即便头顶烈日，走在这山路上也感觉不到多少暑意；最奇异的是，那些树叶在微风中轻轻摇曳时，竟会发出细微的声响，像是有人在低声吟唱。

空气中弥漫着一股浓郁的草木清香，混着泥土的腥味和某种说不清道不明的甜意。那是灵气充沛之地特有的味道，是天地灵气滋养万物的证明。

云泽一边走，一边默默感受着周遭的一切。

他发现了一件奇异的事——这里的树木、草丛、甚至脚下的泥土，都隐隐泛着微弱的光泽。那些光泽肉眼几乎看不见，但在他的灵觉感知中，却清晰得如同夜空中的繁星。

这就是仙家福地吗？

《云游心法》在体内不由自主地运转着，贪婪地吸纳着周围的灵气。那感觉太舒服了，舒服得他几乎想要放慢脚步，好好享受一番。但他知道现在不是时候——他必须集中精神，记住所经过的每一个岔路口、每一处标志物，为日后的行动做准备。

走在队伍前面的是萧云。他走在最前方，步伐不快不慢，却始终与众人保持着固定的距离。他没有回头，也不说话，只是偶尔抬手指一指前方的景物，向众人介绍一二。

"那座山峰叫青冥峰，是天道宗外门弟子的主要活动区域。"

"那片湖叫聚灵湖，湖水由天地灵气凝聚而成，弟子们平日可在湖边打坐修炼。"

"那座塔叫试炼塔，是外门弟子试炼之地，共有七层，每一层都有不同的考验。"

众人听得如痴如醉，眼睛都不够用了。

张小胖趴在路边的一棵古树上，两眼放光："这树得有多少年了？俺爹要是能有一棵这样的树，俺家就发了！"

李狗蛋则蹲在聚灵湖边，伸手去掬水。那湖水清澈见底，泛着淡淡的蓝光，触手冰凉，却又带着一丝温润——像是握着一块温热的玉石。他喝了一口，眼睛顿时亮了："这水好甜！比俺家的井水好喝一百倍！"

"别乱喝！"萧云的声音从前方传来，带着几分无奈，"聚灵湖的水蕴含灵气，没修炼过的人喝多了会肚子疼。"

李狗蛋吓得连忙松手，抹了抹嘴，一脸后怕。

云泽忍不住勾了勾嘴角。

这两个活宝，真是到哪儿都不消停。

又走了一盏茶的功夫，众人终于到达了住处。

那是一片依山而建的院落群，青砖灰瓦，排列整齐，每一排房屋前都种着几株翠竹，微风吹过，竹叶沙沙作响，倒是个清净所在。院落的尽头是一道石门，门楣上刻着"接引居"三个字，笔力遒劲，隐隐透着一股肃杀之气。

"这里是天道宗专门为参加收徒大会的弟子准备的临时住处，"萧云转过身来，对众人说，"每位弟子一间房，房门上有编号，找到自己的编号入住即可。"

"今晚好好休息，明日一早，入门试炼正式开始。"

"入门试炼分为三个环节：心性测试、悟性测试、以及最终的真传试炼。三个环节综合评分，达到标准的弟子，将正式成为天道宗的外门弟子。"

"未能通过的弟子，也不必灰心。天道宗每三年举办一次收徒大会，明年还有机会。"

"好了，诸位早些休息吧。"

说罢，他转身踏云而去，眨眼间便消失在云雾深处。

众少年面面相觑，随即一哄而散，纷纷去找自己的房间。

云泽找到了属于自己的那间——"戊字房第三十七号"。推门进去，里面陈设简单却干净：一张木板床，一张书桌，一把椅子，墙角还有一个洗漱架，架上放着铜盆和巾帕。窗户朝南，透过窗纸能看见外面的翠竹和远山，视野极佳。

他将包袱放在床上，又将游子剑从包袱里取出来，小心地放在枕边。

然后他盘膝坐在床上，闭上眼睛，开始修炼。

浓郁的灵气在体内缓缓流转，《云游心法》的心诀在脑海中默默浮现。他能感觉到丹田中的气流正在以肉眼可见的速度壮大，比在青云镇时快了何止十倍。

照这样的速度，明日的试炼，他信心十足。
