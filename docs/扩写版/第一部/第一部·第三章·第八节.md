# 第一部·第三章·第八节：青岚山·故土

## 一、任务堂

入门第三个月的第一天清晨，天色尚早，窗外的天际才刚刚泛起一线鱼肚白。

云泽便已穿戴整齐，出了弟子房，沿着青石小径向青云峰半山腰的任务堂走去。

清晨的山风带着几分凉意，吹得他衣袂轻轻飘动。空气中弥漫着草木的清香，混着远处炊烟的气息，是他进入天道宗三个月来已经渐渐习惯的味道。一路上遇到不少同样早起的外门弟子，有的行色匆匆赶往藏经阁，有的三五成群地聊着天，还有的干脆找了个僻静的角落盘膝修炼。

天道宗的清晨，和青云镇的清晨截然不同。

在青云镇的时候，这个时辰他应该已经背着柴刀上山了；而在天道宗，同龄的少年们还在睡眼惺忪地打哈欠。

云泽微微一笑，加快了脚步。

任务堂是一座三层高的木楼，坐落在青云峰半山腰的一片平地上，位置极佳，交通便利。从这里出发，无论去往哪个方向的山峰，都十分方便。

楼门口挂着一块匾额，上书"任务堂"三个大字，笔力遒劲，隐隐透着一股庄严肃穆的气势。门前立着两根旗杆，足有丈许之高，旗杆上飘着两面白色的旗帜，旗帜上绣着天道宗的标志——一柄横置的剑与一本翻开的书交叉在一起，在晨风中猎猎作响。

云泽推开沉重的木门，走了进去。

楼内灯火通明，人声鼎沸。

一进门，便是一块巨大的石碑。

那石碑高约两丈，宽约三丈，表面光滑如镜，在长明灯的照耀下泛着淡淡的青光。石碑上密密麻麻地刻着各种任务信息——有采集灵草的，有猎杀妖兽的，有护送商队的，有送信跑腿的，不一而足。每一条任务下面都标注着难度等级、报酬数额和截止日期，密密麻麻，看得人眼花缭乱。

石碑旁边围着不少弟子，有的在仔细查看任务，有的在低声商议策略，还有的已经撕下了自己看中的任务条，准备去柜台登记。人群中时不时传来几声争执，似乎是为了某一任务而起。

云泽穿过人群，走到石碑前，目光在那些任务信息上逐一扫过。

他的目标是采集灵草的任务。

修炼《云游心法》需要大量的灵气支撑，而天道宗每月发放的那点灵石远远不够他用。散修出身的他，没有任何家底可以依仗，唯一的办法就是自己想办法赚取修炼资源。而采集灵草，是最简单、最稳妥的选择——不需要太多战斗力，只需要熟悉山林、懂得辨别灵药，便能有所收获。

石碑上的任务很多，但大多已经被别人挑走了。剩下的要么太简单，报酬太低；要么太难，风险太大；还有一些则是地点太远，或者时间要求太紧，不适合他目前的修为。

就在他准备换个方向寻找的时候，一条任务信息忽然映入眼帘。

他的目光骤然停住了。

"采集青岚草，地点：青岚山深处，数量：十株，报酬：五十灵石。"

青岚草。

青岚山。

云泽的瞳孔微微一缩。

青岚山，正是他出生的地方——青云镇后山的那座大山。那座他从小砍柴、采药、捉虫、掏鸟窝的大山；那座他闭着眼睛都能找到每一条小径、每一块石头的大山；那座埋藏着他所有童年记忆的大山。

时隔数月，他终于要再次踏上那座山了。

那里有云爷爷的坟茔。

那里有他和狗蛋一起长大的回忆。

那里有他作为凡人的一切过往。

他没有犹豫，伸手便将那张任务条撕了下来。

## 二、准备

柜台前排着长长的队伍。

云泽等了约莫一刻钟，才终于轮到他。

"采集青岚草？"柜台后的执事弟子接过任务条，抬眼打量了他一下，"师弟，你是新入门的外门弟子，按宗门规定，不满半年不得接取深山任务。青岚山深处灵气浓郁，妖兽横行，你一个人去太危险了。"

"师兄，"云泽平静地说，"我有不得不去的理由。"

"什么理由？"执事弟子放下笔，神色认真了几分。

云泽沉默了片刻。

"家在那里。"

他的声音很轻，却透着一股不容置疑的坚定。

"家在那里？"执事弟子愣了一下，"你是青云镇人？"

"嗯。"云泽点头，"我是青云镇人，青岚山是我从小砍柴的地方。我对那座山很熟悉，知道哪里能去，哪里不能去。"

他说的是实话。

那座山的每一寸土地他都踏过，每一个角落他都了如指掌。他知道哪里有条小溪可以取水，知道哪里有个山洞可以避雨，知道哪棵老树下藏着一窝山雀蛋，知道哪块石头底下住着一只成了精的穿山甲。

那些记忆，刻在他骨血里，比任何地图都要清晰。

执事弟子沉默了片刻。

他看了看云泽的眼睛，那双眼睛平静而清澈，没有初入门弟子的惶恐与青涩，反而透着一股子与年龄不符的沉稳。那不是一个普通少年该有的眼神——那是经历过什么之后才会有的眼神。

"好吧。"他最终还是点了点头，在任务令牌的背面盖了个章，"既然你对那座山熟悉，我就不多劝了。但是师弟要记住，青岚山深处和你小时候砍柴的地方不一样。那里的妖兽可不是闹着玩的，遇上了一星妖兽，连逃都逃不掉。"

"多谢师兄提醒。"云泽郑重拱手。

他接过任务令牌，将其贴身收好，转身离去。

身后，执事弟子的目光在他背上停留了片刻，轻轻摇了摇头。

"又一个不知天高地厚的小子……但愿他能全须全尾地回来吧。"

## 三、探望

出了任务堂，云泽没有直接回弟子房。

他先拐了个弯，去了一趟杂役堂。

杂役堂位于青云峰的后山，是专门安置那些试炼失败或触犯宗规的弟子的地方。这里的弟子没有资格修炼高深功法，只能做一些洒扫、搬运、劈柴、挑水的粗活，日复一日，年复一年，直到被宗门遗忘。

云泽沿着一条僻静的山路走来，远远地便看见了几个灰衣弟子在院子里忙碌的身影。

其中一个瘦削的背影，正在弯腰搬运一捆木柴。

那背影瘦了一大圈，原本圆滚滚的脸颊如今凹陷下去，皮肤也晒得黝黑发亮，脊背微微弓着，像是被什么东西压弯了似的。哪里还有半点当年在青云镇时那个整天跟在他屁股后面跑、嘴里嚼着烤红薯、笑得没心没肺的少年的模样。

"狗蛋。"云泽轻声唤道。

李狗蛋浑身一震，连忙转过身来。

当他看清来人是谁时，那双黯淡无光的眼睛顿时亮了起来。

"云兄弟！"他扔下手中的柴火，小跑着迎上来，"你咋来了？"

"来看看你。"云泽拍了拍他的肩膀，注意到那肩膀比以前单薄了许多，"这些日子过得怎么样？"

"还行还行……"李狗蛋勉强挤出一个笑容，却怎么也掩盖不住眼底的疲惫与落寞，"就是活儿累了点，不过俺习惯了。"

云泽看着他，心中一阵酸涩。

他看得出来，李狗蛋过得并不好。

堂堂仙门试炼失败者，被发配到杂役堂做下人，这种落差，换作是谁都难以承受。更何况是狗蛋这个从小没吃过什么苦、没受过什么委屈的直肠子汉子。

"狗蛋，"他从怀中取出一个布包，递到李狗蛋手中，"这是我攒下的一些灵石，你留着。"

"这……这是……"李狗蛋打开布包一角，只见里面躺着二十几枚晶莹剔透的灵石，在阳光下闪烁着温润的光芒，顿时瞪大了眼睛，"这太贵重了！云兄弟，你自己修炼要紧，俺不能要！"

"拿着。"云泽按住他的手，不容拒绝，"我在宗门里用不上这些俗物，你比我更需要。"

其实他每个月领的灵石也就那么点，这些是他这三个月从牙缝里省下来的。但他不想说这些，说出来反而像是在邀功。

李狗蛋的眼眶红了。

他张了张嘴，想说什么，喉咙里却像是堵了一团棉花，一个字也吐不出来。

良久，他重重地跪了下去。

"云兄弟！"他抬起头，泪水在眼眶里打转，却倔强地不肯落下，"俺李狗蛋这辈子没什么出息，但俺记得你的恩情！你放心，俺在这里一定好好干活，绝不给你丢脸！"

"起来。"云泽将他扶起，"地上凉，别跪坏了膝盖。"

他扶着李狗蛋的肩膀，目光认真地看着他。

"狗蛋，你记住，今日的失败不代表什么。只要你不放弃，总有一天，你还能站起来。"

"总有一天？"李狗蛋苦笑，眼中的光芒黯淡了几分，"俺的资质……你又不是不知道。连灵根都没测出来，这辈子还能有什么指望？"

"资质算什么？"云泽打断他，声音不大，却掷地有声，"修仙界中，多少天纵奇才半路陨落，又有多少资质平平之辈逆天改命？真正决定一个人能走多远的，从来不是资质，而是心性。"

"你的心性，我了解。你缺的，只是一个机会。"

他松开手，退后一步，目光坚定如磐石。

"等着我，狗蛋。"

三个字，重若千钧。

李狗蛋浑身一震。

他望着云泽的眼睛，只觉得那双眼睛里藏着的东西，是他这辈子从未见过的——那是光，是火，是不甘平庸的意志，是敢与天公试比高的豪气。那不是一个农村少年该有的眼神，那是……那是仙人的眼神。

"好！"他重重地点头，声音沙哑却坚定，"俺等着你！"

"俺李狗蛋，等着你回来！"

云泽微微一笑，转身离去。

他没有回头。

因为他怕一回头，就会看见狗蛋那张写满期盼的脸，而他现在什么都给不了他。

他只有变强。

变得足够强。

强到有一天能够兑现这个承诺的那一天。

## 四、再临青岚山

翌日清晨，天边刚泛起一线金红。

云泽背着行囊，独自一人出了天道宗山门。

他的步子很快，快得像是在赶路，又像是在逃命。脚下的山路弯弯曲曲，像是一条灰白的带子，在山间蜿蜒曲折。路两旁是层叠的梯田，田里的水已经放干了，泥土龟裂，踩上去硬邦邦的。

青岚山距离天道宗并不远，翻过两座小山，再走半日山路，便能到达。

云泽一路疾行，午时刚过，便已站在了青岚山的山脚下。

山风迎面吹来，带着一股熟悉的气息。

泥土、青草、枯叶、还有若有若无的松脂香——这是他从小闻到大的味道，刻在骨子里，融在血液中。无论走到哪里，只要闻到这股气息，他便知道，自己回家了。

他抬起头，望向那座巍峨的大山。

青岚山还是那座青岚山，峰峦叠嶂，云雾缭绕，像是一位披着轻纱的巨人，静静地盘踞在大地之上。山顶的那几块奇石还是那个模样，远远望去，像是一群翘首以盼的老人，在等待着什么人归来。

他的鼻子忽然有些发酸。

他想起了云爷爷。

想起老人坐在门槛上抽旱烟的模样；想起老人半夜腰疾发作时强忍的呻吟；想起老人把舍不得吃的鸡蛋省下来给他吃；想起老人送他离开时说的那句话——

"无论走多远，别忘了家在哪里。"

他深吸一口气，将那股酸涩压下去，迈步登山。

从山脚到山腰的路他已经非常熟悉了。这条路他从小走到大，闭着眼睛都能找到哪里有坑、哪里有坎、哪里该拐弯。

一路走来，风景依旧。

那片野菊花还是开得那么盛，金黄色的花瓣在阳光下舒展开来，像是一颗颗小小的太阳，在秋风中轻轻摇曳。那块他小时候经常坐在上面发呆的大青石还是静静地蹲在那里，石面上被他屁股磨得光溜溜的，在阳光下泛着青光。石缝里的那株野山参果然又长大了一圈，叶片由三片变成了五片，颜色也更加翠绿了。

只是，路还是那条路，山还是那座山，人却已经不是当年那个人了。

曾经，他只是一个砍柴的少年，每日重复着同样的事情——上山、砍柴、下山、卖柴、回家。日复一日，年复一年，看不到尽头，也看不到希望。那时候他以为，这辈子就会这样过下去，直到老去，直到死去，像祖祖辈辈一样。

而现在，他是一名修士了。

炼气境一层。

虽然只是修仙路上最微不足道的一步，却已经让他看到了不一样的风景。他知道了这个世界上有灵气、有境界、有法术、有仙魔之争。他站在了这条路的起点，虽然前方还是一片迷雾，但他知道，只要一直走下去，总有一天会拨云见日。

他继续向上攀登。

越往高处，灵气便越浓郁。

青岚山深处果然不同于外围。这里的灵气浓郁得几乎凝成了实质，呼吸间便有丝丝缕缕的灵气涌入体内，让人心旷神怡，仿佛置身于灵气的海洋之中。与此同时，妖兽的踪迹也多了起来。

路边不时能看到一些妖兽留下的粪便和爪印。那些爪印有大有小，深深浅浅地印在泥土上，有些还很新鲜，边缘的泥土尚未完全干透，显然是不久前才留下的。

云泽提高了警惕，手始终按在游子剑的剑柄上，随时准备应对可能出现的危险。

他沿着一条隐蔽的小路向山中进发，一路走走停停，仔细搜索着青岚草的踪迹。

青岚草是一种常见的灵药，生长在阴凉潮湿的地方，通常长在岩石缝里或者大树的根部。它通体翠绿，叶片上有一层淡淡的白霜，在阳光下会发出幽幽的光芒。这种灵药可以用来炼制培元丹，对于炼气境修士的修炼有不错的辅助效果，因此在宗门内颇受欢迎。

找了大约一个时辰，他终于在一处悬崖下的岩缝里发现了青岚草的踪迹。

那些青岚草长得很茂盛，一丛一丛的，像是一簇簇绿色的火焰在岩石上燃烧。叶片翠绿欲滴，上面的白霜在阳光下闪闪发光，像是被谁撒上了一层细细的盐粒。

"找到了。"

云泽走上前去，蹲下身来。

他伸出手，轻轻拨开岩缝周围的杂草，仔细打量着那些青岚草的长势。每一株都很健康，根系发达，叶片饱满，没有虫蛀的痕迹。他满意地点了点头，开始小心翼翼地将那些青岚草连根拔起，放进背后药篓的软布上。

一株、两株、三株……

他一边采集，一边数着，动作轻柔而仔细，像是在对待什么珍贵的宝物。

十株。

够了。

他将最后一株青岚草小心翼翼地放进药篓，正准备起身离开，忽然——

一阵震天的兽吼从不远处传来！

"吼——！！！"

那声音如同雷霆炸响，震得崖壁上的碎石簌簌落下，地面都在微微颤抖。树叶被声波震得纷纷飘落，像是一场突如其来的叶雨。

云泽脸色骤变。

他猛地站起身来，手已经握上了游子剑的剑柄。

那声音他太熟悉了——那是青鬃野牛的吼声。

在青云镇的时候，他不止一次听到过这种声音。那是一种足以让任何猎人心惊胆战的猛兽，以皮糙肉厚、力大无穷著称。寻常的青鬃野牛就已经够可怕了，而这种栖息在深山里的青鬃野牛，据说比外围的还要凶悍十倍。

还没等他回过神来，树丛中便传来一阵剧烈的晃动。

紧接着，一头足有两人高的巨大身影从林木间冲了出来！
