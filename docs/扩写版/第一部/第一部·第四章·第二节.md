# 第一部·第四章·第二节

## 夜入剑中·故人遗音

入夜，青云峰渐渐安静下来。

外门弟子们陆续回到各自的住处，山峰之上只剩虫鸣与风声。云无咎的住处是一间简陋的木屋，位于青云峰半山腰的一处僻静角落，远离其他弟子的居所。这倒不是因为宗门特别照顾他，而是因为他来得晚，好位置早就被人占完了。

对此，云无咎倒也乐得清静。

他盘膝坐于榻上，将游子剑横放于膝前，深吸一口气，缓缓闭上眼睛。

意识一点一点沉入剑中。

这是一种很奇妙的感觉。最初发现这个能力时，云无咎也吓了一跳——他从未想过，自己竟然能够将意识沉入一柄剑中，与其中的"残念"进行交流。

后来他才渐渐明白，这应该是云游子前辈刻意留下的机制。那位曾经名震修仙界的高手，在临死前将自己的一缕残魂封入剑中，等待有缘人前来继承他的衣钵。而云无咎，恰好就是那个"有缘人"。

意识穿过一层朦胧的光幕，云无咎来到了一个灰蒙蒙的空间之中。

这里没有天，没有地，四周只有无尽的迷雾与淡淡的雾气。空气中弥漫着一股苍老而疲惫的气息，像是一间许久未曾打开的旧屋，积满了岁月的尘埃。

在空间的正中央，一道虚幻的人影盘膝而坐。

那是一个老人的身影，面容模糊，五官难辨，只能依稀看出一个轮廓。他的身形佝偻，周身萦绕着淡淡的光芒，整个人散发着一种行将就木的萧索之感。

"前辈。"

云无咎上前一步，躬身行礼。

云游子的残念缓缓睁开双眼。那双眼睛浑浊而深邃，像是两口望不见底的古井，其中藏着太多太多不为人知的故事。

"无咎。"

云游子的声音沙哑而虚弱，像是从很远很远的地方传来，又像是风中将熄的烛火，随时都可能消散。

"你来了。"

"前辈，您今日感觉如何？"云无咎问道，语气中带着几分担忧。

三个月来，他与这位剑中前辈交流过许多次。起初，他还称呼对方为"前辈"，带着几分敬畏与拘谨。但随着交流的深入，他渐渐发现，云游子虽然修为高深，却并没有架子，反而像是一位慈祥的长辈，愿意倾囊相授。

只是，这位长辈的身体状况，着实令人担忧。

每一次见面，云游子的身影都比上一次更加虚淡，那层萦绕在他周身的光芒也更加黯淡。他就像一盏即将燃尽的油灯，灯油越来越少，火焰越来越微弱。

"老朽还撑得住。"

云游子淡淡一笑，那笑容中带着几分苦涩与释然。

"不必为老朽担心。我这缕残魂，本就是强撑至今，能见到你，已是老天开眼。"

他顿了顿，看向云无咎："今日叫你来，是有一件事要说。"

云无咎心中一动，想起今日在峰顶发生的事："前辈说的是……林惊羽师兄？"

云游子微微点头。

"他看到了剑柄上的铭文。"

"是。"云无咎如实道，"那个字……像是'叶'字。"

"'叶'……"

云游子的声音忽然低了下去，像是在回忆着什么遥远的往事。

那道人影微微颤抖，周身的光芒也随之波动起来。

"无咎，你可知道，叶孤城是谁？"

云无咎摇头。

他只是一个出身穷乡僻壤的少年，所知所学，皆来自云游子前辈留下的玉简，以及这几个月在宗门中的耳濡目染。叶孤城这个名字，他从未听说过。

"叶孤城……"

云游子的声音中带着一丝感慨，一丝敬仰，还有一丝说不清道不明的悲伤。

"他是太古年间，万古以来的剑道第一人。"

"万古第一剑？"

云无咎瞪大了眼睛。

万古以来，剑道高手如云，能人辈出，但被称为"万古第一"的，却是凤毛麟角。据他所知，即便是如今修仙界最负盛名的几位剑道宗师，也无人敢自称"万古第一"。

而这个叶孤城，竟然能得此称号？

"不错。"云游子似乎看出了他心中的震惊，淡淡道，"叶孤城出身平凡，却天赋绝世，年仅三十便已证道成帝，成为那个时代最强的存在。他一生征战无数，未尝一败，更是以一己之力，封印混沌魔兽饕餮，拯救三界于危亡之际。"

"饕餮？"云无咎心中一震，"混沌魔兽饕餮？"

他曾在《修仙杂谈》中读到过关于混沌魔兽的记载。那是远古时代的恐怖存在，据说拥有毁天灭地之能，每一次现身，都会给三界带来滔天大祸。

"不错。"云游子的声音越发低沉，"饕餮是混沌七魔兽之一，性情凶残，嗜杀成性，曾在太古年间掀起一场浩劫，生灵涂炭，血流成河。彼时三界强者联手，都无法将其制服。最后，是叶孤城挺身而出，以绝世剑道将其封印，拯救了无数生灵。"

"那一战，惊天地，泣鬼神。叶孤城以一剑之力，独战饕餮七天七夜，最终虽然将其封印，自己却也油尽灯枯，陨落于九天之上。"

云无咎听得心驰神往。

一个能以一人之力封印混沌魔兽的存在，那是何等的伟岸与强大？那是何等的气魄与胸襟？

"前辈，"他忍不住问道，"您与这位叶剑圣，是什么关系？"

云游子沉默片刻。

那模糊的人影微微低头，像是在掩饰什么。

"关系……"

他的声音忽然变得有些飘忽。

"叶孤城一生铸剑无数，其中最得意的作品，共有三柄。"

"三柄？"

"其一为'天道剑'，乃叶孤城毕生心血所铸，剑成之时，天降异象，万道来朝。叶孤城将此剑赠予了自己的大弟子，也就是后来天道宗的创派祖师。"

云无咎心中一动。

天道宗的创派祖师，竟然是叶孤城的弟子？

"其二为'游子剑'。"云游子抬起手，指向四周的虚空，"便是你手中这柄。"

"其三为'飞仙剑'，是叶孤城最后铸造的一柄神剑，传说被他封于某处秘境之中，至今下落不明。"

云无咎低头看向自己的双手。

原来，他手中的这柄锈迹斑斑的破剑，竟然是太古第一剑铸造的神兵？

"前辈，"他抬起头，眸中闪过一丝疑惑，"若是如此重要的神兵，为何会流落到青云镇那样的穷乡僻壤？又为何会藏在一块石缝之中？"

云游子的身影微微一顿。

那双浑浊的眼睛中，闪过一丝复杂的情绪。

"因为……这柄剑，并非叶孤城主动留在那里的。"

"什么意思？"

"当年叶孤城封印饕餮时，曾留下后手。他知道饕餮不可能被彻底消灭，终有一日会再次苏醒。因此，他将一部分力量封入了三柄剑中，留待后人。"

云游子的声音渐渐低沉。

"然而，就在叶孤城陨落之后，他的弟子们却起了争执。天道剑、游子剑、飞仙剑，这三柄剑各有传承，却在漫长的岁月中渐渐失散。天道剑成为了天道宗的镇宗之宝，而游子剑和飞仙剑，则流落民间，下落成谜。"

"我是叶孤城剑道的传人之一。我的师门，世代守护着游子剑的秘密。然而……"

云游子的声音忽然变得苦涩。

"三百年前，有人发现了游子剑的踪迹。那人勾结天璇峰势力，追杀我满门，将游子剑夺走。我拼死抵抗，最终带着剑逃到了青云镇后山，却因伤势过重，油尽灯枯。"

"临死之前，我将游子剑藏入了后山的石缝之中，设下层层禁制，等待有缘人出现。"

"这一等，就是三百年。"

云无咎沉默了。

他终于明白了云游子前辈为何会困死在青云镇后山。那不是什么意外，而是一场处心积虑的追杀。

"前辈，"他抬起头，声音有些艰涩，"那个追杀您的人……就是天璇真人吗？"

剑中空间骤然安静下来。

云游子的身影剧烈颤抖了一下，周身的光芒闪烁不定，像是被什么东西触动了。

良久，他才开口。

"无咎……有些事，你现在的修为还不够资格知道。"

他的声音很轻，却带着一股不容置疑的坚定。

"但我可以告诉你一点——"

"你要小心天璇峰。不只是天璇真人，还有天璇峰上的一切。"

"我的仇人……远比你想象的要可怕得多。"

话音落下，剑中归于沉寂。

云无咎想要追问，却发现那道人影已经开始消散，像是被风吹散的烟雾。

"前辈！"

"无咎，"云游子的声音越来越虚弱，"今日我耗费太多，需要休息了。你先回去吧。记住我说的话……小心天璇峰……"

光芒彻底黯淡下去。

云无咎的眼前一片空白，当他再次睁开双眼时，已回到了现实之中。

窗外，月色如水。

他盘膝坐于榻上，手中紧握游子剑，额头上已沁出一层细密的汗珠。与剑灵交流极为耗费心神，每一次他都感到一阵强烈的疲惫。

但他心中的疑惑，却更深了。

天璇峰究竟藏着什么秘密？云游子前辈的仇人究竟是谁？那柄被叶孤城铸造的"天道剑"，如今又在何处？

还有……

今日林惊羽对游子剑的反应，又说明了什么？

他知道，自己的修仙之路，远比他想象的要复杂得多。而他手中的这柄锈剑，或许正是解开一切谜团的钥匙。

"叶孤城……万古剑道第一人……"

云无咎轻声呢喃，眼中闪过一丝坚定。

"无论您当年遭遇了什么，无论这其中藏着怎样的阴谋，我都会一步步追寻下去。"

"总有一天，我会揭开所有的真相。"

窗外，月光皎洁，洒落一地清辉。

少年握剑而坐，目光深邃如海。
