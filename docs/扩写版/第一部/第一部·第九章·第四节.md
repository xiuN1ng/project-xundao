# 第一部·第九章·第四节：故人重逢

## 一、湖边故影

湖泊如镜，星河倒映。

云泽望着眼前这个白衣人，一时间竟说不出话来。

那张脸，与他记忆碎片中看到的一模一样——年轻、清秀、嘴角永远带着淡淡的笑意。但近距离看去，那笑意里却藏着太多东西。不是真正的快乐，而是历经沧桑之后的释然，是看透世事之后的淡然，是将所有情感都压在心底深处、不愿让任何人看到的倔强。

"你来了。"白衣人又说了一遍，语气平静得像是在招呼一位老友，"坐吧。"

他伸手向身旁的草地一指，那里凭空出现了一块平整的青石，仿佛一直就在那里等候。

云泽没有动。

他只是静静地望着眼前这个人——这个他"曾经是"的人，心中涌起一种难以言喻的情绪。

"你……是道君？"他开口，声音有些沙哑。

白衣人笑了。

"道君？"他摇了摇头，"那不过是世人给我的称号罢了。"

"我原本没有名字。"他说，"行走于天地之间，默默无闻，从不在意旁人如何称呼我。"

"后来有了些名气，有了些追随者，他们便给我取了个名字——道君。"

"但那不是我的名字。那只是一个……代号罢了。"

云泽沉默了片刻。

"那你的真名……"

"没有。"白衣人说，"从出生到陨落，我从未有过名字。"

"无名氏——这是世人给我的另一个称呼。"

无名。

道君无名。

云泽望着眼前这个白衣人，忽然发现他的身影在星光下有些透明，仿佛随时都会消散。

"你……还活着吗？"云泽问道。

白衣人没有直接回答。

他只是转过身，重新面对那片平静的湖泊，沉默了片刻。

"对于这个问题，"他终于开口，"我也说不清楚。"

"若说肉体，我早已陨落，在万年前的那场大战中灰飞烟灭，连一缕残魂都未能留下。"

"若说意识，我的一丝不灭真灵转世重生，经历无数轮回，终于在这一世化为你——"

他转过头，望着云泽，目光深邃而复杂：

"从某种意义上说，你便是'我'。你是我的延续，是我的转生，也是我留给这世间的一线希望。"

"但你又不仅仅是我。"

"你有你自己的人生，你自己的经历，你自己的羁绊。你不是我——你是云泽。"

"这一点，永远不会改变。"

云泽静静地听着，心中那些一直困扰着他的问题，仿佛在这一刻找到了答案。

他是道君转世，但不仅仅是道君。

他继承了道君的力量与使命，却也拥有独属于自己的人生。

他是云泽，也是无名氏。

他是过去，也是现在，更是未来。

"那……"他开口，"我叫云泽。您……您能接受这个吗？"

白衣人望着他，眼中闪过一丝意外，随即化为温和的笑意。

"接受？"他说，"我当然接受。"

"若我不接受，又怎会将毕生修为都留给你？"

"你是我的孩子——不，你是我自己在这个世间唯一的延续。我怎会不接受你？"

他站起身，走到云泽面前，抬起手，轻轻按在他的额头上。

那手掌温热而干燥，带着一种说不清的力量，让云泽感到一阵电流般的酥麻。

"云泽。"他念着这个名字，声音里带着一丝感慨，"好名字。"

"咎者，过也。云者，腾也。泽者，润也。"

"无有过失，化而为云，润泽天下——这名字，比道君、无名好听多了。"

云泽的鼻子忽然有些发酸。

他不知道这是自己的情感，还是属于"道君"的情感。或许两者皆有，或许他已分不清彼此。

"我……"他开口，却不知该说什么。

"不必说。"白衣人——道君无名——轻轻收回手，退后一步，"你有很多问题，我知道。"

"但现在不是说话的时候。"

"归真试炼，现在开始。"

## 二、万年一梦

话音落下，湖泊忽然泛起层层涟漪。

那些涟漪从湖心向外扩散，一圈一圈，如同投入湖中的石子激起的波纹。但那不是普通的涟漪——每一道涟漪都在散发着微弱的光芒，金色、银色、蓝色、紫色……无数种颜色交织在一起，将整片湖面点缀得如同碎钻铺就。

"闭上眼睛。"道君的声音响起。

云泽依言闭上眼睛。

下一刻，无数画面如潮水般涌入他的脑海。

那不是普通的画面——那是记忆。完整的、连贯的、如同亲身经历般的记忆。

他看到了一个人从混沌中诞生。

那人没有父母，没有来历，只是在一片虚无中忽然睁开眼睛，看到了属于他的第一缕光。他不知道自己是谁，不知道自己从何而来，甚至不知道自己为什么存在。他只知道一件事——他活着。

他开始行走。

从混沌中走出，走进天地初开的洪荒世界。他看到日月星辰在苍穹中运转，看到山川河流在大地上成形，看到第一个生灵在海洋中诞生。

他开始修行。

没有师父，没有功法，没有任何指引。他只是按照本能去吸纳天地灵气，去感受天地运转的规律。日复一日，年复一年，他渐渐从一个懵懂的凡人，成长为能够呼风唤雨的修士。

他开始结交朋友。

那些朋友与他志同道合，一起修行，一起探索这片天地。他们中有温和善良的仙人，有狡黠聪慧的散修，有忠厚老实的凡人。每一个朋友都给他的人生带来不同的色彩，让他第一次感受到"羁绊"的意义。

然后——他开始遭遇敌人。

那些敌人有的觊觎他的修为，有的嫉妒他的天赋，有的只是单纯想要将他消灭。他开始战斗，一次又一次，身边的战友一个个倒下，他却依然站着。

他开始强大。

从真仙到大罗，从大罗到太乙，从太乙到混元。他用了一万三千年，攀登上了这世间修炼的巅峰，成为九纪元以来最强的存在。

然后——他遇到了一个弟子。

那弟子天资聪颖，根骨不凡，对道法有着惊人的悟性。他一眼便看中了这个弟子，将毕生所学倾囊相授，视如己出。

那弟子也没有让他失望。他的修为突飞猛进，很快便成为修仙界的新星，与他并肩而行，被世人尊称为"天玄真人"。

他以为他会与这个弟子一直走下去。

直到那一夜——

画面骤然转暗。

云泽感到一阵刺骨的寒意从心底升起。

他看到那个天玄真人站在一片废墟之上，身后是无数修士的尸体，而废墟的中央，一个白衣人正艰难地站起来。

那是道君。

那是无名。

那是曾经的云泽。

"师尊。"天玄真人的声音冰冷而陌生，"把天道图交出来吧。"

道君——曾经的云泽——望着自己一手培养出来的弟子，眼中满是难以置信的神色。

"为什么？"他问道，声音沙哑，"我视你如己出，将毕生所学都传授给你。你为何要背叛我？"

天玄真人笑了。

那笑容冷酷而疯狂，完全没有了平日的温和："因为我也想要天道图。我也想要成为这世间的主宰。"

"师尊，您太强了，强到让所有人都只能仰望您的背影。他们只知道'道君'，却不知道还有我'天玄'。"

"这种日子，我过够了。"

"我不要做您的影子——我要做自己的太阳。"

话音落下，无数道法术轰然砸向道君。

那不是普通的法术——那是道君亲手传授的绝技，如今被用来对付他自己。道君的身体在那些法术的轰击下节节败退，鲜血从他嘴角溢出，染红了他雪白的衣袍。

但他没有倒下。

他只是望着天玄真人，望着这个他曾经最信任的弟子，眼中渐渐化为一片平静。

"好。"他说，声音轻得像是一声叹息，"你想成道，我便成全你。"

"但你要记住——天道图不是你想象的那么简单。"

"拿到它的那一刻，你便要承担起它背后所有的因果。"

"那些因果，是你承受不起的。"

天玄真人冷笑一声："死到临头，还在故弄玄虚。"

"你以为我会怕你？"

他抬起手，一道更加凌厉的法术在他掌心凝聚。

"去死吧，师尊。"

那道法术轰然射出，直直地命中道君的胸口。

云泽感到自己的胸口仿佛也被那道法术贯穿，一阵剧烈的疼痛从心脏蔓延至全身。

他看到道君的身体在光芒中缓缓倒下。

他看到天玄真人从他怀中取出一块闪烁着金光的玉片——那是天道图。

他看到道君在最后一刻，嘴角浮现一丝意味深长的笑容。

"天道图……不是你的……"

"遁去的一……会回来……"

"找我……报仇……"

声音越来越弱，最终消失在虚空之中。

## 三、泪落星河

记忆的画面戛然而止。

云泽猛地睁开眼睛，发现自己依然站在湖边，而道君——那个白衣人——正静静地望着他。

他的脸上不知何时已经满是泪水。

"那是……那是真的？"他的声音在颤抖，"那真的发生过？"

道君没有说话。

他只是伸出手，轻轻擦去云泽脸上的泪痕，动作轻柔得像是在对待一个受伤的孩子。

"是的。"他说，声音平静，"那都是真的。"

"万年前，我被天玄背叛，陨落于天道秘境的废墟之中。"

"但我在最后一刻，将我的一丝不灭真灵寄寓于'遁去的一'之中，逃脱了天玄的追杀。"

"那一丝真灵在轮回之中飘荡了无数年，最终转世重生在你体内。"

"你，便是那'遁去的一'的延续。"

"你，便是我留给这世间的最后一线希望。"

云泽静静地听着，泪水依然在流。

他不是为自己流泪——他知道，那些泪是属于道君的。那个独自在天地间行走了万年的白衣人，那个被最信任的弟子背叛的师尊，那个即便陨落也要留下一线希望的道君——他将自己所有的情感都封存在那丝真灵之中，穿越万年的时光，最终传递到了云泽这里。

"为什么……"云泽开口，声音哽咽，"为什么您不恨他？"

"恨？"道君微微一笑，"恨又如何？不恨又如何？"

"万年光阴，早已将一切都冲淡了。"

"我只是有些遗憾。"他望向湖泊，望向那片倒映着星光的湖面，"遗憾我没能亲手将真相告诉世人。遗憾我没能保护好那些追随我的朋友。"

"但更多的……是庆幸。"

"庆幸我留下了那'遁去的一'。庆幸我还有机会转世重来。庆幸——天玄的阴谋至今未能得逞。"

他转过头，望着云泽，眼中满是欣慰：

"更庆幸的是，那'遁去的一'，落在了你身上。"

"落在了一个善良、坚韧、有担当的孩子身上。"

"落在了一个比我年轻时更好的云泽身上。"

云泽再也忍不住，泪水夺眶而出。

不是因为悲伤——而是因为一种说不清道不明的情感。那是跨越万年的羁绊，是穿越轮回的传承，是"自己"与"曾经的自己"之间最真挚的共鸣。

他忽然跪了下来。

不是因为敬畏，而是因为某种更深的情感——那是一个孩子对父亲的下跪，是一个后辈对先人的致意，也是"云泽"对"无名"最深的理解与接纳。

"前辈。"他伏在地上，声音颤抖，"我不知道该说什么……"

"不必说。"道君俯下身，将他扶起，"我都明白。"

"万年孤独，不曾磨灭我的心志。万次轮回，不曾消散我的执念。"

"因为我知道，终有一天，会有人继承我的意志，完成我未竟的事业。"

"那个人，就是你。"

"云泽——我最后的传人——"

他的声音忽然变得庄重：

"你可愿接受我的传承？"

"你可愿继承道君之名？"

"你可愿——替我，终结天玄的阴谋？"

云泽抬起头，望着眼前这个白衣人。

星光洒落在道君身上，将他的身影映照得如同神祇一般庄严。他的目光平静而深邃，平静得像是无波的湖水，深邃得像是无尽的星空。

云泽深吸一口气。

他知道自己接下来要说什么。

那不只是一句承诺——那是他此生将要承担的一切的分量。

"我愿意。"

他说，声音坚定得像是钢铁：

"我愿意接受您的传承。"

"我愿意继承您的意志。"

"我愿意——替您，终结天玄的阴谋。"

话音落下，湖面骤然亮起万丈金光。

——第四节完——
