# 第一部·第二章·第二节：三月苦修

## 一、离别

消息在当天便传遍了整个青云镇。

云无咎斩杀妖兽的消息像是长了翅膀一样，从打谷场飞向镇子的每一个角落。到了傍晚时分，整座青云镇都在议论这件事——有人说他是天上的星宿下凡，有人说他是得到了仙人点化，还有人说他是天生的修仙奇才，注定要一飞冲天。

云无咎没有心思理会这些闲言碎语。

他身上大大小小的伤还没处理完，方才那一战虽然有云游子前辈相助，却也让他筋骨错位、肌肉拉伤，整个人像是被人用锤子敲打过一样，浑身上下没有一寸地方不疼。

云爷爷早就等在家门口了。

老人站在那间歪斜的茅屋前，佝偻着背，眯着眼睛朝镇口的方向张望。他的手里攥着一根旱烟杆，烟锅里明明没有点火，他却还是一口一口地嘬着，像是在借此掩饰内心的焦虑。

"爷爷。"

云无咎的声音从巷口传来。云爷爷循声望去，只见孙子一瘸一拐地从街角转出来，衣服上沾满了泥土和血迹，脸上青一块紫一块，胸口的衣服更是被撕开了一道长长的口子，露出里面血肉模糊的伤口。

老人的脸色顿时变了。

"咎儿！你这是……"

"没事，爷爷。"云无咎勉强挤出一个笑容，"路上摔了一跤，不碍事。"

"摔一跤能摔成这样？！"云爷爷三步并作两步迎上去，伸手便要去扶他，"快进屋，让爷爷看看！"

云无咎本想说不用，可云爷爷已经一把扶住了他的手臂。老人的手枯瘦而粗糙，却带着一股让人安心的温度。那只手曾经在无数个寒冷的冬夜里为他掖过被角，也曾经在无数个他发烧生病的夜里彻夜不眠地守在他床边。

他忽然有些鼻酸。

"爷爷，我真的没事……"

"没事？没事你能伤成这样？"云爷爷的声音有些发颤，"你是不是又去后山了？爷爷跟你说过多少次了，后山的深处去不得，那里——"

"爷爷，"云无咎打断了他，声音很轻，却很认真，"我今天杀了妖兽。"

云爷爷的话音戛然而止。

老人怔怔地看着他，浑浊的眼睛里闪过一丝复杂的神色。

"妖兽……"

"嗯。是一头野猪妖，有三百多斤重，比上次我在后山遇到的那头还要大。"云无咎扶着门框脱掉鞋子，走进屋里，"镇上王二牛叔被它咬死了，我路过的时候……就顺手……"

"顺手？"云爷爷的声音忽然拔高了几分，"你拿什么顺手？你一个十五岁的娃娃，拿什么去杀妖兽？！"

老人的眼眶红了。

他不是不知道云无咎这些年在后山采药时遇到过多少危险，也不是不知道这个孙子从小就有一股子不知道打哪儿来的倔劲。可他毕竟是个凡人啊——凡人在妖兽面前那就是蝼蚁一样的存在，这小子怎么敢……

"爷爷，"云无咎坐在床沿上，任由老人翻看他的伤口，"我不是一个人。"

他顿了顿，压低了声音："我身上有一样东西，是云游子前辈留给我的。方才那妖兽要伤我的时候，那东西自己动了，帮我斩杀了妖兽。"

云游子？

云爷爷的动作微微一僵。

这个名字他听云无咎提起过——是后山那位死去仙人的名号，据说是死在了一场惊天动地的阴谋之中，只留下了一柄剑和一枚玉简。云无咎说那是一位值得尊敬的先辈，生前是个光明磊落的修士。

"那东西……能护住你？"云爷爷的声音低了下来。

"能。"云无咎点了点头，"但前辈说，他的残念已经所剩无几了，下一次不一定还能护得住我。"

云爷爷沉默了。

老人默默地翻看着云无咎的伤口，从胸口到后背，从手臂到双腿，每一处都仔仔细细地检查过。还好，都是皮外伤，没有伤到筋骨。

他转身从灶台边的土罐里掏出一把草药，在嘴里嚼烂了，敷在云无咎的伤口上。那草药冰凉而辛辣，敷上去的时候火辣辣地疼，但很快便转为一阵清凉，疼痛也随之减轻了许多。

"咎儿，"老人一边敷药，一边低声说，"你是不是打算去修仙？"

云无咎的动作顿了一下。

"……爷爷都听说了？"

"嗯。李婶子下午来过了，把镇上测灵根的事跟我说了。"云爷爷的声音很平静，"说你测出了什么五行灵根，品相中上。那个姓周的仙师让你三个月后去什么天道宗参加收徒大会。"

云无咎低下头，没有说话。

他知道云爷爷想说什么。这个从小将他拉扯大的老人虽然没读过什么书，却比谁都懂得人情世故。他知道老人心里在担心什么——怕他一去不回，怕他这一走就再也不回来了。

"咎儿，"云爷爷放下手中的草药，在他对面坐下，"爷爷想了一整天，有些话想跟你说。"

"爷爷您说。"

"你长大了。"老人望着他，浑浊的眼睛里透出一丝欣慰，"不再是当年那个躲在我身后的小娃娃了。你有自己的路要走，有自己的事要做。爷爷不拦你。"

"爷爷——"

"让爷爷说完。"云爷爷摆了摆手，"爷爷这辈子没什么本事，没能给你置下什么家业，也没本事让你过上好日子。你能有今天，是你自己争气，是老天爷开眼。爷爷心里高兴，真的高兴。"

老人的声音有些哽咽，却还是努力挤出一个笑容。

"去吧，咎儿。去天道宗，去修仙，去做你该做的事。"

"只是记住一件事——"

云无咎抬起头，看着老人。

"无论走多远，无论飞多高，别忘了你是从哪儿出来的。"

"这里是青云镇，是你的家。爷爷在这里等你回来。"

云无咎的喉头哽住了。

他张了张嘴，想说些什么，却发现嗓子像是被什么东西堵住了一样，一个字也说不出来。

最后，他只是重重地点了点头。

"爷爷，我记住了。"

## 二、闭关

从那天起，云无咎便开始了近乎疯狂的修炼。

白天，他依旧砍柴、挑水、照顾云爷爷的起居，不让镇上的任何人看出异样。他把测灵根的事和王二牛的死告诉了云爷爷，让老人帮忙遮掩——他不想让太多人知道他要去修仙，更不想让那些好事之徒天天来家里打听这打听那。

到了晚上，他便躲进后山那处隐秘的石缝中，按照玉简上的记载，修炼《云游心法》。

那处石缝很隐蔽，藏在一片荆棘丛后面，入口只有一人宽，进去之后却别有洞天。云游子的遗骸还端坐在里面，枯骨森森，却依旧保持着当年盘坐的姿势，仿佛只是在沉睡。云无咎每天都会在遗骸前默默叩拜，感谢前辈的恩情，然后才开始修炼。

《云游心法》是云游子毕生心血所创，博大精深，包罗万象。

玉简中记载的内容分为四个部分：心法、剑诀、丹道、阵法。云游子前辈当初说的"平生所学"，远比他想象的要丰富得多。只是那些内容实在太过高深，以他现在的修为根本看不懂，只能从最基础的心法开始学起。

第一月，他打通奇经八脉中的任督二脉，正式踏入淬体境三层。

所谓淬体境，便是淬炼肉身、强化筋骨的过程。寻常人想要突破淬体境一层，便需要至少半年的苦修。而他在短短一个月内便突破到了三层，修炼速度之快，堪称骇人。

这得益于他五行灵根的特殊体质。五行灵根意味着他可以同时吸纳天地间五种属性的灵气，修炼效率比单灵根或双灵根的修士高了何止十倍。再加上《云游心法》本就是一等一的功法，运转起来事半功倍，他的进境自然一日千里。

第二月，他将《云游心法》第一重练至大成，淬体境六层。

这个速度已经超出了正常修士的认知范畴。按照玉简中的记载，寻常修士从淬体境一层修炼到六层，少说也要三五年的光阴。而他只用了两个月。

但他知道这并不是因为他是什么"天才"。

而是因为他在拼命。

每天晚上，当镇上的灯火渐渐熄灭，他便独自一人钻进后山，忍受着蚊虫叮咬和山风侵袭，一遍又一遍地运转心法。累了便打坐调息，困了便用凉水浇头，硬是把自己逼到了极限。

他知道自己没有时间慢慢来。

三个月，只有三个月。

三个月后就是九宗收徒大会，他必须在那之前尽可能地提升修为，否则根本不可能通过天道宗的入门试炼。

第三月，他突破了淬体境七层，开始冲击第八层。

这个进度，即便放在天道宗外门，也堪称天才！

而他的身体，也在日复一日的修炼中悄然发生着变化。

曾经瘦削的身躯变得精壮有力，肩膀宽阔了，手臂粗了一圈，胸口的肌肉轮廓也隐约可见。皮肤虽然还是黝黑，却透出一层淡淡的光泽，像是被油膜覆盖过一样，在阳光下隐隐泛光。

曾经黯淡的双眼如今精芒内敛，偶尔有灵光一闪而过。镇上的人见到他，都说这小子越长越精神了，和以前那个闷葫芦似的砍柴少年判若两人。

他十五岁了。

十五岁的少年，已经初具修士气象。

## 三、噩耗

然而，天有不测风云。

就在云无咎埋头苦修的这段日子里，云爷爷病了。

起初只是风寒。

老人年纪大了，七十多岁的人，身子骨本就一年不如一年。入秋之后天气转凉，他便染上了咳嗽，起初只是早晚咳几声，后来越来越重，咳得整夜整夜睡不着觉。

云无咎上山采了些寻常草药，熬了汤剂喂老人服下。本以为过几日便能好转，谁知这病来得蹊跷，去得却慢，一拖便是半月，非但不见好转，反而愈发严重起来。

那一日，云无咎从后山修炼回来，刚推开茅屋的门，便闻到一股浓重的药味。

那药味很冲，带着一股苦涩的草药气息，和米粥的香味混在一起，闻起来格外刺鼻。

"爷爷？"他喊了一声，没有得到回应。

他心中一紧，快步走进里屋。

云爷爷躺在床上，面色蜡黄，瘦得皮包骨头。老人的眼眶深深地凹陷下去，颧骨却高高突起，乍一看去，像是一具风干的尸体。

他的呼吸很轻，轻得几乎听不见。胸口微微起伏着，像是一张破旧的风箱，吃力地抽动着。

床头的药碗已经凉透了，结成一层厚厚的药膜。老人显然已经咳得连喝药的力气都没有了。

"爷爷！"云无咎三步并作两步冲到床前，一把扶住老人枯瘦的身躯，"您怎么不叫人喊我？"

云爷爷艰难地睁开眼睛。

那双眼睛浑浊得像是蒙了一层灰蒙蒙的雾，却还是努力聚焦在云无咎脸上。看见孙子的那一刻，老人的嘴角微微牵动了一下，像是想笑，却没笑出来。

"咎儿……你回来了……"他的声音虚弱得像是一阵风就能吹散，"爷爷没事……就是有些累……"

"什么没事！"云无咎的眼眶一下子红了，"您都烧成这样了！"

他伸手探了探老人的额头——滚烫！

那热度烫得吓人，像是一块烧红的铁贴在他掌心。他连忙起身要去请大夫，却被一只枯瘦的手拉住了衣角。

"咎儿……别去……"云爷爷的声音越来越虚弱，"镇上的大夫……治不了我这病……"

"治不了？那就进城去请！"

"进城……也治不了……"老人艰难地摇了摇头，"爷爷这病……不是寻常的病……"

不是寻常的病？

云无咎心中一凛。

他仔细端详着云爷爷的面容，忽然发现了一些不对劲的地方。老人的嘴唇发紫，印堂发黑，舌苔厚腻发黄——这些症状和普通的风寒发热全然不同。

他想起了云游子前辈的玉简中记载过的一种病——灵根反噬。

所谓灵根反噬，是指修士在修炼过程中因为灵气运转不当，导致灵根之力侵入经脉，损伤脏腑。这种病只发生在修士身上，寻常凡人若是染上，轻则缠绵病榻，重则性命不保。

可是云爷爷只是个凡人，怎么会得上这种病？

"爷爷，您老实告诉我，"云无咎压低声音，"您是不是瞒着我做了什么？"

云爷爷沉默了很久。

老人闭上眼睛，胸口起伏得更加剧烈，像是在做一个很艰难的决定。

良久，他才缓缓开口：

"咎儿……你还记得……爷爷跟你说过的那枚玉佩吗？"

玉佩！

云无咎当然记得。那是他身世的唯一线索。

"那玉佩……其实一直都在……"云爷爷颤抖着伸出手，从贴身的衣襟里摸出一样东西，"爷爷……替你保管了十五年……如今……该还给你了……"

那是一枚通体温润的玉佩，巴掌大小，正面刻着一个古朴的"道"字，背面则是密密麻麻的纹路，瞧着像是某种阵法，却又复杂得远超他的认知。

最奇异的是，当他触碰到那枚玉佩的瞬间，一股温热的气流便从玉佩中涌出，顺着他的手臂流遍全身，所过之处，仿佛连灵魂都舒泰了几分。

"这是……"

"这是你亲生父母留给你的东西。"云爷爷的声音愈发虚弱，"十五年前……那个风雪交加的夜里……爷爷在镇外老槐树下捡到你时……你便裹在这玉佩所发的光芒之中……"

云无咎愣住了。

他虽然早就知道自己是被云爷爷捡来的，却从未想过自己竟是被这枚玉佩的光芒包裹着来到这个世界上的。

"那光芒……好生厉害……"老人的眼中闪过一丝追忆，"当时爷爷便知道……你这孩子来历不凡……"

"可是后来……那玉佩的光芒渐渐暗淡了下去，爷爷便把它收了起来，打算等你长大后再交给你。"

"只是爷爷没想到……这玉佩虽然不再发光了，却一直在吸纳周围的灵气……"

云无咎的瞳孔骤然收缩。

他忽然明白了。

这枚道君玉佩虽然不再显现神迹，却一直在暗中吸纳天地灵气。而云爷爷与这枚玉佩朝夕相处了十五年，那些被吸纳的灵气便渐渐侵蚀了老人的身体，最终导致了灵根反噬。

换句话说——是他害了云爷爷。

如果不是这枚玉佩，云爷爷不会染上这种怪病。

如果不是他这十五年来的存在，玉佩不会留在云家，云爷爷更不会被灵气侵蚀。

"爷爷……是我害了您……"云无咎的声音颤抖起来，"都是我的错……"

"傻孩子……"云爷爷艰难地抬手，摸了摸他的头，"爷爷这辈子……能有你这样的孙子……已经很满足了……"

"这玉佩……本就是你的东西……爷爷替你保管了十五年……如今……总算物归原主了……"

老人的眼皮越来越沉，声音也越来越轻。

"咎儿……听爷爷说完……爷爷还有一件事……要告诉你……"

"什么……您说……"

"你的身世……没那么简单……"云爷爷的声音低沉而郑重，"那玉佩上的'道'字……爷爷曾经……在一本古籍上见过……"

"那是上古时期……一位惊天动地的大人物……才能使用的印记……"

"那大人物……人称'道君'……"

道君？！

云无咎脑海中轰然炸响。

他想起了玉简中云游子前辈曾经提到过的那个名字——那位在太古年间，以一己之力封印混沌魔兽饕餮的绝世强者，据说便被称为"道君"！

难道他的身世，竟与那位传说中的大人物有关？！

"爷爷……您说的是真的吗？"

"爷爷……不知道……"云爷爷摇了摇头，"但爷爷知道……你是好孩子……无论你是谁的孩子……无论你从哪里来……你都是爷爷的孙子……"

"去吧……咎儿……去天道宗……去追寻你的道……"

"别让爷爷……成为你的羁绊……"

老人的声音越来越轻，眼皮越来越沉，手也渐渐松开。

"爷爷！"

"……去……"

这是云爷爷留给他的最后一个字。

## 四、后山一夜

那一夜，青云镇后山，多了一个疯狂攀爬的身影。

云无咎将云爷爷的遗体安置在床上，盖好被子，然后独自一人钻进了夜色之中。

他的眼眶红肿，却没有眼泪。

不是不悲伤，而是悲伤到了极点，反倒哭不出来了。

他只知道一件事——他不能让云爷爷就这样死去。

玉简中记载过一种能续命救人的灵药，名为"九转还魂草"。那是一种只生长在悬崖峭壁之上、终年不见阳光的珍稀草药，据说有起死回生之效。

云爷爷还没死。

老人只是昏迷了，如果能在天亮之前找到九转还魂草，或许还有救。

云无咎沿着后山的小路飞奔，脚下的青石板被露水打湿，滑得几乎无法落脚，他却浑然不觉。他的脑海中只有一个念头——

快一点，再快一点。

九转还魂草生长在青岚山主峰的悬崖绝壁上，寻常人根本爬不上去。但云无咎不是寻常人。

他修炼了三个月的《云游心法》，虽然只是淬体境七层的修为，却已经远超凡人。他可以徒手攀附在悬崖峭壁上，可以借力在陡坡上跳跃，甚至可以短暂地凌空借力——这些都是淬体境修士才有的能力。

夜色深沉，山风呼啸。

月亮被乌云遮住，整座山林陷入了一片漆黑之中。只有偶尔传来的虫鸣声和远处不知名鸟类的啼叫声，在这寂静的夜里显得格外清晰。

云无咎借着体内灵力的微光辨认方向，沿着熟悉的山路向上攀爬。

他从小就跟着云爷爷在这片山林里穿行，对这里的每一块石头、每一棵树都了如指掌。可此刻在夜色中行走，一切都变得陌生起来——那些白日里清晰可见的路径隐没在黑暗里，取而代之的是嶙峋的岩石和密布的荆棘。

一百丈……两百丈……三百丈……

他不知疲倦地向上攀爬，手指抠进岩缝，脚尖踩着凸起的岩石，一寸一寸地往上挪动。崖壁上满是青苔，湿滑得几乎无法落脚；山风凌厉，吹得他摇摇欲坠，好几次险些坠落崖底。

可他没有停下。

他不能停。

每耽搁一刻，云爷爷便少一分生机。

就在他即将触及那株在月光下隐隐泛着银光的草药时——

"吼——！"

一声低沉的嘶吼从头顶传来。

云无咎抬头一看，顿时浑身汗毛倒竖。

一头比那日野猪妖还要庞大数倍的黑影，正盘踞在崖顶，一双猩红的眼睛死死盯着他，像是在看一只送上门的猎物。

不，不止一头！

借着月光，他看到那崖顶上竟盘踞着密密麻麻的黑影——十余头大小不一的妖兽，正用猩红的眼睛盯着他。它们的体型各异，有的如牛，有的如狼，有的干脆就是一团看不清形状的黑雾。但无一例外，每一头都散发着浓郁的妖气。

那是妖兽群！

它们是被那头野猪妖引来的同伴！又或者是这青云镇后山的妖兽原住民！无论如何，他都闯入了它们的地盘！

"吼——！"

一声尖锐的嘶吼响起，一头形如巨狼的妖兽率先扑下崖壁！

紧接着是第二头、第三头……

十余头妖兽如同下山的猛虎，争先恐后地朝云无咎扑来！
