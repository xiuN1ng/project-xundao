# 第一部·第一章·第四节：归来

## 一、山下重逢

不知过了多久，云泽缓缓睁开了眼。

阳光的角度变了。从方才的斜斜照射，变成了近乎垂直的倾泻而下——已是正午时分，日头正烈。

他在石台上修炼了整整半日。

可奇怪的是，他丝毫也不觉得饿，反而神清气爽，浑身上下似有使不完的力气。丹田中的那缕气流比方才粗壮了一圈，正沿着他的经脉缓缓流转，所过之处，暖融融的，像是有什么东西在滋养着他的血肉。

他站起身来，活动了一下筋骨。

断掉的两根肋骨已基本愈合，只剩下微微的酸胀感。他试着挥动了一下拳头，拳风虎虎，竟比从前有力了许多。

他低头看了看自己身上的伤——手掌上被藤条勒出的血痕已经结痂，肩膀上被野猪獠牙划出的伤口也已收拢，只有几道浅浅的疤痕还在。

这愈合的速度，比从前快了一倍不止。

他将游子剑从腰间解下，仔细端详了一番。

那剑身上的锈迹似乎淡了一些，原本模糊的"游子"二字，此刻在阳光下泛着莹莹的光泽。他能感觉到剑中似乎有什么东西在沉睡——极微弱，却极真实。那是云游子前辈留下的剑灵，虽然尚未完全苏醒，却已与他的气息产生了某种微妙的联结。

他将剑重新别在腰间，又将那枚玉简贴身收好，这才开始打量四周，寻找下山的方向。

方才他是顺着那道隐蔽的岩缝爬上来的，此刻要原路返回，却并非易事——那岩缝狭窄陡峭，上来时已是拼尽全力，若要再下去，只怕凶多吉少。

好在这石台三面皆是绝壁，唯有一面是一条蜿蜒而下的山道。那山道被杂草和灌木遮掩，若非刻意寻找，根本不会注意到它的存在。

云泽沿着山道一路向下。

山道曲折迂回，时而穿过密林，时而越过溪涧，时而要手脚并用地攀爬陡坡。他一边走，一边默默记下沿途的地标——哪块石头、哪棵老树、哪个转弯，都是他日后可能还要用到的参考。

约莫走了一炷香的功夫，前方的林木忽然变得稀疏起来，阳光大片大片地倾泻而下。

他走出林子，眼前豁然开朗——

那是他方才遇险的地方。

那头野猪妖已经不见了踪影，只留下地面上杂乱的蹄印和几滩黑色的血迹。云泽低头看了看那些蹄印——有大有小，深浅不一，显然那头畜生不止一次从这里经过。

它追他追到崖边，却被他跳崖逃脱。想来那畜生也知道这崖壁的厉害，不敢冒然追下来，只能在崖边干等了半晌，最终才悻悻离去。

只是……它会就此善罢甘休吗？

云泽的眉头微微皱起。

他在青云镇生活了十五年，从小便听云爷爷讲过不少关于妖兽的传说。那些故事里，妖兽大多记仇得很，一旦与人结下仇怨，便是不死不休的局面。今日他砍伤了那畜生的后腿，又在它眼皮子底下跳崖逃脱，那畜生怎肯善罢甘休？

只怕这山，往后是不好再来了。

他叹了口气，继续朝山下走去。

穿过半山腰的那片杂树林时，他的脚步忽然一顿。

林边的青石上，不知何时多了一个人影。

那是个与他年纪相仿的少年，生得虎头虎脑，皮肤黝黑，一双眼睛滴溜溜地转，透着几分机灵劲儿。他的衣服上打着补丁，却洗得干干净净，袖口处还特意缝了一圈新布，显然是家中长辈用心打理过的。见云泽从山上下来，那少年顿时跳了起来，远远地便挥手嚷道：

"无咎哥！你可算下来了！我等了你一个多时辰了！"

来人正是云泽的发小——李狗蛋。

这名字听着粗鄙，却是青云镇的老传统。乡间百姓信奉贱名好养，孩子生下来便取个狗蛋、猪娃、石头之类的乳名，待长大成人后方才另取大名。李狗蛋的大名唤作李昭，也是镇上老秀才给取的，寓意光明磊落，只是镇中百姓叫惯了"狗蛋"，反倒没几人记得他的大名。

云泽见到是他，紧绷的心弦顿时松了下来，嘴角不由浮现一丝笑意。

"狗蛋，你怎么在这儿？"

"哎，别提了！"李狗蛋一蹦三尺高，神色激动得满脸通红，"今儿一早镇上来了个大人物，你猜是谁？"

"大人物？"云泽微微挑眉，"什么大人物？"

"仙人！"李狗蛋两眼放光，声音都压低了几分，仿佛怕被旁人听去，"从天道宗来的大仙人！就在咱们镇口的打谷场上，说是要给咱们青云镇的娃娃们测灵根！说是只要有灵根，就能去天道宗当弟子，修仙问道，长生不老！"

他说得口沫横飞，手舞足蹈，仿佛已经看见自己成了仙人的模样。

"我娘一早就催我去了，我想着你肯定也要去测，便想着在这下山的必经之路上等你。结果一等就是一个多时辰！你上哪儿去了？我还以为你被妖兽拖走了呢！"李狗蛋絮絮叨叨地抱怨着，目光却不由自主地落在了云泽腰间的长剑上。

"咦？无咎哥，你这是……"

他的眼睛一下子瞪圆了，"剑？！你从哪儿弄来的剑？！"

## 二、发小情谊

"路上捡的。"

云泽轻描淡写地说，一边将柴捆从肩膀上卸下，搁在了路边的大石头上。

"捡的？"李狗蛋凑近看了看，眼睛里闪过一丝羡慕，"这剑瞧着挺旧的……不过还真有几分气势！无咎哥，你不会是遇到什么世外高人了吧？然后人家看你骨骼清奇、天资聪颖，就把这柄神兵利器传授给你了？"

他说着说着，连自己都信了，脸上露出神往的神色："就像话本里写的那样！侠客落难，偶遇少年，然后将毕生绝学倾囊相授，然后少年仗剑天下、行侠仗义……啧啧啧，无咎哥，你该不会就是那个少年吧？"

云泽被他逗笑了，伸手在他脑袋上拍了一下："想什么呢。哪有那么多的世外高人，这剑就是我在山里捡的，兴许是哪位过路的旅人落下的。"

他说着，将话题岔开："对了，你测灵根的结果怎么样？"

李狗蛋的脸色顿时垮了下来。

"别提了。"他垂头丧气地说，"那仙人拿了一面镜子一样的东西，让我把手按上去。我照做了，结果那镜子里就亮了一下，然后那仙人就说我是什么……火土双灵根，下品之中属上等，可入外门候选。"

他说得闷闷不乐，显然对这个结果并不满意。

"下品之中属上等，那不挺好的吗？"云泽微微挑眉，"你没听那仙人说？今日青云镇总共就测出了十五个有灵根的，你一个就占了俩，还是'上等'，这还有什么不满意的？"

"可是……"李狗蛋挠了挠头，"可是无咎哥，我听说那些大宗门的仙人，最低的要求都是中品灵根。下品灵根的话，能不能进天道宗还两说呢。那仙人也说了，三个月后的入门试炼，得通过了才能正式入门。通不过的话，就只能做个外门杂役，连正式弟子都当不上。"

他的声音越说越低，到最后几乎像是在自言自语：

"我就是想着……万一我进不了天道宗，那不就白测了吗？我娘还指望我能出人头地呢，她一个人把我拉扯大不容易，我总不能让她失望吧……"

云泽望着他那副垂头丧气的模样，心中忽然涌起一股复杂的情绪。

他知道李狗蛋家里的情况——父亲早逝，母亲一人将他拉扯大，平日里靠给镇上人家浆洗衣服度日，日子过得比他还清苦。这小子从小便有一股不服输的劲头，旁人越是看不起他，他越是要证明给别人看。

这一点，倒是与他颇为相似。

"行了。"云泽拍了拍他的肩膀，"三个月后的试炼，还没开始呢，你急什么？这三个月好好修炼，把根基打好，到时候未必过不了。再说了——"

他顿了顿，嘴角浮现一丝笑意：

"你不是说要当仙人吗？怎么，这就打退堂鼓了？"

李狗蛋愣了一下，随即眼睛一亮，用力拍了拍自己的脸颊："对啊！我李狗蛋怎么能这么没出息！不就是个入门试炼吗？别人能过，我凭什么过不了！"

他的眼眶微微发红，却硬是挤出一个笑容："无咎哥，你说得对！我不能让我娘失望！就算拼了这条命，我也要进天道宗！"

"好。"云泽点了点头，"那便一起去试试。"

"一起？"李狗蛋的眼睛再次亮了起来，"无咎哥，你也要去测灵根？"

"嗯。"云泽淡淡应了一声。

他并没有打算将石台上的奇遇告诉任何人。不是不信任李狗蛋，而是这件事太过重大，在没有弄清来龙去脉之前，还是谨慎为上。

再说了，他现在已是云游子前辈的传承者，身怀修仙功法丹田有气感——这些事若是被旁人知道了，只怕会招来不必要的麻烦。

"太好了！"李狗蛋兴奋得几乎要跳起来，"无咎哥，你要是也去测灵根，咱们俩就能一起去天道宗了！到时候咱们互帮互助，说不定都能成为大仙人！"

"先别高兴得太早。"云泽泼了他一盆冷水，"灵根这东西，万人中未必有一个。咱们青云镇这种穷乡僻壤，怕是十年八年也出不了一个。"

"呸呸呸！"李狗蛋连忙啐了几口，"大吉大利！无咎哥，你可别乌鸦嘴！我李狗蛋这辈子就是砍柴的命？不，我不信！我要当仙人！我要让所有人看看，我李狗蛋也能出人头地！"

他拍了拍胸脯，一脸认真地说："我爹娘没本事，给不了我好日子。我就只有靠自己。仙人又怎样？都是从凡人变来的。我就不信，我李狗蛋比谁差！"

云泽望着他那副不服输的模样，心中暗暗为他祈祷。

希望这小子真的能通过入门试炼吧。

"好了，别在这儿磨蹭了。"他扛起柴捆，迈步朝山下走去，"测灵根的地点在镇口打谷场，咱们快走吧，别错过了。"

"等等我！"李狗蛋连忙跟上，"无咎哥，你等等我啊！你还没告诉我那剑是从哪儿捡的呢！"

"就是山里捡的，没什么好说的。"

"骗人！你肯定有奇遇！无咎哥，你可不许瞒着我！咱们可是穿开裆裤一起长大的好兄弟！"

"……"

两人一边说一边走，身影渐渐消失在山道尽头。

## 三、打谷场上

待回到镇上时，已是午后时分。

镇口的打谷场上仍聚着不少人，都是携儿带女来看热闹的村民。场中央搭起了一座高台，台上铺着青色的地毯，四周插着几面杏黄色的旗帜，在午后的阳光下猎猎作响。高台之上，几名身着月白道袍的年轻人正在忙碌着，或整理器具，或低声交谈，神态从容，气度超然。

据说天道宗每三年会派弟子下山，在中州各城镇寻访有灵根的少年，择其优异者带回宗门培养。这便是所谓的"三年一度的收徒大典"。对寻常百姓而言，家中若能出一名修仙弟子，便是光宗耀祖的大喜事。即便只是被测出有灵根而未能入选，也足以让乡邻刮目相看。

云泽和李狗蛋挤进人群，在高台下方找了个位置站定。

"那便是仙人吗？"李狗蛋瞪大眼睛，仰头望着台上那几名年轻道士，"瞧着和咱们也差不多嘛，也是两个眼睛一条鼻子……"

"别胡说。"云泽低声呵斥，"那些都是天道宗的正式弟子，比咱们不知道高到哪里去了。"

"知道了知道了……"李狗蛋吐了吐舌头，却还是忍不住偷偷打量着台上的那些人。

云泽也在打量着四周。

今日来参加测灵根的人还真不少。粗粗一看，少说也有七八十人，男女老少都有，多是十二到十八岁的少年少女，也有少数几个年过二十的青壮年。他们或紧张、或期待、或忐忑、或兴奋，神态各异，却都伸长了脖子望着台上。

人群的议论声此起彼伏。

"听说张家庄有个娃娃测出了上品灵根，整个村子都轰动了！"

"上品？那还了得？怕不是要被当作核心弟子培养吧？"

"咱们青云镇能出一个就不错了，这穷山恶水的，哪来的好苗子……"

"可不是嘛，上一次测灵根还是三年前，咱们镇上连一个有灵根的都没有……"

云泽听着那些议论，心中暗暗盘算。

上品灵根……那是万中无一的资质。若真有人测出了上品，只怕天道宗的人会亲自出面招揽。

不过他也没太把这事放在心上。他来测灵根，主要是想确认一下自己究竟是什么资质——毕竟他现在已经有了气感，算是半只脚踏入了修仙的门槛。测灵根对他而言，不过是一个验证罢了。

"无咎哥，你紧不紧张？"李狗蛋扯了扯他的袖子，"我瞧你脸色一点都不变，就跟没事人一样。"

"有什么好紧张的？"云泽淡淡道，"成与不成，测了便知。紧张也没用。"

"你这心态也太好了吧……"李狗蛋羡慕地望着他，"我就不行，我腿都软了……"

正说着，高台上忽然传来一阵锣鼓声。

人群顿时安静下来，所有人的目光都朝台上望去。

只见一名中年男子从后台走了出来，身着月白道袍，头戴玉冠，周身隐隐有云雾缭绕，气度超然出尘，恍若画中走出的神仙人物。他的面容清癯，颌下留着一缕短须，眉宇间透着几分威严，一出场便给人一种不怒自威的感觉。

"那是主持今日测灵根的仙师。"李狗蛋凑到云泽耳边，压低声音说，"我上午来的时候就看见他了，据说是天道宗的外门执事，专门负责这次收徒大典的。"

云泽点了点头，目光落在那中年男子身上。

不知为何，他总觉得那道目光有些锐利——像是鹰隼在俯瞰猎物，让人有一种被看穿的感觉。

他不动声色地收回目光，心中却暗暗多了几分警惕。

测灵根的仪式很快便开始了。
