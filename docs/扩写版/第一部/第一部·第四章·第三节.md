# 第一部·第四章·第三节

## 任务堂·故友相逢

三日后，辰时。

天道宗外门任务堂，位于青云峰半山腰的一处开阔地带，是一座三层高的阁楼，青砖黛瓦，飞檐翘角，看上去古朴而庄严。

清晨的阳光洒在阁楼的琉璃瓦上，折射出淡淡的金芒。来往的外门弟子络绎不绝，有的手中提着药篓，显然是刚从外面历练归来；有的则行色匆匆，像是赶着去接任务。

云泽迈步走入任务堂的大门。

一股混合着墨香与草药气息的空气扑面而来。任务堂内部宽敞明亮，四壁悬挂着密密麻麻的任务牌，每一块任务牌上都写着一项宗门任务，按照难度分为一星至五星，密密麻麻，看得人眼花缭乱。

按照天道宗的规矩，所有外门弟子每月必须完成一定数量的任务，否则将扣除修炼资源。这是宗门激励弟子出门历练的一种手段，毕竟修仙之道，不能只知埋头苦修，也需要经风雨、见世面，在生死之间磨砺心性。

云泽这三个月来，一直埋头修炼，除了最初报到时领过一次基础物资之外，还没有正式接过宗门任务。明日便是本月的最后期限，若再不来完成任务，只怕下个月的灵石供给就要被扣减了。

他正四处张望，观察着墙上的任务牌，忽然听见身后传来一个爽朗的声音。

"云兄弟！这么巧！"

云泽回头一看，只见一个圆脸少年笑嘻嘻地跑了过来。

那少年约莫十六七岁，身材圆润，一张圆脸白里透红，笑起来眼睛眯成两道月牙，看着便让人觉得亲近。他穿着一件洗得有些发白的青色道袍，腰间挂着一个鼓鼓囊囊的布袋，整个人透着一股子喜感。

正是当初在前往天道宗路上结识的散修少年——张小胖。

张小胖本名张云鹤，因体型圆润、性格随和，被同门的师兄弟们戏称为"小胖"。他与云泽一见如故，两人都是散修出身，没有世家子弟的傲慢与偏见，一路相伴而来，结下了深厚的友谊。进入天道宗后，虽然分属不同山峰，但两人也常有来往，算是云泽在宗门中为数不多的朋友之一。

"张兄！"云泽笑着打招呼，"你也来接任务？"

"那可不！"张小胖凑上前，压低声音道，"再不完成任务，明儿我的灵石就没了，得喝西北风去。"

他说着，目光在四周扫了一圈，忽然一把拉住云泽的胳膊，将他拽到一个人少的角落里。

"云兄弟，有件事我得提醒你。"

张小胖的神色忽然变得严肃起来，胖乎乎的脸上难得露出几分凝重的表情。

"怎么了？"云泽问道。

"赵乾坤那小子，放出话来了。"张小胖压低声音，"他说要在大比之前好好收拾你，让你知道什么叫规矩。"

云泽眉头微皱。

这三日来，他一直在闭关修炼，不曾出门，倒还真没听说赵乾坤有什么动作。没想到这家伙竟然这么沉不住气，当众丢了脸，就要急着报复回来。

"他还说什么了？"云泽问道。

"还能说什么？"张小胖撇撇嘴，"说你不知天高地厚，敢在林师兄面前出风头，说你一个乡下来的野小子，凭什么让林师兄正眼相看……反正就是那些难听的话，我学都学不来。"

他顿了顿，又道："最可气的是，孙麻那狗腿子还在到处宣扬，说大比之前，一定让你好看。"

云泽沉默片刻。

赵乾坤的报复，早在他预料之中。以对方的性格，当众丢了那么大的脸，不找回场子才怪。

"他打算怎么动手？"云泽问道，"宗门不是有规矩，禁止私斗吗？"

"明面上当然不敢。"张小胖摇摇头，"但你别忘了，大比可不是只有正规比试。每年大比期间，宗门都会开放秘境试炼，让弟子们在秘境中采集灵药、猎杀妖兽，以此作为大比的加分项。"

"而青峰岭，正是外门弟子最常历练的秘境之一。"

他压低声音，眼中闪过一丝精光："据说，赵乾坤已经派人在青峰岭等着了。只要你敢去，他们就敢动手。"

云泽心中一凛。

张小胖说得没错。大比期间的秘境试炼，虽然有宗门长老巡视，但青峰岭地域广阔，若是深入太远，难免会有"意外"发生。往年也不是没有弟子在秘境试炼中"意外"受伤，甚至"意外"身亡的事情，只是大多数都被压了下来，无人追究。

赵乾坤那种人，明着不敢来，暗着却什么事都做得出来。

"那我该怎么办？"云泽问道，"总不能一直躲着不出门吧？"

张小胖眨眨眼，压低声音道："我倒是有个主意。"

"什么主意？"

"这次大比的秘境试炼，正好有一项是前往青峰岭采集灵药。我打听过了，青峰岭虽然是外围区域，但其中也不乏一些珍稀灵药和低级妖兽。若是运气好，说不定能有不小的收获。"

他顿了顿，又道："而且，据我所知，林惊羽师兄每年大比期间都会去青峰岭闭关修炼。以林师兄的修为，整个青峰岭都在他神识覆盖范围之内。若你能在那里遇到他……"

张小胖没有说完，但意思已经很明白了。

林惊羽对云泽的态度，众人都看在眼里。若云泽能再次遇到林惊羽，有这位天道宗第一剑修护着，赵乾坤的人就算再嚣张，也不敢轻举妄动。

云泽沉吟片刻。

张小胖的建议，倒也不失为一个办法。林惊羽对游子剑的反应，让他一直耿耿于怀。他也想知道，这位传说中的天才剑修，究竟与叶孤城、与游子剑有什么渊源。

而更重要的是，他需要在大比之前尽可能提升自己的实力。赵乾坤既然已经放出话来要对付他，他总不能坐以待毙。

"多谢张兄提点。"云泽抱拳道，"这个情，我记下了。"

"嗨，小事小事！"张小胖摆摆手，脸上重新露出笑容，"咱俩谁跟谁啊？你要是有出息了，我不也跟着沾光吗？"

他凑上前，眨眨眼："对了，我正准备接青峰岭的任务，要不咱俩一起去？也好有个照应！"

云泽想了想，点头答应。

张小胖虽然修为不高，但人机灵得很，而且对青峰岭的地形似乎颇为熟悉。有他带路，总比自己一个人瞎摸要强。

两人一同来到任务接取处，各自领了青峰岭灵药采集的任务。

任务的内容并不复杂：在三日之内，采集至少十株二品灵药"青灵草"，以及任意数量的其他灵药。青灵草是炼制聚气丹的辅材，在宗门内有一定的需求量，因此被列为了常规任务之一。

两人又凑了些灵石，在任务堂的兑换处兑换了基础的疗伤丹药和防身符箓。

"这枚回灵丹，一枚五十灵石，关键时刻能救你一命。"张小胖将一枚散发着淡淡药香的丹药塞到云泽手中，"这符篆是'金刚符'，能挡炼气境后期以下的全力一击，只有一枚，省着点用。"

云泽本想推辞，但看张小胖一脸认真的样子，终究没有拒绝。

他收下丹药和符篆，又将自己身上仅有的三十灵石全给了张小胖，算是分摊费用。张小胖本不肯收，但见云泽态度坚决，只好作罢。

走出任务堂时，已是午后。

阳光正好，天道宗山门笼罩在一片金色的光辉之中。云泽抬头望去，只见远处云雾缭绕的山峰之上，隐约可见一座巍峨的宫殿，那便是天道宗的核心——云顶天宫。

传说那里是天道宗历代宗主修行之所，也是九宗大比的举办之地。

云泽看着那座宫殿，心中暗暗发誓：总有一天，他要踏上那座山峰，让所有人看到，一个出身青云镇的孤儿，究竟能在这修仙之路上走多远。

"走吧！"张小胖拍拍他的肩膀，"回去准备一下，明日一早出发！"

云泽点点头，转身向自己的住处走去。

身后，任务堂的喧嚣渐渐远去，前方是未知的挑战与机遇。

他没有回头。

回到住处后，云泽将自己关在房间里，仔细整理着行囊。

包袱里装着几套换洗的道袍、一包干粮、十余枚下品灵石，以及从宗门兑换处领来的基础丹药。这些东西加起来，不过区区数十灵石的财产，却是他全部的家当。

他将那枚回灵丹贴身放好，又将金刚符收入袖中，确保危急时刻能够第一时间取出。

做完这些，他才在窗边坐下，望着窗外渐渐西沉的夕阳出神。

明日，便要启程前往青峰岭了。

青峰岭的历练，对他而言，既是机遇，也是挑战。那里虽然危险，但同样蕴藏着无数珍稀的灵药和妖兽内丹。若能在那里有所收获，不仅仅是完成任务那么简单，更能让他在接下来的外门大比中占据优势。

但更让他在意的，是张小胖说的那句话——林惊羽每年大比期间都会去青峰岭闭关修炼。

林惊羽对游子剑的反应，让他一直耿耿于怀。这位传说中的天才剑修，显然知道一些关于叶孤城、关于游子剑的秘密。若能再次遇到他，或许能得到一些解答。

只是，他也知道，这种机缘可遇不可求。修仙界中，强者为尊，弱者只能仰人鼻息。他如今不过是炼气境三层的修为，在林惊羽眼中，只怕连蝼蚁都算不上。对方愿意与他说话，愿意出手救他，已是看在游子剑的份上。

"归根结底，还是实力不够。"

云泽轻声自语，眼中闪过一丝坚定。

"只有足够强，才有资格追寻真相。只有足够强，才有资格保护自己在乎的人。"

他站起身，走到床边，从枕下取出那本被他翻得起了毛边的《云游心法》，继续研读起来。

夜色渐深，烛火摇曳。

少年伏案苦读的身影，被昏黄的灯光拉得很长很长。

明日，便是新的征程。
