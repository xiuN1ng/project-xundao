# 第一部·第一章·第五节：五行灵根

## 一、照灵镜

测灵根的仪式并不复杂。

负责测灵根的是一枚巴掌大小的铜镜，名为"照灵镜"，是天工宗专为天道宗打造的法器。那铜镜古朴厚重，镜面泛着一层幽幽的青光，一看便知不是寻常之物。

每一个参与测试的少年只需将手掌按在镜面上，镜中便会显出不同的颜色——金、木、水、火、土，五行之色，对应五种灵根。若镜中只显一色，便是天品或上品灵根；若颜色驳杂，则是中品或下品；若镜中一片灰暗，则是无灵根，终生与仙道无缘。

测试进行得很快。

一名天道宗的年轻弟子站在照灵镜旁，负责引导测试者上台、记录结果；另一名弟子则手持笔墨，将每个人的测试结果一一登记在册。

"下一个——张铁蛋！"

"下一个——李二狗！"

"下一个——王大丫！"

一个又一个少年走上前去，将手掌按在镜面上。镜中或亮或暗，有人欢喜有人愁。

"张铁蛋，无灵根。下一个。"

"李二狗，下品火灵根，可入外门候选。下一个。"

"王大丫，中品水灵根，优秀。下一位。"

人群中的议论声此起彼伏。

"十个人里头才出一个有灵根的，这比例也太低了吧……"

"可不是嘛，我听说那些大宗门的弟子，个个都是万中无一的天才。咱们这种穷乡僻壤，能出什么好苗子？"

"听说张家庄出了个上品灵根，不知道是不是真的……"

云泽听着那些议论，心中暗暗盘算。从目前的测试结果来看，约莫十人中有一人能测出灵根，而其中大多数都是下品，能达到中品的寥寥无几，至于上品灵根，更是凤毛麟角。

"无咎哥，快轮到咱们了！"李狗蛋在身旁扯了扯他的衣袖，一脸紧张，"我腿都软了……"

"怕什么。"云泽淡淡道，"成与不成，测了便知。"

"说得也是。"李狗蛋深吸一口气，"无咎哥，你说得对！大不了就是个没灵根呗！我李狗蛋又不是没被拒绝过！"

他说得硬气，可攥紧的拳头还是出卖了他内心的紧张。

云泽望着他那副模样，嘴角微微上扬。

就在这时，高台之上忽然传来一阵骚动。

"上品木灵根！这是上品木灵根！"

那负责记录的天道宗弟子声音里带上了几分激动，原本平静的面容也微微动容："这位小友，你叫什么名字？家住在何处？"

人群中爆发出一阵惊呼，所有人都朝那个方向望去。

云泽顺着众人的目光看去，只见一个年约十四五岁的少年站在高台之上，面容清秀，衣着华贵，一看便是哪家大户人家的公子。那少年的身旁站着一名中年仆从，正一脸骄傲地替他整理衣襟。

"那是张家庄张员外家的小公子，叫什么来着……"

"张灵轩！我记得他！去年赶集的时候见过一面，那孩子从小就聪明伶俐，书读得好，剑也舞得像模像样……"

"上品木灵根啊……啧啧啧，这下可是一飞冲天了……"

人群中议论纷纷，那张灵轩却只是微微昂着头，脸上带着几分矜持的笑意，目光扫过人群，像是在寻找什么。

他的目光忽然在云泽身上停了一瞬。

那目光带着几分好奇，几分审视，像是在打量一件有趣的物什。

然后他便收回了目光，嘴角的笑意更深了几分。

云泽注意到了那道目光，却并未放在心上。他只是静静地站在原地，等待着属于自己的测试。

"下一个——云泽！"

那负责唱名的执事声音清亮，传遍了整个打谷场。

云泽深吸一口气，缓步走上高台。

他的脚步平稳，神色从容，仿佛即将面对的不过是寻常小事，而非决定一生命运的测试。

高台之上，那中年男子的目光终于正式落在了他身上。

云泽能感觉到那道目光的重量——沉稳、锐利，像是要将他看穿。那是一双极深邃的眼睛，眼底仿佛藏着一汪深不见底的潭水，让人无法窥探其中的秘密。

"你叫云泽？"那中年仙师开口，声音不高，却清晰地传入他耳中。

"是。"云泽不卑不亢地答道。

"嗯。"那仙师微微点头，没有多说什么，只是指了指照灵镜，"将手掌按上去便是。"

云泽依言上前，将右手按在了那枚铜镜之上。

铜镜冰凉，入手如同触着一块寒玉。他能感觉到镜面下有什么东西在涌动，像是在探查他的身体，探查他的经脉，探查他体内那缕刚刚萌生的气流。

一息。

两息。

三息。

照灵镜中，忽然亮起了一片光芒。

那光芒五彩斑斓，交织缠绕，仿佛有五种颜色在他体内同时苏醒——金色、青色、蓝色、红色、黄色，五行之色，轮番流转，最终缓缓融合在一起，化作一片柔和的混沌之光。

那混沌之光越来越亮，越来越盛，最终竟透出镜面，将云泽的整个手掌都笼罩其中！

高台之上，那中年仙师的瞳孔骤然紧缩。

"这……"他的声音微微一滞，眼中闪过一丝难以置信的神色。

他霍然站起身来，原本威严的面容上第一次出现了动容之色。

"五行灵根！"

他的声音回荡在打谷场上空，所有人都听得清清楚楚。

"竟然是五行灵根！"

片刻的寂静之后，他忽然仰天长笑：

"好！好！好！五行灵根！中品之上的五行灵根！此等资质，在中州边陲之地，当真是天大的造化！"

他的声音如同惊雷，在打谷场上空炸响。

"五行灵根"四个字，如同一块巨石投入平静的湖面，顿时激起千层浪花。人群中爆发出一阵惊呼，所有人都朝云泽投去或羡慕或嫉妒的目光。

"五行灵根？那是什么？"

"就是金木水火土五种灵根都有！听说这种灵根什么功法都能修炼！"

"天哪，咱们青云镇竟出了这样的天才！"

"云家那个孤儿？啧啧，真是祖坟冒青烟了……"

议论声如潮水般涌来，将云泽淹没其中。

而他只是静静地站在原地，神色平静，仿佛那些议论声与他毫无关系。

他的目光越过人群，越过喧嚣，落在了远方青岚山的轮廓上。

那里，断崖之上，有一具枯骨正静静躺在云雾之中。

云游子前辈，您看到了吗？

您的传承，后继有人了。

## 二、余波

测灵根的结果很快便传遍了整个青云镇。

五行灵根，中品之上——这个结果像是一阵旋风，在短短一个时辰内便刮遍了镇上的每一条大街小巷。有人羡慕，有人嫉妒，有人惊叹，有人质疑，各种议论甚嚣尘上。

"云家那个孤儿？真的假的？"

"人家仙师亲自说的，还能有假？"

"啧啧啧，这命也太好了吧……一个孤儿，居然能测出五行灵根，老天爷也太不开眼了……"

"你懂什么？这叫大难不死必有后福！我早就说过，那孩子不是寻常人物，你们还不信！"

"五行灵根啊……我活了这么大把年纪，还是头一回听说这种灵根。也不知道这孩子日后能有什么出息……"

镇上的茶馆里、酒肆中、田埂地头，到处都在议论这件事。有人替云泽高兴，也有人酸溜溜地冷嘲热讽，更有那等长舌妇人编排出各种离奇的故事，说什么云泽是天上星宿下凡、什么他本就是修仙世家的遗孤、什么他的亲生父母其实是某位大能……

云泽没有心思去理会那些闲言碎语。

测完灵根之后，那中年仙师特意将他叫到一旁，详细询问了他的来历。当得知他是孤儿出身、由镇上的老猎户抚养长大时，那仙师的眼神里闪过一丝复杂的神色，却没有多说什么，只是拍了拍他的肩膀，让他好好准备三个月后的入门试炼。

"以你的资质，只要试炼时不犯大的过错，进入天道宗不成问题。"那仙师说，"这三个月，好好修炼，切莫荒废了这一身的好资质。"

云泽恭敬地应了，又问了几句关于入门试炼的事。那仙师倒也没有藏私，简略地告诉了他一些关于试炼的内容——主要考的是灵根资质、体魄根基、心性悟性三个方面，其中灵根资质只占三成，体魄和心性各占三成半。

"修仙之道，灵根只是门槛，修行还要看个人的努力和机缘。"那仙师意味深长地说，"五行灵根虽好，却也意味着你要付出比常人更多的努力。五行兼修，难度远超单一灵根，若没有足够的毅力和悟性，只怕到头来一事无成。"

云泽将这些话牢牢记在心里。

他离开高台时，李狗蛋早已等在了台下。

"无咎哥！"那小子两眼放光，一把将他抱住，"五行灵根！你是五行灵根！我就知道你不是一般人！咱们镇上那些瞎了眼的混蛋，以后谁还敢说你是孤儿命硬？"

"行了行了。"云泽将他推开，"别在这儿嚷嚷，怪丢人的。"

"丢人？哪里丢人了？"李狗蛋不以为然，"无咎哥，你现在是咱们青云镇的大名人了！五行灵根啊！整个中州能有几个？你知道吗？那些大宗门的天才弟子，很多也就是单灵根或双灵根，五行灵根简直是凤毛麟角！"

他越说越激动，声音也越来越大，引得周围的人都朝这边看过来。

云泽无奈地摇了摇头，也懒得再劝他。

测灵根的大会一直持续到日暮时分，方才结束。

据那执事统计，今日青云镇及周边三村四寨，共有四十三名适龄少年参与了灵根检测。其中测出灵根的，共有一十五人——比例约为三成，比往年高出不少，也算是一桩难得的盛事。

会后，那中年仙师将所有测出灵根的少年召集到高台之下，郑重宣布：

"天道宗三年一度收徒大典将在三个月后举行。届时，凡有灵根者，皆可自行前往中州云雾山天道宗山门，参与入门试炼。通过试炼者，便可正式成为天道宗弟子。"

他的目光在人群中缓缓扫过，最后在云泽和李狗蛋身上停留了一瞬。

"希望诸位好自为之，勿要虚度了这三个月的时光。若能勤加修炼，打好根基，试炼之时便多几分把握。"

说完，他袖袍一挥，一道云雾自脚下升起，将他托上半空，须臾之间便消失在夜色之中，只留下一群目瞪口呆的少年和满地的惊叹之声。

"仙人啊……"

"真是仙人！"

"三个月后，我一定要去天道宗！"

少年们沸腾了。云泽却只是静静地站在原地，抬头望着那已经空无一人的夜空。

三个月。

他只有三个月的时间。
