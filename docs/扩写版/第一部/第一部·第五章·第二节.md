# 第一部·第五章·第二节：三年蜕变

## 一、外门少年

消息传出的当天下午，云帆特意来到云泽的住处，将试炼名单的消息告诉了他。

"云师弟！"云帆的身影落在屋前的空地上，脸上带着掩饰不住的笑意，"好消息！好消息啊！"

云泽正在屋前劈柴，听到声音抬起头来。他手中的柴刀悬在半空，斧刃上还沾着新鲜的木屑，在午后的阳光下泛着淡淡的黄色。

"云帆师兄？"他将柴刀搁在一旁，拍了拍手上的碎屑，"什么好消息？"

"试炼名单出来了！"云帆快步走到他面前，压低声音说道，"我刚从执事堂那边打听到的——你榜上有名！"

"哦。"云泽的反应出乎意料地平淡，只是微微点了点头，"我知道了。"

云帆愣了一下，似乎没想到他会这么平静："云师弟，你就……就这点反应？那可是内门试炼啊！十年才一次！一旦通过就能进入内门，享用上等修炼资源，甚至有机会拜入化神境强者门下！"

"我知道。"云泽弯腰捡起地上的柴刀，继续劈柴，"但激动解决不了问题。"

他举起柴刀，对准一截粗壮的木头，手起刀落，只听"咔嚓"一声，木头应声而裂，断面整齐如镜。

云帆看着他的动作，心中不由得生出几分感慨。

三年了。

三年前那个从青云镇走出的少年砍柴郎，如今已不再是当初那个懵懂无知的少年。他的身形拔高了几寸，肩膀宽阔了些许，眉宇间多了几分沉稳与锐利。但最让云帆印象深刻的，是他身上那股与年龄不符的沉稳——这个少年遇事不惊，即便泰山崩于前也能面不改色，仿佛这世上没有什么能让他动容。

但云帆知道，这不过是表象罢了。

三年前初入天道宗时，云泽还是个毛头小子，见到什么都新鲜，遇到什么事都藏不住表情。这三年来，他亲眼看着这个少年一点点地改变，一点一点地将自己的锋芒收敛起来，变成如今这副深不可测的模样。

有时候云帆会觉得，云泽像是一潭静水——表面上波澜不惊，但水面之下，谁也不知道藏着怎样的暗流涌动。

"云师弟。"云帆忽然开口，"这三年，我看着你一步步走到今天。说实话，我很佩服你。"

云泽劈柴的动作微微一顿，随即又继续。

"师兄过奖了。"他头也不抬，"我不过是运气好罢了。"

"运气？"云帆摇了摇头，"这世上哪有什么运气？你入门时才十三岁，修为比同届弟子差了不止一星半点。但你用了三年时间，硬是追了上来，如今已是外门弟子中的佼佼者。"

他顿了顿，声音里透着几分认真："这份毅力和心性，可不是谁都有的。"

云泽没有说话，只是将最后一截木头劈开，然后直起身来，将柴刀别在腰间。

他转过头，看向云帆。

午后的阳光从他背后洒落，在他脸上投下一片阴影，让人看不清他的表情。但那双眼睛却亮得惊人，像是两汪深不见底的墨潭，里面藏着太多云帆读不懂的东西。

"师兄。"云泽开口，声音很轻，"这次试炼，你怎么看？"

云帆愣了一下："我？"

"嗯。"云泽点点头，"你觉得这次试炼……有什么不对劲的地方吗？"

云帆沉吟片刻，脸上的笑意渐渐收敛，取而代之的是一抹凝重。

"确实有些蹊跷。"他压低声音，"昆仑秘境虽然是我天道宗的试炼场所，但那里面的危险程度，远超普通弟子的应对能力。按照惯例，内门试炼通常会选择一些相对安全的地方，比如宗门境内的几处洞天福地。但这次偏偏选了昆仑秘境……"

他没有说下去，但云泽已经明白了他的意思。

"而且，"云帆的声音更低了，"我听说这次试炼，内门派了一位化神境的长老亲自坐镇。"

"化神境？"云泽眉毛微微一挑，"是哪位？"

云帆看了他一眼，似乎在犹豫要不要说出口。

片刻之后，他深吸一口气，压着嗓子说道："是天璇真人。"

这个名字落入耳中，云泽的瞳孔微微收缩了一下，但只是一瞬间便恢复正常。

"天璇真人……"他轻声重复，语气听不出喜怒，"我听说他是内门德高望重的长老，怎么会对这种试炼感兴趣？"

"谁知道呢。"云帆摇了摇头，"天璇真人在宗门内的地位极高，轻易不会出面主持这类活动。他这次主动请缨，确实让人有些意外。"

他看向云泽，眼中闪过一丝担忧："云师弟，我知道你实力不俗，但这次试炼……你一定要小心。"

"小心？"云泽微微侧头，"师兄指的是什么？"

云帆张了张嘴，似乎想说什么，却又咽了回去。

他总不能直说"我总觉得天璇真人对你不怀好意"吧？这种话传出去，不仅没人会信，反而会给他自己招来麻烦。

"没什么。"他最终只是叹了口气，"就是……昆仑秘境里妖兽横行，危险重重，你千万要注意安全。"

云泽静静地看了他一眼。

他看得出来，云帆没有说实话。但他也知道，云帆之所以不说，不是因为不信任他，而是因为那些话实在太过荒唐——一个外门弟子，怎么会与化神境的内门长老扯上关系？

"多谢师兄提醒。"云泽拱了拱手，"我会注意的。"

云帆点点头，又叮嘱了几句，便转身离去。

待云帆的身影消失在后山的小径上，云泽才收回目光。

他抬起头，望向天璇峰的方向。

那座山峰高耸入云，山顶隐没在云层之中，看不真切。但云泽知道，那云雾深处，藏着一个他必须面对的敌人。

天璇真人。

"无咎小子。"剑灵的声音在脑海中响起，"那个姓云的小子倒是个好人。"

"嗯。"云泽轻声应道，"他是这三年里，为数不多真心对我好的人。"

"那你打算怎么办？"剑灵问道，"这次试炼，你不可避免地要与天璇老贼打交道。你打算瞒着他？"

云泽沉默了片刻。

"不必。"他摇了摇头，"云帆师兄对我有恩，我不愿欺瞒他。但现在还不是说的时候。等这次试炼结束……若是我还能活着回来，我会把一切都告诉他。"

"你倒是挺有自信。"剑灵嗤笑一声，"就不怕自己回不来？"

"怕。"云泽坦然承认，"但怕也没用。该来的总会来，与其逃避，不如正面迎上。"

他低下头，看着自己掌心的茧子，嘴角微微勾起一丝弧度。

"更何况……这三年来，我也不是白练的。"

## 二、夜深人思

入夜。

云泽独自坐在屋中，面前摆着一盏昏黄的油灯。灯火摇曳，在他脸上投下忽明忽暗的光影，让他的面容显得格外深邃。

他伸出手，从怀中取出一卷泛黄的竹简。

那竹简是他偶然间在一处古籍市场发现的，上面记载的是《云游心法》的一些进阶心法。虽然只是残篇，但对于他的修炼依然有不小的帮助。

这三年里，他除了日常的修炼之外，还花了大量的时间去搜集与云游子有关的资料。他想知道更多关于这位前辈的故事，想了解他当年的修炼历程，想知道他为何会被背叛，又为何会落得那般下场。

然而，关于云游子的记载实在太少了。

这个曾经名震天道宗的天才弟子，在宗门的历史中几乎没有任何痕迹。他就像是被人刻意抹去了一般，消失在了岁月的长河之中。

云泽知道，这是天璇真人做的。

二十年前，天璇真人不仅害死了云游子，还抹去了所有关于他的记载。他将自己伪装成正道楷模，将云游子塑造成一个"外出历练遭遇意外"的倒霉鬼，从此高枕无忧地做他的内门长老。

"无咎小子。"剑灵的声音打断了他的思绪，"你在想什么？"

"在想这三年。"云泽轻声回答，"想想自己都经历了什么。"

他将竹简收起，靠在墙上，闭上眼睛。

三年前的青云镇，那个懵懂无知的少年，还不知道自己的命运将走向何方。

他以为自己只是一个普通的孤儿，被云爷爷收养，在青云镇砍柴采药度日。他以为自己这辈子就会这样平淡地过下去，娶妻生子，养老送终，如同镇上世世代代的百姓一样。

然而那场测灵根，改变了一切。

他至今还记得那一日的情景——镇外搭建的高台上，来自天道宗的仙人居高临下地俯视着他们这群孩子，目光如同在看一群蝼蚁。轮到他的灵根被测出时，那位仙人的眼睛亮了一下，随即便问他愿不愿意修仙。

他当时不知道"修仙"意味着什么，只知道云爷爷对他说过："小咎，去吧。去看看外面的世界，别像爷爷一样，一辈子困在这穷山沟里。"

于是他去了。

踏入天道宗的那一刻，他以为自己迎来了新生。但很快，他便发现这个世界远比青云镇复杂得多。

外门弟子的生活并不好过。

这里有出身世家的少爷，有背景深厚的公子，有天赋异禀的天才，也有像他这样什么都没有的穷小子。在这些人眼中，他不过是一个笑话——一个从穷山沟里走出来的土包子，竟然妄想与他们平起平坐？

最开始的半年，是他人生中最艰难的一段时光。

他被分配到最差的住处，干最苦最累的杂活，还要忍受同门师兄弟的冷嘲热讽。没有人看得起他，没有人愿意与他来往，他就像是一株生长在石缝里的小草，拼命地想要往上爬，却总是被人踩在脚下。

但他没有放弃。

他知道自己没有退路。青云镇已经回不去了——不是因为他不思念那里，而是因为他知道，若是在这里失败，他将永远没有机会改变自己的命运。

于是他拼命地修炼。

每日天不亮便起床打坐，夜里其他弟子都睡了，他还在灯下苦读经文。宗门分发给他的丹药，他舍不得吃，全部存下来换成灵石，用来购买更多的修炼资源。

这种苦行僧般的日子，他整整过了三年。

三年里，他几乎没有睡过一个安稳觉。每一个夜晚，他都在思考如何提升自己的实力；每一个清晨，他都比别人更早地出现在修炼场上。

而更重要的是，他遇到了云游子。

准确地说，是遇到了云游子的遗骸和游子剑。

那是入门第二年的事。他在一次宗门任务中偶然闯入了一处禁地，在那里发现了云游子的枯骨和那柄陪伴了他一生的佩剑。

也是在那一次，他得到了云游子留下的传承——不仅是《云游心法》和游子九式剑诀，还有云游子临死前的记忆碎片。

那段记忆碎片里，记载着二十年前那场背叛的真相。

他至今还记得看到那段记忆时的感受——愤怒、悲伤、不甘，还有一种深深的无力感。他为云游子感到不值，为天璇真人的卑鄙感到愤恨，更为自己如今的实力不足感到羞耻。

但他没有让这些情绪冲昏头脑。

他知道自己现在还太弱了。化神境与筑基境之间的差距，如同天堑一般，以他现在的实力，根本无法与天璇真人抗衡。

所以他选择了隐忍。

这三年来，他一直在隐藏自己的真实实力。在外人面前，他只是一个"还不错"的外门弟子，天赋尚可，修炼刻苦，但远没有达到让人忌惮的地步。

他刻意地放慢了自己的修炼速度，让自己看起来"按部就班"地成长，而非一飞冲天式的突进。

他知道，这种隐忍不会持续太久。

只要他还在修炼《云游心法》，只要他还拿着游子剑，天璇真人就迟早会盯上他。

而那一天……似乎已经不远了。

## 三、明月如钩

夜深了。

云泽走出屋外，抬头望向天空。

一轮明月高悬，清辉洒落，将青云峰笼罩在一片银白色的光芒之中。山间的夜风带着几分凉意，吹动他的衣袂，发出轻微的声响。

他站在屋前，静静地望着那轮明月，心中思绪万千。

三年前的这个时候，他还在青云镇。

那时候他也常常在夜里抬头望月，思念着云爷爷，想念着青云镇的一草一木。而现在……

他低下头，看了看自己的手掌。

三年过去了，他已经不再是当初那个只会在夜里望着月亮发呆的少年。他变了——变得沉稳，变得内敛，变得学会了隐藏自己的情绪和锋芒。

但有些东西，始终没有变。

他依然记得自己为何而出发，依然记得自己踏上这条路时的誓言。

变强。

保护想保护的人。

找到真相。

还有……复仇。

"爷爷……"他轻声呢喃，声音被夜风吹散，"您在天上看着吗？我会变强的。总有一天，我会让您看到——您的云泽，真的成为了大英雄。"

他闭上眼睛，深深地吸了一口气。

然后，他转身走回屋内。

明天还有很多事要做。昆仑秘境的试炼即将开始，他需要做好万全的准备。

天璇真人既然布下了这个局，那他就奉陪到底。

来吧。
