# 第一部·第四章·第七节

## 剑道传承·隐秘渊源

两人离开那片隐秘山谷时，天色已近黄昏。

夕阳西沉，将连绵的青峰岭染成一片金红之色。晚霞如火，铺满了半边天际，将远处的山峰映照得如同燃烧的火焰。

林惊羽御剑而行，速度不快，似乎有意放慢，等待着云无咎。

云无咎施展轻功，紧紧跟在他身后，心中却仍在回味方才在山谷中经历的一切。

那具枯骨……究竟是谁？

他为何知道叶孤城？又为何会出现在青峰岭深处？

最关键的是，他为何要将自己残存的残念传递给云无咎？

这些问题，像是一团乱麻，缠绕在云无咎的心头。

"无咎师弟。"

林惊羽的声音忽然响起。

"林师兄。"云无咎抬头。

林惊羽没有回头，只是淡淡说道："你今日在山谷中看到了什么，不妨与我细说。"

他的声音依旧清冷，却多了几分认真与郑重。

云无咎沉默片刻，斟酌着措辞。

他本想将那段记忆完整地告诉林惊羽，但话到嘴边，却又咽了回去。

那段记忆中，涉及到天道宗的建立、天玄子的背叛、以及天道剑的下落。这些事情若是属实，那天道宗的根基将从根本上被动摇。而林惊羽身为天道宗弟子，对这些事情的接受程度如何，云无咎并不确定。

更重要的是，他与林惊羽相识不过数日，虽然对方对他似乎颇为欣赏，但终究不算深交。在没有完全信任对方之前，他不敢轻易将这等惊天秘密全盘托出。

"我看到了一段残破的记忆。"云无咎最终选择了一个折中的说法，"关于叶孤城前辈封印饕餮时的情景。"

"还有呢？"林惊羽问道。

"还有一些……关于游子剑来历的记载。"云无咎顿了顿，"具体的内容，我一时也说不清楚。只是隐约觉得，这柄剑比我想象的要复杂得多。"

他没有撒谎，只是隐去了关于天玄子的部分。

林惊羽沉默良久。

"无咎师弟，"他忽然开口，"你可知道，我为何会出现在青峰岭？"

云无咎摇头。

"三个月前，我在一次闭关修炼中，忽然感应到一股熟悉的气息。"林惊羽的声音低沉，"那是祖师传承的剑意……不，不止是剑意，还有一缕残魂的气息。"

"残魂？"

"不错。"林惊羽转头看向云无咎，目光深邃，"我的师承，来自太古剑圣叶孤城一脉。祖师当年陨落时，将毕生剑道传承分作三份，分别封入三柄神剑之中。"

"天道剑、游子剑、飞仙剑……你手中的游子剑，便是其中之一。"

云无咎心中一震。

这个说法，与云游子前辈告诉他的基本吻合。只是云游子说得比较简略，而林惊羽则更为详细。

"我一直在寻找这三柄剑的下落。"林惊羽继续说道，"三月前，我感应到青峰岭方向有异动，便前来查探。没想到……竟在那里遇到了你。"

他看向云无咎腰间的游子剑，目光中闪过一丝复杂的情绪。

"当年祖师陨落时，曾留下遗言：'天道有变，吾剑流落凡间。有缘者得之，当承吾志，斩尽世间不公。'"

"我一直在寻找这三柄剑的踪迹，却始终一无所获。没想到……游子剑竟然在你手中。"

林惊羽的声音中带着一丝感慨，一丝释然，还有一丝说不出的羡慕。

云无咎心中百感交集。

原来，林惊羽对游子剑的反应，竟是源于这层关系。他是叶孤城的传人，一直在寻找祖师留下的三柄神剑。而游子剑，正是其中之一。

"林师兄，"云无咎忽然问道，"祖师的剑中既然封有残魂，那这缕残魂……是否还有意识？"

林惊羽脚步一顿。

他转头看向云无咎，目光中闪过一丝讶异，似乎没想到他会问出这样的问题。

片刻后，他摇了摇头。

"祖师的残魂早已消散，如今三柄剑中残留的，只是祖师生前留下的一道意志烙印，并无独立意识。"

他顿了顿，又道："但每一柄剑中封存的剑道感悟，都是祖师毕生心血的结晶。若能完全继承其中剑道，假以时日，未必不能达到祖师当年的高度。"

"当然，前提是……你能活到那一天。"

这句话说得平淡，却带着一股说不出的寒意。

云无咎心中一凛。

他隐隐觉得，林惊羽话中有话。

"林师兄，"他试探着问道，"您是否知道……当年叶祖师陨落的真相？"

林惊羽的脚步再次顿住。

他沉默了很长一段时间，长到云无咎以为他不会回答。

"不知道。"

最终，林惊羽只是淡淡说了三个字。

但云无咎分明看到，他说这两个字的时候，眼中闪过一丝极淡极淡的……怨毒？

那怨毒不是对云无咎，而是对某个不在场的人，或者某段被掩埋的历史。

两人一路无话，直到回到青峰岭外围时，天色已经完全暗了下来。

张小胖早已等得心急如焚。

他躲在一块大石头后面，探出脑袋四处张望。见到两人平安归来，顿时喜出望外，连滚带爬地跑了过来。

"云兄弟！你没事吧？里面怎么样了？"

云无咎笑着拍了拍他的肩膀："没事，遇到了一点小麻烦，不过已经解决了。"

张小胖松了口气，又好奇地看向林惊羽。

"那这位是……"

"林惊羽师兄。"云无咎介绍道。

张小胖瞪大了眼睛，满脸难以置信。

"林……林惊羽？！天道宗第一剑修林惊羽？！"

他吓得差点跳起来，连忙躬身行礼。

"小的张云鹤，见过林师兄！久仰大名，如雷贯耳！没想到小的今日竟有幸得见林师兄真容，三生有幸啊三生有幸！"

林惊羽眉头微皱。

他显然不太习惯这种热情，只是淡淡"嗯"了一声，算是回应。

"今日时辰已晚，你们先回宗门休息。"林惊羽转向云无咎，"三月后的外门大比，你好好准备。若能进入前十，可来东峰寻我。"

说罢，他御剑而起，化作一道白光，消失在夜空之中。

张小胖目瞪口呆地看着林惊羽消失的方向，半晌才回过神来。

他一把抓住云无咎的胳膊，满脸震惊与兴奋。

"云兄弟！你什么时候认识的林师兄？！这等人物，你怎么不早告诉我！"

云无咎无奈地笑了笑："说来话长，我们边走边说。"

"还有！你在山谷里遇到了什么？那头青鳞狼是怎么回事？林师兄为什么会突然出现？"

张小胖的问题像连珠炮一样，一个接一个。

云无咎摇摇头，避开了关于枯骨和记忆的话题，只是简单地说遇到了林惊羽，被他顺手救了。

张小胖虽然有些失望，但也知道有些事情不方便问，便没有继续追问。

两人踏上了返回天道宗的路途。

夜色深沉，星月无光。

云无咎走在山间小径上，心中却依然无法平静。

今日在山谷中经历的一切，像是一场梦。那具枯骨传递给他的记忆、那段关于天玄子的历史、以及林惊羽对剑道传承的解释……太多的信息涌入他的脑海，让他一时之间难以消化。

天玄子。

这个名字，像是一根刺，扎在他心上。

从云游子前辈的口中，他得知天璇真人便是陷害云游子的仇人。而从今日那段记忆中，他又得知天玄子是天璇峰的幕后黑手，是背叛叶孤城的元凶。

这两者之间，是否有什么联系？

天璇真人与天玄子，是否是同一个人？

还是说，天璇真人只是天玄子的后继者或代言人？

云无咎不知道答案。

但他知道，自己的修仙之路，远比他想象的要危险得多。

而他手中的游子剑，正是这一切的关键。

两人在山间小径上走了约莫一个时辰，便回到了青峰岭外围与张小胖约定好的汇合点。

此时已是深夜，天上繁星点点，月光如水银般洒落大地。远处的山峰在月色中显得格外静谧，仿佛沉睡的巨兽。

张小胖早已等得心急如焚。

他躲在一块大石头后面，探出脑袋四处张望，见到两人的身影后，顿时喜出望外，连滚带爬地跑了过来。

"云兄弟！你没事吧？里面怎么样了？林师兄怎么也跟你们一起出来了？"

他的问题像连珠炮一样，一个接一个，脸上的表情又是担忧又是好奇。

云无咎笑了笑，拍拍他的肩膀："没事，遇到了一点小麻烦，不过已经解决了。林师兄是路过，顺手帮了一把。"

他没有提山谷中的枯骨，也没有提那段震撼的记忆。有些事情，知道的人越少越好。

张小胖虽然有些失望，但也知道有些事情不方便问，便没有继续追问。

三人一同踏上了返回天道宗的路途。

一路上，张小胖叽叽喳喳地说着话，讲述着他在外围等待时遇到的种种趣事——什么遇到了一头迷路的小狐狸啦，什么在一棵老树上发现了灵雀的巢穴啦，什么差点被一只发狂的野猪追得满山跑啦……

云无咎听着他的絮叨，心情也渐渐放松了下来。

在这个尔虞我诈、步步惊心的修仙界里，张小胖是为数不多真心待他的人。这种纯粹的友情，让他感到一丝难得的温暖。

"无论前方有什么在等待着我……"

云无咎握紧手中的剑，目光坚定。

"我都会一步步走下去。"

夜风呼啸，吹动他的衣袂。

少年握剑而行，身影渐渐消失在夜色之中。
