# 大型活动方案

## 一、仙道大会（单人竞技）

仙道大会是面向全体修士的单人竞技玩法，分为预选赛、正赛、决赛三个阶段。玩家通过匹配对手进行实时对战，胜负取决于功法搭配、法宝运用与策略选择。预选赛采用积分制，正赛采用淘汰赛，最终决出前十名获得稀世称号与丰厚奖励。

## 二、宗门战（宗门对抗）

宗门战是两个宗门之间的团队对抗玩法，每周限时开启。每个宗门最多50人参与，由宗主或长老指挥作战。战斗分为据点争夺、资源抢夺、最终决战三个环节，占领更多据点并在最终决战中获胜的宗门将获得宗门荣耀宝箱，成员可获得大量贡献度与稀有道具。

## 三、活动时间、奖励与参与条件

**活动时间**：仙道大会每月举办一次，为期7天；宗门战每周六20:00-22:00开启。

**参与条件**：仙道大会需角色等级≥30级且战力达到要求；宗门战需加入宗门且宗门等级≥2级。

**活动奖励**：仙道大会冠军可获称号"仙道魁首"、稀有仙器和大量灵石；宗门战获胜宗门全员获得宗门贡献度、限定时装碎片和稀有材料。
