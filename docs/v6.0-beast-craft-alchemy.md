## 1. 灵兽系统

### 1.1 系统概述

灵兽系统是玩家在修仙世界中的忠实伙伴，为玩家提供战斗辅助、资源产出、属性加成等多维度价值。玩家可通过捕捉、孵化、繁殖等方式获得灵兽，并对其进行培养进化，使其成为修仙路上的强大助力。

### 1.2 灵兽类型

| 灵兽分类 | 品质 | 定位 | 特点 |
|----------|------|------|------|
| 兽类 | 白/绿/蓝/紫/金 | 物理输出 | 高物理攻击，防御较低 |
| 禽类 | 白/绿/蓝/紫/金 | 速度控制 | 高速，先手控制 |
| 虫类 | 白/绿/蓝/紫/金 | 持续伤害 | 中毒debuff，持久战 |
| 鳞类 | 白/绿/蓝/紫/金 | 防御辅助 | 高防御，护盾技能 |
| 精怪类 | 白/绿/蓝/紫/金 | 法术输出 | 高法术伤害，范围攻击 |
| 神兽类 | 紫色/金色 | 全能 | 高成长，全属性加成 |

### 1.3 核心玩法

#### 灵兽捕捉
- **捕捉场景**: 野外挂机时有概率遇到野生灵兽
- **捕捉方式**: 消耗「灵兽袋」或「捕兽网」进行捕捉
- **捕捉概率**: 灵兽品质越高，捕捉成功率越低

| 灵兽品质 | 捕捉成功率 | 所需道具 |
|----------|------------|----------|
| 白色 | 80% | 捕兽网 |
| 绿色 | 60% | 捕兽网 |
| 蓝色 | 40% | 灵兽袋 |
| 紫色 | 20% | 高级灵兽袋 |
| 金色 | 5% | 锁灵阵 |

#### 灵兽培养
- **升级**: 战斗中自动积累经验，服用「兽灵丹」可额外获得经验
- **进化**: 达到等级上限后，消耗进化材料进行进化
- **资质提升**: 使用「资质丹」提升灵兽资质
- **技能学习**: 通过「技能书」让灵兽学习技能

#### 灵兽战斗
- 玩家可携带最多3只灵兽参战
- 灵兽自动释放技能
- 灵兽死亡后需休息一段时间才能再次出战

### 1.4 灵兽数值设计

#### 灵兽属性公式

基础属性 = 品质系数 × 种族系数 × 等级
品质系数: 白色1.0 / 绿色1.5 / 蓝色2.0 / 紫色3.0 / 金色5.0

## 2. 炼器系统

### 2.1 系统概述

炼器系统是玩家打造装备、法宝的核心系统。通过收集材料、学习配方、掌握火候，玩家可以炼制出强大的武器、防具和各类法宝。炼器产物直接提升玩家战斗力，是修仙路上不可或缺的成长途径。

### 2.2 法宝类型

| 分类 | 作用 | 代表法宝 |
|------|------|----------|
| 武器 | 提升攻击 | 青虹剑、太乙金仙剑 |
| 防具 | 提升防御/气血 | 玄冰甲、金缕玉衣 |
| 饰品 | 特殊效果 | 乾坤圈、混天绫 |
| 法宝 | 辅助战斗 | 九龙神火罩、翻天印 |

### 2.3 核心玩法

#### 炼器流程
1. **选择配方**: 从已学习的配方中选择要炼制的装备
2. **放入材料**: 按配方要求放入对应材料和灵石
3. **控制火候**: 通过小游戏或自动模式控制炼器火候
4. **开炉取宝**: 炼器完成后获得装备

#### 配方系统
- 配方分为「武器」「防具」「饰品」「法宝」四类
- 配方需要通过任务、副本、探索等方式解锁
- 高品质配方炼制出的装备属性更强

#### 炼器技巧
- **火候控制**: 完美火候可提升装备品质
- **材料加成**: 放入高品质材料可提升属性
- **幸运祝福**: 使用「幸运符」可提升出高品质概率

## 3. 丹药系统

### 3.1 系统概述

丹药系统是修仙世界中提升修炼效率、突破境界瓶颈的关键系统。玩家通过收集灵草、矿石，学习丹方，炼制各种功效的丹药。丹药可用于加速修炼、突破境界、提升属性、治疗伤病等多重用途。

### 3.2 丹药分类

| 分类 | 功效 | 代表丹药 |
|------|------|----------|
| 修炼类 | 提升修炼速度 | 聚灵丹、引气丹 |
| 突破类 | 增加突破成功率 | 破境丹、渡劫丹 |
| 属性类 | 临时提升属性 | 力量丹、敏捷丹 |
| 恢复类 | 恢复气血/灵气 | 回春丹、灵气丹 |
| 特殊类 | 特殊效果 | 悟道丹、轮回丹 |

### 3.3 核心玩法

#### 炼丹流程
1. **选择丹方**: 从已学习的丹方中选择要炼制的丹药
2. **放入药材**: 按丹方要求放入对应药材
3. **控制火候**: 通过控制火力大小影响成丹品质
4. **开炉取丹**: 炼丹完成后获得丹药

#### 丹方系统
- 丹方分为「人」「地」「天」「仙」四品
- 高品丹方需要更稀有的药材和更高的炼丹技术
- 丹方通过任务、探索、师徒传承等方式获得

#### 炼丹技巧
- **火力控制**: 文火稳定，武火迅猛，三味真火最高效
- **药材品质**: 高品质药材提升成丹品质
- **手法技巧**: 搅拌时机、封印技巧影响最终品质

## 4. 系统联动设计

### 4.1 灵兽与战斗系统

```
灵兽系统
    ↓
    ├─→ 出战 → 战斗伤害/防御加成
    ├─→ 技能 → 战斗控制/治疗/增益
    └─→ 死亡 → 休息时间 → 无法出战
```

- 灵兽直接参与玩家战斗
- 灵兽技能可与玩家功法叠加
- 灵兽死亡不影响玩家但影响整体战斗力

### 4.2 灵兽与炼器系统

```
炼器系统
    ↓
    ├─→ 装备强化 → 灵兽装备
    └─→ 炼器材料 → 灵兽进化材料
```

- 玩家可为灵兽炼制专属装备
- 炼器副产物可作为灵兽进化材料
- 灵兽装备同样可以强化和进阶

### 4.3 灵兽与炼丹系统

```
炼丹系统
    ↓
    ├─→ 兽灵丹 → 灵兽升级
    └─→ 资质丹 → 灵兽资质提升
```

- 炼丹产出灵兽培养所需丹药
- 灵兽专用的兽灵丹和资质丹
- 高品质灵兽需要高品质丹药培养

### 4.4 炼器与炼丹系统

```
炼器系统 ←→ 炼丹系统
    ↓              ↓
 废弃装备 → 回炉 → 炼丹材料
    ↓              ↓
 药材 → 炼器 → 法宝/容器
```

- 废弃装备可回收为炼器材料
- 部分药材需要特殊容器保存
- 炼器产出可作为炼丹辅助道具

### 4.5 综合联动

```
         ┌──────────────┐
         │   灵兽系统    │
         │  战斗伙伴     │
         └──────┬───────┘
                │
    ┌───────────┼───────────┐
    ↓           ↓           ↓
┌───────┐  ┌───────┐  ┌───────┐
│炼器系统│  │炼丹系统│  │境界系统│
│装备强化│  │修炼加速│  │突破辅助│
└───┬───┘  └───┬───┘  └───┬───┘
    ↓          ↓          ↓
    └──────────┬──────────┘
               ↓
        ┌──────────────┐
        │   战斗系统    │
        │ 综合战力提升  │
        └──────────────┘
```

## 5. 数据结构

### 5.1 灵兽数据

```javascript
beast: {
    beasts: {
        [beastId]: {
            templateId: string,      // 灵兽模板ID
            name: string,            // 灵兽名称
            quality: string,         // 品质 white/green/blue/purple/gold
            level: number,           // 当前等级
            exp: number,             // 当前经验
            aptitude: number,        // 资质 1-100
            skills: string[],        // 技能列表
            equipped: {              // 装备槽位
                weapon: string,
                armor: string,
                accessory: string
            },
            status: 'idle' | 'fighting' | 'resting',
            restUntil: timestamp,    // 休息结束时间
            acquiredAt: timestamp    // 获取时间
        }
    },
    battleTeam: [beastId, beastId, beastId],  // 出战队列
    totalBeasts: number
}
```

### 5.2 炼器数据

```javascript
smithing: {
    formulas: string[],               // 已解锁配方
    equipment: {
        [equipmentId]: {
            templateId: string,      // 装备模板ID
            quality: string,          // 品质
           强化等级: number,          // 强化等级
            effects: string[],        // 特效列表
            equipped: boolean,        // 是否已装备
            createdAt: timestamp
        }
    },
    enhanceLevel: number              // 炼器台等级
}
```

### 5.3 炼丹数据

```javascript
alchemy: {
    formulas: string[],               // 已解锁丹方
    pills: {
        [pillId]: {
            templateId: string,      // 丹药模板ID
            quality: string,          // 品质
            quantity: number          // 数量
        }
    },
    furnace: {
        level: number,               // 丹炉等级
        exp: number                  // 熟练度
    },
    herbs: {
        [herbId]: {
            quantity: number,
            quality: string
        }
    }
}
```
