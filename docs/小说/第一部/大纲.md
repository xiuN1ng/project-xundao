# 第一部：凡人之始

> 寻道修仙 · 起点篇

---

## 概述

**主角**：云泽（普通少年，意外获得上古散修"云游子"传承）
**时间线**：第九纪元，灵气复苏初期
**地点**：中州 → 青云镇 → 天道宗
**境界跨度**：淬体境 → 筑基境

---

## 大纲

### 第一章：山野少年
**等级**：1-5级（淬体境）

- 云泽，青云镇孤儿，以砍柴为生
- 意外在后山悬崖石缝中发现盘坐枯骨（云游子遗骸）
- 获得《云游心法》玉简与"游子剑"
- 触碰玉简瞬间觉醒气感，踏入修炼之途
- 第一次感受灵气，体验淬体境的感觉

### 第二章：仙缘初显
**等级**：5-8级（淬体境）

- 青云镇出现妖兽伤人事件
- 云泽初战妖兽野猪妖，险胜
- 镇上来了天道宗外门弟子检测灵根
- 云泽被测出"五行灵根"——非天品但可修炼
- 仙师留下三个月后九宗收徒大会的消息

### 第三章：初入道途
**等级**：8-10级（淬体境巅峰）

- 云泽开始正式修炼《云游心法》
- 修炼中遇到瓶颈，在李狗蛋帮助下突破
- 选择：救助被妖兽抓走的李狗蛋
- 云泽独自追击，深入妖兽领地
- 险象环生但获得重要机缘：灵果

### 第四章：九宗大选（上）
**等级**：10-15级（炼气境初入）

- 前往天道宗参加九宗收徒大会
- 路途经历：东州边境遭遇劫匪，云泽展现修炼者实力
- 天道宗山门初现，云海亭台，如入仙境
- 外门选拔第一关：资质检测
- 云泽的五行灵根引起小范围关注

### 第五章：九宗大选（下）
**等级**：15-20级（炼气境）

- 外门选拔第二关：心性试炼（幻境）
- 云泽在幻境中直面内心恐惧，展现坚韧心性
- 第三关：实战考核
- 云泽对战各路高手，展现游子剑法的雏形
- 成功进入外门，获得外门弟子身份

### 第六章：外门岁月
**等级**：20-25级（炼气境中期）

- 外门弟子生活：与云帆师兄相遇
- 宗门任务系统开放
- 第一次执行宗门任务：青云秘境外围探索
- 获得基础装备，开始锻造修炼
- 了解宗门基本格局

### 第七章：暗中观察
**等级**：25-30级（炼气境后期）

- 云泽在宗门中低调修炼，暗中调查游子剑来历
- 发现云游子千年前的死亡疑点重重
- 游子剑剑灵首次显化，给出模糊指引
- 偶然得知外门大比即将开始
- 结识同门师姐苏晴雨

### 第八章：锋芒初露
**等级**：30级（筑基境突破）

- 外门大比开始，云泽一路过关斩将
- 关键对手：外门高手周天成
- 游子剑剑灵在战斗中觉醒部分力量
- 云泽险胜周天成，震惊外门
- 引起内门长老注意——天璇真人

### 尾声：新的起点
- 天璇真人出现，收云泽为记名弟子
- 云泽得知天璇真人与云游子的渊源
- 师父透露：游子剑背后藏着惊天秘密
- 第一部结束，云泽正式踏入修仙界

1. **游子剑秘密**：云游子之死与天璇真人有关
2. **天道图碎片**：云游子生前曾参与争夺
3. **大劫征兆**：灵气复苏并非偶然
4. **五行灵根**：看似普通，实则暗藏玄机
5. **道君转世**：主角真实身份的伏笔
