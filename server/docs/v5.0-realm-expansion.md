# 挂机修仙 v5.0 境界系统扩展

**版本**: v5.0  
**主题**: 境界系统深度扩展  
**预计开发周期**: 2周

## 1. 境界突破机制优化

### 1.1 突破准备阶段

#### 灵气阈值系统
- 每个境界设置灵气上限要求
- 突破需要收集对应的天材地宝作为引子
- 新增「悟道茶」「渡劫丹」等突破辅助道具

| 境界 | 灵气要求 | 突破道具 |
|------|----------|----------|
| 练气→筑基 | 1000 | 引气符×3 |
| 筑基→金丹 | 5000 | 聚灵丹×5 |
| 金丹→元婴 | 20000 | 养魂玉×3 |
| 元婴→化神 | 80000 | 渡劫丹×1 |
| 化神→炼虚 | 300000 | 悟道茶×3 |
| 炼虚→合体 | 1000000 | 天地灵乳×5 |
| 合体→大乘 | 5000000 | 混沌之气×1 |
| 大乘→仙人 | 20000000 | 仙根×1 |

#### 突破状态
- **准备期**: 可进行突破仪式
- **进行中**: 突破动画与交互
- **成功**: 境界提升，属性飞跃
- **失败**: 返还部分材料，保留进度

### 1.2 突破难度系统

#### 心魔系统
- 突破时有概率遭遇心魔攻击
- 心魔强度随境界提升
- 战胜心魔获得额外属性加成
- 失败不影响突破成功

#### 护法系统
- 可邀请好友担任护法
- 护法可协助击退心魔
- 护法获得贡献奖励

## 2. 渡劫系统

### 2.1 天劫类型

#### 雷劫
- 最常见的劫难
- 需要硬抗雷电伤害
- 法宝可减免部分伤害
- 渡劫成功获得雷劫之体

#### 风劫
- 考验身法与防御
- 持续掉血机制
- 需要及时补充气血
- 渡劫成功获得风灵之体

#### 火劫
- 范围持续伤害
- 需要水系法宝或功法
- 渡劫成功获得火德之身

#### 混合劫
- 高境界渡劫出现
- 多种劫难同时降临
- 考验综合实力

### 2.2 渡劫准备

#### 渡劫阵法
- 可在洞府布置渡劫阵
- 阵法提供属性加成
- 需要收集阵法材料

#### 渡劫道具
- 替身符：抵挡一次伤害
- 护体金莲：持续回血
- 避雷针：减免雷劫伤害

### 2.3 渡劫奖励

#### 仙位册封
- 渡劫成功后获得仙位
- 仙位提供永久属性加成
- 仙位可升级

| 仙位 | 属性加成 |
|------|----------|
| 地仙 | 气血+10%, 攻击+5% |
| 天仙 | 气血+20%, 攻击+10% |
| 金仙 | 气血+30%, 攻击+15% |
| 大罗金仙 | 气血+50%, 攻击+25% |
| 准圣 | 气血+80%, 攻击+40% |
| 圣人 | 气血+100%, 攻击+50% |

## 3. 境界专属功法

### 3.1 功法解锁机制

- 每个境界有专属功法
- 突破到新境界自动解锁
- 专属功法无法通过其他途径获得

### 3.2 各境界专属功法

#### 筑基期
- **太乙心法**: 气血上限+15%
- **御剑飞行**: 移动速度+20%

#### 金丹期
- **金丹真解**: 灵气恢复速度+25%
- **三昧真火**: 战斗伤害+15%

#### 元婴期
- **元婴出窍**: 可分离元婴进行探索
- **神念化形**: 视野范围+30%

#### 化神期
- **虚空行走**: 穿越地形限制
- **领域展开**: 战斗时生成领域

#### 炼虚期
- **化身千万**: 分身协同战斗
- **法则感悟**: 功法效果+20%

#### 合体期
- **天人合一**: 全属性+10%
- **大道共鸣**: 渡劫伤害-15%

#### 大乘期
- **仙人模式**: 战斗时有概率触发仙人形态
- **飞升之光**: 散仙状态下属性不衰减

## 5. 数据结构

### 5.1 新增数据表

```javascript
// 突破数据
breakthrough: {
    stage: 'preparing' | 'processing' | 'success' | 'failed',
    attempts: number,
    lastAttempt: timestamp,

// 渡劫数据
tribulation: {
    times: number,           // 渡劫次数
    types: string[],         // 渡劫类型
    successes: number,       // 成功次数
    currentType: string,     // 当前天劫类型
    equipment: string[],     // 装备的道具
    realm: string            // 仙位
}

// 专属功法
realmSkills: {
    unlocked: string[],      // 已解锁功法ID
    equipped: string         // 当前装备的专属功法
}
```
