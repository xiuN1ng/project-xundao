# 挂机修仙 - 养成深度数值调整方案

## 一、当前问题分析

### 1.1 境界系统
- ❌ 存在两套境界数值（主角120-131行 vs 灵兽134-203行），需统一
- ❌ 突破消耗曲线前期过缓，后期陡峭，玩家容易流失
- ❌ 灵气获取效率在后期相对不足

### 1.2 功法系统
- ⚠️ 功法加成系数偏低（最高4.5倍）
- ⚠️ 辅助功法效果较弱
- ❌ 功法升级/突破机制缺失

### 1.3 灵兽系统
- ❌ 灵兽境界数值与主角不统一
- ❌ 灵兽属性加成较低
- ❌ 灵兽进化机制需完善

### 1.4 装备/法宝系统
- ⚠️ 法宝属性加成不明显
- ❌ 强化/精炼机制需完善

### 1.5 宗门系统
- ⚠️ 建筑效果加成较弱
- ❌ 弟子系统数值需优化

### 1.6 仙侣系统
- ⚠️ 双修加成效果较弱
- ❌ 亲密度系统数值需完善

## 二、调整方案

### 2.1 境界养成线优化

**目标：前期平滑过渡，后期有挑战性，保持玩家成长体验**

| 境界 | 突破消耗(原→新) | 灵气速率(原→新) | 属性成长(HP/ATK/DEF) |
|------|----------------|----------------|---------------------|
| 凡人→练气 | 50→30 | 6→8 | 50/5/0 → 80/8/2 |
| 练气→筑基 | 200→100 | 30→35 | 150/15/5 → 200/20/8 |
| 筑基→金丹 | 7500→2000 | 90→120 | 400/40/15 → 600/60/25 |
| 金丹→元婴 | 300000→30000 | 150→200 | 1000/100/40 → 1500/150/60 |
| 元婴→化神 | 8000000→500000 | 150→300 | 3000/300/120 → 5000/500/200 |
| 化神→炼虚 | 50000000→5000000 | 500→800 | 10000/1000/400 → 15000/1500/600 |
| 炼虚→合体 | 200M→30M | 1500→2000 | 30000/3000/1200 → 45000/4500/1800 |
| 合体→大乘 | 1000M→100M | 5000→6000 | 80000/8000/3000 → 120K/12K/4500 |
| 大乘→渡劫 | 10000M→1000M | 20000→25000 | 200K/25K/10K → 300K/35K/15K |
| 渡劫→仙人 | 100000M→10000M | 100000→120000 | 1M/100K/50K → 1.5M/150K/75K |

**境界突破消耗降低50%（前期）到90%（后期），灵气速率提升20-30%**

### 2.3 灵兽养成线优化

**目标：灵兽成为重要战力辅助，与境界联动**

- 统一灵兽境界数值与主角一致
- 灵兽属性加成：HP/ATK/DEF 获得 境界等级×5% 加成
- 灵兽进化机制：每3个境界可进化一次
- 灵兽技能：增加主动技能和被动技能

### 2.4 装备/法宝系统优化

**目标：法宝成为核心战力之一**

- 法宝基础属性提升50%
- 强化效果翻倍（每级+5% → +10%）
- 增加法宝精炼系统（额外属性加成）
- 增加套装效果（2件/4件套）

### 2.5 宗门养成优化

**目标：宗门成为资源产出重要来源**

- 建筑效果提升50%
- 弟子效果提升30%
- 增加宗门技能树（消耗贡献学习）

### 2.6 仙侣养成优化

**目标：仙侣成为双修核心辅助**

- 双修效率提升50%
- 亲密度效果增强：
  - 1000亲密度：修炼+5%
  - 5000亲密度：修炼+15%
  - 10000亲密度：修炼+30%
- 增加仙侣技能（被动加成）

### 2.7 多系统协同

**目标：各系统相互关联，形成养成闭环**

| 组合效果 | 加成 |
|---------|------|
| 功法+境界 | 功法效果受境界加成（境界等级×2%） |
| 灵兽+仙侣 | 灵兽训练效率+10%/亲密度等级 |
| 宗门+法宝 | 炼器成功率+5%/宗门等级 |
| 全部满级 | 最终成为真正的仙人 |
