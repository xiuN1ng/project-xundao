# 挂机修仙 v5.2 社交扩展

**版本**: v5.2  
**主题**: 宗门战、跨服竞技  
**预计开发周期**: 3周

## 1. 宗门战系统

### 1.1 宗门战类型

#### 资源战
- 争夺灵矿归属权
- 占领时间越长收益越高
- 可使用战术策略

#### 领地战
- 争夺宗门领地
- 领地提供属性加成
- 失败方失去领地

#### 复仇战
- 宗门间仇杀触发
- 生死状机制
- 胜利方获得赔偿

### 1.2 战斗机制

#### 参赛条件
- 宗门等级≥3
- 宗门成员≥10人
- 报名费用：10000灵石

#### 战斗流程
```
报名阶段 → 准备阶段 → 战斗阶段 → 结算阶段
```

#### 战斗模式
- **据点争夺**：占领3个据点获胜
- **宗门对决**：击败对方掌门获胜
- **资源收集**：规定时间内收集更多资源

### 1.3 宗门职位与权限

| 职位 | 人数 | 权限 |
|------|------|------|
| 掌门 | 1 | 全部权限 |
| 副掌门 | 2 | 管理成员、发起战争 |
| 长老 | 5 | 指导弟子、任命职务 |
| 护法 | 10 | 参战、防守 |
| 弟子 | 无限制 | 基本权限 |

### 1.4 宗门技能（战斗加成）

| 技能名称 | 效果 | 升级消耗 |
|----------|------|----------|
| 护宗大阵 | 宗门成员防御+10% | 5000贡献 |
| 同心协力 | 成员伤害+5% | 3000贡献 |
| 鼓舞士气 | 暴击率+5% | 4000贡献 |
| 防御强化 | 气血上限+10% | 4000贡献 |
| 进攻号角 | 攻击速度+10% | 5000贡献 |

### 1.5 战斗奖励

#### 胜利奖励
- 灵石：100000
- 贡献：5000
- 领地：1块
- 排名积分：+100

#### 失败奖励
- 灵石：10000
- 贡献：500
- 排名积分：-20

#### 参与奖励
- 灵石：5000
- 贡献：200

## 2. 跨服竞技系统

### 2.1 竞技场类型

#### 个人竞技
- 1v1对决
- 实时匹配
- 赛季制度

#### 团队竞技
- 3v3组队
- 需要宗门支持
- 配合要求高

#### 全服争霸
- 大型混战
- 每服限定人数
- 最终存活获胜

### 2.2 匹配机制

#### 段位系统
```
青铜 → 白银 → 黄金 → 铂金 → 钻石 → 星耀 → 王者
```

| 段位 | 基础战力要求 | 每周奖励 |
|------|--------------|----------|
| 青铜 | 0 | 1000灵石 |
| 白银 | 5000 | 2000灵石 |
| 黄金 | 20000 | 5000灵石 |
| 铂金 | 50000 | 10000灵石 |
| 钻石 | 100000 | 20000灵石 |
| 星耀 | 300000 | 50000灵石 |
| 王者 | 800000 | 100000灵石 |

#### 匹配规则
- 战力接近的玩家优先匹配
- 相差不超过20%战力
- 等待时间最长60秒

### 2.3 战斗机制

#### 战斗准备
- 法宝选择
- 功法切换
- 灵兽配置
- 药品储备

#### 战斗过程
- 自动战斗为主
- 可手动释放技能
- 药品自动使用
- AI辅助战斗

#### 战斗结算
- 胜：获得积分和奖励
- 负：扣除少量积分
- 连胜：额外奖励

### 2.4 赛季制度

#### 赛季时长
- 2周为一个赛季
- 赛季末结算奖励
- 段位重置

#### 赛季奖励
| 段位 | 灵石 | 专属称号 | 特殊奖励 |
|------|------|----------|----------|
| 王者 | 500000 | 至尊王者 | 神器碎片×10 |
| 星耀 | 200000 | 绝代天骄 | 仙器自选×1 |
| 钻石 | 100000 | 旷世奇才 | 稀有功法×1 |
| 铂金 | 50000 | 后起之秀 | 灵兽蛋×1 |
| 黄金 | 20000 | 小有名气 | 贡献×5000 |
| 白银 | 10000 | 初露锋芒 | 贡献×2000 |
| 青铜 | 5000 | 锋芒毕露 | 贡献×500 |

## 3. 跨服战系统

### 3.1 跨服战类型

#### 仙道大会
- 全服玩家参与
- 淘汰赛制
- 冠军奖励神器

#### 宗门争霸
- 以宗门为单位
- 多服混战
- 胜者称霸

#### 阵营战争
- 正邪两道对立
- 阵营贡献系统
- 阵营专属奖励

### 3.2 参战条件

#### 仙道大会
- 境界≥金丹
- 报名参赛
- 战力前100名

#### 宗门争霸
- 宗门等级≥5
- 宗门总战力前20
- 参赛费用：100000

#### 阵营战争
- 选择阵营
- 贡献值要求
- 阵营战开启时参与

### 3.3 战斗规则

#### 仙道大会规则
- 单败淘汰赛
- 3局2胜制
- 每局3分钟限制
- 禁止使用药品

#### 宗门争霸规则
- 采资源、占旗帜、击败对手
- 宗门成员可随时加入
- 结束时结算总分

#### 阵营战争规则
- 大型团战
- 死亡可复活
- 阵营总积分决胜

### 3.4 奖励分配

| 季军 | 300000 | 天骄 | 仙器×1 |
| 4-10 | 100000 | 仙子/散仙 | 稀有材料 |

#### 宗门争霸

## 4. 社交功能扩展

### 4.1 结缘系统

#### 结婚条件
- 双方境界≥金丹
- 友好度≥10000
- 同宗门或异性好友

#### 结婚奖励
- 双修经验加成+20%
- 战斗属性+10%
- 专属结缘技能

#### 离婚惩罚
- 友好度清零
- 返还50%财产
- 3天冷却时间

### 4.2 师徒扩展

#### 出师条件
- 徒弟境界≥元婴
- 师徒时间≥7天

#### 出师奖励
- 师父：贡献+5000
- 徒弟：突破成功率+5%

### 4.3 联盟系统

#### 联盟创建
- 至少3个宗门联合
- 盟主宗门主导
- 共同防御

#### 联盟功能
- 联盟聊天室
- 资源共享
- 联合副本
- 互助传承
