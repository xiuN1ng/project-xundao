# 挂机修仙 - 界面关系地图

## 一级界面（底部导航）

| 界面 | 路由 | 功能 |
|------|------|------|
| 首页 | / | 角色总览/快速入口 |
| 修炼 | /cultivation | 境界提升/属性增强 |
| 宗门 | /sect | 宗门系统 |
| 战斗 | /battle | 竞技挑战 |
| 我的 | /profile | 设置/背包 |

## 关键测试点

### 1. 路由测试
- 所有一级界面可正常切换
- 所有二级/三级界面可正常打开
- 返回按钮正常工作

### 2. 弹窗测试
- 点击入口弹出正确弹窗
- 弹窗可正常关闭
- 弹窗内按钮可点击

### 3. 数据测试
- 模拟数据显示正常
- 数值变化正确
- 状态更新及时

### 4. 交互测试
- 按钮点击反馈
- 加载状态显示
- 错误提示正确
